# 🟩 Koordináta-rendszer

A Clip tartalma x/y koordináta-rendszert használ, ahol az origó (0,0) a képernyő közepén van.

* A vízszintes x tengelyen a bal oldal negatív, a jobb oldal pozitív.
* A függőleges y tengelyen a felfelé irány negatív, a lefelé irány pozitív.

A Clip szélessége és magassága 400 pixel, ezért a látható terület koordinátái -200 és 200 között vannak.

{% hint style="info" %}
A clip editor _vektoros_ alakzatokat hoz létre. Ez azt jelenti, hogy nem pixelekként tárolja őket, hanem olyan koordinátakészletekként, amelyek meghatározzák az alakzatok kirajzolását. Ez hasonló ahhoz, ahogyan az Inkscape vagy az Illustrator kezeli a tartalmat, szemben a Photoshoppal.
{% endhint %}

### 3D

Emellett 3D térben is mozoghatsz, előre és hátra a z tengely mentén. Alapértelmezés szerint minden a 0 z pozícióban van.

* A z tengelyen a tőled távolodó hátrafelé irány negatív, a feléd közeledő előrefelé irány pozitív.
