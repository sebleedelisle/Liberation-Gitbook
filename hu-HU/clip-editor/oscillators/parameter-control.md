# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

A _Parameter Control_ módszerrel egy effektuson vagy clipen belüli különböző beállítások vezérelhetők a clip editoron kívülről.

* **range min / range max** - azok a minimum- és maximumértékek, amelyekre a paraméter szintje leképeződik
* **level** - a paraméter alapértelmezett beállítása. Ez az az érték, amely kívülről módosítható. Kézzel is állíthatod, hogy lásd, milyen hatása lesz.
* **parameter number** - minden _parameter control_ kaphat egy számot, és ez jelenik meg a clipen kívüli vezérlőkön is. Több parameter control is használhatja ugyanazt a számot; ilyenkor mindegyik egyszerre módosul.

{% hint style="warning" %}
Az APC40 és a támogatott live controller eszközök alapértelmezés szerint minden effektushely 1-es paraméterét vezérlik. Az effektusok és Clips összes paramétere a jobb kattintásos menüben is elérhető.
{% endhint %}
