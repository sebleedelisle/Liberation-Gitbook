# ✅ Telepítés Windowsra (örökölt, 0.9 előtti verziók)

### Kompatibilitás

Windows 10 vagy 11, 64 bites gépeken. Nagyobb show-k futtatásához erősebb gép ajánlott. Minél gyorsabb a számítógéped, annál simább lesz az animált lézerkimenet.

### Telepítési folyamat

Miután aktív előfizetésed van, kapsz egy hivatkozást, vagy rákattinthatsz a _Download_ hivatkozásra a [_My subscriptions_](https://liberationlaser.com/account/my-products) oldalon.

Ezt az oldalt fogod látni:

<figure><img src="../../.gitbook/assets/windows-download-page.png" alt=""><figcaption></figcaption></figure>

Kattints a _Download latest for Windows_ gombra (a verziószámok változhatnak), és látni fogod, ahogy a letöltésjelző mutatja a folyamatot (általában jobb felül; a képernyőkép Chrome-ból készült). Amikor elkészült, vidd az egeret a Liberation letöltés fölé, majd kattints a megjelenő mappa ikonra.

<figure><img src="../../.gitbook/assets/windows-download-chrome.png" alt="" width="260"><figcaption></figcaption></figure>

Ez megnyitja a letöltések mappáját. Keresd meg a letöltött fájlt (általában felül található), kattints rá jobb gombbal, majd válaszd az "Extract all" lehetőséget.

<figure><img src="../../.gitbook/assets/windows-extract-all.png" alt="" width="248"><figcaption></figcaption></figure>

Megnyílik egy új panel, ahol megadhatod, hová szeretnéd telepíteni a Liberationt. Bárhová teheted a merevlemezen, ahol van írási jogosultságod, de ha bizonytalan vagy, tedd egy mappába a C: meghajtón.

<figure><img src="../../.gitbook/assets/windows-select-destination.png" alt="" width="375"><figcaption></figcaption></figure>

Miután a Windows befejezte a kicsomagolást, megjeleníti a telepítőfájlt. Az alkalmazás futtatásához kattints duplán a Liberation.exe fájlra.

<figure><img src="../../.gitbook/assets/windows-open-exe.png" alt="" width="375"><figcaption></figcaption></figure>

A Liberationnek engedélyre van szüksége a fájlok és a hálózat eléréséhez, ezért a Windows valószínűleg több engedélykérő ablakot is megjelenít. Fogadj el mindent, különben a Liberation nem biztos, hogy megfelelően fog működni.

<figure><img src="../../.gitbook/assets/windows-protection-window.png" alt="" width="375"><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

A Liberationnek most már futnia kell. A lézerekre történő kimenethez most végig kell menned az [engedélyezési folyamaton.](../authorising-and-de-authorising.md)
