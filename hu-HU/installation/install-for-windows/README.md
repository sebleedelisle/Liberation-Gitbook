# ✅ Telepítés Windowsra

### Kompatibilitás

Windows 10 vagy 11, 64 bites gépeken. Nagyobb műsorok futtatásához erősebb gép ajánlott. Minél gyorsabb a számítógép, annál simább lesz az animált lézerkimenet.

### Telepítési folyamat

Ha már van aktív előfizetésed, kapsz egy hivatkozást, vagy rákattinthatsz a _Download_ linkre a[ _Your subscriptions_](https://liberationlaser.com/account/my-products) oldalon.

Ezt az oldalt fogod látni:

<figure><img src="../../.gitbook/assets/windows-install-download-page.png" alt=""><figcaption></figcaption></figure>

Kattints a _Download latest for Windows_ gombra (a verziószámok változhatnak), és megjelenik a letöltés előrehaladása (általában a jobb felső sarokban; a képernyőkép Chrome-ból származik). Ha elkészült, nyisd meg a letöltéseket, és kattints a letöltött telepítőfájlra.

<figure><img src="../../.gitbook/assets/windows-install-download-list.png" alt=""><figcaption></figcaption></figure>

Ez elindítja a telepítőt.

{% hint style="info" %}
Bizonyos esetekben megjelenhet egy figyelmeztetés, hogy a Windows megvédte a számítógépedet – a Windows óvatos az új szoftverek futtatásakor. Ahogy egyre többen futtatják és megbízhatónak jelölik a Liberationt, ennek el kell tűnnie.

Ha megjelenik ez a figyelmeztetés, kattints a _More info_ linkre, ellenőrizd, hogy a _Publisher_ mezőben _SEB.LY LIMITED_ szerepel, majd kattints a _Run anyway_ gombra.

<img src="../../.gitbook/assets/windows-install-windows-protection-warning.png" alt="" data-size="original"><img src="../../.gitbook/assets/windows-install-warning-run-anyway.png" alt="" data-size="original">
{% endhint %}

A telepítő indulása előtt megjelenhet egy figyelmeztetés, amely megkérdezi, engedélyezed-e, hogy a telepítő módosításokat végezzen. Kattints a _Yes_ gombra.

<figure><img src="../../.gitbook/assets/windows-install-warning-make-changes.png" alt=""><figcaption></figcaption></figure>

Most elindulhat a telepítő. Menj végig az oldalakon.

<figure><img src="../../.gitbook/assets/windows-installer-1.png" alt=""><figcaption></figcaption></figure>

Ajánlott az alapértelmezett telepítési helyet használni, ezért egyszerűen kattints a _Next_ gombra.

<figure><img src="../../.gitbook/assets/windows-installer-2.png" alt=""><figcaption></figcaption></figure>

Ezután beállíthatod az asztali ikont és a Start menü mappát (ajánlott).

<figure><img src="../../.gitbook/assets/windows-installer-3.png" alt=""><figcaption></figcaption></figure>

Várj egy rövid ideig, amíg a Liberation települ.

<figure><img src="../../.gitbook/assets/windows-installer-4.png" alt=""><figcaption></figcaption></figure>

A telepítő elkészült. Kattints a _Finish_ gombra a bezárásához és a Liberation elindításához.

<figure><img src="../../.gitbook/assets/windows-installer-5.png" alt=""><figcaption></figcaption></figure>

A Liberationnek engedélyre van szüksége a fájlok és a hálózat eléréséhez, ezért a Windows valószínűleg néhány engedélykérést is megjelenít. Ügyelj rá, hogy mindent engedélyezz, különben a Liberation nem biztos, hogy megfelelően működik.

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

A Liberationnek most már futnia kell. A lézerekre történő kimenethez ezután végig kell menned az [engedélyezési folyamaton.](../authorising-and-de-authorising.md)
