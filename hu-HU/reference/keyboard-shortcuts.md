# ✅ Billentyűparancsok

A billentyűparancsokat `Cmd / Ctrl` formában adjuk meg: macOS-en a `Cmd`, Windows alatt a `Ctrl` billentyűt használd.

| Billentyűparancs                  | Művelet                                                       |
| --------------------------------- | ------------------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Az összes lézer élesítése                                     |
| `Escape`                          | Az összes lézer hatástalanítása                               |
| `Tab`                             | Nézetek váltása és lézerek közötti léptetés                   |
| `Shift + Tab`                     | Nézetek váltása és lézerek közötti léptetés visszafelé        |
| `1-9`                             | Ugrás egy lézerre                                             |
| `Left / Right Arrow`              | Clip Deck mozgatása balra / jobbra                            |
| `Cmd / Ctrl + Left / Right Arrow` | Clip Deck mozgatása egy oldallal balra / jobbra               |
| `Shift + Left / Right Arrow`      | Zónák mozgatása egy oldallal balra / jobbra                   |
| `Space`                           | Lejátszás / szünet (amikor a _Timeline_ tempóforrás aktív)    |
| `T`                               | Tempó ütögetése                                               |
| `R`                               | Ütem visszaállítása                                           |
| `Y`                               | Tempó kerekítése a legközelebbi BPM-re                        |
| `Cmd / Ctrl + ,`                  | A beállítások panel megnyitása                                |
| `Cmd / Ctrl + F`                  | Teljes képernyős mód be-/kikapcsolása                         |
| `Cmd / Ctrl + L`                  | A lézerkimeneti beállítások panel megnyitása                  |
| `Cmd / Ctrl + K`                  | A lézervezérlő-hozzárendelési panel megnyitása                |
| `Cmd / Ctrl + J`                  | A 3D Visualiser beállítási panel megnyitása                   |
| `Cmd / Ctrl + Shift + Plus (+)`   | Felület méretezésének növelése                                |
| `Cmd / Ctrl + Shift + Minus (-)`  | Felület méretezésének csökkentése                             |
| `Cmd / Ctrl + 0`                  | Felület méretezésének visszaállítása                          |
| `Cmd / Ctrl + A`                  | Timeline: összes kijelölése                                   |
| `Delete`                          | Timeline: kijelölt blokkok törlése                            |
| `Delete`                          | Clip editor: kijelölt csomópontok törlése                     |
| `Cmd + Q` (macOS)                 | Kilépés az alkalmazásból                                      |
| `Alt + F4` (Windows)              | Kilépés az alkalmazásból                                      |
