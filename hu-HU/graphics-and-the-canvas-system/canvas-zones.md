# 🟩 Canvas zónák

A canvas zóna a canvas nézetben rózsaszín körvonalú téglalapként jelenik meg. Arra szolgál, hogy kijelölj egy téglalap alakú területet a canvason belül, és annak tartalmát egy vagy több lézerre küldd.

A zónákat egérrel kattintva és húzva mozgathatod. A méret és az alak módosításához húzd a sarkokat.

### Új canvas zóna hozzáadása

Kattints az _Add a new canvas zone_ gombra az eszköztáron.

### Canvas zóna hozzárendelése lézerhez

Kattints jobb gombbal a zónára, majd a számozott gombokkal válaszd ki, melyik lézer(ek)hez legyen hozzárendelve ez a canvas zóna.

{% hint style="danger" %}
FIGYELMEZTETÉS – ha a lézer élesítve van, hirtelen tartalom vetítése indulhat el egy alapértelmezett canvas zónában. Canvas zónák hozzárendelése előtt érdemes hatástalanítani a lézert.
{% endhint %}

Ha ezután megnyitod az adott lézer _OUTPUT_ nézetét, ott látni fogod a canvas zónát, amelyet ugyanúgy szerkeszthetsz, mint bármelyik beam zónát. Lásd: [zones.md](../output-view/zones.md).

{% hint style="info" %}
Canvas zónát közvetlenül az _OUTPUT_ nézetből is hozzáadhatsz egy lézerhez – kattints a bal oldali eszköztáron az _Add existing canvas zone_ gombra.
{% endhint %}

### Jobb kattintásos menü

Kattints jobb gombbal a zónára, és megjelennek a lehetőségek a canvas zóna törlésére, hátraküldésére és zárolására.
