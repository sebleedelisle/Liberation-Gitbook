# 🟩 Oversikt over MIDI-kontroll

Liberation bruker MIDI på flere måter:

* Som en live-kontroller, med APC 40. Se [Live-styring med APC40](live-control-with-the-apc40.md "mention").
* Som en kilde for klokkesynkronisering, ved hjelp av MIDI clock og MIDI song position-meldinger. Se [#midi-clock](../tempo-synchronisation.md#midi-clock "mention")
* Som interaktiv inngang på MIDI notes-noden for å lage effekter i «laserharpe»-stil. Se [MIDI notes](../clip-editor/operator-nodes/midi-notes.md "mention")
* Som et mer generelt system for inngang/utgang ved hjelp av MIDI Send/Receive-systemet. Se [MIDI Send/Receive](midi-send-receive.md "mention")
