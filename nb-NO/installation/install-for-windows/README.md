# ✅ Installer for Windows

### Kompatibilitet

Windows 10 eller 11 på 64-biters maskiner. Maskiner med høy ytelse anbefales hvis du vil kjøre store show. Jo raskere datamaskinen er, desto jevnere blir den animerte laserutgangen.

### Installasjonsprosess

Når du har et aktivt abonnement, får du tilsendt en lenke. Du kan også klikke på _Download_-lenken på siden[ _Your subscriptions_](https://liberationlaser.com/account/my-products).

Du får se denne siden:

<figure><img src="../../.gitbook/assets/windows-install-download-page.png" alt=""><figcaption></figcaption></figure>

Klikk på _Download latest for Windows_ (versjonsnumrene kan endre seg), og du ser at nedlastingsindikatoren viser fremdriften (vanligvis øverst til høyre; skjermbildet er fra Chrome). Når nedlastingen er ferdig, åpner du nedlastingene dine og klikker på den nedlastede installasjonsfilen

<figure><img src="../../.gitbook/assets/windows-install-download-list.png" alt=""><figcaption></figcaption></figure>

Dette starter installasjonsprogrammet.

{% hint style="info" %}
I noen tilfeller kan du få en advarsel som sier _Windows protected your PC_ – Windows er forsiktig med å kjøre ny programvare! Etter hvert som flere bruker og stoler på Liberation, bør denne advarselen forsvinne.

Hvis du får denne advarselen, klikker du på lenken _More info_, kontrollerer at _Publisher_ vises som _SEB.LY LIMITED_, og klikker deretter på _Run anyway_.

<img src="../../.gitbook/assets/windows-install-windows-protection-warning.png" alt="" data-size="original"><img src="../../.gitbook/assets/windows-install-warning-run-anyway.png" alt="" data-size="original">
{% endhint %}

Før installasjonsprogrammet starter, kan du få en advarsel som spør om du tillater at installasjonsprogrammet gjør endringer. Klikk på _Yes_.

<figure><img src="../../.gitbook/assets/windows-install-warning-make-changes.png" alt=""><figcaption></figcaption></figure>

Nå kan installasjonsprogrammet starte! Klikk deg gjennom sidene

<figure><img src="../../.gitbook/assets/windows-installer-1.png" alt=""><figcaption></figcaption></figure>

Det anbefales å bruke standard installasjonsplassering, så klikk bare på _Next_

<figure><img src="../../.gitbook/assets/windows-installer-2.png" alt=""><figcaption></figcaption></figure>

Deretter får du valg for skrivebordsikon og mappe i Start-menyen (anbefalt)

<figure><img src="../../.gitbook/assets/windows-installer-3.png" alt=""><figcaption></figcaption></figure>

Vent et lite øyeblikk mens Liberation installeres

<figure><img src="../../.gitbook/assets/windows-installer-4.png" alt=""><figcaption></figcaption></figure>

Installasjonsprogrammet er ferdig! Klikk på _Finish_ for å lukke det og starte Liberation.

<figure><img src="../../.gitbook/assets/windows-installer-5.png" alt=""><figcaption></figcaption></figure>

Liberation trenger tillatelse til å få tilgang til filer og nettverket, så Windows vil sannsynligvis vise flere forespørselsvinduer. Pass på å godta alt, ellers kan det hende Liberation ikke fungerer som det skal.

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation skal nå kjøre! For å sende utdata til lasere må du nå gå gjennom [Autorisere og deautorisere](../authorising-and-de-authorising.md)
