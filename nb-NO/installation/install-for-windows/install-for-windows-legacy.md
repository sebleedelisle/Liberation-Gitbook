# ✅ Installer for Windows (eldre versjon før 0.9)

### Kompatibilitet

Windows 10- eller 11-maskiner med 64-biters system. Maskiner med høy ytelse anbefales hvis du vil kjøre store show. Jo raskere datamaskinen din er, desto jevnere blir den animerte laseroutputen.

### Installasjonsprosess

Når du har et aktivt abonnement, får du tilsendt en lenke, eller du kan klikke på _Download_-lenken på siden [_My subscriptions_](https://liberationlaser.com/account/my-products).

Du får se denne siden:

<figure><img src="../../.gitbook/assets/windows-download-page.png" alt=""><figcaption></figcaption></figure>

Klikk på _Download latest for Windows_ (versjonsnumrene kan endre seg), så ser du nedlastingsindikatoren vise fremdriften (vanligvis øverst til høyre; skjermbildet er fra Chrome). Når nedlastingen er ferdig, holder du musepekeren over Liberation-nedlastingen og klikker på mappeikonet som vises.

<figure><img src="../../.gitbook/assets/windows-download-chrome.png" alt="" width="260"><figcaption></figcaption></figure>

Dette åpner nedlastingsmappen din. Finn den nedlastede filen (vanligvis øverst), høyreklikk og velg "Extract all".

<figure><img src="../../.gitbook/assets/windows-extract-all.png" alt="" width="248"><figcaption></figcaption></figure>

Et nytt panel åpnes og spør hvor du vil installere Liberation. Du kan plassere den hvor som helst på harddisken der du har skrivetilgang, men hvis du er usikker, kan du bare legge den i en mappe på C:-disken.

<figure><img src="../../.gitbook/assets/windows-select-destination.png" alt="" width="375"><figcaption></figcaption></figure>

Når Windows er ferdig med å pakke ut filene, viser den installasjonsfilen. Dobbeltklikk på Liberation.exe for å kjøre appen.

<figure><img src="../../.gitbook/assets/windows-open-exe.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation trenger tillatelse til å få tilgang til filer og nettverket, så Windows vil sannsynligvis vise flere dialogbokser. Sørg for å godta alt, ellers kan det hende at Liberation ikke fungerer som den skal.

<figure><img src="../../.gitbook/assets/windows-protection-window.png" alt="" width="375"><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation skal nå kjøre! For å sende output til lasere må du nå gå gjennom [autoriseringsprosessen.](../authorising-and-de-authorising.md)
