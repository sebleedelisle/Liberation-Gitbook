# ✅ Tastatursnarveier

Snarveier er skrevet som `Cmd / Ctrl`: bruk `Cmd` på macOS og `Ctrl` på Windows.

| Snarvei                           | Handling                                              |
| --------------------------------- | ----------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Armér alle lasere                                     |
| `Escape`                          | Avarmér alle lasere                                   |
| `Tab`                             | Bytt visning og bla gjennom lasere                    |
| `Shift + Tab`                     | Bytt visning og bla bakover gjennom lasere            |
| `1-9`                             | Gå til laser                                          |
| `Left / Right Arrow`              | Flytt Clip Deck til venstre / høyre                   |
| `Cmd / Ctrl + Left / Right Arrow` | Flytt Clip Deck én side til venstre / høyre           |
| `Shift + Left / Right Arrow`      | Flytt soner én side til venstre / høyre               |
| `Space`                           | Spill av / pause (når _Timeline_ er aktiv tempokilde) |
| `T`                               | Tap tempo                                             |
| `R`                               | Nullstill takt                                        |
| `Y`                               | Rund av tempo til nærmeste BPM                        |
| `Cmd / Ctrl + ,`                  | Åpne innstillingspanelet                              |
| `Cmd / Ctrl + F`                  | Slå fullskjermmodus av/på                             |
| `Cmd / Ctrl + L`                  | Åpne panelet for laserutgangsinnstillinger            |
| `Cmd / Ctrl + K`                  | Åpne panelet for tilordning av laserkontrollere       |
| `Cmd / Ctrl + J`                  | Åpne innstillingspanelet for 3D Visualiser            |
| `Cmd / Ctrl + Shift + Plus (+)`   | Øk skalering av grensesnittet                         |
| `Cmd / Ctrl + Shift + Minus (-)`  | Reduser skalering av grensesnittet                    |
| `Cmd / Ctrl + 0`                  | Tilbakestill skalering av grensesnittet               |
| `Cmd / Ctrl + A`                  | Timeline: velg alt                                    |
| `Delete`                          | Timeline: slett valgte blokker                        |
| `Delete`                          | Clip editor: slett valgte noder                       |
| `Cmd + Q` (macOS)                 | Avslutt appen                                         |
| `Alt + F4` (Windows)              | Avslutt appen                                         |
