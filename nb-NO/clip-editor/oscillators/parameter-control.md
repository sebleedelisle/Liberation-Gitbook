# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_Parameter Control_ er en metode for å styre ulike innstillinger i en effekt eller et clip utenfor clip-editoren.

* **range min / range max** - minimums- og maksimumsverdiene som parameternivået mappes til
* **level** - standardinnstillingen for denne parameteren. Dette er verdien som kan endres utenfra. Du kan justere den manuelt for å se hvilken effekt det har.
* **parameter number** - hver _parameter control_ kan tilordnes et nummer, og dette vises i kontrollene utenfor clipet. Du kan ha mer enn én parameter control med samme nummer, og alle justeres samtidig.

{% hint style="warning" %}
APC40 og støttede live-controllere styrer parameter 1 for hvert effect slot som standard. Alle parametere i effekter og clips er også tilgjengelige i høyreklikkmenyen.
{% endhint %}
