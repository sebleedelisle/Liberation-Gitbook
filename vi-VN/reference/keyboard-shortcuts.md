# ✅ Phím tắt

Phím tắt được viết theo dạng `Cmd / Ctrl`: dùng `Cmd` trên macOS và `Ctrl` trên Windows.

| Phím tắt                          | Hành động                                                |
| --------------------------------- | ------------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Arm tất cả laser                                        |
| `Escape`                          | Disarm tất cả laser                                     |
| `Tab`                             | Chuyển đổi các chế độ xem và lần lượt chuyển qua các laser |
| `Shift + Tab`                     | Chuyển đổi các chế độ xem và lần lượt chuyển qua các laser theo chiều ngược lại |
| `1-9`                             | Chuyển nhanh đến laser                                  |
| `Left / Right Arrow`              | Di chuyển Clip Deck sang trái / phải                    |
| `Cmd / Ctrl + Left / Right Arrow` | Di chuyển Clip Deck sang trái / phải một trang          |
| `Shift + Left / Right Arrow`      | Di chuyển zones sang trái / phải một trang              |
| `Space`                           | Phát / tạm dừng (khi nguồn tempo _Timeline_ đang hoạt động) |
| `T`                               | Tap tempo                                               |
| `R`                               | Đặt lại ô nhịp                                          |
| `Y`                               | Làm tròn tempo đến BPM gần nhất                         |
| `Cmd / Ctrl + ,`                  | Mở panel cài đặt                                        |
| `Cmd / Ctrl + F`                  | Bật/tắt chế độ toàn màn hình                            |
| `Cmd / Ctrl + L`                  | Mở panel cài đặt đầu ra laser                           |
| `Cmd / Ctrl + K`                  | Mở panel gán laser controller                           |
| `Cmd / Ctrl + J`                  | Mở panel cài đặt trình hiển thị 3D                      |
| `Cmd / Ctrl + Shift + Plus (+)`   | Tăng tỷ lệ giao diện                                    |
| `Cmd / Ctrl + Shift + Minus (-)`  | Giảm tỷ lệ giao diện                                    |
| `Cmd / Ctrl + 0`                  | Đặt lại tỷ lệ giao diện                                 |
| `Cmd / Ctrl + A`                  | Timeline: chọn tất cả                                   |
| `Delete`                          | Timeline: xóa các khối đã chọn                          |
| `Delete`                          | Clip editor: xóa các nodes đã chọn                      |
| `Cmd + Q` (macOS)                 | Thoát ứng dụng                                          |
| `Alt + F4` (Windows)              | Thoát ứng dụng                                          |
