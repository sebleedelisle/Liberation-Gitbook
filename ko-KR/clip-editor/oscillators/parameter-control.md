# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_Parameter Control_은 Clip Editor 외부에서 effect 또는 clip 내부의 다양한 설정을 제어하는 방법입니다.

* **range min / range max** - parameter level이 매핑되는 최소값과 최대값입니다.
* **level** - 이 parameter의 기본 설정입니다. 외부에서 변경할 수 있는 값입니다. 직접 조정하여 어떤 효과가 나타나는지 확인할 수 있습니다.
* **parameter number** - 각 _parameter control_에는 번호를 할당할 수 있으며, 이 번호는 clip 외부의 컨트롤에 반영됩니다. 동일한 번호를 가진 parameter control을 여러 개 둘 수 있으며, 모두 동시에 조정됩니다.

{% hint style="warning" %}
APC40 및 지원되는 live controller는 기본적으로 각 effect slot의 parameter 1을 제어합니다. effect와 Clip 내부의 모든 parameter는 오른쪽 클릭 메뉴에서도 사용할 수 있습니다.
{% endhint %}
