# ✅ Liberation에 오신 것을 환영합니다



<figure><img src=".gitbook/assets/LiberationTease.jpeg" alt=""><figcaption></figcaption></figure>

Liberation은 레이저 작업 방식에 새로운 흐름을 제시합니다. **현대적**이고 **효율적**이며, **Mac 및 PC**에서 사용할 수 있고, 오늘날 이벤트 프로덕션 현장의 실제 요구를 중심으로 설계되었습니다. 빠르고 표현력 있는 창작 도구와 내장된 안전 기능을 하나로 묶고, 유연하고 직관적이며 미래 지향적으로 느껴지는 워크플로를 제공해 레이저 소프트웨어를 21세기에 맞게 발전시킵니다.

Liberation은 같은 계열의 오픈 프로토콜 컨트롤러를 사용하는 레이저 제어 소프트웨어 생태계의 일부입니다. TouchDesigner, MadLaser, LaserShowGen 또는 Liberation 중 무엇으로 작업하든 특정 제조사에 묶이지 않습니다. 사용자의 장비 구성, 예산, 아이디어에 맞는 하드웨어를 선택할 수 있습니다.

무엇보다 Liberation은 표현력 있는 창작 도구로 만들어졌습니다. 배우기 쉽고, 실제로 사용하기 재미있으며, 단일 컴퓨터만으로도 가장 까다로운 페스티벌, 설치 작업, 아레나 투어를 안정적으로 처리할 만큼 강력합니다.

이 가이드는 설치와 안전 기본 사항부터 클립 제작, 존 매핑, 전체 쇼 운영까지 Liberation에 익숙해질 수 있도록 도와줍니다. 중요한 개념과 워크플로에 초점을 맞추며, Liberation은 사용자가 소프트웨어와 씨름하는 시간보다 창작에 더 많은 시간을 쓸 수 있도록 작업을 방해하지 않게 설계되었습니다.

## 이 가이드의 대상

* 안전을 건너뛰지 않고 명확한 시작 경로를 원하는 신규 사용자.
* 공연장 셋업이나 리허설 중 신뢰할 수 있는 참고 자료가 필요한 쇼 오퍼레이터.
* Clip Editor, MIDI, DMX, Canvas 도구 같은 고급 기능을 살펴보는 호기심 많은 사용자.

## 가이드 사용 방법

* 하드웨어를 연결하기 전에 [**Important! Read this first**](important.md)에서 중요한 안전 및 라이선스 관련 사항을 먼저 확인하세요.
* 빠르게 출력을 실행하고 싶다면 [**Quick start guide**](getting-started.md)를 따라 진행한 다음, 더 깊은 설정을 위해 [**Hardware**](hardware/)와 [**Setting up**](setting-up/)을 살펴보세요.
* 섹션 헤더를 사용해 원하는 주제로 바로 이동할 수 있습니다(예: [**Clips & Clip deck**](clips/), [**Output view**](output-view/), **etc**). 정확한 답이 필요할 때는 [**Reference**](reference/) 또는 [**Troubleshooting**](troubleshooting/)으로 돌아가세요.

## 표기 규칙 및 팁

* 상태 이모지는 완료 수준을 나타냅니다: ✅ 완료, 🟧 일부 완료, 🟩 이미지 필요, ◼️ 자리표시자.
* 링크는 현재 릴리스를 기준으로 합니다. 이전 빌드를 사용하는 경우 스크린샷과 메뉴가 약간 다를 수 있습니다.
* 하드웨어 단계에서는 안전을 우선한 기본값을 안내합니다. 인터록을 절대 우회하지 말고, 빔이 사람을 향하지 않도록 유지하세요.

문제가 생기면 지원 요청 전에 먼저 [**FAQ**](faq.md)를 확인한 뒤, [**Troubleshooting**](troubleshooting/) 섹션에서 해당 문제에 맞는 해결 방법을 찾아보세요.
