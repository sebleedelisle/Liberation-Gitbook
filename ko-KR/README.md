# ✅ Liberation에 오신 것을 환영합니다



<figure><img src=".gitbook/assets/LiberationTease.jpeg" alt=""><figcaption></figcaption></figure>

Liberation 소프트웨어는 레이저 작업 방식에 새로운 접근을 제공합니다. **현대적**이고 **효율적**이며, **Mac과 PC**를 지원하고, 오늘날 이벤트 프로덕션 현장의 실제 요구에 맞춰 설계되었습니다. 빠르고 표현력 있는 크리에이티브 도구, 내장 안전 기능, 유연하고 직관적인 워크플로를 하나로 결합하여, 레이저 소프트웨어를 21세기에 맞게 발전시킵니다.

Liberation 소프트웨어는 동일한 계열의 개방형 프로토콜 controller를 사용하는 레이저 제어 소프트웨어 생태계의 일부입니다. TouchDesigner, MadLaser, LaserShowGen, Liberation 중 무엇을 사용하든 특정 제조사 하나에 묶이지 않습니다. 리그, 예산, 아이디어에 맞는 하드웨어를 선택할 수 있습니다.

무엇보다 Liberation 소프트웨어는 표현력 있는 창작 도구로 만들어졌습니다. 배우기 쉽고, 실제로 사용하기 즐거우며, 가장 까다로운 페스티벌, 설치 작업, 아레나 투어까지 한 대의 컴퓨터에서 안정적으로 처리할 수 있을 만큼 강력합니다.

이 가이드는 설치와 안전 기본 사항부터 Clip 제작, zone 매핑, 전체 쇼 운영까지 Liberation 소프트웨어에 익숙해질 수 있도록 도와줍니다. 중요한 개념과 워크플로에 집중하며, Liberation 소프트웨어는 사용자가 소프트웨어와 씨름하는 시간을 줄이고 창작에 더 많은 시간을 쓸 수 있도록 설계되었습니다.

## 이 가이드의 대상

* 안전을 건너뛰지 않으면서 명확한 시작 경로를 원하는 신규 사용자.
* 공연장 셋업이나 리허설 중 신뢰할 수 있는 참고 자료가 필요한 쇼 operator.
* Clip Editor, MIDI, DMX, canvas 도구 같은 고급 기능을 살펴보려는 호기심 많은 사용자.

## 가이드 사용 방법

* 하드웨어를 연결하기 전에 [**중요! 먼저 읽어 주세요**](important.md)에서 핵심 안전 및 라이선스 안내를 확인하세요.
* 빠르게 Output을 실행하고 싶다면 [**빠른 시작 가이드**](getting-started.md)를 따른 다음, 더 자세한 구성은 [**하드웨어**](hardware/)와 [**설정하기**](setting-up/)를 확인하세요.
* 섹션 제목을 사용해 원하는 주제로 바로 이동할 수 있습니다(예: [**Clips 및 Clip Deck**](clips/), [**Output 보기**](output-view/), **등**). 정확한 답이 필요할 때는 [**참조**](reference/) 또는 [**문제 해결**](troubleshooting/)로 돌아가세요.

* 상태 이모지는 완성도를 나타냅니다: ✅ 완료, 🟧 일부 완료, 🟩 이미지 필요, ◼️ 자리 표시자.
* 링크는 현재 릴리스를 기준으로 합니다. 이전 빌드를 사용하는 경우 스크린샷과 메뉴가 약간 다를 수 있습니다.
* 하드웨어 단계에서는 안전 우선 기본값을 명시합니다. 인터록을 절대 우회하지 말고, beam이 사람을 향하지 않도록 하세요.
