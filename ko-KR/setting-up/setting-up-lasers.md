# ✅ 레이저 설정 절차 개요

### 레이저를 안전하게 켜기 위한 절차 개요

이 매뉴얼은 공식 레이저 안전 교육을 대체하지 않습니다. 공공장소에서 레이저를 사용하려면 반드시 정식 교육이 필요합니다. 지역에 따라 추가적인 법적 요구 사항이 있을 수 있으며, 어떤 경우든 항상 안전과 전문성에 관한 모범 사례를 따라야 합니다.

PLASA에서는 일반적으로 모범 사례로 인정받는 레이저 안전 가이드를 무료로 다운로드할 수 있도록 제공하고 있습니다: [https://www.plasa.org/guidance-for-display-lasers/](https://www.plasa.org/guidance-for-display-lasers/)

사용하기 전에 레이저와 관련된 안전상의 영향을 반드시 이해해야 합니다!

#### 소개

이 페이지는 레이저 전원을 안전하게 켜는 절차를 전체적으로 이해할 수 있도록 구성되어 있습니다. 각 단계를 정확히 수행하는 방법은 이 섹션의 뒤쪽에서 자세히 다루지만, 이 페이지를 통해 전체 흐름을 파악할 수 있으며, 이후 레이저를 설정할 때마다 참고 자료로 다시 확인할 수 있습니다.

### 하드웨어 설정:

1. 레이저의 **개구부 커버를 닫습니다**
2. **레이저를 단단히 고정**하고 올바른 방향으로 향하게 합니다
3. **정지 버튼을 레이저에 연결**합니다
4. **laser controller를 컴퓨터에 연결**합니다
5. 레이저 **전원을 켭니다**

### Liberation 설정:

1. **모든 레이저를 disarmed 상태로 둔 뒤**, Liberation에서 controller 장치를 찾아 연결합니다
2. **_Global Brightness_ 설정을 0으로 낮춥니다**(아이콘 바의 슬라이더 또는 APC40의 _Master Fader_ 사용)
3. **레이저를 armed 상태로 전환합니다** - 개구부 커버는 계속 닫아 둔 상태에서, 현재 활성화된 Clip이 없는지 확인한 뒤 레이저를 armed 상태로 전환합니다(_Laser Overview_ 패널의 _Arm_ 버튼 사용)
4. **test pattern을 켭니다**(아이콘 바의 ☒ 버튼을 사용하고, 십자 표시가 있는 초록색 사각형인 패턴 1을 선택)
5. **출력 zone 설정을 조정합니다** - 가장 안전한 zone 크기와 위치를 대략적으로 잡습니다. 예를 들어 천장 위쪽으로 높게 설정할 수 있지만, 이는 실제 환경에 따라 달라집니다
6. **레이저가 작동하는지 확인합니다** - 개구부 창 뒤에서 빛이 보일 때까지 밝기를 천천히 올립니다. 그런 다음 밝기를 다시 0으로 낮춥니다.
7. **정지 버튼을 테스트**하여 버튼을 눌렀을 때 모든 레이저 출력이 꺼지는지 확인합니다

### 레이저 출력 시작

1. **노출 영역을 비웁니다** - 누구도 레이저에 노출되지 않도록 하고, 레이저 설정 중에는 모든 작업자가 노출 영역에 들어가지 않도록 안내합니다. (카메라와 프로젝터도 가려 두거나 렌즈 캡을 씌워 두어야 합니다!)
2. **개구부 커버를 엽니다** - 출력 방향을 피해서 옆에 선 상태로, 개구부 커버를 아래로 밀어 엽니다. zone 위치가 높게 설정되어 있다면 일부만 열어 두는 것이 좋을 수 있습니다.
3. **레이저가 겨우 보일 정도까지 밝기를 올립니다** - zone을 확인하는 데 필요한 최소 밝기로만 설정합니다
4. **zone 설정을 조정합니다** - 모든 공개 접근 가능 영역에서 바닥 기준 3m 이상 떨어지도록 zone 크기, 모양, 위치를 설정하고, 레이저가 다른 공개 접근 가능 영역에 도달하지 않도록 합니다
5. **물리적 마스킹을 추가합니다** - 개구부 커버 및/또는 검은색 포일 테이프를 사용하여 원하는 zone 바깥 영역을 물리적으로 가립니다. 레이저 하드웨어나 소프트웨어는 오작동할 수 있으므로 이 단계는 매우 중요합니다.
6. **소프트웨어 mask를 추가합니다** - Liberation의 소프트웨어 mask는 카메라와 프로젝터를 보호하는 데 사용할 수 있지만, 사람을 보호하기 위한 물리적 마스킹을 대체하는 용도로는 **절대** 사용하면 안 됩니다. 어떤 소프트웨어나 하드웨어도 완벽하지 않으므로, 소프트웨어 mask를 사용하기 전에 관련 위험을 반드시 이해해야 합니다.

{% hint style="warning" %}
이 가이드는 실내 설정을 전제로 합니다. 야외에서 작업하는 경우 항공기 안전을 보장하기 위해 다음을 포함하되 이에 국한되지 않는 추가 조치를 취해야 합니다.

* FAA 또는 CAA와 같은 항공 당국의 필요한 허가 받기
* 인근 공항 및 비행장과 협의하기
* 공개 항공 레이더를 확인하고, 항공기를 감시할 관측 담당자를 배치하기

안전 기준치를 훨씬 밑도는 레이저라도 조종사에게 치명적인 주의 분산을 일으킬 수 있습니다.

어떤 레이저든 공역을 향해 조사하기 전에 필요한 자격, 면허, 허가를 갖추었는지 확인하십시오.
{% endhint %}
