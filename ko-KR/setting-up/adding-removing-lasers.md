# ✅ 레이저 추가 / 제거

_Laser overview_ 패널에서 _ADD LASER_ 버튼을 클릭하여 새 레이저를 추가합니다.

<figure><img src="../.gitbook/assets/Remove Laser in Laser Overview.png" alt="" width="292"><figcaption></figcaption></figure>

또는 _Output_ 보기에서 _+_ 버튼을 클릭합니다.<br>

<figure><img src="../.gitbook/assets/Laser Output View (1).png" alt="" width="176"><figcaption></figcaption></figure>

새 레이저를 추가하면 빔 존이 자동으로 함께 추가됩니다. 레이저와 존은 추가한 순서대로 정렬됩니다. [re-ordering-beam-zones.md](../output-view/re-ordering-beam-zones.md)를 참조하세요.

_Laser overview_ 패널에서 빨간색 "-" 버튼을 클릭하여 레이저를 제거합니다. 이 작업을 하면 해당 레이저의 모든 설정이 삭제되므로 주의하세요!

{% hint style="info" %}
다른 레이저에서 존이나 기타 설정을 복사하려면 [copy-laser-settings.md](copy-laser-settings.md)를 사용하세요.
{% endhint %}

{% hint style="info" %}
레이저 삭제처럼 위험할 수 있는 작업을 수행할 때마다 확인 대화상자가 표시됩니다. 이를 건너뛰려면 버튼을 클릭할 때 `Cmd / Ctrl`을 누른 상태로 클릭하세요. 물론 주의해서 사용하세요!
{% endhint %}
