# 🟩 Canvas zones

Canvas zone은 Canvas view에서 분홍색 윤곽선의 사각형으로 표시됩니다. Canvas 안의 사각형 영역을 선택하고, 해당 영역의 콘텐츠를 하나 이상의 레이저로 보내는 데 사용됩니다.

마우스로 zone을 클릭하고 드래그하여 이동할 수 있습니다. 모서리를 드래그하면 크기와 형태를 변경할 수 있습니다.

### 새 Canvas zone 추가

툴바에서 _Add a new canvas zone_ 버튼을 클릭합니다.

### Canvas zone을 레이저에 할당

zone을 마우스 오른쪽 버튼으로 클릭한 뒤, 숫자 버튼을 토글하여 이 Canvas zone을 할당할 레이저를 선택합니다.

{% hint style="danger" %}
WARNING - 레이저가 armed 상태이면 기본 Canvas zone에서 콘텐츠 투사가 갑자기 시작될 수 있습니다. Canvas zone을 레이저에 할당하기 전에 레이저를 disarm하는 것이 좋습니다.
{% endhint %}

이제 해당 레이저의 _OUTPUT_ view로 이동하면 Canvas zone이 표시됩니다. 이 zone은 다른 beam zone과 동일하게 편집할 수 있습니다. [Zones](../output-view/zones.md "mention")를 참조하세요.

{% hint style="info" %}
_OUTPUT_ view에서 레이저에 Canvas zone을 직접 추가할 수도 있습니다. 왼쪽 툴바에서 _Add existing canvas zone_ 버튼을 클릭하세요.
{% endhint %}

### 오른쪽 클릭 메뉴

zone을 마우스 오른쪽 버튼으로 클릭하면 이 Canvas zone을 삭제하거나, 뒤로 보내거나, 잠그는 옵션이 표시됩니다.
