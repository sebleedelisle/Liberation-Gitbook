# 🟩 Canvas settings

왼쪽 툴바 상단의 _Canvas settings_ 버튼을 클릭하면 _Canvas settings_ 패널이 열립니다.

### Width and Height

드래그 가능한 이 슬라이더를 사용해 전체 Canvas의 전반적인 너비와 높이를 변경합니다. 이 치수는 개념적으로 픽셀 크기처럼 생각할 수 있지만, 모두 상대적인 값입니다.

슬라이더를 더블 클릭하면 값을 직접 입력할 수 있습니다.

{% hint style="info" %}
조금 더 기술적인 내용을 덧붙이면, 레이저는 zone 크기가 아니라 Canvas 해상도를 기준으로 움직입니다. Canvas에서 큰 zone은 실제로도 더 넓은 영역에 투사될 가능성이 높기 때문입니다.

대부분의 경우 크게 신경 쓰지 않아도 되는 작은 차이지만, Canvas zone 전반에서 성능과 가시성을 더 고르게 만드는 데 도움이 됩니다.
{% endhint %}

### Zone brightness

Canvas view에서 분홍색 zone 윤곽선이 얼마나 밝게 표시되는지 조정합니다. 콘텐츠를 더 명확하게 보이게 하고, 이 view가 너무 복잡해 보이지 않도록 하는 데 도움이 됩니다.
