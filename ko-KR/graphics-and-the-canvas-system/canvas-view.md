# 🟩 Canvas view

상단 툴바의 _CANVAS_ 버튼을 클릭해 _CANVAS_ view를 선택합니다.

Canvas view는 전체 Canvas를 보여주는 화면입니다. 여기에서 콘텐츠를 배치하고 작업할 수 있습니다. 이 view에서는 Canvas의 어느 부분을 어떤 zone으로 보낼지 선택하고, 가이드 이미지를 추가할 수 있습니다.

### Canvas view에서 이동하기

빈 공간(또는 잠긴 zone)을 클릭한 채 드래그하면 view를 이동할 수 있습니다.

### Canvas view 확대 및 축소하기

마우스 휠을 사용해 확대하거나 축소합니다.

### 툴바 아이콘

view의 왼쪽에 툴바가 표시됩니다. 각 아이콘 위에 마우스를 올리면 해당 기능을 확인할 수 있습니다.

* _Canvas settings_ 패널 열기
* 새 Canvas zone 추가
* 새 가이드 이미지 추가
* 그리드 스냅 켜기/끄기
* 그리드 크기 조정(0이 가장 작고 8이 가장 큼). (그리드 크기는 2의 거듭제곱 단위로 증가하므로, 그리드 크기를 줄여도 스냅된 모든 포인트는 계속 같은 선상에 유지됩니다.)
* 그리드 표시 / 숨기기
