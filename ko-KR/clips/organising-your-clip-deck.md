# 🟩 Clip Deck 구성하기

### 클립 이동/복제

클립을 클릭한 상태로 드래그하여 이동합니다. 여러 클립이 선택되어 있으면([Clip 시작 / 중지](starting-stopping-clips.md "mention") 참조) 선택된 클립이 모두 함께 이동합니다. 복제하려면 `Alt / Option`을 누른 상태로 드래그합니다. 클립을 빈 공간에 놓으면 됩니다.

{% hint style="info" %}
많은 수의 Clip을 선택해야 하는 경우, 클릭한 뒤 드래그하여 lasso를 만든 다음, 계속 드래그한 상태에서 Clip Deck을 스크롤할 수 있습니다(커서 키, APC40 또는 터치패드/마우스의 가로 스크롤 사용). lasso는 Clip Deck의 가장자리 밖까지 확장됩니다.
{% endhint %}

### 클립 삭제

Clip 하나를 삭제하려면 우클릭 메뉴를 사용합니다.

여러 클립을 삭제하려면 `Alt / Option + Shift`-클릭 또는 lasso 드래그로 선택한 뒤, Clip Deck 밖으로 드래그합니다.

### 빈 공간 추가

클립 버튼을 우클릭하고 _ADD EMPTY COLUMN TO RIGHT_ 를 누르면 Clip Deck에 빈 열을 추가할 수 있습니다.

### 여러 Clip Deck 결합

기존 Clip Deck을 가져와 현재 Clip Deck의 끝에 추가할 수 있습니다. 메뉴에서 _File -> Clips/Effects -> Append Clip Deck._ 을 사용합니다.

클립을 선택한 뒤(lasso 또는 `Alt / Option + Shift`-클릭 사용) _File -> Clips/Effects -> Export Selected Clips._ 를 선택합니다. 이렇게 하면 선택한 클립만 포함된 Clip Deck이 내보내지며, 이후 다른 Clip Deck에 추가할 수 있습니다.

### 타임라인별로 클립 정리

타임라인을 가져올 때 몇 가지 클립 옵션도 함께 사용할 수 있습니다. 현재 Clip Deck을 교체하거나, 타임라인의 전체 Clip Deck을 추가하거나, 해당 타임라인에서 사용하는 클립만 추가할 수 있습니다.

프로젝트에 여러 타임라인이 있는 경우, 타임라인별로 클립을 다시 정리할 수 있습니다. _Timeline selection_ 패널을 열고 _ORGANISE CLIPS BY TIMELINE_ 을 클릭합니다. 그러면 모든 클립이 해당 타임라인을 기준으로 그룹화되어 표시됩니다. 하나의 클립이 여러 타임라인에서 사용되는 경우, 각 그룹마다 해당 클립이 복제됩니다.
