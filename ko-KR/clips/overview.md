# 🟩 개요

### Clip이란 무엇인가요?

Liberation의 모든 레이저 콘텐츠는 Clip 안에 저장됩니다.

Clip에는 atmospheric beam effect나 graphical animation을 넣을 수 있으며, 일반적으로 반복 재생되는 루프 형태입니다. Clip은 원하는 zone(또는 Canvas Target Area)으로 보낼 수 있고, Clip Deck 안의 Clip 버튼을 사용해 트리거합니다.

Clip 콘텐츠는 유연한 노드 기반 Clip Editor 시스템을 사용해 만듭니다. (자세한 내용은 [clip-editor-intro.md](../clip-editor/clip-editor-intro.md)를 참조하세요)

### Clip Deck

<figure><img src="../.gitbook/assets/clips-clip-deck" alt=""><figcaption><p>기본 Clip Deck</p></figcaption></figure>

Clip Deck은 화면에 표시되는 Clip 그리드이며, 보통 현재 보이는 영역보다 더 넓게 확장되어 있습니다.

Clip Deck을 좌우로 스크롤하려면 다음을 사용하세요.

* 왼쪽 및 오른쪽 화살표 키
* 트랙패드 두 손가락 스와이프(또는 마우스가 지원한다면 좌우 스크롤)
* APC40의 _Cue Level_ 스크롤 휠

Clip Deck을 한 페이지씩 좌우로 스크롤하려면 다음을 사용하세요.

* `Cmd / Ctrl + Left / Right Arrow`
* APC40의 _Device <- and ->_ 버튼

#### Mini Clip Visualiser

Clip Deck 상단에는 전체 Clip Deck을 작게 보여주는 표시 영역이 있습니다. 밝게 표시된 부분은 현재 화면에 보이는 Clip 영역을 나타냅니다.
