# ✅ 키보드 단축키

단축키는 `Cmd / Ctrl` 형식으로 표기됩니다. macOS에서는 `Cmd`, Windows에서는 `Ctrl`을 사용하세요.

| Shortcut                          | Action                                             |
| --------------------------------- | -------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | 모든 레이저를 Arm 상태로 설정                     |
| `Escape`                          | 모든 레이저를 Disarm 상태로 설정                  |
| `Tab`                             | 뷰를 전환하고 레이저를 순환 선택                  |
| `Shift + Tab`                     | 뷰를 전환하고 레이저를 역방향으로 순환 선택       |
| `1-9`                             | 해당 번호의 레이저로 이동                         |
| `Left / Right Arrow`              | Clip Deck을 왼쪽/오른쪽으로 이동                  |
| `Cmd / Ctrl + Left / Right Arrow` | Clip Deck을 한 페이지 왼쪽/오른쪽으로 이동        |
| `Shift + Left / Right Arrow`      | Zone을 한 페이지 왼쪽/오른쪽으로 이동             |
| `Space`                           | 재생/일시정지(_Timeline_ tempo source가 활성화된 경우) |
| `T`                               | 탭 템포                                           |
| `R`                               | 마디 리셋                                         |
| `Y`                               | 템포를 가장 가까운 BPM으로 반올림                 |
| `Cmd / Ctrl + ,`                  | settings panel 열기                               |
| `Cmd / Ctrl + F`                  | 전체 화면 모드 전환                               |
| `Cmd / Ctrl + L`                  | laser output settings panel 열기                  |
| `Cmd / Ctrl + K`                  | laser controller assignment panel 열기            |
| `Cmd / Ctrl + J`                  | 3D visualiser settings panel 열기                 |
| `Cmd / Ctrl + Shift + Plus (+)`   | 인터페이스 배율 확대                              |
| `Cmd / Ctrl + Shift + Minus (-)`  | 인터페이스 배율 축소                              |
| `Cmd / Ctrl + 0`                  | 인터페이스 배율 재설정                            |
| `Cmd / Ctrl + A`                  | Timeline: 모두 선택                               |
| `Delete`                          | Timeline: 선택한 블록 삭제                        |
| `Delete`                          | Clip editor: 선택한 노드 삭제                     |
| `Cmd + Q` (macOS)                 | 앱 종료                                           |
| `Alt + F4` (Windows)              | 앱 종료                                           |
