# 🟩 테스트 패턴

존을 편집하고 조정할 때는 테스트 패턴을 출력하면 유용합니다. 상단 아이콘 바의 토글 버튼으로 활성화된 모든 존에 테스트 패턴을 켤 수 있으며, 그 오른쪽의 슬라이더로 테스트 패턴을 선택할 수 있습니다.

현재 조닝 중인 레이저에만 테스트 패턴을 사용하려면 _Output_ view 왼쪽 툴바의 토글 버튼을 사용하세요. 그 아래의 숫자를 클릭하거나 드래그하여 테스트 패턴을 선택합니다.

### 테스트 패턴 목록

아래는 테스트 패턴의 전체 목록입니다. 별도로 명시되지 않은 경우, 패턴은 각 존의 전체 출력 공간을 채웁니다.

1. 기본 패턴 - 십자선이 그어진 초록색 사각형입니다. 낮은 밝기에서도 더 잘 보이도록 초록색을 사용합니다.
2. 정렬 패턴 - 파란색 및 초록색 원과 흰색 및 빨간색 그리드의 조합입니다. 원의 형태를 이용해 올바른 종횡비를 맞출 수 있으므로 건축물 매핑에 유용합니다.
3. 초록색 가로선 - 균등한 간격의 가로선 4개입니다.
4. 초록색 세로선 - 균등한 간격의 세로선 4개입니다.

색상 보정 패턴 5-8입니다. 이 패턴들은 레이저 설정의 색상 보정 시스템과 함께 사용할 수 있습니다. [colour-calibration.md](../advanced/colour-calibration.md)를 참고하세요. 이 패턴들은 전체 출력 공간을 채우지 않습니다.

5. 색상 보정 패턴 - 빨간색.
6. 색상 보정 패턴 - 초록색.
7. 색상 보정 패턴 - 파란색.
8. 색상 보정 패턴 - 흰색.

{% hint style="danger" %}
**주의** - 테스트 패턴 9는 매우 위험할 수 있으므로, 해당 존이 안전하다고 판단한 후에만 사용해야 합니다. 밝고 움직이지 않는 점은 표면에 그을림이나 화상을 일으킬 수 있습니다. 하늘을 향해 사용하는 경우 필요한 면허/허가를 갖추고 있는지 확인하고, 항공기가 있는지 계속 확인하세요.
{% endhint %}

9. 밝은 흰색 점 - 각 모서리에 최대 출력의 점 하나가 표시됩니다. 가장 잘 보이는 패턴이므로 밝은 장소나 하늘을 향해 조닝할 때 적합합니다.
10. 스캔하는 흰색 선 - 존의 전체 폭을 좌우로 스캔하는 세로 흰색 선입니다. 존 전체를 커버하므로 마스킹을 테스트하고 미러볼이나 기타 위험 요소를 피하고 있는지 확인하기에 좋습니다.
11. 흰색 사각형 - 밝은 흰색 사각형입니다.
