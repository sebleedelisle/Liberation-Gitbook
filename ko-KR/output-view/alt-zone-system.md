# 🟩 Alt zone 시스템

특정 zone들을 즉시 이동하거나 비활성화해야 하는 상황이 있었나요? 예를 들면 다음과 같습니다.

* 하늘로 향하는 레이저가 있을 때, 항공기가 지나가면 건물에 종료되도록 방향을 바꿔야 하는 경우.
* 사용하지 않는 발코니 층으로 들어가는 레이저가 있고, 쇼의 특정 구간에서 스태프가 그곳에 있을 때 방향을 바꿔야 하는 경우.

Alt zone 시스템을 사용하면 기존 zone에 대한 대체 zone을 만들고, 버튼 하나로 모든 대체 zone으로 한 번에 전환할 수 있습니다.

### 대체 zone 추가하기

Output view에서 zone을 오른쪽 클릭하고 _ADD ALT ZONE_을 클릭합니다. 새 녹색 zone이 나타나는 것을 확인할 수 있습니다. 이 zone은 다른 zone과 동일한 방식으로 편집하고 조정할 수 있습니다. 테스트 패턴을 보려면 상단 아이콘 바에서 Alt zones를 활성화해야 합니다.

### Alt zone으로 전환하기

alt zone을 하나라도 추가하면 상단 아이콘 바에 _ALT ZONES._라는 토글 버튼이 나타납니다. 이 버튼을 클릭하면 alt zone과 일반 zone 사이를 전환할 수 있습니다.

### zone을 이동하지 않고 비활성화하는 방법

Alt zone을 추가한 다음, 해당 zone을 오른쪽 클릭하고 _Disable output_ 버튼(선이 그어진 원)을 클릭합니다. 이제 Alt zones를 활성화하면 콘텐츠가 이 비활성화된 alt zone으로 리디렉션됩니다.
