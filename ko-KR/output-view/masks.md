# 🟩 마스크

{% hint style="danger" %}
**안전 고지**

소프트웨어 마스크를 레이저 출력으로부터 사람을 보호하는 유일한 방법으로 사용해서는 절대 안 됩니다. 레이저와 사람이 노출될 가능성이 있는 모든 위치 사이에는 항상 물리적 차단 장치/마스킹을 설치해야 합니다. 이 시스템은 2차 중복 보호 수단이나 안전이 중요하지 않은 용도(예: 카메라 또는 프로젝터 보호)로 고려할 수 있지만, 어떤 소프트웨어든 실패할 수 있다는 점을 기억하세요.
{% endhint %}

***

특정 위치에서 레이저 출력을 제거하거나 줄이기 위해 하나 이상의 가상 마스크를 추가할 수 있습니다.

툴바의 아이콘을 클릭하여 마스크를 추가합니다. Output view에 빨간색 사각형이 나타나며, 이것이 새 마스크입니다. 클릭 후 드래그하여 위치를 이동할 수 있고, 모서리를 드래그하여 크기를 변경할 수 있습니다. 균일하지 않은 형태를 만들고 싶다면 모서리를 드래그하는 동안 `Alt / Option`을 누르세요. (마스크가 한 번 균일하지 않은 형태가 되면 더 이상 `Alt / Option` 키를 누를 필요가 없습니다.)

마스크를 마우스 오른쪽 버튼으로 클릭하면 추가 설정을 볼 수 있습니다.

* Reduction percentage - 마스크 내부에서 레이저 출력이 얼마나 감소되는지를 나타냅니다. 기본값은 100%로, 마스크 내부가 완전히 차단됩니다. 원하는 만큼 레이저 출력을 줄일 수 있습니다. 완전 차단이 아닌 경우 마스크 색상이 주황색으로 바뀝니다.
* DELETE MASK 버튼도 이 마우스 오른쪽 클릭 메뉴에 있습니다.
