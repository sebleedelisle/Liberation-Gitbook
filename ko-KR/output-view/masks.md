# 🟩 Masks

{% hint style="danger" %}
**안전 안내**

소프트웨어로 설정한 masks만으로 사람이 laser 출력에 노출되는 것을 보호해서는 절대 안 됩니다. laser와 사람이 노출될 수 있는 모든 영역 사이에는 항상 물리적 차단물 또는 masking을 설치해야 합니다. 이 시스템은 보조적인 중복 안전 장치 또는 안전이 핵심이 아닌 용도(예: 카메라나 프로젝터 보호)로 사용하되, 어떤 소프트웨어도 실패할 수 있다는 점을 항상 기억하세요.
{% endhint %}

***

특정 위치에서 laser 출력을 제거하거나 줄이기 위해 가상 mask 하나 이상을 추가할 수 있습니다.

툴바의 아이콘을 클릭하면 mask가 추가됩니다. Output view에 빨간색 사각형이 나타나며, 이것이 새 mask입니다. 클릭하여 드래그하면 위치를 이동할 수 있고, 모서리를 드래그하면 크기를 변경할 수 있습니다. 균일하지 않은 형태를 만들려면 모서리를 드래그하는 동안 `Alt / Option`을 누르세요. (한번 비균일 형태가 된 뒤에는 더 이상 `Alt / Option` 키를 누르지 않아도 됩니다.)

mask 위에서 마우스 오른쪽 버튼을 클릭하면 추가 설정이 표시됩니다.

* Reduction percentage - mask 안에서 레이저의 출력 감소량을 나타냅니다. 기본값은 100%로 설정되어 있으며, 이는 mask 안에서 완전히 차단된다는 뜻입니다. 원하는 만큼 레이저 출력을 줄일 수 있습니다. 완전히 차단하지 않는 경우 mask 색상이 주황색으로 바뀝니다.
* DELETE MASK 버튼도 이 오른쪽 클릭 메뉴에 있습니다.
