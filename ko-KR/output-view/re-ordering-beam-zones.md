# 🟩 빔 존 재정렬

빔 존은 생성한 순서대로 표시되지만, 나중에 순서를 다시 정렬할 수 있습니다. _View -> Beam Zone Organiser_를 엽니다.

빔 존 목록과 각 빔 존이 연결된 레이저를 확인할 수 있습니다. 원하는 순서가 되도록 존을 드래그 앤 드롭하거나, _Autosort by laser_를 클릭해 각 존이 속한 레이저에 따라 자동으로 연속 정렬되게 할 수 있습니다.

{% hint style="info" %}
빔 존에 사용자 지정 이름을 지정하지 않은 경우 기본 이름으로 _Beam 1, Beam 2,_ 등이 사용됩니다. 순서를 다시 정렬하면 정렬 순서에 따라 기본 이름도 함께 업데이트됩니다. 혼동될 수 있으므로 존을 재정렬할 때 이 점에 유의하세요.
{% endhint %}
