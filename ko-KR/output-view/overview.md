# ✅ 개요

Output view는 각 레이저의 전체 출력 범위를 보여줍니다. 이 보기에서 zone을 조정하고 mask 영역을 추가할 수 있습니다.

<figure><img src="../.gitbook/assets/Output View.png" alt="" width="375"><figcaption><p>beam zone, canvas zone, mask가 있는 <em>OUTPUT</em> view입니다.</p></figcaption></figure>

상단 툴바에서 _OUTPUT_ 버튼을 클릭하여 Output view를 선택합니다.

<figure><img src="../.gitbook/assets/Top icon bar with output selected.png" alt=""><figcaption></figcaption></figure>

### 레이저 선택

상단의 숫자 버튼을 사용해 편집할 레이저를 선택합니다. 또는 `Tab` 키로 차례대로 전환하거나 숫자 키를 누를 수 있습니다.

### Output view에서 이동하기

빈 공간(또는 잠긴 zone)을 클릭한 상태로 드래그하면 보기를 이동할 수 있습니다.

### Output view 확대 및 축소

마우스 휠을 사용해 확대하거나 축소합니다.

### 툴바 아이콘

<figure><img src="../.gitbook/assets/Output Tool Bar Icon.png" alt="" width="269"><figcaption><p><em>OUTPUT</em> view 왼쪽의 툴바</p></figcaption></figure>

보기 왼쪽에 툴바가 표시됩니다. 아이콘 위에 마우스를 올리면 해당 기능을 확인할 수 있습니다.

* 레이저에 beam zone 추가
* 이 레이저에 포함할 canvas zone 선택
* 레이저에 mask 추가
* 이 레이저의 test pattern 켜기/끄기
* 이 레이저의 test pattern 선택(기본 test pattern보다 우선 적용)
* 그리드에 스냅 켜기/끄기
* 그리드 크기 조정(0은 가장 작음, 8은 가장 큼). (그리드 크기는 2의 거듭제곱 단위로 증가하므로, 그리드 크기를 줄여도 스냅된 모든 포인트는 계속 정렬된 상태를 유지합니다).
