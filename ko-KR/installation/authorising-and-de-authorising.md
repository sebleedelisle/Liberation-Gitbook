# ✅ 인증 및 인증 해제

### Liberation 인증하기

Liberation을 처음 열면 _free mode_로 실행되며 _About panel_이 표시됩니다.

<figure><img src="../.gitbook/assets/authorisation-about-panel.png" alt="" width="375"><figcaption></figcaption></figure>

_AUTHORISE ONLINE_ 버튼을 클릭하면 웹 브라우저가 열립니다. 아직 로그인하지 않았다면 이 단계에서 로그인을 요청받습니다.

이제 시스템이 라이선스로 설치된 Liberation을 자동으로 인증하며, 확인 메시지가 표시됩니다.

Liberation으로 돌아오면 _About panel_이 업데이트된 것을 볼 수 있습니다(몇 초 정도 기다려야 할 수 있습니다).

<figure><img src="../.gitbook/assets/authorise-about-panel-authorised" alt="" width="375"><figcaption></figcaption></figure>

{% hint style="warning" %}
이미 라이선스에서 허용하는 최대 컴퓨터 수를 인증했다면, 다른 컴퓨터 중 하나를 인증 해제하거나 라이선스를 업그레이드해야 합니다.
{% endhint %}

{% hint style="info" %}
라이선스가 여러 개인 경우, 해당 컴퓨터에 할당할 라이선스를 선택하라는 메시지가 표시됩니다.
{% endhint %}

축하합니다! 이제 Liberation 설치가 인증되었으며 레이저로 출력할 수 있습니다. 하지만 레이저를 arm 상태로 전환하기 전에 [빠른 시작 가이드](../getting-started.md "mention") 및 [레이저 설정 절차 개요](../setting-up/setting-up-lasers.md "mention")를 먼저 읽어 주세요.

{% hint style="info" %}
_Liberation -> About Liberation_ 또는 _Liberation -> Authorise/Deauthorise this computer_ 메뉴를 통해 언제든지 _About panel_을 열 수 있습니다.
{% endhint %}

### Liberation 인증 해제하기

**Liberation 안에서** - _Liberation -> Authorise / De-authorise this Computer_ 메뉴를 열고 _DEAUTHORISE COMPUTER_ 버튼을 클릭합니다. 이것으로 끝입니다! 이 기능을 사용하려면 온라인 상태여야 합니다.

<figure><img src="../.gitbook/assets/Screenshot 2024-08-28 at 10.18.57.png" alt=""><figcaption></figcaption></figure>

또는 웹사이트에서도 진행할 수 있습니다. 메뉴에서 _Your licences_를 선택한 다음, 해당 라이선스의 _MANAGE_를 클릭합니다. 계정에 라이선스가 하나만 있는 경우 _Your licences_를 선택하면 해당 라이선스 페이지로 바로 이동합니다. 라이선스 정보와 인증한 컴퓨터 목록이 표시됩니다.

<figure><img src="../.gitbook/assets/account-your-licences.png" alt=""><figcaption></figcaption></figure>

인증 해제하려는 컴퓨터 옆의 _De-authorise_ 링크를 클릭합니다.

마지막 라이선스 갱신 이후 해당 컴퓨터가 온라인 상태였던 적이 없다면 즉시 인증 해제됩니다. 그렇지 않은 경우 해당 컴퓨터는 인증 해제 대기열에 들어갑니다. 즉, 다음에 해당 컴퓨터가 인터넷에 연결되거나 다음 라이선스 갱신일이 되었을 때, 둘 중 먼저 발생하는 시점에 인증 해제가 자동으로 진행됩니다.
