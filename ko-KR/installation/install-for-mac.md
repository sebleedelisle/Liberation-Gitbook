# ✅ Mac에 설치하기

### 호환성

macOS 버전 12(Monterey) 이상이 필요합니다. 대규모 쇼를 실행하려는 경우 특히 Apple Silicon을 권장합니다. 컴퓨터 성능이 빠를수록 애니메이션 레이저 출력이 더 부드럽게 재생됩니다.

### 설치 과정

활성 구독이 있으면 링크가 전송됩니다. 또는 [_My subscriptions_](https://liberationlaser.com/account/my-products) 페이지에서 _Download_ 링크를 클릭할 수 있습니다.

다음 페이지가 표시됩니다:

<figure><img src="../.gitbook/assets/windows-download-page.png" alt=""><figcaption></figcaption></figure>

_Download latest for Mac_ 을 클릭하면(버전 번호는 시간이 지나면서 변경될 수 있습니다) Liberation DMG 설치 파일이 Downloads 폴더에 저장됩니다. 파일을 찾아 더블 클릭하여 엽니다.

<figure><img src="../.gitbook/assets/mac-dmg-window" alt="" width="375"><figcaption></figcaption></figure>

왼쪽의 Liberation 아이콘을 오른쪽의 Applications 폴더로 드래그합니다(관리자 비밀번호가 필요할 수 있습니다). 이제 Applications 폴더에서 Liberation을 찾아 더블 클릭하여 열 수 있습니다.

이제 Liberation이 실행됩니다! 레이저로 출력하려면 이제 [인증 과정](authorising-and-de-authorising.md)을 진행해야 합니다.
