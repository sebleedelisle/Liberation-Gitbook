# ✅ 라이선스 작동 방식

컴퓨터에 Liberation을 설치하면 _인증_하기 전까지 _demo mode_로 실행됩니다.

{% hint style="info" %}
_demo mode_에서는 레이저 하드웨어로 출력하는 것 외에는 제한이 없습니다. 쇼를 연습하고, 타임라인을 만들고, 파일을 저장하고 불러오는 작업을 제한 없이 할 수 있습니다. Liberation은 원하는 만큼 많은 컴퓨터에 _demo mode_로 설치할 수 있으며, 레이저를 사용할 준비가 되었을 때만 설치를 인증하면 됩니다!
{% endhint %}

활성화(arm)할 수 있는 레이저 수 외에도 DMX fixtures를 사용할 수 있습니다. [dmx-control](../dmx-control/)를 참조하세요.

Liberation을 인증할 수 있는 컴퓨터 수에도 제한이 있으며, 이는 구독 등급에 따라 다릅니다.

{% hint style="danger" %}
각 라이선스는 사용자 1명에게만 제공되며, 한 번에 하나의 쇼에만 사용할 수 있습니다. 사용자당 설치를 여러 대에 할 수 있는 이유는 **항상** 예비로 인증된 컴퓨터를 최소 한 대 준비해 두어야 하기 때문입니다!
{% endhint %}

### 인터넷 연결

Liberation의 라이선스 및 인증은 인터넷을 통해 작동합니다. 앱이 서버에 연결되어 설치를 사용자의 구독과 연결합니다.

설치가 인증되면 구독이 갱신될 때까지 다시 온라인 상태일 필요는 없습니다. 갱신 날짜는 _About panel_에서 확인할 수 있습니다. 짧은 유예 기간(월간 구독은 7일)이 있지만, 인증이 갱신되고 활성 상태를 유지하도록 갱신 날짜 이후 가능한 한 빨리 온라인으로 접속하는 것이 좋습니다.
