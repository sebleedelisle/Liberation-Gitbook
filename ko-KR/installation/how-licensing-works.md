# ✅ 라이선스 작동 방식

컴퓨터에 Liberation을 설치하면 _무료 모드_로 실행되며, 사용하려면 _인증_해야 합니다.

{% hint style="info" %}
_무료 모드_에서는 레이저 하드웨어로 Output을 내보내는 것 외에는 제한이 없습니다. 쇼를 연습하고, 타임라인을 만들고, 파일을 저장하고 불러오는 작업을 제한 없이 할 수 있습니다. Liberation은 원하는 수의 컴퓨터에 _무료 모드_로 설치할 수 있으며, 레이저를 사용할 준비가 되었을 때만 설치를 인증하면 됩니다!
{% endhint %}

arm 상태로 둘 수 있는 레이저 수 외에도 DMX fixture를 사용할 수 있습니다. 자세한 내용은 [DMX / Art-Net](../dmx-control/ "mention")을 참고하세요.

또한 라이선스 등급에 따라 Liberation을 인증할 수 있는 컴퓨터 수에도 제한이 있습니다.

{% hint style="danger" %}
각 라이선스는 사용자 1명에게만 허용되며, 동시에 하나의 쇼에만 사용할 수 있습니다. 사용자당 여러 대의 설치를 사용할 수 있는 이유는 **항상** 최소 한 대의 인증된 예비 컴퓨터를 준비해 두어야 하기 때문입니다!
{% endhint %}

### 인터넷 연결

Liberation의 라이선스 및 인증은 인터넷을 통해 작동합니다. 앱이 서버에 연결되어 설치를 사용자의 유료 라이선스와 연결합니다.

인증이 완료되면 반복 결제 라이선스가 다음 결제일(_About panel_에 표시됨)에 도달할 때까지 다시 온라인 상태일 필요가 없습니다. 월간 반복 결제 라이선스에는 일반적으로 짧은 유예 기간(7일)이 포함되지만, 유료 기능을 중단 없이 계속 사용하려면 결제일 이후 가능한 한 빨리 온라인으로 연결해야 합니다.

유예 기간은 갱신 및 결제 문제를 위한 것입니다. 유료 기간이 끝난 뒤 취소되었거나 의도적으로 일시 중지된 라이선스에는 적용되지 않습니다. 결제에 실패한 경우 유예 기간 내에 카드 정보를 업데이트해야 유료 기능을 계속 사용할 수 있습니다.

현재 라이선스 등급도 상단 바에 표시됩니다. 등급 이름을 클릭하거나, 라이선스가 없는 경우 _FREE MODE_를 클릭하면 About panel이 열립니다.
