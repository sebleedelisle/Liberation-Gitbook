# ✅ 라이선스 작동 방식

컴퓨터에 Liberation을 설치하면 _데모 모드_로 실행되며, 사용하려면 _인증_해야 합니다.

{% hint style="info" %}
_데모 모드_에서는 레이저 하드웨어로 Output을 내보내는 것 외에는 제한이 없습니다. 쇼를 연습하고, 타임라인을 만들고, 파일을 저장하고 불러오는 작업을 제한 없이 할 수 있습니다. Liberation은 원하는 수의 컴퓨터에 _데모 모드_로 설치할 수 있으며, 레이저를 사용할 준비가 되었을 때만 설치를 인증하면 됩니다!
{% endhint %}

arm 상태로 둘 수 있는 레이저 수 외에도 DMX fixture를 사용할 수 있습니다. 자세한 내용은 [DMX 제어](../dmx-control/)를 참고하세요.

또한 구독 등급에 따라 Liberation을 인증할 수 있는 컴퓨터 수에도 제한이 있습니다.

{% hint style="danger" %}
각 라이선스는 사용자 1명에게만 허용되며, 동시에 하나의 쇼에만 사용할 수 있습니다. 사용자당 여러 대의 설치를 사용할 수 있는 이유는 **항상** 최소 한 대의 인증된 예비 컴퓨터를 준비해 두어야 하기 때문입니다!
{% endhint %}

### 인터넷 연결

Liberation의 라이선스 및 인증은 인터넷을 통해 작동합니다. 앱이 서버에 연결되어 설치를 사용자의 구독과 연결합니다.

설치가 인증되면 구독이 갱신될 때까지 다시 온라인 상태일 필요가 없습니다. 갱신 날짜는 _About panel_에서 확인할 수 있습니다. 짧은 유예 기간(월간 구독의 경우 7일)이 있지만, 인증이 갱신되고 활성 상태로 유지되도록 갱신 날짜 이후 가능한 한 빨리 온라인으로 연결하는 것이 좋습니다.
