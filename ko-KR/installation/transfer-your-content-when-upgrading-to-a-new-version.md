# 🟩 새 버전으로 업그레이드할 때 콘텐츠 이전하기

Liberation은 콘텐츠를 작업 폴더에 저장하며([where-to-find-the-working-folder.md](../troubleshooting/where-to-find-the-working-folder.md) 참조), 이 폴더는 새 버전이 릴리스될 때마다 변경됩니다. 이전 버전의 작업 파일을 새 버전에서도 유지하려면:&#x20;

1. 기존 버전의 Liberation을 엽니다.
2. _File->Project->Export Project_ 를 사용합니다. 이렇게 하면 Liberation의 모든 항목, 즉 레이저 설정, 타임라인, Clip Deck 등 모든 것이 저장됩니다!&#x20;
3. 이 버전을 닫은 다음 새 버전의 Liberation을 엽니다.
4. _File->Project->Import Project_ 를 사용하고, 2단계에서 내보낸 프로젝트 파일을 선택합니다.&#x20;

이제 모든 콘텐츠가 새 버전에 있어야 합니다.

#### Liberation 여러 버전 유지하기

새 버전을 설치하기 전에 이전 버전의 이름을 변경하세요. 저는 보통 버전 번호를 사용합니다. 예: Liberation103.app(Windows에서는 .exe) 원하는 만큼 여러 버전의 Liberation을 보관할 수 있지만, 한 번에 하나만 실행할 수 있습니다. <br>

#### 이전 버전의 Liberation을 새 버전으로 덮어쓴 경우

가장 안정적인 방법은 이전 버전의 Liberation을 다시 설치한 다음 위에 설명된 과정을 진행하는 것입니다.&#x20;

또는 작업 폴더를 복사한 뒤 새 버전 번호로 이름을 변경해 볼 수도 있습니다.&#x20;
