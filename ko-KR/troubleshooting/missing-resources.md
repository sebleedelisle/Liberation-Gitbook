# ✅ 시작 시 Missing resources 오류

시작할 때 "missing resources" 오류가 표시되면, Liberation 실행에 필요한 중요한 파일이 누락되었다는 뜻입니다. Windows에서는 설치 폴더의 파일을 실수로 삭제하거나 이동한 경우 발생할 수 있습니다. Mac에서는 설치 파일이 어떤 이유로 손상된 경우 발생할 수 있습니다.

### 해결 방법

이 문제를 해결하는 가장 좋은 방법은 Liberation을 다시 설치하는 것입니다.
