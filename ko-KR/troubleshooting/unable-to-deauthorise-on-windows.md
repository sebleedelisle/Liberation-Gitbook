# ✅ Windows에서 인증 해제가 되지 않나요?

#### Windows에서 인증 해제가 되지 않나요?

Windows에서 컴퓨터의 인증을 해제할 수 없다면, 먼저 다른 버전에서 다시 인증하기 전에 처음 인증할 때 사용한 것과 동일한 Liberation 버전으로 라이선스 인증 해제를 시도하세요.

그래도 해결되지 않고 1.0 이전 버전을 사용 중이라면, 이전 Windows용 Liberation 빌드가 컴퓨터를 식별하던 방식 때문에 발생한 문제일 가능성이 큽니다. 해당 버전에서는 머신 ID를 생성하는 시스템의 신뢰성이 낮았고, 경우에 따라 하드웨어가 명확히 변경되지 않았더라도 재시작 후 ID가 바뀔 수 있었습니다.

버전을 변경하지 않았는데도 인증 해제를 진행할 수 없다면 support@liberationlaser.com 으로 문의해 주세요. 저희가 해당 머신의 인증을 수동으로 해제해 드릴 수 있습니다.

**이 문제가 발생하는 이유**

초기 Windows용 Liberation 빌드(1.0 이전)에서는 머신 ID 생성을 위해 권장되는 Windows 시스템 방식을 사용했습니다. 하지만 이 방식은 일부 상황에서 일관성이 떨어지는 것으로 확인되었습니다. 이 때문에 버전 1.0에서 라이선스 시스템을 다시 작성하여 더 안정적인 여러 방식을 조합하도록 변경했으며, 현재는 안정적으로 작동합니다.

그 결과, 이전 버전의 Liberation에서 사용하는 컴퓨터 ID가 현재 버전에서 사용하는 ID와 다를 수 있습니다. ID가 이미 변경된 경우에는 지원팀에서 수동으로 인증 해제를 처리해야 합니다.
