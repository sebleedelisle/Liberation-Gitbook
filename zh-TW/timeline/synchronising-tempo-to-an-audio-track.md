# 🟩 將速度同步到音訊軌

<iframe src="https://www.youtube.com/embed/sL0SsuTf7Pc" title="將速度同步到音訊軌" style="width: 100%; aspect-ratio: 16 / 9; border: 0;" allowfullscreen></iframe>

Liberation 的時間軸可配合固定或變動的速度使用，但要取得可靠的同步，第一步一定是找出速度，並正確對齊音訊。本節說明建議的工作流程。

#### 1. 對齊第一個強拍

將音訊軌加入時間軸，並確認它已吸附到拍點，讓音軌的**第一個音樂強拍**精準對齊小節的起點。

這個步驟非常重要。

如果音訊不是自然地從強拍開始，你有兩個選項：

* **調整 Clip 延遲**\
  右鍵點擊音訊 Clip，調整 Delay 值，直到第一個強拍對齊拍點或小節標記。
* **在外部修剪音訊**\
  使用 Audacity 等外部編輯器編輯音訊檔，讓檔案精準從第一個強拍開始，然後重新匯入。

{% hint style="info" %}
**重要：**\
如果音訊開頭沒有對齊拍點或小節，當你變更速度時，音樂聽起來的起始位置會前後偏移。這會讓準確比對速度變得非常困難。
{% endhint %}

#### 2. 設定初始速度

如果你大概知道 BPM，請將它輸入時間軸的速度控制欄位，並從開頭開始播放。

播放時仔細觀察拍點與小節標記。

* 如果標記逐漸跑到音樂前面，請稍微降低速度。
* 如果標記落在音樂後面，請稍微提高速度。
* 停止播放、調整速度，然後再試一次。

對大多數現代音樂來說，速度是固定的整數 BPM。找到正確數值後，它應該會在整首音軌中保持鎖定。

#### 3. 使用波形作為視覺參考

比對速度時，音訊波形是很有用的參考。

* 瞬態與峰值應該對齊垂直的小節標記。
* 放大檢查多個小節範圍內的對齊狀況。

**提示：**\
使用滑鼠滾輪或觸控板手勢縮放時間軸。使用水平滾輪或手勢左右移動。在放大狀態下操作，會讓細微調整容易得多。

#### 4. 非整數 BPM 的音軌

如果音軌不是使用整數 BPM，漂移會比較緩慢。

* 進一步放大。
* 使用更小幅度的速度調整。
* 不要只檢查前幾個小節，而是檢查音軌中較長區段的對齊狀況。

#### 5. 有速度變化的音樂

如果音樂會加速或減速，請使用 Tempo Map：

* 播放音軌並觀察拍點標記。
* 當漂移變得明顯時，在該位置加入速度變化。
* 調整新區段的速度，直到它再次鎖定。

針對音樂中的每一次速度變化，重複這個流程。

#### 6. 外部速度分析（選用）

最後不得已時，你可以在 Logic Pro 等 DAW 中分析音軌，並自動產生 Tempo Map。請注意，這通常會產生大量速度變化，有時甚至每個小節一個；對大多數演出來說，這可能比實際需要更詳細。
