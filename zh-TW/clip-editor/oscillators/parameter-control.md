# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_Parameter Control_ 是一種方法，可從 Clip Editor 外部控制 effect 或 Clip 內的各項設定。

* **range min / range max** - 參數 level 會對應到的最小值與最大值
* **level** - 此參數的預設設定。這就是可從外部變更的值。你也可以手動調整，查看它會產生的效果。
* **parameter number** - 每個 _Parameter Control_ 都可以指定一個編號，這個編號會反映在 Clip 外部的控制項上。你可以讓多個 Parameter Control 使用相同編號，它們會同時被調整。

{% hint style="warning" %}
目前只有 effect 的 parameter 1 可透過 APC40 控制。effect 與 Clip 內的所有參數都可在右鍵選單中使用。未來將會有更多這類參數可對應到 MIDI 控制器。
{% endhint %}
