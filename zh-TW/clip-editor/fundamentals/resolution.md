# 🟩 解析度

大多數 _Creators_ 都有 _resolution_ 選項。這是進階設定，用來決定形狀有多少細分，也會影響形狀之後被拉伸或由修改器處理時，邊緣看起來有多鋸齒。

這個值代表細分的大小，因此數值越低，解析度越高。預設值 4 對大多數 Clips 來說都沒問題；如果你需要更細緻的品質，可以考慮降低這個數值。請注意，較高的解析度會需要更多 CPU 資源。
