# ✅ 鍵盤快捷鍵

快捷鍵以 `Cmd / Ctrl` 表示：在 macOS 上使用 `Cmd`，在 Windows 上使用 `Ctrl`。

| 快捷鍵                            | 動作                                                  |
| --------------------------------- | ----------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Arm 所有雷射                                         |
| `Escape`                          | Disarm 所有雷射                                      |
| `Tab`                             | 切換 view，並在各雷射之間循環切換                    |
| `Shift + Tab`                     | 切換 view，並反向在各雷射之間循環切換                |
| `1-9`                             | 跳到指定雷射                                         |
| `Left / Right Arrow`              | 將 Clip Deck 向左／右移動                            |
| `Cmd / Ctrl + Left / Right Arrow` | 將 Clip Deck 向左／右移動一頁                        |
| `Shift + Left / Right Arrow`      | 將 zone 向左／右移動一頁                             |
| `Space`                           | 播放／暫停（啟用 _Timeline_ tempo source 時）        |
| `T`                               | Tap tempo                                            |
| `R`                               | 重設小節                                             |
| `Y`                               | 將 tempo 四捨五入到最接近的 BPM                     |
| `Cmd / Ctrl + ,`                  | 開啟設定面板                                         |
| `Cmd / Ctrl + F`                  | 切換全螢幕模式                                       |
| `Cmd / Ctrl + L`                  | 開啟雷射輸出設定面板                                 |
| `Cmd / Ctrl + K`                  | 開啟 laser controller 指派面板                       |
| `Cmd / Ctrl + J`                  | 開啟 3D 視覺化器設定面板                             |
| `Cmd / Ctrl + Shift + Plus (+)`   | 提高介面縮放比例                                     |
| `Cmd / Ctrl + Shift + Minus (-)`  | 降低介面縮放比例                                     |
| `Cmd / Ctrl + 0`                  | 重設介面縮放比例                                     |
| `Cmd / Ctrl + A`                  | Timeline：全選                                       |
| `Delete`                          | Timeline：刪除選取的區塊                             |
| `Delete`                          | Clip Editor：刪除選取的 nodes                        |
| `Cmd + Q` (macOS)                 | 結束應用程式                                         |
| `Alt + F4` (Windows)              | 結束應用程式                                         |
