# 🟩 整理你的 Clip Deck

### 移動／複製 Clips

按住並拖曳即可移動 Clip。如果已選取多個 Clip（請參閱 [啟動／停止 Clips](starting-stopping-clips.md)），它們會一起移動。按住 `Alt / Option` 可複製它們。將 Clip 放到空白位置即可。

{% hint style="info" %}
如果需要選取大量 Clips，可以按住並拖曳套索框選，然後在仍然拖曳的狀態下捲動 Clip Deck（可使用方向鍵、APC 40，或觸控板／滑鼠的橫向捲動）。套索會延伸到 Clip Deck 邊緣之外。
{% endhint %}

### 刪除 Clips

若要刪除單一 Clip，請使用右鍵選單。

若要刪除多個 Clips，請先選取它們（使用 `Alt / Option + Shift` 點擊，或拖曳套索框選），再將它們拖出 Clip Deck。

### 加入空白空間

你可以在 Clip 按鈕上按右鍵，然後按 _ADD EMPTY COLUMN TO RIGHT_，在 Clip Deck 中加入一個空白欄。

### 合併多個 Clip Deck

你可以匯入既有的 Clip Deck，並將它附加到目前 Clip Deck 的結尾。使用選單 _File -> Clips/Effects -> Append Clip Deck._

選取一組 Clips（使用套索或 `Alt / Option + Shift` 點擊），然後選擇 _File -> Clips/Effects -> Export Selected Clips._ 這會匯出只包含這些 Clips 的 Clip Deck，之後可再附加到另一個 Clip Deck。

### 依時間軸整理 Clips

匯入時間軸時，也會有一些 Clip 選項。你可以選擇取代目前的 Clip Deck、附加該時間軸的整個 Clip Deck，或只附加它使用的 Clips。

如果專案中有多個時間軸，你可以依時間軸重新整理 Clips。開啟 _Timeline selection_ 面板，然後點擊 _ORGANISE CLIPS BY TIMELINE_。你現在會看到所有 Clips 依其時間軸分組顯示。如果某個 Clip 用於多個時間軸，它會在每個群組中各複製一份。
