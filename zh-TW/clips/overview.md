# 🟩 概觀

### 什麼是 Clip？

Liberation 中的所有雷射內容都儲存在 Clips 裡。

Clips 可以包含空中光束效果或圖形動畫，通常會以循環方式播放。它們可以指定輸出到任何 zone（或 Canvas 目標區域），並透過 Clip Deck 內的 Clip 按鈕觸發。

Clip 內容是使用功能完整、以 node 為基礎的 Clip Editor 系統建立的。（詳情請參閱 [Clip Editor 簡介](../clip-editor/clip-editor-intro.md)）

### Clip Deck

<figure><img src="../.gitbook/assets/clips-clip-deck" alt=""><figcaption><p>預設的 Clip Deck</p></figcaption></figure>

Clip Deck 是螢幕上的 Clips 格狀區域，通常會延伸到你目前可見範圍之外。

可使用下列方式將 Clip Deck 向左或向右捲動：

* 左右方向鍵
* 觸控板雙指滑動（如果你的滑鼠支援，也可以用滑鼠左右捲動！）
* APC40 上的 _Cue Level_ 滾輪

可使用下列方式將 Clip Deck 向左或向右捲動一整頁：

* `Cmd / Ctrl + Left / Right Arrow`
* APC40 上的 _Device <- and ->_ 按鈕

#### 迷你 Clip 視覺化檢視器
