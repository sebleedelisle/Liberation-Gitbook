# ✅ 啟動／停止 Clips

{% hint style="info" %}
你可以使用畫面上的按鈕來啟動和停止 Clip，但使用 APC40 MIDI controller 會好得多。Liberation 已針對這項硬體最佳化，讓它能順暢搭配使用；在投入雷射設備時，應該把它視為重要的一部分。
{% endhint %}

### 啟動和停止 Clip

若要啟動某個 Clip，按下該 Clip 的按鈕（在畫面上或在你的 MIDI controller 上皆可）。

若要停止該 Clip，再按一次即可。

如果你啟動新的 Clip（在同一個群組中），第一個 Clip 會自動停止。

若要啟動新的 Clip，同時不停止其他 Clip，可以使用以下任一方式：

* 啟動新的 Clip 時按住 `Shift` 鍵（或 APC40 Shift 按鈕）；或
* 在啟動新的 Clip 時，同時重新按下目前正在播放的 Clip。

各個 Clip 群組彼此獨立，因此在某個群組中啟動 Clip，不會影響其他群組中的 Clip。請參閱[群組](groups.md "mention")

### Flash mode

如果某個 Clip 位於設定為 _flash mode_ 的群組中，它的行為會稍微不同；只有在你按住該 Clip 按鈕時，Clip 才會持續播放。一放開就會停止。（預設情況下，Clip 群組 3，也就是紅色的群組，會設為 _flash mode_。）

### 停止所有 Clip

若要**停止所有正在播放的 Clip**，請按下 **STOP** 按鈕。

{% hint style="info" %}
按 **STOP** 按鈕**兩次**，可略過 Clip 的淡出時間並立即黑場。
{% endhint %}

若要停止特定群組中所有正在播放的 Clip，先按下該群組按鈕，然後按 **STOP** 按鈕。

若要停止所有 Clip 但保留其中一個，請按住你想保留的正在播放 Clip，然後在按住時同時按下 **STOP** 按鈕。接著放開該 Clip 按鈕。（這也適用於多個 Clip，只要你一次按得住都可以！）

### 目前選取的 Clip

<figure><img src="../.gitbook/assets/clips-selected-active.png" alt="" width="269"><figcaption><p>目前正在播放的兩個 Clip。白色外框表示右側的 Clip 是<em>目前選取的 Clip。</em></p></figcaption></figure>

當 Clip 目前正在播放時，它會在畫面上亮起（它的迷你 Clip 視覺化顯示也會閃爍）。你也會注意到，最後按下的 Clip 周圍有白色外框。這表示它是_目前選取的 Clip_。

在 Clip 上按住 `Alt / Option` 並點選，可以選取它而不啟動它。按住 `Alt / Option + Shift` 並點選，可選取多個 Clip。

你也可以在 Clip Deck 上拖曳框選範圍，以選取多個 Clip。

{% hint style="info" %}
APC40 也有 `Alt` 和 `Shift` 按鈕可用來進行選取。
{% endhint %}
