# 🟩 色彩校正

色彩校正可確保投影機的紅、綠、藍雷射在所有亮度等級下，都能以平順且可預期的方式輸出光線。不同投影機可能有非線性的亮度曲線，也就是 50% 紅色看起來可能比 100% 紅色的一半亮度亮很多，或暗很多。校正會修正這個問題，讓色彩混合更乾淨、漸層更平順，白色也能保持平衡。

#### 讓投影機預熱

雷射二極體會隨著溫度升高而改變特性。校正前務必先讓投影機穩定：

* 投射明亮的畫面，例如 **White rectangle test pattern (11)**，至少 **15–20 分鐘**。
* 這可確保你設定的色彩平衡在演出期間保持一致。

#### 校正測試的運作方式

請使用 test pattern 進行校正（請參閱[測試圖樣](../output-view/test-patterns.md)）

* **5** – Red
* **6** – Green
* **7** – Blue
* **8** – White

每個 test pattern 都會顯示四條移動線：

* **最上方線條** – 100% 亮度、全速
* **第二條線** – 75% 亮度、75% 速度
* **第三條線** – 50% 亮度、50% 速度
* **第四條線** – 25% 亮度、25% 速度

因為亮度和_速度_會一起縮放，這些線條看起來應該要有相同亮度。如果其中一條看起來較亮或較暗，請調整對應的滑桿，直到它們一致。

每個 test pattern 也有第五條線，亮度為 **0%**，理論上不應該看見。這用來修正某些雷射在非常低的輸出等級下完全不出光的情況。如果你的雷射在低亮度時仍然不可見，請逐步提高 **0% setting**，直到線條剛好可見，接著再稍微調低，直到它再次消失。目標是找出雷射開始發光的臨界點，然後停在略低於該臨界點的位置——這樣淡入淡出會更自然，不會直接切掉最低亮度範圍。

#### 使用 Colour Calibration panel

此 panel 可讓你針對每個色彩通道（紅、綠、藍）在 100、75、50、25 和 0% 等級下分別控制。

1. **選擇一個 test pattern**（先從紅色開始）。
2. **調整滑桿**，讓 100、75、50 和 25% 的線條看起來有相同亮度。
3. 如果「關閉」的線條仍隱約可見，請**微調 0% 滑桿**。
4. **針對綠色和藍色重複操作。**
5. 切換到 **White test pattern (8)**。四條線看起來應該亮度一致，白色也應該呈現中性（沒有偏色）。

#### 調整白平衡

你也可以使用這套系統來調整**白平衡**。個別校正每個通道後，切換到 **White test pattern (8)**。如果輸出看起來有偏色（例如太綠或太藍），請調整紅、綠、藍通道的相對等級，直到線條呈現中性白。即使你的雷射功率差異相當大，校正仍能協助讓它們更接近彼此，並產生更乾淨、平衡的色彩混合。

#### 儲存校正

* 使用 **Store** 覆寫目前的 preset。
* 使用 **Store As** 建立新的 preset（如果你會使用多台雷射，這會很有用）。
