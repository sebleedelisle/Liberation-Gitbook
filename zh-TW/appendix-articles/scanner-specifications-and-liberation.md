# 🟩 掃描振鏡規格與 Liberation

### 掃描振鏡規格的混亂現實

點數率與掃描振鏡規格可能有點令人困惑。你常會看到像是 30kpps @ 8º 或 50kpps @ 4º 這類規格，但這些數字實際代表什麼並不一定很清楚。

{% hint style="info" %}
免責說明 — 我不是掃描振鏡硬體專家，但我有多年實務經驗，透過軟體與點串流產生方式，而不是硬體調校，讓各種不同品質的掃描振鏡呈現良好效果。以下內容是基於這些經驗。
{% endhint %}

#### **這些數字從哪裡來**

像「30K」和「50K」這類說法，是根據掃描振鏡在特定條件下，以這些點數率顯示 ILDA test pattern 時的評估結果所形成的簡寫。

當某個掃描振鏡標示為類似：_30Kpps @ 8°_，它真正的意思是：

> 「這組掃描振鏡在正確調校後，可以用 30,000 點/秒、8° 掃描角度重現 ILDA test pattern。」

這不是對實際使用效能的完整或完全標準化測量。事實上，它一開始根本不是設計來當作基準測試，而是用於**調校程序**。你以固定點數率（例如 30,000 點/秒）輸出已知圖案，然後調整阻尼與增益，直到圖案看起來正確。

但它仍然是目前最廣泛使用的參考，至少在信譽良好的製造商那裡，可以讓你大致了解掃描振鏡的品質。不過如果是_比較沒那麼可靠_的廠商，就另當別論了……

#### 如果你想依照標稱規格測試掃描振鏡

{% hint style="danger" %}
**這是進階技巧，如果不小心可能會損壞掃描振鏡。除非你很清楚自己在做什麼，否則不建議使用。**
{% endhint %}

你需要找到能輸出 [ILDA 測試圖案](https://ilda.com/technical.htm?r=7950)的軟體——我想 LaserShowGen 可能可以做到——並將輸出尺寸調整到符合指定的掃描角度（例如 8°）。請參考 ILDA 文件，了解如何分析輸出結果。

#### 為什麼它不一定是好的基準測試

首先，即使你的掃描振鏡品質很好，測試圖案仍可能顯示不正確，因為它們的調校方式不一定是針對這個測試最佳化。

它可以作為判斷「好」與「差」掃描振鏡的實用概略指南，但製造商有時會對這些規格採取比較寬鬆的解讀。

#### 那我要怎麼選擇好的掃描振鏡？

我認為只要確認它們是由有信譽的製造商製造就好。像 Cambridge Technology (CT)、Eye Magic (EMS) 和 ScannerMAX（Pangolin 的子公司）這類昂貴的高階掃描振鏡製造商一直都很可靠，基本上不會出錯。但當一對掃描振鏡要價大約 $1000 時，對很多剛起步的人來說，這甚至比雷射本體還貴！

所以我大多使用中國製造商的產品。Dragon Tiger (DT) 掃描振鏡品質不錯且價格合理，我認為 LightSpace 使用的就是它們。許多其他製造商（包括 OPT，以及 Able 的某些型號）也使用基於 DT 的系統。

Phenix Technology (PT) 通常屬於較低階的等級，但老實說，對大多數用途來說應該也夠用。

**如果你的掃描振鏡沒有品牌，那才可能是你需要擔心品質的時候！**

#### Liberation 如何協助

首先，對大多數用途來說，你不需要非常昂貴的掃描振鏡！價格合理的 30kpps DT，甚至 PT，都可以正常使用。預設的掃描振鏡設定刻意保守，而且大多數情況下_你不需要調整它們_（除了 _Scanner sync_ 之外）。

即使你有更好的掃描振鏡，也沒有必要讓它們以超出需求的方式運作。這會大幅延長它們的壽命。

#### 什麼是「point stream」？

你可能以前聽過這個詞——它是我們控制掃描振鏡路徑的方式。

point stream 是一串 X/Y 位置清單，每個位置都帶有顏色。若要繪製像白線這樣的內容，你會沿著那條線送出一連串點，並將它們全部設為白色。接著掃描振鏡會以固定速率，也就是每秒點數率 (PPS)，從一個點移動到下一個點，光束就會描繪出形狀。

#### 傳統雷射軟體中的運作方式

傳統雷射軟體會為每個 cue 儲存一組 point stream。對動畫 cue 來說，這通常表示每一影格都有獨立的 point stream。重點是所有內容都已完全預先決定。因此，提高點數率會讓掃描振鏡在同一份預先定義的資料中移動得更快。

{% hint style="info" %}
對舊式軟體來說，這種做法是必要的——當時的電腦速度根本不足以即時產生 point stream。因此通常會有獨立的 cue 編輯器，用來預先產生每一格動畫的資料。

這也解釋了為什麼內容可能會佔用數 GB 的空間。你實際上是在以相當高的取樣率儲存大量未壓縮的波形資料。
{% endhint %}

#### 為什麼「point rate」在 Liberation 中比較沒那麼有意義

Liberation 會即時產生 point stream，這給了我們很大的彈性。請注意 Laser Settings 面板中的 "Scanner speed" 設定。它讓我們可以加快或減慢掃描振鏡，但重要的是，它**不會**改變底層的點數率 (PPS)。

#### 等等，什麼？怎麼做到的？

我知道，一開始聽起來很奇怪。

因為 Liberation 會即時產生 point stream，所以它可以調整這個 point stream。點與點之間分得越開，掃描振鏡移動得越快；點與點越接近，掃描振鏡移動得越慢。

{% hint style="info" %}
在近期版本的 Liberation 中，實際的 _point rate_（位於進階設定中）完全不會影響掃描振鏡速度。相反地，renderer 會調整點的分布，以符合所選的點數率，同時保持掃描振鏡的運動不變。

這確實會影響點路徑的「解析度」，但主要只會在 graphics 上造成差異（也有助於更細緻地調整 _scanner sync_ 設定）。
{% endhint %}

#### 很好！所以這到底代表什麼？

好問題。以下是我的建議：

* 避免使用無品牌的掃描振鏡。如果能取得更快的掃描振鏡，就選快一點的——最低 30KPPS。
* 在大多數情況下，DT30 掃描振鏡就夠用；在較便宜的雷射中，PT30 掃描振鏡大概也可以。
* 如果你要做 graphics，在大多數情況下，增加雷射數量會比使用更快的掃描振鏡更有幫助。
* 當你進入較高階的系統後，任何成熟的高階品牌都可以。
* 如果你只能取得最便宜的無品牌掃描振鏡，Liberation 的預設設定相當保守，你應該仍能在基本 beam 工作上得到還可以的結果。如果它顯得吃力，請降低 **Speed** 設定（但不要改變點數率！）。

#### 那 ILDA Test Pattern 呢？

……它作為校正與參考工具仍然非常有用，但它從來不是設計成完整的基準測試，而且可能被製造商誤用或寬鬆解讀。
