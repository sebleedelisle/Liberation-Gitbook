# ✅ 授權運作方式

在電腦上安裝 Liberation 後，軟體會以 _free mode（免費模式）_ 執行，直到你完成_授權_為止。

{% hint style="info" %}
_free mode（免費模式）_ 除了無法輸出到雷射硬體之外，沒有其他限制。你可以練習演出、建立 timeline，並且不受限制地儲存與載入檔案。你可以在任意數量的電腦上以 _free mode（免費模式）_ 安裝 Liberation；只有在準備好使用雷射設備時，才需要授權你的安裝！
{% endhint %}

除了可以 arm 的雷射數量之外，你也可以使用 DMX 燈具。請參閱 [DMX](../dmx-control/ "mention")。

你可以授權 Liberation 的電腦數量也會依訂閱方案等級而有限制。

{% hint style="danger" %}
每個授權僅限一位使用者使用，且同一時間只能用於一場演出。每位使用者可以有一個以上的安裝，因為你**一定**應該隨時準備至少一台已授權的備用電腦！
{% endhint %}

### 網際網路連線

Liberation 的授權與驗證是透過網際網路運作；應用程式會連線到我們的伺服器，將某個安裝與你的訂閱建立關聯。

安裝完成授權後，在訂閱續期前不需要再次上線。你可以在 _About panel_ 中查看續期日期。系統提供一段短暫的寬限期（月繳訂閱為 7 天），但建議在續期日期後盡快上線，以確保你的授權完成續期並維持有效。

目前的 licence tier 也會顯示在頂端列。按一下 tier 名稱，或在未授權時按一下 _FREE MODE_，即可開啟 About panel。
