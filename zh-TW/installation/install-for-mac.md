# ✅ 在 Mac 上安裝

### 相容性

macOS 12（Monterey）或更新版本。建議使用 Apple Silicon，尤其是你想執行大型 show 時。電腦速度越快，雷射動畫輸出就會越順暢。

### 安裝流程

取得有效訂閱後，你會收到一個連結；也可以在 [_My subscriptions_](https://liberationlaser.com/account/my-products) 頁面點擊 _Download_ 連結。

你會看到這個頁面：

<figure><img src="../.gitbook/assets/windows-download-page.png" alt=""><figcaption></figcaption></figure>

點擊 _Download latest for Mac_（版本號會隨時間變動），Liberation DMG 安裝檔就會儲存到你的下載資料夾。找到檔案後，按兩下開啟。

<figure><img src="../.gitbook/assets/mac-dmg-window" alt="" width="375"><figcaption></figcaption></figure>

將左側的 Liberation 圖示拖曳到右側的 Applications 資料夾（可能需要輸入管理員密碼）。現在你可以在 Applications 資料夾中找到 Liberation，按兩下即可開啟。

Liberation 現在應該可以執行了！若要輸出到雷射，你接下來需要完成[授權流程](authorising-and-de-authorising.md)。
