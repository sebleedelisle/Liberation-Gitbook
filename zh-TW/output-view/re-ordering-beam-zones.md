# 🟩 重新排序 beam zones

Beam zones 會依照你建立的順序顯示，但你也可以事後重新排序。開啟 _View -> Beam Zone Organiser_。

你會看到 beam zones 的清單，以及它們連接到哪一台雷射。拖放 zones，將它們排成你想要的順序；或按一下 _Autosort by laser_，依照它們所在的雷射自動連續排序。

{% hint style="info" %}
如果你沒有為 beam zones 設定自訂名稱，它們會使用預設名稱，例如 _Beam 1, Beam 2,_ 等等。如果你重新排序，這些預設名稱也會依照排序順序更新。重新排序 zones 時請注意這一點，否則可能會造成混淆。
{% endhint %}
