# 🟩 Masks

{% hint style="danger" %}
**安全注意事項**

請絕對不要只依賴軟體中的 masks 來保護人員免於雷射輸出照射。雷射與任何可能有人員暴露的位置之間，應一律設置實體屏障或遮蔽。請將此系統視為第二層備援，或用於非安全關鍵的應用（例如保護攝影機或投影機），但請記住，任何軟體都有可能失效。
{% endhint %}

***

你可以新增一個或多個虛擬 masks，用來移除（或降低）特定位置的雷射輸出。

按一下工具列中的圖示即可新增 mask；你會在 Output view 中看到一個紅色方形，這就是新的 mask。你可以按住並拖曳來移動它，也可以拖曳角落來調整大小。如果想要非等比例的形狀，拖曳角落時請按住 `Alt / Option`。（一旦 mask 變成非等比例形狀，就不需要再按住 `Alt / Option` 鍵。）

在 mask 上按一下右鍵，可查看其他設定：

* Reduction percentage - 這代表 mask 範圍內的雷射功率降低比例。預設為 100%，也就是在 mask 範圍內完全遮黑。不過你可以依需要將雷射功率降低任意比例。請注意，如果不是完全降低，mask 的顏色會變成橘色。
* DELETE MASK 按鈕也在這個右鍵選單中。
