# ✅ 概觀

Output view 會顯示每台雷射的完整輸出範圍。在此 view 中，你可以調整它的 zones，並新增 mask 區域。

<figure><img src="../.gitbook/assets/Output View.png" alt="" width="375"><figcaption><p><em>OUTPUT</em> view，其中包含一個 beam zone、一個 canvas zone 和一個 mask。</p></figcaption></figure>

按一下上方工具列中的 _OUTPUT_ 按鈕，即可選取 Output view。

<figure><img src="../.gitbook/assets/Top icon bar with output selected.png" alt=""><figcaption></figcaption></figure>

### 選取雷射

使用上方的數字按鈕，選擇要編輯的雷射。你也可以使用 `Tab` 鍵依序切換，或直接按數字鍵。

### 在 Output view 中移動

在任何空白處（或已鎖定的 zones）按住並拖曳，即可移動 view。

### 放大與縮小 Output view

使用滑鼠滾輪放大或縮小。

### 工具列圖示

<figure><img src="../.gitbook/assets/Output Tool Bar Icon.png" alt="" width="269"><figcaption><p><em>OUTPUT</em> view 左側的工具列</p></figcaption></figure>

在 view 左側可以看到工具列。將滑鼠游標停在任何圖示上，即可查看該圖示的功能。

* 將 beam zone 新增到雷射
* 選擇此雷射中包含哪些 canvas zones
* 將 mask 新增到雷射
* 切換此雷射的 test pattern
* 選擇此雷射的 test pattern（會覆蓋主要的 test pattern）
* 切換是否對齊格線
* 調整格線大小（0 最小，8 最大）。（請注意，格線大小會以 2 的次方增加，因此當你縮小格線大小時，所有已對齊的點仍會保持在同一直線上）。
