# 🟩 Test patterns

在編輯與調整 zone 時，輸出 test pattern 會很有幫助。你可以使用上方圖示列中的切換按鈕，在每個已啟用的 zone 上開啟 test pattern，並用其右側的滑桿選擇 test pattern。

如果你只想在目前正在設定 zone 的雷射上顯示 test pattern，請使用 _Output_ view 左側工具列上的切換按鈕。點按或拖曳下方的數字即可選擇 test pattern。

### Test pattern 清單

以下是完整的 test pattern 清單。除非另有說明，這些 pattern 會填滿每個 zone 的整個輸出空間。

1. 預設 pattern：綠色矩形，中間有一條十字線。使用綠色是為了在低亮度時更容易看見。
2. 對位 pattern：由藍色與綠色圓形，加上白色與紅色格線組成。適合建築投影對位，因為你可以利用圓形的形狀來取得正確的長寬比。
3. 水平綠線：4 條等距的水平線。
4. 垂直綠線：4 條等距的垂直線。

色彩校正 pattern 5–8。這些 pattern 可搭配雷射設定中的色彩校正系統使用。請參閱[色彩校正](../advanced/colour-calibration.md)。這些 pattern 不會填滿整個輸出空間。

5. 色彩校正 pattern：紅色。
6. 色彩校正 pattern：綠色。
7. 色彩校正 pattern：藍色。
8. 色彩校正 pattern：白色。

{% hint style="danger" %}
**注意**：test pattern 9 可能非常危險，只有在你已確認該 zone 安全後才應使用。明亮且不移動的光點可能會燒灼表面。如果要照射天空，請確認你具備必要的執照／許可，並持續留意是否有航空器。
{% endhint %}

9. 明亮白點：每個角落各有一個全功率光點。這是最顯眼的 pattern，因此很適合在明亮區域或對天空進行 zone 設定時使用。
10. 掃描白線：一條垂直白線，會在 zone 的完整寬度範圍內左右掃描。它會覆蓋整個 zone，因此非常適合用來測試 mask，並確認鏡球或其他危險物已避開。
11. 白色矩形：明亮的白色矩形。
