# ✅ 輸出間歇／閃爍

開啟 _Laser Overview_ 面板，查看出問題的雷射旁邊的連線指示燈。

**如果連線指示燈不是持續綠色：**\
那麼可能是網路或 CPU 效能問題：

**網路效能 -**

* 確認你沒有連上 Wi-Fi 網路。你應該一律使用有線連線。確認作業系統會優先使用有線網路而不是 Wi-Fi；如果不確定，請停用 Wi-Fi。
* 檢查你的網路轉接器，並試試不同的 USB-C 轉接器。
* Windows 使用者：檢查／更新你的網路驅動程式，並執行適合的網路測試工具。
* 檢查所有線材、交換器、路由器。按部就班地更換並測試每一條線。
* 重新啟動所有網路設備，包括交換器、路由器，以及每一台 Ether Dream。

**CPU 效能**

如果你的電腦較舊或規格較低，可能太慢而無法順暢執行 Liberation。請查看圖示列右側的幀率指示器。

那裡有兩個數字：實際幀率和目標幀率。如果實際幀率低於 30，可能會出現問題。

以下做法可能有幫助：

* 移除未使用的雷射，也就是如果你只連接一台雷射，就刪除其他雷射。
* 切換到 Output view 或 Canvas view。
* 關閉所有其他程式、檢查網路防火牆設定、關閉防毒軟體、Dropbox 等。
* 降低顯示解析度，並縮小 Liberation 視窗。

如果以上都無效，請考慮升級你的電腦。

***

**如果連線指示燈持續綠色：**

那麼很可能是硬體問題。這超出本手冊的範圍，但你可以嘗試以下做法：

* 停用 SFS（Scan Fail Safety，掃描失效安全）系統。有些雷射具備一項功能，會在掃描器停止移動時停用輸出，也就是避免產生強烈的靜態光束。這類系統有時可能過於敏感／不可靠。

{% hint style="danger" %}
停用掃描失效安全系統時，請務必格外小心。強烈的靜態光束可能造成燃燒！請確保手邊有停止按鈕和滅火器。
{% endhint %}

* 檢查 安全連鎖線材與系統。
* 檢查控制器與雷射之間的所有線材。
