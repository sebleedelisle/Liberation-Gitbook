# ✅ 無法在 Windows 上取消授權？

#### 無法在 Windows 上取消授權？

如果你無法在 Windows 上取消某台電腦的授權，請先確認你是使用當初授權該電腦的同一個 Liberation 版本來取消授權，然後再用其他版本重新授權。

如果這樣仍無法解決，而且你使用的是 1.0 以前的版本，問題很可能與較舊的 Windows 版 Liberation 識別電腦的方式有關。在那些版本中，用來產生機器 ID 的系統較不可靠；在某些情況下，即使硬體看起來沒有變更，ID 也可能在重新啟動後改變。

如果你無法取消授權，而且並沒有切換版本，請聯絡 support@liberationlaser.com，我們可以手動為你取消該機器的授權。

**為什麼會發生這種情況**

在早期的 Windows 版 Liberation（1.0 以前）中，我們使用 Windows 建議的系統方法來產生機器 ID。很遺憾，這在某些情況下被證實不一致。因此，1.0 版重新設計了授權系統，改用更穩定的多種方法組合，現在已能可靠運作。

因此，較舊版本 Liberation 使用的電腦 ID，可能會與目前版本使用的 ID 不同。如果 ID 已經改變，就必須由支援人員手動處理取消授權。
