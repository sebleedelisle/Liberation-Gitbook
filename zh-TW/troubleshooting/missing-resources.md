# ✅ 啟動時出現缺少資源錯誤

如果啟動時出現「missing resources」錯誤，表示 Liberation 執行所需的重要檔案遺失。這可能發生在 Windows 上，例如你不小心刪除或移動了安裝資料夾中的檔案；也可能發生在 macOS 上，例如安裝內容因某些原因損毀。

### 修正方式

修正這個問題的最佳方式是重新安裝 Liberation。
