# ✅ 新增／移除雷射

在 _Laser Overview_ 面板中點擊 _ADD LASER_ 按鈕，即可新增雷射。

<figure><img src="../.gitbook/assets/Remove Laser in Laser Overview.png" alt="" width="292"><figcaption></figcaption></figure>

也可以在 _Output_ view 中點擊 _+_ 按鈕。<br>

<figure><img src="../.gitbook/assets/Laser Output View (1).png" alt="" width="176"><figcaption></figcaption></figure>

新增雷射時，系統會自動為它加入一個 beam zone。請注意，雷射和 zones 會依照你新增它們的順序排序。請參閱[重新排序 beam zones](../output-view/re-ordering-beam-zones.md)。

在 _Laser Overview_ 面板中點擊紅色的「-」按鈕，即可移除雷射。請注意，這會讓你失去所有雷射設定！

{% hint style="info" %}
如果你想從其他雷射複製 zones 或其他設定，請使用[複製雷射設定](copy-laser-settings.md)
{% endhint %}

{% hint style="info" %}
請注意，每當你執行可能有風險的操作（例如刪除雷射）時，都會出現確認對話框。如果你想略過確認，請在點擊按鈕時按住 `Cmd / Ctrl`。當然，請謹慎使用！
{% endhint %}
