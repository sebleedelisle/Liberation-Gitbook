# 🟩 簡介

近年來，雷射更常用來在空中製造光束與煙霧中的空間效果，而不是把視覺圖像投射到表面上。我甚至可以說，用雷射投影圖形常常被認為不太酷，這很大程度上是因為品質不佳的剪貼畫動畫與可疑內容大量出現在預設素材中。

但重點是：雷射圖形其實可以非常靈活，能提供大量創作可能。從簡單閃亮的企業標誌（是的，我們都做過）到完整 mapping 的建築立面，我們都能支援。

透過使用 mapping 圖形與自訂動畫，你可以開啟一整套全新的創作可能，加入你的工具箱中。
