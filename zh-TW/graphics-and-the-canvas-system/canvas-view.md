# 🟩 Canvas view

按一下頂端工具列中的 _CANVAS_ 按鈕，選取 _CANVAS_ view。

Canvas view 代表完整的 Canvas，也就是你可以操作並加入內容的區域。在這個 view 中，你可以選擇 Canvas 的哪些部分要送到哪些 zones，也可以加入輔助參考圖片。

### 在 Canvas view 中移動

在任何空白處（或已鎖定的 zones 上）按住並拖曳，即可移動畫面視角。

### 放大和縮小 Canvas view

使用滑鼠滾輪放大或縮小。

### 工具列圖示

在 view 左側會看到工具列。將滑鼠移到任何圖示上，即可查看它的功能。

* 開啟 _Canvas settings_ 面板
* 新增 canvas zone
* 新增輔助參考圖片
* 切換是否對齊格線
* 調整格線大小（0 最小 - 8 最大）。（請注意，格線大小會以 2 的次方增加，因此當你縮小格線大小時，所有已對齊的點仍會保持在同一直線上）。
* 顯示／隱藏格線
