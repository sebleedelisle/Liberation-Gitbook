# ✅ 歡迎使用 Liberation



<figure><img src=".gitbook/assets/LiberationTease.jpeg" alt=""><figcaption></figcaption></figure>

Liberation 開啟了一種全新的雷射工作方式：**現代化**、**高效率**、適用於 **Mac 和 PC**，並以當代活動製作的實際需求為核心設計。它結合快速、富表現力的創作工具、內建安全機制，以及流暢直覺、真正面向未來的工作流程，讓雷射軟體進入 21 世紀。

Liberation 是不斷成長的雷射控制軟體生態系的一部分，這些軟體都使用同一系列的開放通訊協定控制器。無論你使用的是 TouchDesigner、MadLaser、LaserShowGen 還是 Liberation，都不會被綁定在單一製造商上。你可以依照你的設備配置、預算和創作想法，選擇最適合的硬體。

最重要的是，Liberation 是為了成為一套富表現力的創作工具而打造。它容易上手、使用起來真的很有趣，也具備足以應付最高要求的音樂節、裝置藝術與大型場館巡演的強大能力，並能在單一電腦上提供穩定可靠的表現。

本指南會協助你熟悉 Liberation，從安裝與安全基礎，到建立 Clip、對應 zone，以及執行完整演出。內容聚焦在真正重要的概念與工作流程；Liberation 的設計目標，是讓軟體不妨礙你創作，讓你把更多時間用在創造，而不是與軟體搏鬥。

## 本指南適合誰

* 想要有清楚入門路徑，同時不跳過安全事項的新使用者。
* 在場地設定或彩排時，需要可靠參考資料的演出操作人員。
* 想探索 Clip Editor、MIDI、DMX 與 Canvas 工具等進階功能的好奇玩家。

## 如何使用本指南

* 在連接硬體之前，請先閱讀 [**重要！請先閱讀**](important.md)，確認關鍵安全與授權注意事項。
* 如果你想快速開始輸出，請依照 [**快速入門指南**](getting-started.md) 操作，接著再深入閱讀 [**硬體**](hardware/) 與 [**設定**](setting-up/) 以完成更完整的配置。
* 使用各章節標題直接跳到需要的主題（例如 [**Clips 與 Clip Deck**](clips/)、[**Output view**](output-view/)、**等等**），需要精確解答時，再回到 [**參考資料**](reference/) 或 [**疑難排解**](troubleshooting/)。

## 慣例與提示

* 狀態 emoji 代表完成度：✅ 已完成、🟧 部分完成、🟩 需要圖片、◼️ 佔位。
* 連結內容以目前版本為準；如果你使用較舊版本，截圖和選單可能會略有不同。
* 硬體步驟會標出以安全優先的預設做法：絕不要繞過安全連鎖裝置，並且務必讓光束避開人員。

如果遇到問題，請先查看 [**常見問題**](faq.md)，再到 [**疑難排解**](troubleshooting/) 章節尋找針對性的修正方式，之後再聯絡支援。
