# ✅ Copier les réglages entre les lasers

Cliquez sur le bouton _COPY LASER SETTINGS_ dans _Laser Output_ : un nouveau panneau s’ouvre.&#x20;

Cette fonction est très utile et fait gagner beaucoup de temps pour copier rapidement des zones et d’autres réglages d’un laser vers les autres.&#x20;

Assurez-vous d’avoir sélectionné le laser dont vous voulez copier les réglages.&#x20;

<figure><img src="../../.gitbook/assets/Copy Laser Setting in Laser Output.png" alt="" width="326"><figcaption></figcaption></figure>

Activez tous les autres lasers vers lesquels vous voulez copier les réglages. (Utilisez _ALL_ et _NONE_ pour aller plus vite si vous avez beaucoup de lasers)

<figure><img src="../../.gitbook/assets/Copy Laser Settings Details.png" alt="" width="293"><figcaption></figcaption></figure>

Sélectionnez les réglages que vous voulez copier :&#x20;

* **Copy output zone positions&#x20;**_**-**_ copie les positions / tailles / types de zones d’un laser vers les autres. Notez que les zones sont copiées dans l’ordre : les zones 1 et 9 du laser 1 seront donc copiées vers les zones 2 et 10 du laser 2. Les zones Alt seront également copiées.&#x20;
* **Copy scanner / speed settings** - copie la vitesse du scanner et le préréglage du scanner, mais _pas_ _Colour shift_.
* **Copy colour settings -** copie tous les réglages d’étalonnage des couleurs
* **Copy advanced settings -** inclut le point rate, la smooth home position, le target frame rate et tous les autres réglages avancés.&#x20;

Une fois votre sélection effectuée, cliquez sur le bouton _COPY SETTINGS_ pour terminer l’opération.&#x20;
