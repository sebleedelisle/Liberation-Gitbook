# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

Le _Parameter Control_ est une méthode permettant de contrôler différents réglages d’un effet ou d’un Clip en dehors de l’éditeur de Clip.

* **range min / range max** - les valeurs minimale et maximale auxquelles le niveau du paramètre est associé
* **level** - le réglage par défaut de ce paramètre. C’est la valeur qui peut être modifiée depuis l’extérieur. Vous pouvez l’ajuster manuellement pour voir l’effet obtenu.
* **parameter number** - chaque _parameter control_ peut recevoir un numéro, qui se retrouve dans les contrôles en dehors du Clip. Vous pouvez avoir plusieurs parameter controls avec le même numéro ; ils seront alors tous ajustés en même temps.

{% hint style="warning" %}
Actuellement, seul le paramètre 1 d’un effet peut être contrôlé avec l’APC40. Tous les paramètres des effets et des Clips sont disponibles dans le menu du clic droit. À l’avenir, davantage de ces paramètres pourront être mappés avec des contrôleurs MIDI.
{% endhint %}
