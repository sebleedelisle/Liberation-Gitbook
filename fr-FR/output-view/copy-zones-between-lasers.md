# 🟩 Copier des zones entre lasers

Cliquez sur le bouton _COPY LASER SETTINGS_ dans _Laser Output_ : un nouveau panneau s’ouvre.

Cette fonction est extrêmement utile et fait gagner beaucoup de temps pour copier rapidement des zones et d’autres réglages d’un laser vers les autres.

Assurez-vous d’avoir sélectionné le laser _source_, depuis lequel vous voulez copier les réglages.

Activez tous les autres lasers vers lesquels vous voulez copier les réglages. (Utilisez _ALL_ et _NONE_ pour aller plus vite si vous avez beaucoup de lasers)

Sélectionnez l’option _Copy output zone settings_, puis cliquez sur le bouton _COPY SETTINGS_ pour terminer l’opération.

{% hint style="info" %}
Notez que cette opération n’ajoute ni ne supprime de zones sur les lasers. Elle copie simplement la position et le type de zone pour chaque zone qui existe dans les lasers cibles.
{% endhint %}

Voir aussi [Copier les réglages entre lasers](../setting-up/copy-laser-settings.md "mention") pour une description complète des autres fonctions de ce système.
