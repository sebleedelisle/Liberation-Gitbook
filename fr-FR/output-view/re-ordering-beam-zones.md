# 🟩 Réorganiser les zones de faisceau

Les zones de faisceau apparaissent dans l’ordre dans lequel vous les créez, mais vous pouvez les réorganiser par la suite. Ouvrez _View -> Beam Zone Organiser._

Vous verrez la liste des zones de faisceau et le laser auquel elles sont connectées. Faites glisser-déposer les zones pour les placer dans l’ordre souhaité, ou cliquez sur _Autosort by laser_ pour les trier automatiquement de manière consécutive selon le laser auquel elles appartiennent.

{% hint style="info" %}
Si vous n’avez pas donné de noms personnalisés à vos zones de faisceau, elles utiliseront un nom par défaut du type _Beam 1, Beam 2,_ etc. Si vous les réorganisez, leurs noms par défaut seront également mis à jour selon leur ordre de tri. Gardez cela à l’esprit lorsque vous réorganisez les zones, car cela peut prêter à confusion.
{% endhint %}
