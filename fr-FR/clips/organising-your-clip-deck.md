# 🟩 Organiser votre Clip Deck

### Déplacer/dupliquer des clips

Cliquez et faites glisser pour déplacer un clip. Si plusieurs clips sont sélectionnés (voir [démarrer/arrêter des clips](starting-stopping-clips.md "mention")), ils seront tous déplacés. Maintenez `Alt / Option` enfoncé pour les dupliquer. Déposez le ou les clips sur un espace vide.

{% hint style="info" %}
Si vous devez sélectionner un grand nombre de clips, vous pouvez cliquer et faire glisser un lasso, puis faire défiler le Clip Deck tout en continuant à faire glisser (avec les touches fléchées, l’APC 40 ou le défilement horizontal du pavé tactile/de la souris). Le lasso s’étendra au-delà du bord du Clip Deck.
{% endhint %}

### Supprimer des clips

Pour supprimer un seul clip, utilisez le menu du clic droit.

Pour supprimer plusieurs clips, sélectionnez-les (avec `Alt / Option + Shift`-clic ou en faisant glisser le lasso), puis faites-les glisser hors du Clip Deck.

### Ajouter un espace vide

Vous pouvez faire un clic droit sur un bouton de clip et appuyer sur _ADD EMPTY COLUMN TO RIGHT_ pour ajouter une colonne vide dans le Clip Deck.

### Combiner plusieurs Clip Decks

Vous pouvez importer un Clip Deck existant et l’ajouter à la fin du Clip Deck actuel. Utilisez le menu _File -> Clips/Effects -> Append Clip Deck._

Faites une sélection de clips (avec le lasso ou `Alt / Option + Shift`-clic), puis utilisez _File -> Clips/Effects -> Export Selected Clips._ Cela exportera un Clip Deck contenant uniquement ces clips, qui pourra ensuite être ajouté à un autre Clip Deck.

### Organiser les clips par timeline

Lorsque vous importez une timeline, vous disposez également de plusieurs options pour les clips. Vous pouvez remplacer le Clip Deck actuel, ajouter l’intégralité du Clip Deck de la timeline, ou seulement les clips qu’elle utilise.

Si votre projet contient plusieurs timelines, vous pouvez réorganiser vos clips par timeline. Ouvrez le panneau _Timeline selection_, puis cliquez sur _ORGANISE CLIPS BY TIMELINE_. Vous verrez alors tous vos clips regroupés selon leur timeline. Si un clip est utilisé dans plusieurs timelines, il sera dupliqué dans chaque groupe.
