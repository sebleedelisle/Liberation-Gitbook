# 🟩 Groupes de Clips

Vous remarquerez que chaque Clip possède un contour coloré, et que cette couleur indique à quel _groupe_ il appartient. Les boutons de Clip de l’APC40 s’allument également dans la couleur du groupe correspondant.

<table data-header-hidden><thead><tr><th width="108"></th><th></th></tr></thead><tbody><tr><td>Group 1</td><td>Cyan</td></tr><tr><td>Group 2</td><td>Orange</td></tr><tr><td>Group 3</td><td>Rouge</td></tr><tr><td>Group 4</td><td>Indigo</td></tr><tr><td>Group 5</td><td>Vert</td></tr></tbody></table>

Le système de groupes est très flexible et vous permet de :

* Laisser les Clips d’un groupe continuer à jouer, pendant que vous activez ou désactivez des groupes dans un autre
* Assigner rapidement des zones et des inversions X/Y à tous les Clips d’un groupe
* Définir le _Flash mode_ pour un Clip (Group 3 est en _Flash mode_ par défaut)
* Définir les durées de transition d’entrée/sortie par défaut, qui peuvent être héritées par les Clips ou remplacées.

Vous êtes libre d’utiliser le système de _groupes_ comme vous le souhaitez, mais si vous avez beaucoup de zones/lasers, il peut être utile d’utiliser les différents groupes pour différents ensembles de lasers. ### Les boutons _Group_

Les boutons _Group_ se trouvent sur le côté droit du Clip Deck. Lorsque vous en activez un, vous pouvez :

* assigner des Clips à un groupe
* modifier les réglages de zone et l’inversion x/y pour tous les Clips du groupe

Pour activer le bouton _Group_ :

* Maintenez enfoncé le bouton _Group_ sur l’APC40
* Cliquez avec la souris sur le bouton _Group_ à l’écran. Cliquez à nouveau pour le désactiver.

Vous pouvez facilement assigner des Clips au groupe en appuyant dessus pendant que le bouton _Group_ est actif. (Une autre méthode consiste à utiliser le menu contextuel du Clip.)

### Modifier les réglages de zone pour tous les Clips d’un groupe

Lorsqu’un bouton _Group_ est actif, les boutons de _zone_ (BEAM 1 - 8, CANVAS 1, etc.) s’allument comme suit :

* OFF : cette zone n’est définie pour **aucun** des Clips de ce groupe
* FLASHING : cette zone est définie pour _**certains**_**&#x20;Clips, mais pas pour&#x20;**_**tous**_ les Clips du groupe
* ON : cette zone est définie pour _**tous**_ les Clips du groupe

Appuyez sur le bouton de zone pour activer ou désactiver cette zone dans _**tous**_ les Clips du groupe.

Le même principe s’applique aux boutons d’inversion X/Y.
