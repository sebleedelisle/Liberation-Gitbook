# 🟩 Vue d’ensemble du contrôle MIDI

Liberation utilise le MIDI de plusieurs façons :

* Comme contrôleur live, avec l’APC 40. Voir [live-control-with-the-apc40.md](live-control-with-the-apc40.md "mention").
* Comme source de synchronisation d’horloge, à l’aide de l’horloge MIDI et des messages de position dans le morceau MIDI. Voir [#midi-clock](../tempo-synchronisation.md#midi-clock "mention")
* Comme entrée interactive sur le nœud de notes MIDI, pour créer des effets de type « harpe laser ». Voir [midi-notes.md](../clip-editor/operator-nodes/midi-notes.md "mention")
* Comme système d’entrée/sortie plus général, avec le système MIDI Send/Receive. Voir [midi-send-receive.md](midi-send-receive.md "mention")
