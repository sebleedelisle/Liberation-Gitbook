# ✅ Contrôle live

### Les clips ne s’animent plus

Vous avez probablement arrêté le système de tempo ou modifié la source de tempo. Assurez-vous que la source de tempo _Live_ est sélectionnée, puis appuyez sur le bouton de lecture. (Si votre source de tempo est _Timeline_, vous pouvez démarrer et arrêter avec la barre d’espace).
