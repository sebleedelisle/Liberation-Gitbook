# ✅ Impossible de désautoriser sur Windows ?

#### Impossible de désautoriser sur Windows ?

Si vous ne parvenez pas à désautoriser un ordinateur sous Windows, vérifiez d’abord que vous désautorisez la licence avec la même version de Liberation que celle utilisée pour l’autoriser à l’origine, avant de l’autoriser à nouveau dans une autre version.

Si cela ne fonctionne pas et que vous utilisez une version antérieure à la 1.0, le problème est probablement lié à la façon dont les anciennes versions Windows de Liberation identifiaient l’ordinateur. Dans ces versions, le système utilisé pour générer un ID machine était moins fiable et, dans certains cas, l’ID pouvait changer entre deux redémarrages, même si aucun matériel n’avait visiblement été modifié.

Si vous êtes bloqué lors de la désautorisation et que vous n’avez pas changé de version, veuillez contacter support@liberationlaser.com et nous pourrons désautoriser la machine manuellement pour vous.

**Pourquoi cela se produit**

Dans les premières versions Windows de Liberation (avant la 1.0), nous utilisions la méthode système recommandée par Windows pour générer un ID machine. Malheureusement, elle s’est révélée incohérente dans certaines situations. C’est pourquoi le système de licences a été réécrit pour la version 1.0 afin d’utiliser une combinaison de méthodes plus robuste, qui fonctionne désormais de manière fiable.

Par conséquent, l’ID ordinateur utilisé par les anciennes versions de Liberation peut différer de celui utilisé par les versions actuelles. Si l’ID a déjà changé, la désautorisation doit être effectuée manuellement par l’assistance.
