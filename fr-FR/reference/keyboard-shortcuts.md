# ✅ Raccourcis clavier

Les raccourcis sont indiqués au format `Cmd / Ctrl` : utilisez `Cmd` sur macOS et `Ctrl` sur Windows.

| Shortcut                          | Action                                                               |
| --------------------------------- | -------------------------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Armer tous les lasers                                                |
| `Escape`                          | Désarmer tous les lasers                                             |
| `Tab`                             | Basculer entre les vues et parcourir les lasers                      |
| `Shift + Tab`                     | Basculer entre les vues et parcourir les lasers en sens inverse      |
| `1-9`                             | Aller au laser                                                       |
| `Left / Right Arrow`              | Déplacer le Clip Deck vers la gauche / droite                        |
| `Cmd / Ctrl + Left / Right Arrow` | Déplacer le Clip Deck d’une page vers la gauche / droite             |
| `Shift + Left / Right Arrow`      | Déplacer les zones d’une page vers la gauche / droite                |
| `Space`                           | Lecture / pause (lorsque la source de tempo _Timeline_ est active)   |
| `T`                               | Définir le tempo au tap                                              |
| `R`                               | Réinitialiser la mesure                                              |
| `Y`                               | Arrondir le tempo au BPM le plus proche                              |
| `Cmd / Ctrl + ,`                  | Ouvrir le panneau des réglages                                       |
| `Cmd / Ctrl + F`                  | Basculer en mode plein écran                                         |
| `Cmd / Ctrl + L`                  | Ouvrir le panneau des réglages de sortie laser                       |
| `Cmd / Ctrl + K`                  | Ouvrir le panneau d’affectation des contrôleurs laser                |
| `Cmd / Ctrl + J`                  | Ouvrir le panneau des réglages du 3D Visualiser                      |
| `Cmd / Ctrl + Shift + Plus (+)`   | Augmenter la mise à l’échelle de l’interface                         |
| `Cmd / Ctrl + Shift + Minus (-)`  | Réduire la mise à l’échelle de l’interface                           |
| `Cmd / Ctrl + 0`                  | Réinitialiser la mise à l’échelle de l’interface                     |
| `Cmd / Ctrl + A`                  | Timeline : tout sélectionner                                         |
| `Delete`                          | Timeline : supprimer les blocs sélectionnés                          |
| `Delete`                          | Éditeur de Clip : supprimer les nœuds sélectionnés                   |
| `Cmd + Q` (macOS)                 | Quitter l’application                                                |
| `Alt + F4` (Windows)              | Quitter l’application                                                |
