# ✅ Autorisation et désautorisation

### Autoriser Liberation

Lorsque vous ouvrez Liberation pour la première fois, il s’exécute en _mode gratuit_ et le _About panel_ s’affiche :

<figure><img src="../.gitbook/assets/authorisation-about-panel.png" alt="" width="375"><figcaption></figcaption></figure>

Cliquez sur le bouton _AUTHORISE ONLINE_ : votre navigateur web s’ouvre. Si vous n’êtes pas déjà connecté, vous serez invité à le faire maintenant.

Le système autorise alors automatiquement votre installation avec votre abonnement, et vous verrez ce message :

<figure><img src="../.gitbook/assets/authorisation-machine-added" alt=""><figcaption></figcaption></figure>

Lorsque vous revenez dans Liberation, vous verrez que le _About panel_ a été mis à jour (vous devrez peut-être patienter quelques secondes).

<figure><img src="../.gitbook/assets/authorise-about-panel-authorised" alt="" width="375"><figcaption></figcaption></figure>

{% hint style="warning" %}
Si vous avez déjà autorisé le nombre maximal d’ordinateurs pour votre abonnement, vous devrez désautoriser l’une de vos autres machines ou mettre à niveau votre abonnement.
{% endhint %}

{% hint style="info" %}
Si vous avez plusieurs abonnements, vous serez invité à choisir celui auquel vous souhaitez associer l’ordinateur.
{% endhint %}

Félicitations ! Votre installation de Liberation est maintenant autorisée et vous pouvez envoyer la sortie vers des lasers ! Mais veuillez lire [Guide de démarrage rapide](../getting-started.md "mention") et [Présentation du processus de configuration des lasers](../setting-up/setting-up-lasers.md "mention") avant d’armer vos lasers.

{% hint style="info" %}
Vous pouvez ouvrir le _About panel_ à tout moment via le menu _Liberation -> About Liberation_ ou _Liberation -> Authorise/Deauthorise this computer_
{% endhint %}

### Désautoriser Liberation

**Depuis Liberation** — Ouvrez le menu _Liberation -> Authorise / De-authorise this Computer_ et cliquez sur le bouton _DEAUTHORISE COMPUTER_. C’est tout ! Vous devez être en ligne pour que cela fonctionne.

<figure><img src="../.gitbook/assets/Screenshot 2024-08-28 at 10.18.57.png" alt=""><figcaption></figcaption></figure>

Vous pouvez également le faire depuis le site web : sélectionnez _Your subscriptions_ dans le menu, puis cliquez sur _Manage Subscription_ pour ouvrir la page de votre abonnement. Vous devriez voir les informations relatives à votre abonnement ainsi qu’une liste des ordinateurs que vous avez autorisés.

Cliquez sur le lien _De-authorise_ à côté de la machine que vous souhaitez désautoriser.

<figure><img src="../.gitbook/assets/Screenshot 2024-08-28 at 10.16.40.png" alt=""><figcaption></figcaption></figure>

Si votre machine n’a pas été en ligne depuis votre dernier renouvellement, elle sera désautorisée immédiatement. Sinon, la machine est mise en _file d’attente_ pour désautorisation. Cela signifie que la désautorisation se fera automatiquement la prochaine fois que la machine sera connectée à Internet, ou à la date de votre prochain renouvellement, selon ce qui arrive en premier.
