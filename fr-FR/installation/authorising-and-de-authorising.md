# ✅ Autorisation et désautorisation

### Autoriser Liberation

Lorsque vous ouvrez Liberation pour la première fois, il s’exécute en _mode gratuit_ et le _About panel_ s’affiche :

<figure><img src="../.gitbook/assets/authorisation-about-panel.png" alt="" width="375"><figcaption></figcaption></figure>

Cliquez sur le bouton _AUTHORISE ONLINE_ : votre navigateur web s’ouvre. Si vous n’êtes pas déjà connecté, vous serez invité à le faire maintenant.

Le système autorise alors automatiquement votre installation avec votre licence, et vous verrez un message de confirmation.

Lorsque vous revenez dans Liberation, vous verrez que le _About panel_ a été mis à jour (vous devrez peut-être patienter quelques secondes).

<figure><img src="../.gitbook/assets/authorise-about-panel-authorised" alt="" width="375"><figcaption></figcaption></figure>

{% hint style="warning" %}
Si vous avez déjà autorisé le nombre maximal d’ordinateurs pour votre licence, vous devrez désautoriser l’une de vos autres machines ou mettre à niveau votre licence.
{% endhint %}

{% hint style="info" %}
Si vous avez plusieurs licences, vous serez invité à choisir celle à laquelle vous souhaitez associer l’ordinateur.
{% endhint %}

Félicitations ! Votre installation de Liberation est maintenant autorisée et vous pouvez envoyer la sortie vers des lasers ! Mais veuillez lire le [Guide de démarrage rapide](../getting-started.md "mention") et la [Présentation du processus de configuration des lasers](../setting-up/setting-up-lasers.md "mention") avant d’armer vos lasers.

{% hint style="info" %}
Vous pouvez ouvrir le _About panel_ à tout moment via le menu _Liberation -> About Liberation_ ou _Liberation -> Authorise/Deauthorise this computer_
{% endhint %}

### Désautoriser Liberation

**Depuis Liberation** — Ouvrez le menu _Liberation -> Authorise / De-authorise this Computer_ et cliquez sur le bouton _DEAUTHORISE COMPUTER_. C’est tout ! Vous devez être en ligne pour que cela fonctionne.

<figure><img src="../.gitbook/assets/Screenshot 2024-08-28 at 10.18.57.png" alt=""><figcaption></figcaption></figure>

Vous pouvez également le faire depuis le site web. Sélectionnez _Your licences_ dans le menu, puis cliquez sur _MANAGE_ pour la licence concernée. Si votre compte ne comporte qu’une seule licence, _Your licences_ vous mènera directement à la page de cette licence. Vous devriez voir les informations relatives à votre licence ainsi qu’une liste des ordinateurs que vous avez autorisés.

<figure><img src="../.gitbook/assets/account-your-licences.png" alt=""><figcaption></figcaption></figure>

Cliquez sur le lien _De-authorise_ à côté de la machine que vous souhaitez désautoriser.

Si votre machine ne s’est pas connectée à Internet depuis votre dernier rafraîchissement de licence, elle sera désautorisée immédiatement. Dans le cas contraire, la machine est mise en _file d’attente_ pour désautorisation. Cela signifie que la désautorisation aura lieu automatiquement la prochaine fois que la machine sera connectée à Internet, ou à la prochaine date de rafraîchissement de la licence, selon ce qui arrive en premier.
