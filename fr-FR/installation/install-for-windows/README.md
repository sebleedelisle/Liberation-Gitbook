# ✅ Installation pour Windows

### Compatibilité

Ordinateurs Windows 10 ou 11 64 bits. Des machines puissantes sont recommandées si vous souhaitez exécuter de grands shows. Plus votre ordinateur est rapide, plus la sortie laser animée sera fluide.

### Processus d’installation

Une fois votre abonnement actif, vous recevrez un lien, ou vous pouvez cliquer sur le lien _Download_ sur la page[ _Your subscriptions_](https://liberationlaser.com/account/my-products).

Vous verrez cette page :

<figure><img src="../../.gitbook/assets/windows-install-download-page.png" alt=""><figcaption></figcaption></figure>

Cliquez sur _Download latest for Windows_ (les numéros de version peuvent changer) et l’indicateur de téléchargement affichera la progression (généralement en haut à droite ; la capture d’écran vient de Chrome). Une fois le téléchargement terminé, ouvrez vos téléchargements et cliquez sur le fichier d’installation téléchargé.

<figure><img src="../../.gitbook/assets/windows-install-download-list.png" alt=""><figcaption></figcaption></figure>

Cela lancera l’installateur.

{% hint style="info" %}
Dans certains cas, un avertissement peut indiquer que Windows a protégé votre PC : Windows est prudent lorsqu’il s’agit d’exécuter de nouveaux logiciels ! À mesure que davantage de personnes utilisent Liberation et lui font confiance, cet avertissement devrait disparaître.

Si vous voyez cet avertissement, cliquez sur le lien _More info_, vérifiez que le _Publisher_ est bien _SEB.LY LIMITED_, puis cliquez sur _Run anyway_.

<img src="../../.gitbook/assets/windows-install-windows-protection-warning.png" alt="" data-size="original"><img src="../../.gitbook/assets/windows-install-warning-run-anyway.png" alt="" data-size="original">
{% endhint %}

Avant le démarrage de l’installateur, un avertissement peut vous demander si vous autorisez l’installateur à apporter des modifications. Cliquez sur _Yes_.

<figure><img src="../../.gitbook/assets/windows-install-warning-make-changes.png" alt=""><figcaption></figcaption></figure>

L’installateur peut maintenant démarrer ! Avancez dans les pages.

<figure><img src="../../.gitbook/assets/windows-installer-1.png" alt=""><figcaption></figcaption></figure>

Il est recommandé d’utiliser l’emplacement d’installation par défaut ; cliquez donc simplement sur _Next_.

<figure><img src="../../.gitbook/assets/windows-installer-2.png" alt=""><figcaption></figcaption></figure>

Ensuite, vous avez des options pour créer une icône sur le bureau et un dossier dans le menu Démarrer (recommandé).

<figure><img src="../../.gitbook/assets/windows-installer-3.png" alt=""><figcaption></figcaption></figure>

Patientez quelques instants pendant l’installation de Liberation.

<figure><img src="../../.gitbook/assets/windows-installer-4.png" alt=""><figcaption></figcaption></figure>

L’installation est terminée ! Cliquez sur _Finish_ pour fermer l’installateur et lancer Liberation.

<figure><img src="../../.gitbook/assets/windows-installer-5.png" alt=""><figcaption></figcaption></figure>

Liberation a besoin d’autorisations pour accéder aux fichiers et au réseau ; Windows affichera donc probablement plusieurs demandes. Veillez à tout accepter, sinon Liberation pourrait ne pas fonctionner correctement.

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation devrait maintenant se lancer ! Pour envoyer la sortie vers des lasers, vous devez maintenant suivre le [processus d’autorisation.](../authorising-and-de-authorising.md)
