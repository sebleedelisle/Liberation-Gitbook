# 🟩 Transférer votre contenu lors de la mise à niveau vers une nouvelle version

Liberation stocke le contenu dans son dossier de travail (voir [Où trouver le dossier de travail](../troubleshooting/where-to-find-the-working-folder.md "mention")), et celui-ci change à chaque nouvelle version. Si vous souhaitez conserver vos fichiers de travail de l’ancienne version vers la nouvelle :&#x20;

1. Ouvrez la version d’origine de Liberation
2. Utilisez _File->Project->Export Project_. Cela enregistre tout ce qui se trouve dans votre Liberation : la configuration laser, les timelines, le Clip Deck, tout !&#x20;
3. Fermez cette version, puis ouvrez la nouvelle version de Liberation
4. Utilisez _File->Project->Import Project_ et sélectionnez le fichier de projet que vous avez exporté à l’étape 2.&#x20;

Tout votre contenu devrait maintenant se trouver dans la nouvelle version.

#### Conserver plusieurs versions de Liberation

Avant d’installer une nouvelle version, renommez l’ancienne version. J’utilise généralement son numéro de version, par exemple Liberation103.app (ou .exe sous Windows). Vous pouvez conserver autant de versions de Liberation que vous le souhaitez, mais vous ne pouvez en exécuter qu’une seule à la fois. <br>

#### Si vous avez remplacé votre ancienne version de Liberation par une version plus récente

L’option la plus fiable consiste à réinstaller l’ancienne version de Liberation, puis à suivre la procédure décrite ci-dessus.&#x20;

Vous pouvez également essayer de copier votre dossier de travail et de le renommer avec le numéro de la nouvelle version.&#x20;
