# 🟩 Zones Canvas

Une zone Canvas apparaît dans la vue Canvas sous forme de rectangle au contour rose. Elle sert à sélectionner une zone rectangulaire dans le Canvas et à envoyer son contenu vers un ou plusieurs lasers.

Vous pouvez cliquer sur les zones et les faire glisser à la souris. Faites glisser les coins pour modifier leur taille et leur forme.

### Ajouter une nouvelle zone Canvas

Cliquez sur le bouton _Add a new canvas zone_ dans la barre d’outils.

### Assigner la zone Canvas à un laser

Faites un clic droit sur la zone et activez ou désactivez les boutons numérotés pour sélectionner le ou les lasers auxquels cette zone Canvas est assignée.

{% hint style="danger" %}
AVERTISSEMENT - si le laser est armé, vous pourriez commencer soudainement à projeter du contenu dans une zone Canvas par défaut. Il est préférable de désarmer le laser avant de lui assigner des zones Canvas.
{% endhint %}

Si vous allez maintenant dans la vue _OUTPUT_ de ce laser, vous y verrez la zone Canvas, qui peut désormais être modifiée exactement comme n’importe quelle zone de faisceau. Voir [zones.md](../output-view/zones.md "mention").

{% hint style="info" %}
Vous pouvez également ajouter une zone Canvas à un laser directement depuis la vue _OUTPUT_ : cliquez sur le bouton _Add existing canvas zone_ dans la barre d’outils de gauche.
{% endhint %}

### Menu contextuel

Faites un clic droit sur la zone pour afficher les options permettant de supprimer cette zone Canvas, de la déplacer à l’arrière-plan et de la verrouiller.
