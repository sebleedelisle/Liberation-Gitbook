# ✅ Flýtilyklar

Flýtilyklar eru skrifaðir sem `Cmd / Ctrl`: notaðu `Cmd` á macOS og `Ctrl` á Windows.

| Flýtilykill                       | Aðgerð                                                |
| --------------------------------- | ----------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Virkja arm á öllum leysitækjum                        |
| `Escape`                          | Virkja disarm á öllum leysitækjum                     |
| `Tab`                             | Skipta á milli views og fletta í gegnum leysitæki     |
| `Shift + Tab`                     | Skipta á milli views og fletta afturábak í gegnum leysitæki |
| `1-9`                             | Hoppa í leysitæki                                     |
| `Left / Right Arrow`              | Færa Clip Deck til vinstri / hægri                    |
| `Cmd / Ctrl + Left / Right Arrow` | Færa Clip Deck um eina síðu til vinstri / hægri       |
| `Shift + Left / Right Arrow`      | Færa zones um eina síðu til vinstri / hægri           |
| `Space`                           | Spila / setja í bið (þegar _Timeline_ tempo source er virkt) |
| `T`                               | Slá inn takt                                           |
| `R`                               | Endurstilla takt                                      |
| `Y`                               | Rúna tempo að næsta BPM                               |
| `Cmd / Ctrl + ,`                  | Opna stillingaspjaldið                                |
| `Cmd / Ctrl + F`                  | Kveikja/slökkva á fullskjá                            |
| `Cmd / Ctrl + L`                  | Opna stillingaspjald fyrir laser output               |
| `Cmd / Ctrl + K`                  | Opna spjald fyrir úthlutun laser controller           |
| `Cmd / Ctrl + J`                  | Opna stillingaspjald fyrir 3D visualiser              |
| `Cmd / Ctrl + Shift + Plus (+)`   | Auka stærð viðmóts                                    |
| `Cmd / Ctrl + Shift + Minus (-)`  | Minnka stærð viðmóts                                  |
| `Cmd / Ctrl + 0`                  | Endurstilla stærð viðmóts                             |
| `Cmd / Ctrl + A`                  | Timeline: velja allt                                  |
| `Delete`                          | Timeline: eyða völdum blokkum                        |
| `Delete`                          | Clip Editor: eyða völdum nodes                        |
| `Cmd + Q` (macOS)                 | Hætta í appinu                                        |
| `Alt + F4` (Windows)              | Hætta í appinu                                        |
