# ✅ Flýtilyklar

Flýtilyklar eru skrifaðir sem `Cmd / Ctrl`: notaðu `Cmd` á macOS og `Ctrl` á Windows.

| Flýtilykill                       | Aðgerð                                                |
| --------------------------------- | ----------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Virkja alla leysigeisla                               |
| `Escape`                          | Afvirkja alla leysigeisla                             |
| `Tab`                             | Skipta um sýn og fletta í gegnum leysigeisla          |
| `Shift + Tab`                     | Skipta um sýn og fletta aftur á bak í gegnum leysigeisla |
| `1-9`                             | Hoppa í leysigeisla                                   |
| `Left / Right Arrow`              | Færa clip deck til vinstri / hægri                    |
| `Cmd / Ctrl + Left / Right Arrow` | Færa clip deck um eina síðu til vinstri / hægri       |
| `Shift + Left / Right Arrow`      | Færa svæði um eina síðu til vinstri / hægri           |
| `Space`                           | Spila / gera hlé (þegar _Timeline_ tempógjafi er virkur) |
| `T`                               | Slá inn tempó                                         |
| `R`                               | Endurstilla takt                                      |
| `Y`                               | Rúnna tempó að næsta BPM                              |
| `Cmd / Ctrl + ,`                  | Opna stillingaspjaldið                                |
| `Cmd / Ctrl + F`                  | Kveikja eða slökkva á fullskjásstillingu              |
| `Cmd / Ctrl + L`                  | Opna stillingaspjald fyrir laser output               |
| `Cmd / Ctrl + K`                  | Opna spjald fyrir úthlutun laser controller           |
| `Cmd / Ctrl + J`                  | Opna stillingaspjald fyrir 3D Visualiser              |
| `Cmd / Ctrl + Shift + Plus (+)`   | Auka skölun viðmóts                                   |
| `Cmd / Ctrl + Shift + Minus (-)`  | Minnka skölun viðmóts                                 |
| `Cmd / Ctrl + 0`                  | Endurstilla skölun viðmóts                            |
| `Cmd / Ctrl + A`                  | Timeline: velja allt                                  |
| `Delete`                          | Timeline: eyða völdum blokkum                         |
| `Delete`                          | Clip editor: eyða völdum hnútum                       |
| `Cmd + Q` (macOS)                 | Hætta í forritinu                                     |
| `Alt + F4` (Windows)              | Hætta í forritinu                                     |
