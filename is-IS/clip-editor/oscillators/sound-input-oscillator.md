# ✅ Sveiflugjafi fyrir Sound input

## <img src="../../.gitbook/assets/sound-input-icon.png" alt="" data-size="line"> Sound input

Breytir styrk hljóðinntaksins í gildi fyrir eiginleika.

{% hint style="info" %}
Sound input sveiflugjafinn notar sjálfgefið hljóðviðmót, en þú getur breytt því í _Preferences_. Opnaðu valmyndina _Liberation -> Preferences._

Í stillingunum _Sound Input_ getur þú valið hvaða hljóðviðmót þú vilt nota, ásamt nokkrum öðrum stillingum til að laga hljóðstyrkinn.
{% endhint %}

* **range min / range max** - lágmarks- og hámarksgildin sem bylgjuformið er varpað yfir á
* **channel** - ef hljóðviðmótið þitt er með fleiri en eina inntaksrás getur þú valið hér hvaða rás þú vilt nota.

{% hint style="info" %}
Skemmtileg aðferð er að fá fleiri en eina hljóðrás frá hljóðblönduninni. Þannig getur þú látið mismunandi Clips bregðast við mismunandi hljóðfærum.
{% endhint %}

{% hint style="info" %}
Þú getur aðeins notað eitt hljóðviðmót í einu fyrir öll _Sound Inputs_ (valið í _App Settings_ panel). Ef þú vilt nota fleiri en eitt viðmót í þessum tilgangi getur þú á macOS [búið til Aggregate Device](https://support.apple.com/en-gb/HT202000) til að sameina inntök í eina sýndarhljóðgjafa. (Það eru líka til mörg forrit fyrir Windows sem gera þetta, en ég hef ekki prófað þau.)
{% endhint %}

* **clamp min / clamp max** - notaðu þetta til að velja hvaða styrksvið þú vilt að bregðist við. Þú ættir ekki að þurfa að breyta þessu ef gate- og limit-stillingarnar (í _App Settings_ panel) eru rétt stilltar, en hægt er að nota þetta fyrir skapandi áhrif.

{% hint style="info" %}
Þú sérð lítið hljóðnematákn á Clip hnappnum þegar hann er með _Sound Input_ sveiflugjafa.
