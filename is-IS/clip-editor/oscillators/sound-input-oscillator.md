# ✅ Sound input sveiflugjafi

## <img src="../../.gitbook/assets/sound-input-icon.png" alt="" data-size="line"> Sound input

Breytir hljóðinntaksstyrknum í gildi eiginleika.

{% hint style="info" %}
Sound input sveiflugjafinn notar sjálfgefið hljóðviðmót, en þú getur breytt því í _Preferences_. Opnaðu valmyndina _Liberation -> Preferences._

Í _Sound Input_ stillingunum geturðu valið hvaða hljóðviðmót þú vilt nota, ásamt nokkrum öðrum stillingum til að stilla hljóðstyrkinn.
{% endhint %}

* **range min / range max** - lágmarks- og hámarksgildin sem bylgjuformið er varpað yfir á
* **channel** - ef hljóðviðmótið þitt er með fleiri en eina inntaksrás geturðu valið hér hvaða rás þú vilt taka inn.

{% hint style="info" %}
Það er skemmtileg aðferð að fá fleiri en eina hljóðrás frá hljóðblönduninni. Þannig geturðu látið mismunandi clips bregðast við mismunandi hljóðfærum.
{% endhint %}

{% hint style="info" %}
Þú getur aðeins notað eitt hljóðviðmót í einu fyrir öll _Sound Inputs (_&#x73;em er valið í _App Settings_ panel). Ef þú vilt nota fleiri en eitt viðmót fyrir þetta geturðu á macOS [búið til Aggregate Device](https://support.apple.com/en-gb/HT202000) til að sameina inntök í einn sýndarhljóðgjafa. (Það eru líka mörg forrit á Windows sem gera þetta, en ég hef ekki prófað þau).
{% endhint %}

* **clamp min / clamp max** - notaðu þetta til að velja hvaða styrksvið þú vilt að bregðist við. Þú ættir ekki að þurfa að breyta þessu ef gate- og limit-stillingarnar (í _App Settings_ panel) eru rétt stilltar, en hægt er að nota þetta fyrir skapandi áhrif.

{% hint style="info" %}
Þú sérð lítið hljóðnematákn á clip-hnappnum þegar hann er með _Sound Input_ sveiflugjafa.
