# ✅ Klávesové zkratky

Zkratky jsou zapsané jako `Cmd / Ctrl`: na macOS použijte `Cmd`, ve Windows `Ctrl`.

| Zkratka                           | Akce                                                  |
| --------------------------------- | ----------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Přepnout všechny lasery do stavu armed                |
| `Escape`                          | Přepnout všechny lasery do stavu disarmed             |
| `Tab`                             | Přepínat zobrazení a postupně procházet lasery        |
| `Shift + Tab`                     | Přepínat zobrazení a procházet lasery opačným směrem  |
| `1-9`                             | Přejít na laser                                       |
| `Left / Right Arrow`              | Posunout Clip Deck doleva / doprava                   |
| `Cmd / Ctrl + Left / Right Arrow` | Posunout Clip Deck o jednu stránku doleva / doprava   |
| `Shift + Left / Right Arrow`      | Posunout zones o jednu stránku doleva / doprava       |
| `Space`                           | Přehrát / pozastavit, když je aktivní zdroj tempa _Timeline_ |
| `T`                               | Vyťukat tempo                                         |
| `R`                               | Reset taktu                                           |
| `Y`                               | Zaokrouhlit tempo na nejbližší BPM                    |
| `Cmd / Ctrl + ,`                  | Otevřít panel nastavení                               |
| `Cmd / Ctrl + F`                  | Přepnout režim celé obrazovky                         |
| `Cmd / Ctrl + L`                  | Otevřít panel nastavení výstupu laseru                |
| `Cmd / Ctrl + K`                  | Otevřít panel přiřazení laser controller              |
| `Cmd / Ctrl + J`                  | Otevřít panel nastavení 3D vizualizéru                |
| `Cmd / Ctrl + Shift + Plus (+)`   | Zvětšit měřítko rozhraní                              |
| `Cmd / Ctrl + Shift + Minus (-)`  | Zmenšit měřítko rozhraní                              |
| `Cmd / Ctrl + 0`                  | Resetovat měřítko rozhraní                            |
| `Cmd / Ctrl + A`                  | Timeline: vybrat vše                                  |
| `Delete`                          | Timeline: odstranit vybrané bloky                     |
| `Delete`                          | Clip Editor: odstranit vybrané nodes                  |
| `Cmd + Q` (macOS)                 | Ukončit aplikaci                                      |
| `Alt + F4` (Windows)              | Ukončit aplikaci                                      |
