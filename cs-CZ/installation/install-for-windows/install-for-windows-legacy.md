# ✅ Instalace pro Windows (starší verze před 0.9)

### Kompatibilita

64bitové počítače s Windows 10 nebo 11. Pokud chcete spouštět větší show, doporučujeme výkonnější počítač. Čím rychlejší počítač máte, tím plynulejší bude animovaný laserový výstup.

### Postup instalace

Jakmile budete mít aktivní předplatné, dostanete odkaz, případně můžete kliknout na odkaz _Download_ na stránce [_My subscriptions_](https://liberationlaser.com/account/my-products).

Zobrazí se tato stránka:

<figure><img src="../../.gitbook/assets/windows-download-page.png" alt=""><figcaption></figcaption></figure>

Klikněte na _Download latest for Windows_ (čísla verzí se mohou lišit) a uvidíte indikátor průběhu stahování (obvykle vpravo nahoře; snímek obrazovky je z Chrome). Po dokončení najeďte myší na stažený soubor Liberation a klikněte na ikonu složky, která se zobrazí.

<figure><img src="../../.gitbook/assets/windows-download-chrome.png" alt="" width="260"><figcaption></figcaption></figure>

Otevře se složka se staženými soubory. Najděte stažený soubor (obvykle bude nahoře), klikněte na něj pravým tlačítkem a vyberte „Extract all“.

<figure><img src="../../.gitbook/assets/windows-extract-all.png" alt="" width="248"><figcaption></figcaption></figure>

Otevře se nový panel s dotazem, kam chcete Liberation nainstalovat. Můžete jej umístit kamkoli na pevný disk, kde máte oprávnění k zápisu. Pokud si nejste jistí, jednoduše jej dejte do složky na disku C:.

<figure><img src="../../.gitbook/assets/windows-select-destination.png" alt="" width="375"><figcaption></figcaption></figure>

Jakmile Windows dokončí rozbalování, zobrazí instalační soubor. Aplikaci spustíte dvojitým kliknutím na Liberation.exe.

<figure><img src="../../.gitbook/assets/windows-open-exe.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation potřebuje oprávnění k přístupu k souborům a síti, takže Windows pravděpodobně zobrazí několik výzev k povolení. Nezapomeňte vše povolit, jinak Liberation nemusí fungovat správně.

<figure><img src="../../.gitbook/assets/windows-protection-window.png" alt="" width="375"><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation by se nyní měl spustit. Abyste mohli vysílat výstup do laserů, je potřeba projít [procesem autorizace](../authorising-and-de-authorising.md).
