# ✅ Instalace pro Windows

### Kompatibilita

Počítače s 64bitovým systémem Windows 10 nebo 11. Pokud chcete spouštět velké show, doporučujeme výkonnější počítač. Čím rychlejší počítač máte, tím plynulejší bude animovaný laserový výstup.

### Postup instalace

Jakmile budete mít aktivní předplatné, dostanete odkaz, nebo můžete kliknout na odkaz _Download_ na stránce[ _Your subscriptions_](https://liberationlaser.com/account/my-products).

Zobrazí se tato stránka:

<figure><img src="../../.gitbook/assets/windows-install-download-page.png" alt=""><figcaption></figcaption></figure>

Klikněte na _Download latest for Windows_ (čísla verzí se mohou změnit) a uvidíte indikátor průběhu stahování (obvykle vpravo nahoře, snímek obrazovky je z prohlížeče Chrome). Po dokončení otevřete stažené soubory a klikněte na stažený instalační soubor.

<figure><img src="../../.gitbook/assets/windows-install-download-list.png" alt=""><figcaption></figcaption></figure>

Tím se spustí instalátor.

{% hint style="info" %}
V některých případech se může zobrazit varování, že Windows ochránil váš počítač – Windows je při spouštění nového softwaru opatrný. Jak bude Liberation používat a důvěřovat mu více lidí, mělo by toto varování zmizet.

Pokud se toto varování zobrazí, klikněte na odkaz _More info_, zkontrolujte, že _Publisher_ je uveden jako _SEB.LY LIMITED_, a potom klikněte na _Run anyway_.

<img src="../../.gitbook/assets/windows-install-windows-protection-warning.png" alt="" data-size="original"><img src="../../.gitbook/assets/windows-install-warning-run-anyway.png" alt="" data-size="original">
{% endhint %}

Před spuštěním instalátoru se může zobrazit varování s dotazem, zda souhlasíte s tím, aby instalátor provedl změny. Klikněte na _Yes_.

<figure><img src="../../.gitbook/assets/windows-install-warning-make-changes.png" alt=""><figcaption></figcaption></figure>

Teď se instalátor může spustit. Proklikejte jednotlivé stránky.

<figure><img src="../../.gitbook/assets/windows-installer-1.png" alt=""><figcaption></figcaption></figure>

Doporučujeme použít výchozí umístění instalace, takže stačí kliknout na _Next_.

<figure><img src="../../.gitbook/assets/windows-installer-2.png" alt=""><figcaption></figcaption></figure>

Dále máte k dispozici volby pro ikonu na ploše a složku v nabídce Start (doporučeno).

<figure><img src="../../.gitbook/assets/windows-installer-3.png" alt=""><figcaption></figcaption></figure>

Chvíli počkejte, než se Liberation nainstaluje.

<figure><img src="../../.gitbook/assets/windows-installer-4.png" alt=""><figcaption></figcaption></figure>

Instalátor je hotový. Kliknutím na _Finish_ ho zavřete a spustíte Liberation.

<figure><img src="../../.gitbook/assets/windows-installer-5.png" alt=""><figcaption></figcaption></figure>

Liberation potřebuje oprávnění k přístupu k souborům a k síti, takže Windows pravděpodobně zobrazí několik výzev k povolení. Nezapomeňte vše povolit, jinak Liberation nemusí fungovat správně.

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation by se teď měl spustit. Abyste mohli posílat výstup do laserů, pokračujte částí [Autorizace a zrušení autorizace](../authorising-and-de-authorising.md).
