# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_Paremeter Control_ je způsob, jak ovládat různá nastavení efektu nebo Clip mimo Clip Editor.

* **range min / range max** – minimální a maximální hodnota, na které se mapuje úroveň parametru
* **level** – výchozí nastavení tohoto parametru. Tuto hodnotu lze měnit zvenku. Můžete ji ručně upravit a podívat se, jaký bude mít vliv.
* **parameter number** – každému _parameter control_ lze přiřadit číslo, které se pak projeví na ovládacích prvcích mimo Clip. Můžete mít více než jeden parameter control se stejným číslem; všechny se pak nastavují současně.

{% hint style="warning" %}
APC40 a podporované live kontrolery ve výchozím nastavení ovládají parameter 1 pro každý slot efektu. Všechny parametry v efektech a v Clip jsou dostupné také v nabídce po kliknutí pravým tlačítkem.
{% endhint %}
