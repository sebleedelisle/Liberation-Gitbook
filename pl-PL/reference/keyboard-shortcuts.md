# ✅ Skróty klawiaturowe

Skróty zapisano jako `Cmd / Ctrl`: użyj `Cmd` w systemie macOS i `Ctrl` w systemie Windows.

| Skrót                             | Działanie                                             |
| --------------------------------- | ----------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Uzbrój wszystkie lasery                               |
| `Escape`                          | Rozbrój wszystkie lasery                              |
| `Tab`                             | Przełącz widoki i przechodź między laserami           |
| `Shift + Tab`                     | Przełącz widoki i przechodź między laserami wstecz    |
| `1-9`                             | Przejdź do lasera                                     |
| `Left / Right Arrow`              | Przesuń Clip Deck w lewo / w prawo                    |
| `Cmd / Ctrl + Left / Right Arrow` | Przesuń Clip Deck o jedną stronę w lewo / w prawo     |
| `Shift + Left / Right Arrow`      | Przesuń strefy o jedną stronę w lewo / w prawo        |
| `Space`                           | Odtwórz / wstrzymaj (gdy aktywne jest źródło tempa _Timeline_) |
| `T`                               | Wystukaj tempo                                        |
| `R`                               | Reset taktu                                           |
| `Y`                               | Zaokrąglij tempo do najbliższego BPM                  |
| `Cmd / Ctrl + ,`                  | Otwórz panel ustawień                                 |
| `Cmd / Ctrl + F`                  | Przełącz tryb pełnoekranowy                           |
| `Cmd / Ctrl + L`                  | Otwórz panel ustawień wyjścia laserowego              |
| `Cmd / Ctrl + K`                  | Otwórz panel przypisywania kontrolerów laserowych     |
| `Cmd / Ctrl + J`                  | Otwórz panel ustawień 3D Visualiser                   |
| `Cmd / Ctrl + Shift + Plus (+)`   | Zwiększ skalowanie interfejsu                         |
| `Cmd / Ctrl + Shift + Minus (-)`  | Zmniejsz skalowanie interfejsu                        |
| `Cmd / Ctrl + 0`                  | Zresetuj skalowanie interfejsu                        |
| `Cmd / Ctrl + A`                  | Timeline: zaznacz wszystko                            |
| `Delete`                          | Timeline: usuń zaznaczone bloki                       |
| `Delete`                          | Clip editor: usuń zaznaczone węzły                    |
| `Cmd + Q` (macOS)                 | Zamknij aplikację                                     |
| `Alt + F4` (Windows)              | Zamknij aplikację                                     |
