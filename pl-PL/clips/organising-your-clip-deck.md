# 🟩 Organizowanie Clip Deck

### Przenoszenie/duplikowanie klipów

Kliknij i przeciągnij, aby przenieść klip. Jeśli zaznaczony jest więcej niż jeden klip (zobacz [starting-stopping-clips.md](starting-stopping-clips.md)), wszystkie zostaną przeniesione. Przytrzymaj `Alt / Option`, aby je zduplikować. Upuść klip(y) na puste miejsce.

{% hint style="info" %}
Jeśli musisz zaznaczyć dużą liczbę klipów, możesz kliknąć i przeciągnąć lasso, a następnie przewijać Clip Deck, nadal przeciągając (za pomocą klawiszy kursora, APC 40 albo przewijania w poziomie na touchpadzie/myszy). Lasso zostanie rozszerzone poza krawędź Clip Deck.
{% endhint %}

### Usuwanie klipów

Aby usunąć pojedynczy klip, użyj menu pod prawym przyciskiem myszy.

Aby usunąć wiele klipów, zaznacz je (klikając z `Alt / Option + Shift` albo przeciągając lasso) i przeciągnij poza Clip Deck.

### Dodawanie pustego miejsca

Możesz kliknąć prawym przyciskiem myszy przycisk klipu i nacisnąć _ADD EMPTY COLUMN TO RIGHT_, aby dodać pustą kolumnę miejsca w Clip Deck.

### Łączenie wielu Clip Deck

Możesz zaimportować istniejący Clip Deck i dołączyć go na końcu bieżącego Clip Deck. Użyj menu _File -> Clips/Effects -> Append Clip Deck._

Zaznacz klipy (lasso albo kliknięciem z `Alt / Option + Shift`), a następnie wybierz _File -> Clips/Effects -> Export Selected Clips._ Spowoduje to wyeksportowanie Clip Deck zawierającego tylko te klipy, który można potem dołączyć do innego Clip Deck.

### Organizowanie klipów według timeline

Podczas importowania timeline masz też dostępne opcje klipów. Możesz zastąpić bieżący Clip Deck, dołączyć cały Clip Deck z timeline albo tylko klipy, których używa.

Jeśli projekt zawiera wiele timeline, możesz przeorganizować klipy według timeline. Otwórz panel _Timeline selection_ i kliknij _ORGANISE CLIPS BY TIMELINE_. Zobaczysz teraz wszystkie klipy w grupach odpowiadających ich timeline. Jeśli klip jest używany w wielu timeline, zostanie zduplikowany dla każdej grupy.
