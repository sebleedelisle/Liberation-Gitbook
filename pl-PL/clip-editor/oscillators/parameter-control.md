# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_Parameter Control_ to metoda sterowania różnymi ustawieniami efektu lub klipu poza edytorem klipu.

* **range min / range max** - minimalne i maksymalne wartości, na które mapowany jest poziom parametru
* **level** - domyślne ustawienie tego parametru. Jest to wartość, którą można zmieniać z zewnątrz. Możesz dostosować ją ręcznie, aby zobaczyć, jaki będzie efekt.
* **parameter number** - każdemu _parameter control_ można przypisać numer, który jest odzwierciedlany w kontrolkach poza klipem. Możesz mieć więcej niż jeden parameter control z tym samym numerem; wszystkie będą regulowane jednocześnie.

{% hint style="warning" %}
Obecnie za pomocą APC40 można sterować tylko parametrem 1 efektu. Wszystkie parametry efektów i klipów są dostępne w menu prawego przycisku myszy. W przyszłości więcej z tych parametrów będzie można mapować na kontrolery MIDI.
{% endhint %}
