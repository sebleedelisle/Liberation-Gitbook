# 🟩 Strefy Canvas

Strefa Canvas jest widoczna w widoku Canvas jako różowy prostokąt obrysu. Służy do zaznaczenia prostokątnego obszaru w Canvas i wysłania jego zawartości do jednego lub kilku laserów.

Strefy można przeciągać myszą. Przeciągnij narożniki, aby zmienić rozmiar i kształt.

### Dodawanie nowej strefy Canvas

Kliknij przycisk _Add a new canvas zone_ na pasku narzędzi.

### Przypisywanie strefy Canvas do lasera

Kliknij strefę prawym przyciskiem myszy i przełącz przyciski z numerami, aby wybrać, do którego lasera lub laserów ma być przypisana ta strefa Canvas.

{% hint style="danger" %}
OSTRZEŻENIE — jeśli laser jest uzbrojony, możesz nagle rozpocząć projekcję zawartości w domyślnej strefie Canvas. Najlepiej rozbroić laser przed przypisaniem do niego stref Canvas.
{% endhint %}

Jeśli przejdziesz teraz do widoku _OUTPUT_ dla tego lasera, zobaczysz tam strefę Canvas, którą można edytować dokładnie tak jak każdą strefę wiązki. Zobacz [zones.md](../output-view/zones.md).

{% hint style="info" %}
Możesz też dodać strefę Canvas do lasera bezpośrednio z widoku _OUTPUT_ — kliknij przycisk _Add existing canvas zone_ na lewym pasku narzędzi.
{% endhint %}

### Menu pod prawym przyciskiem myszy

Kliknij strefę prawym przyciskiem myszy, aby zobaczyć opcje usunięcia tej strefy Canvas, przeniesienia jej na spód oraz zablokowania.
