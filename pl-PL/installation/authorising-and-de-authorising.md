# ✅ Autoryzacja i cofanie autoryzacji

### Autoryzacja Liberation

Przy pierwszym uruchomieniu Liberation program będzie działać w _trybie bezpłatnym_ i zobaczysz _About panel:_

<figure><img src="../.gitbook/assets/authorisation-about-panel.png" alt="" width="375"><figcaption></figcaption></figure>

Kliknij przycisk _AUTHORISE ONLINE_, a otworzy się przeglądarka internetowa. Jeśli nie jesteś jeszcze zalogowany, pojawi się prośba o zalogowanie.

System automatycznie autoryzuje teraz Twoją instalację w ramach licencji. Zobaczysz komunikat z potwierdzeniem.

Po powrocie do Liberation zobaczysz, że _About panel_ został zaktualizowany (może to potrwać kilka sekund).

<figure><img src="../.gitbook/assets/authorise-about-panel-authorised" alt="" width="375"><figcaption></figcaption></figure>

{% hint style="warning" %}
Jeśli autoryzowano już maksymalną liczbę komputerów dostępną w Twojej licencji, musisz cofnąć autoryzację jednej z pozostałych maszyn albo podnieść plan licencji.
{% endhint %}

{% hint style="info" %}
Jeśli masz wiele licencji, pojawi się prośba o wybranie licencji, do której chcesz przypisać komputer.
{% endhint %}

Gratulacje! Twoja instalacja Liberation została autoryzowana i możesz wysyłać sygnał do laserów. Przed uzbrojeniem laserów przeczytaj jednak [Szybki start](../getting-started.md "mention") oraz [Przegląd procesu konfiguracji laserów](../setting-up/setting-up-lasers.md "mention").

{% hint style="info" %}
Możesz otworzyć _About panel_ w dowolnym momencie z menu _Liberation -> About Liberation_ albo _Liberation -> Authorise/Deauthorise this computer_
{% endhint %}

### Cofanie autoryzacji Liberation

**Z poziomu Liberation** — otwórz menu _Liberation -> Authorise / De-authorise this Computer_ i kliknij przycisk _DEAUTHORISE COMPUTER_. To wszystko! Aby to zadziałało, musisz być online.

<figure><img src="../.gitbook/assets/Screenshot 2024-08-28 at 10.18.57.png" alt=""><figcaption></figcaption></figure>

Możesz też zrobić to ze strony internetowej. Wybierz z menu _Your licences_, a następnie kliknij _MANAGE_ przy odpowiedniej licencji. Jeśli na Twoim koncie jest tylko jedna licencja, _Your licences_ przeniesie Cię od razu na jej stronę. Powinny być tam widoczne informacje o Twojej licencji oraz lista autoryzowanych komputerów.

<figure><img src="../.gitbook/assets/account-your-licences.png" alt=""><figcaption></figcaption></figure>

Kliknij link _De-authorise_ obok maszyny, której autoryzację chcesz cofnąć.

Jeśli komputer nie był online od ostatniego odświeżenia licencji, jego autoryzacja zostanie cofnięta natychmiast. W przeciwnym razie komputer zostanie _dodany do kolejki_ cofnięcia autoryzacji. Oznacza to, że cofnięcie autoryzacji nastąpi automatycznie przy następnym połączeniu komputera z internetem albo w następnym dniu odświeżenia licencji — w zależności od tego, co nastąpi wcześniej.
