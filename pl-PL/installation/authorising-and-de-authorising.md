# ✅ Autoryzacja i cofanie autoryzacji

### Autoryzacja Liberation

Przy pierwszym uruchomieniu Liberation program będzie działać w _trybie demo_ i zobaczysz _About panel:_

<figure><img src="../.gitbook/assets/authorisation-about-panel.png" alt="" width="375"><figcaption></figcaption></figure>

Kliknij przycisk _AUTHORISE ONLINE_, a otworzy się przeglądarka internetowa. Jeśli nie jesteś jeszcze zalogowany, pojawi się prośba o zalogowanie.

System automatycznie autoryzuje teraz Twoją instalację w ramach subskrypcji. Zobaczysz ten komunikat:

<figure><img src="../.gitbook/assets/authorisation-machine-added" alt=""><figcaption></figcaption></figure>

Po powrocie do Liberation zobaczysz, że _About panel_ został zaktualizowany (może to potrwać kilka sekund).

<figure><img src="../.gitbook/assets/authorise-about-panel-authorised" alt="" width="375"><figcaption></figcaption></figure>

{% hint style="warning" %}
Jeśli autoryzowano już maksymalną liczbę komputerów dostępną w Twojej subskrypcji, musisz cofnąć autoryzację jednej z pozostałych maszyn albo podnieść plan subskrypcji.
{% endhint %}

{% hint style="info" %}
Jeśli masz wiele subskrypcji, pojawi się prośba o wybranie subskrypcji, do której chcesz przypisać komputer.
{% endhint %}

Gratulacje! Twoja instalacja Liberation została autoryzowana i możesz wysyłać sygnał do laserów. Przed uzbrojeniem laserów przeczytaj jednak [Szybki start](../getting-started.md) oraz [Przegląd procesu konfiguracji laserów](../setting-up/setting-up-lasers.md).

{% hint style="info" %}
Możesz otworzyć _About panel_ w dowolnym momencie z menu _Liberation -> About Liberation_ albo _Liberation -> Authorise/Deauthorise this computer_
{% endhint %}

### Cofanie autoryzacji Liberation

**Z poziomu Liberation** — otwórz menu _Liberation -> Authorise / De-authorise this Computer_ i kliknij przycisk _DEAUTHORISE COMPUTER_. To wszystko! Aby to zadziałało, musisz być online.

<figure><img src="../.gitbook/assets/Screenshot 2024-08-28 at 10.18.57.png" alt=""><figcaption></figcaption></figure>

Możesz też zrobić to ze strony internetowej — wybierz z menu _Your subscriptions_, a następnie kliknij _Manage Subscription_, aby otworzyć stronę subskrypcji. Powinny być tam widoczne informacje o Twojej subskrypcji oraz lista autoryzowanych komputerów.

Kliknij link _De-authorise_ obok maszyny, której autoryzację chcesz cofnąć.

<figure><img src="../.gitbook/assets/Screenshot 2024-08-28 at 10.16.40.png" alt=""><figcaption></figcaption></figure>

Jeśli od ostatniego odnowienia subskrypcji Twoja maszyna nie była online, jej autoryzacja zostanie cofnięta natychmiast. W przeciwnym razie maszyna zostanie dodana do _kolejki_ cofnięcia autoryzacji. Oznacza to, że cofnięcie autoryzacji nastąpi automatycznie przy następnym połączeniu maszyny z internetem albo w dniu kolejnego odnowienia subskrypcji — zależnie od tego, co nastąpi wcześniej.
