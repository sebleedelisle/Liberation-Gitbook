# ✅ Instalacja na Macu

### Zgodność

macOS w wersji 12 (Monterey) lub nowszej. Zalecany jest Apple Silicon, szczególnie jeśli chcesz uruchamiać duże pokazy. Im szybszy komputer, tym płynniejszy będzie animowany obraz laserowy.

### Proces instalacji

Gdy masz aktywną płatną licencję, otrzymasz link. Możesz też kliknąć _Download_ na stronie [_Twoje licencje_](https://liberationlaser.com/account/my-products).

Zobaczysz tę stronę:

<figure><img src="../.gitbook/assets/account-download-page.png" alt=""><figcaption></figcaption></figure>

Kliknij _Download for Mac_ (numery wersji będą się z czasem zmieniać), a plik instalacyjny Liberation DMG zostanie zapisany w folderze Downloads. Znajdź plik i kliknij go dwukrotnie, aby go otworzyć.

<figure><img src="../.gitbook/assets/mac-dmg-window" alt="" width="375"><figcaption></figcaption></figure>

Przeciągnij ikonę Liberation po lewej stronie do folderu Applications po prawej (może być wymagane hasło administratora). Liberation znajdziesz teraz w folderze Applications — kliknij dwukrotnie, aby go otworzyć.

Liberation powinien się teraz uruchomić. Aby wysyłać obraz do laserów, przejdź przez [Autoryzowanie i cofanie autoryzacji](authorising-and-de-authorising.md).
