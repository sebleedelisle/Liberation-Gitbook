# ✅ Instalacja w Windows (starsza metoda, przed wersją 0.9)

### Zgodność

Komputery z 64-bitowym systemem Windows 10 lub 11. Jeśli chcesz uruchamiać duże pokazy, zalecany jest komputer o wysokiej wydajności. Im szybszy komputer, tym płynniejszy będzie animowany obraz laserowy.

### Proces instalacji

Gdy masz aktywną subskrypcję, otrzymasz link. Możesz też kliknąć link _Download_ na stronie [_My subscriptions_](https://liberationlaser.com/account/my-products).

Zobaczysz tę stronę:

<figure><img src="../../.gitbook/assets/windows-download-page.png" alt=""><figcaption></figcaption></figure>

Kliknij _Download latest for Windows_ (numery wersji mogą się zmieniać). Zobaczysz wskaźnik postępu pobierania (zwykle w prawym górnym rogu; zrzut ekranu pochodzi z Chrome). Po zakończeniu najedź kursorem na pobrany plik Liberation i kliknij ikonę folderu, która się pojawi.

<figure><img src="../../.gitbook/assets/windows-download-chrome.png" alt="" width="260"><figcaption></figcaption></figure>

Otworzy się folder pobranych plików. Znajdź pobrany plik (zwykle na samej górze), kliknij go prawym przyciskiem myszy i wybierz „Extract all”.

<figure><img src="../../.gitbook/assets/windows-extract-all.png" alt="" width="248"><figcaption></figcaption></figure>

Otworzy się nowy panel z pytaniem, gdzie chcesz zainstalować Liberation. Możesz umieścić go w dowolnym miejscu na dysku twardym, do którego masz uprawnienia zapisu. Jeśli nie masz pewności, po prostu umieść go w folderze na dysku C:.

<figure><img src="../../.gitbook/assets/windows-select-destination.png" alt="" width="375"><figcaption></figcaption></figure>

Po zakończeniu rozpakowywania Windows pokaże plik instalacyjny. Kliknij dwukrotnie Liberation.exe, aby uruchomić aplikację.

<figure><img src="../../.gitbook/assets/windows-open-exe.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation potrzebuje uprawnień dostępu do plików i sieci, więc Windows prawdopodobnie wyświetli kilka komunikatów z prośbą o zgodę. Zaakceptuj wszystkie, w przeciwnym razie Liberation może nie działać poprawnie.

<figure><img src="../../.gitbook/assets/windows-protection-window.png" alt="" width="375"><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation powinien teraz działać. Aby wysyłać sygnał do laserów, musisz teraz przejść przez [proces autoryzacji.](../authorising-and-de-authorising.md)
