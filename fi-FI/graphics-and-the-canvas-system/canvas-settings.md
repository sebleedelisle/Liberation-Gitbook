# 🟩 Canvas-asetukset

Avaa _Canvas settings_ -paneeli napsauttamalla vasemman työkalupalkin yläosassa olevaa _Canvas settings_ -painiketta.

### Width and Height

Muuta koko canvasin kokonaisleveyttä ja -korkeutta näillä vedettävillä liukusäätimillä. Mittoja voi ajatella kuvitteellisesti pikselikokona, mutta kaikki on suhteellista.

Kaksoisnapsauta liukusäädintä, jos haluat kirjoittaa arvon.

{% hint style="info" %}
Tässä hieman nörttimäistä lisätietoa: laser liikkuu suhteessa canvasin resoluutioon, ei suhteessa zonen kokoon. Tämä johtuu siitä, että suuri zone canvasilla projisoidaan todennäköisesti suuremmalle alueelle myös todellisuudessa.

Tämä on pieni yksityiskohta, jolla ei todennäköisesti ole sinulle suurta merkitystä, mutta se auttaa tasoittamaan suorituskykyä ja näkyvyyttä canvasin eri zoneissa.
{% endhint %}

### Zone brightness

Tällä säädät, kuinka kirkkaina vaaleanpunaiset zone-ääriviivat näkyvät Canvas-näkymässä. Tämä voi parantaa sisällön selkeyttä ja estää näkymää muuttumasta liian sekavaksi.
