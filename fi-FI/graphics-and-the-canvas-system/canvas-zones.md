# 🟩 Canvas-zonet

Canvas-zone näkyy Canvas-näkymässä vaaleanpunaisena suorakulmiona, jossa on vain reunaviiva. Sillä valitaan Canvasilta suorakulmainen alue ja lähetetään sen sisältö yhdelle tai useammalle laserille.

Voit siirtää zoneja hiirellä napsauttamalla ja vetämällä. Muuta kokoa ja muotoa vetämällä kulmista.

### Lisää uusi Canvas-zone

Napsauta työkalurivillä painiketta _Add a new canvas zone_.

### Määritä Canvas-zone laserille

Napsauta zonea hiiren oikealla painikkeella ja valitse numeropainikkeilla, mille lasereille tämä Canvas-zone määritetään.

{% hint style="danger" %}
VAROITUS – jos laser on armed-tilassa, saatat alkaa yhtäkkiä projisoida sisältöä oletusarvoisella Canvas-zonella. Laser kannattaa asettaa disarmed-tilaan ennen Canvas-zonejen määrittämistä sille.
{% endhint %}

Jos siirryt nyt kyseisen laserin _OUTPUT_-näkymään, näet Canvas-zonen siellä. Voit muokata sitä täsmälleen kuten mitä tahansa beam zonea. Katso [zones.md](../output-view/zones.md).

{% hint style="info" %}
Voit myös lisätä Canvas-zonen laserille suoraan _OUTPUT_-näkymästä – napsauta vasemman työkalurivin _Add existing canvas zone_ -painiketta.
{% endhint %}

### Hiiren oikean painikkeen valikko

Napsauta zonea hiiren oikealla painikkeella. Näet valinnat, joilla voit poistaa tämän Canvas-zonen, siirtää sen taakse ja lukita sen.
