# ✅ Pikanäppäimet

Pikanäppäimet on merkitty muodossa `Cmd / Ctrl`: käytä macOS:ssä `Cmd`-näppäintä ja Windowsissa `Ctrl`-näppäintä.

| Shortcut                          | Action                                                           |
| --------------------------------- | ---------------------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Aseta kaikki laserit käyttövalmiiksi                             |
| `Escape`                          | Poista kaikkien lasereiden käyttövalmius                         |
| `Tab`                             | Vaihda näkymää ja siirry laserista toiseen                       |
| `Shift + Tab`                     | Vaihda näkymää ja siirry laserista toiseen taaksepäin            |
| `1-9`                             | Siirry laseriin                                                  |
| `Left / Right Arrow`              | Siirrä Clip Deckiä vasemmalle / oikealle                         |
| `Cmd / Ctrl + Left / Right Arrow` | Siirrä Clip Deckiä sivu vasemmalle / oikealle                    |
| `Shift + Left / Right Arrow`      | Siirrä alueita sivu vasemmalle / oikealle                        |
| `Space`                           | Toista / keskeytä (kun _Timeline_-tempolähde on aktiivinen)      |
| `T`                               | Määritä tempo napauttamalla                                      |
| `R`                               | Tahdin nollaus                                                   |
| `Y`                               | Pyöristä tempo lähimpään BPM-arvoon                              |
| `Cmd / Ctrl + ,`                  | Avaa asetuspaneeli                                               |
| `Cmd / Ctrl + F`                  | Ota koko näytön tila käyttöön / pois käytöstä                    |
| `Cmd / Ctrl + L`                  | Avaa laserulostulon asetuspaneeli                                |
| `Cmd / Ctrl + K`                  | Avaa laserohjainten määrityspaneeli                              |
| `Cmd / Ctrl + J`                  | Avaa 3D Visualiser -asetuspaneeli                                |
| `Cmd / Ctrl + Shift + Plus (+)`   | Suurenna käyttöliittymän skaalausta                              |
| `Cmd / Ctrl + Shift + Minus (-)`  | Pienennä käyttöliittymän skaalausta                              |
| `Cmd / Ctrl + 0`                  | Palauta käyttöliittymän skaalaus                                 |
| `Cmd / Ctrl + A`                  | Timeline: valitse kaikki                                         |
| `Delete`                          | Timeline: poista valitut lohkot                                  |
| `Delete`                          | Clip editor: poista valitut solmut                               |
| `Cmd + Q` (macOS)                 | Sulje sovellus                                                   |
| `Alt + F4` (Windows)              | Sulje sovellus                                                   |
