# ✅ Miten lisensointi toimii

Kun olet asentanut Liberationin tietokoneellesi, se toimii _demo-tilassa_, kunnes _valtuutat_ sen.

{% hint style="info" %}
_demo-tilassa_ ei ole muita rajoituksia kuin lähetys laserlaitteistolle. Voit harjoitella esityksiä, rakentaa aikajanoja sekä tallentaa ja ladata tiedostoja ilman rajoituksia. Voit asentaa Liberationin _demo-tilassa_ niin monelle tietokoneelle kuin haluat, ja sinun tarvitsee valtuuttaa asennus vasta, kun olet valmis käyttämään lasereita!
{% endhint %}

Sen lisäksi, kuinka monta laseria voit ottaa käyttövalmiiksi, voit käyttää myös DMX-laitteita. Katso [dmx-control](../dmx-control/).

Myös niiden tietokoneiden määrää, joilla voit valtuuttaa Liberationin, rajoitetaan tilausportaan mukaan.

{% hint style="danger" %}
Kukin lisenssi on tarkoitettu vain yhdelle käyttäjälle ja yhdelle esitykselle kerrallaan. Sinulla voi olla käyttäjää kohti useampi kuin yksi asennus, koska sinulla tulisi **aina** olla vähintään yksi varalle valtuutettu tietokone käyttövalmiina!
{% endhint %}

### Internet-yhteys

Liberationin lisensointi ja valtuutus toimivat internetin kautta; sovellus muodostaa yhteyden palvelimeemme liittääkseen asennuksen tilaukseesi.

Kun asennuksesi on valtuutettu, sen ei tarvitse olla uudelleen verkossa ennen kuin tilauksesi uusiutuu. Näet uusimispäivän _About panel_ -paneelissa. Käytössä on lyhyt lisäaika (7 päivää kuukausitilauksissa), mutta on suositeltavaa muodostaa internet-yhteys mahdollisimman pian uusimispäivän jälkeen, jotta valtuutus uusiutuu ja pysyy aktiivisena.
