# ✅ Asennus Windowsille

### Yhteensopivuus

Windows 10- tai 11-koneet, 64-bittinen versio. Suurten show’iden ajamiseen suositellaan tehokasta tietokonetta. Mitä nopeampi tietokone on, sitä sujuvampaa animoitu laserulostulo on.

### Asennusprosessi

Kun sinulla on aktiivinen tilaus, saat linkin, tai voit napsauttaa _Download_-linkkiä [ _Your subscriptions_](https://liberationlaser.com/account/my-products) -sivulla.

Näet tämän sivun:

<figure><img src="../../.gitbook/assets/windows-install-download-page.png" alt=""><figcaption></figcaption></figure>

Napsauta _Download latest for Windows_ (versionumerot voivat muuttua). Latauksen edistyminen näkyy latausilmaisimessa, yleensä oikeassa yläkulmassa (kuvakaappaus on Chromesta). Kun lataus on valmis, avaa lataukset ja napsauta ladattua asennustiedostoa.

<figure><img src="../../.gitbook/assets/windows-install-download-list.png" alt=""><figcaption></figcaption></figure>

Tämä käynnistää asennusohjelman.

{% hint style="info" %}
Joissakin tapauksissa saatat nähdä varoituksen, jossa kerrotaan, että Windows suojasi tietokoneesi. Windows suhtautuu varovaisesti uusien ohjelmistojen suorittamiseen. Kun useammat käyttäjät käyttävät Liberationia ja luottavat siihen, tämän varoituksen pitäisi poistua.

Jos näet tämän varoituksen, napsauta _More info_ -linkkiä, varmista että _Publisher_ näkyy muodossa _SEB.LY LIMITED_, ja valitse sitten _Run anyway_.

<img src="../../.gitbook/assets/windows-install-windows-protection-warning.png" alt="" data-size="original"><img src="../../.gitbook/assets/windows-install-warning-run-anyway.png" alt="" data-size="original">
{% endhint %}

Ennen asennusohjelman käynnistymistä saatat nähdä varoituksen, jossa kysytään, sallitko asennusohjelman tehdä muutoksia. Napsauta _Yes_.

<figure><img src="../../.gitbook/assets/windows-install-warning-make-changes.png" alt=""><figcaption></figcaption></figure>

Nyt asennusohjelma voi käynnistyä. Siirry sivujen läpi napsauttamalla eteenpäin.

<figure><img src="../../.gitbook/assets/windows-installer-1.png" alt=""><figcaption></figcaption></figure>

Suosittelemme käyttämään oletusasennussijaintia, joten napsauta vain _Next_.

<figure><img src="../../.gitbook/assets/windows-installer-2.png" alt=""><figcaption></figcaption></figure>

Seuraavaksi voit valita työpöytäkuvakkeen ja Start Menu -kansion (suositeltu).

<figure><img src="../../.gitbook/assets/windows-installer-3.png" alt=""><figcaption></figcaption></figure>

Odota hetki, kun Liberation asennetaan.

<figure><img src="../../.gitbook/assets/windows-installer-4.png" alt=""><figcaption></figcaption></figure>

Asennus on valmis. Sulje asennusohjelma ja käynnistä Liberation napsauttamalla _Finish_.

<figure><img src="../../.gitbook/assets/windows-installer-5.png" alt=""><figcaption></figcaption></figure>

Liberation tarvitsee oikeuden käyttää tiedostoja ja verkkoa, joten Windows näyttää todennäköisesti useita pyyntöikkunoita. Hyväksy kaikki pyynnöt, muuten Liberation ei välttämättä toimi oikein.

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

Liberationin pitäisi nyt käynnistyä. Jotta voit lähettää ulostuloa lasereille, sinun täytyy seuraavaksi käydä läpi [valtuutusprosessi.](../authorising-and-de-authorising.md)
