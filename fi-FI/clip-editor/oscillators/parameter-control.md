# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_Parameter Control_ on menetelmä, jolla voit ohjata efektin tai klipin eri asetuksia klippieditorin ulkopuolelta.

* **range min / range max** - vähimmäis- ja enimmäisarvot, joihin parametrin taso skaalataan
* **level** - tämän parametrin oletusasetus. Tätä arvoa voidaan muuttaa ulkopuolelta. Voit säätää sitä käsin ja nähdä, miten se vaikuttaa.
* **parameter number** - jokaiselle _parameter control_ -ohjaukselle voidaan määrittää numero, ja tämä näkyy klipin ulkopuolisissa ohjaimissa. Samalla numerolla voi olla useita parameter control -ohjauksia, ja niitä kaikkia säädetään samanaikaisesti.

{% hint style="warning" %}
APC40 ja tuetut live-ohjaimet ohjaavat oletusarvoisesti kunkin efektipaikan parametria 1. Kaikki efektien ja Clips-kohteiden parametrit ovat käytettävissä myös oikean napsautuksen valikossa.
{% endhint %}
