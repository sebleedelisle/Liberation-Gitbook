# 🟩 Canvas settings

Klicke oben in der linken Werkzeugleiste auf die Schaltfläche _Canvas settings_, um das Panel _Canvas settings_ zu öffnen.

### Width and Height

Mit diesen ziehbaren Slidern änderst du die Gesamtbreite und -höhe des gesamten Canvas. Diese Maße können sinngemäß als Pixelgröße betrachtet werden, sind aber vollständig relativ.

Doppelklicke auf einen Slider, um einen Wert einzugeben.

{% hint style="info" %}
Hier noch ein zusätzliches Nerd-Detail: Der Laser bewegt sich relativ zur Canvas-Auflösung, nicht relativ zur Zonengröße. Das liegt daran, dass eine große Zone im Canvas in der Realität wahrscheinlich auf eine größere Fläche projiziert wird.

Das ist ein kleines Detail, das für dich vermutlich keine große Rolle spielt, aber es hilft dabei, Performance und Sichtbarkeit über deine Canvas-Zonen hinweg gleichmäßiger zu halten.
{% endhint %}

### Zone brightness

Hiermit änderst du, wie hell die pinkfarbenen Zonen-Umrisse in der Canvas-Ansicht dargestellt werden. Das kann die Übersichtlichkeit deiner Inhalte verbessern und verhindern, dass diese Ansicht zu überladen wirkt.
