# ✅ Tastaturkürzel

Tastaturkürzel sind als `Cmd / Ctrl` angegeben: Verwende `Cmd` unter macOS und `Ctrl` unter Windows.

| Shortcut                          | Action                                                                    |
| --------------------------------- | ------------------------------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Alle Laser aktivieren                                                     |
| `Escape`                          | Alle Laser deaktivieren                                                   |
| `Tab`                             | Ansichten umschalten und durch die Laser wechseln                         |
| `Shift + Tab`                     | Ansichten umschalten und rückwärts durch die Laser wechseln               |
| `1-9`                             | Zu einem Laser springen                                                   |
| `Left / Right Arrow`              | Clip Deck nach links / rechts verschieben                                 |
| `Cmd / Ctrl + Left / Right Arrow` | Clip Deck um eine Seite nach links / rechts verschieben                   |
| `Shift + Left / Right Arrow`      | Zonen um eine Seite nach links / rechts verschieben                       |
| `Space`                           | Wiedergabe / Pause (wenn die Tempoquelle _Timeline_ aktiv ist)            |
| `T`                               | Tempo eintippen                                                           |
| `R`                               | Takt zurücksetzen                                                         |
| `Y`                               | Tempo auf die nächste BPM-Zahl runden                                     |
| `Cmd / Ctrl + ,`                  | Settings Panel öffnen                                                     |
| `Cmd / Ctrl + F`                  | Vollbildmodus umschalten                                                  |
| `Cmd / Ctrl + L`                  | Laser Output Settings Panel öffnen                                        |
| `Cmd / Ctrl + K`                  | Laser Controller Assignment Panel öffnen                                  |
| `Cmd / Ctrl + J`                  | 3D Visualiser Settings Panel öffnen                                       |
| `Cmd / Ctrl + Shift + Plus (+)`   | Skalierung der Benutzeroberfläche erhöhen                                 |
| `Cmd / Ctrl + Shift + Minus (-)`  | Skalierung der Benutzeroberfläche verringern                              |
| `Cmd / Ctrl + 0`                  | Skalierung der Benutzeroberfläche zurücksetzen                            |
| `Cmd / Ctrl + A`                  | Timeline: alles auswählen                                                 |
| `Delete`                          | Timeline: ausgewählte Blöcke löschen                                      |
| `Delete`                          | Clip Editor: ausgewählte Nodes löschen                                    |
| `Cmd + Q` (macOS)                 | App beenden                                                               |
| `Alt + F4` (Windows)              | App beenden                                                               |
