# ✅ Fehler „Missing resources“ beim Start

Wenn beim Starten ein Fehler „missing resources“ angezeigt wird, bedeutet das, dass wichtige Dateien fehlen, die Liberation zum Ausführen benötigt. Unter Windows kann das passieren, wenn du versehentlich Dateien im Installationsordner löschst oder verschiebst. Auf dem Mac kann es auftreten, wenn deine Installation auf irgendeine Weise beschädigt wurde.

### So behebst du das Problem

Die beste Lösung für dieses Problem ist, Liberation neu zu installieren.
