# ✅ Installation für Mac

### Kompatibilität

macOS Version 12 (Monterey) oder höher. Apple Silicon wird empfohlen, besonders wenn du große Shows ausführen möchtest. Je schneller dein Computer ist, desto flüssiger wird die animierte Laserausgabe.

### Installationsprozess

Sobald du ein aktives Abonnement hast, erhältst du einen Link. Alternativ kannst du auf der Seite [_My subscriptions_](https://liberationlaser.com/account/my-products) auf den Link _Download_ klicken.

Du siehst dann diese Seite:

<figure><img src="../.gitbook/assets/windows-download-page.png" alt=""><figcaption></figcaption></figure>

Klicke auf _Download latest for Mac_ (die Versionsnummern ändern sich im Laufe der Zeit). Die Liberation-DMG-Installationsdatei wird dann in deinem Downloads-Ordner gespeichert. Suche die Datei und öffne sie per Doppelklick.

<figure><img src="../.gitbook/assets/mac-dmg-window" alt="" width="375"><figcaption></figcaption></figure>

Ziehe das Liberation-Symbol links in den Ordner Applications rechts (möglicherweise benötigst du dafür ein Administratorpasswort). Du findest Liberation jetzt in deinem Applications-Ordner und kannst es per Doppelklick öffnen.

Liberation sollte jetzt laufen! Um Laser auszugeben, musst du nun den [Autorisierungsprozess](authorising-and-de-authorising.md) durchlaufen.
