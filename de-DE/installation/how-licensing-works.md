# ✅ So funktioniert die Lizenzierung

Sobald du Liberation auf deinem Computer installiert hast, läuft es im _demo mode_, bis du es _autorisierst_.

{% hint style="info" %}
Im _demo mode_ gibt es keine Einschränkungen, außer bei der Ausgabe an Laser-Hardware. Du kannst Shows üben, Timelines erstellen sowie deine Dateien ohne Einschränkungen speichern und laden. Du kannst Liberation im _demo mode_ auf beliebig vielen Computern installieren und musst deine Installation erst autorisieren, wenn du bereit bist, Laser zu verwenden!
{% endhint %}

Zusätzlich zur Anzahl der Laser, die du aktivieren kannst, kannst du auch DMX-Geräte verwenden. Siehe [DMX / Art-Net](../dmx-control/ "mention").

Außerdem gibt es je nach Abostufe eine Begrenzung dafür, auf wie vielen Computern du Liberation autorisieren kannst.

{% hint style="danger" %}
Jede Lizenz gilt nur für einen Benutzer und jeweils nur für eine Show gleichzeitig. Du kannst mehr als eine Installation pro Benutzer haben, weil du **immer** mindestens einen autorisierten Ersatzcomputer bereithalten solltest!
{% endhint %}

### Internetverbindung

Die Lizenzierung und Autorisierung von Liberation funktioniert über das Internet; die App verbindet sich mit unserem Server, um eine Installation deiner Subscription zuzuordnen.

Sobald deine Installation autorisiert ist, muss sie erst wieder online sein, wenn deine Subscription verlängert wird. Das Verlängerungsdatum findest du im _About panel_. Es gibt eine kurze Kulanzfrist (7 Tage bei monatlichen Subscriptions), aber es wird empfohlen, nach dem Verlängerungsdatum so bald wie möglich online zu gehen, damit deine Autorisierung erneuert wird und aktiv bleibt.
