# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_Parameter Control_ ist eine Methode, um verschiedene Einstellungen innerhalb eines Effekts oder Clips außerhalb des Clip-Editors zu steuern.

* **range min / range max** - die Minimal- und Maximalwerte, auf die der Parameter-Level abgebildet wird
* **level** - die Standardeinstellung für diesen Parameter. Das ist der Wert, der von außen geändert werden kann. Du kannst ihn manuell anpassen, um zu sehen, welche Wirkung er hat.
* **parameter number** - jeder _parameter control_ kann eine Nummer zugewiesen werden, und diese wird in den Steuerelementen außerhalb des Clips angezeigt. Du kannst mehrere Parameter Controls mit derselben Nummer verwenden; sie werden dann alle gleichzeitig angepasst.

{% hint style="warning" %}
Der APC40 und unterstützte Live-Controller steuern standardmäßig Parameter 1 für jeden Effekt-Slot. Alle Parameter innerhalb von Effekten und Clips sind außerdem im Rechtsklickmenü verfügbar.
{% endhint %}
