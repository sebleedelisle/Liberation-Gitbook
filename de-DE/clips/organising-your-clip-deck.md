# 🟩 Dein Clip Deck organisieren

### Clips verschieben/duplizieren

Klicke und ziehe, um einen Clip zu verschieben. Wenn mehr als ein Clip ausgewählt ist (siehe [Clips starten / stoppen](starting-stopping-clips.md "mention")), werden alle ausgewählten Clips verschoben. Halte `Alt / Option` gedrückt, um sie zu duplizieren. Lege den/die Clip(s) auf einer freien Stelle ab.

{% hint style="info" %}
Wenn du eine große Anzahl von Clips auswählen musst, kannst du per Klicken/Ziehen ein Lasso aufziehen und dann weiterziehen, während du im Clip Deck scrollst (entweder mit den Pfeiltasten, der APC 40 oder seitlichem Scrollen per Touchpad/Maus). Das Lasso wird über den Rand des Clip Deck hinaus erweitert.
{% endhint %}

### Clips löschen

Um einen einzelnen Clip zu löschen, verwende das Rechtsklick-Menü.

Um mehrere Clips zu löschen, wähle sie aus (entweder mit `Alt / Option + Shift`-Klick oder indem du das Lasso ziehst) und ziehe sie aus dem Clip Deck heraus.

### Leeren Platz hinzufügen

Du kannst mit der rechten Maustaste auf einen Clip-Button klicken und _ADD EMPTY COLUMN TO RIGHT_ drücken, um dem Clip Deck eine leere Spalte hinzuzufügen.

### Mehrere Clip Decks kombinieren

Du kannst ein vorhandenes Clip Deck importieren und an das Ende des aktuellen Clip Deck anhängen. Verwende dazu das Menü _File -> Clips/Effects -> Append Clip Deck._

Wähle Clips aus (entweder mit dem Lasso oder per `Alt / Option + Shift`-Klick) und wähle dann _File -> Clips/Effects -> Export Selected Clips._ Dadurch wird ein Clip Deck exportiert, das nur diese Clips enthält und anschließend an ein anderes Clip Deck angehängt werden kann.

### Clips nach Timeline organisieren

Wenn du eine Timeline importierst, stehen dir auch einige Clip-Optionen zur Verfügung. Du kannst entweder das aktuelle Clip Deck ersetzen, das gesamte Clip Deck der Timeline anhängen oder nur die Clips, die sie verwendet.

Wenn dein Projekt mehrere Timelines enthält, kannst du deine Clips nach Timeline neu organisieren. Öffne das Panel _Timeline selection_ und klicke auf _ORGANISE CLIPS BY TIMELINE_. Du siehst nun alle deine Clips in Gruppen, basierend auf ihrer Timeline. Wenn ein Clip in mehreren Timelines verwendet wird, wird er für jede Gruppe dupliziert.
