# ✅ ライセンスの仕組み

コンピューターに Liberation をインストールすると、_free mode_ で起動します。使用を開始するには、インストールを_認証_する必要があります。

{% hint style="info" %}
_free mode_ では、レーザーハードウェアへの出力以外に制限はありません。ショーの練習、タイムラインの作成、ファイルの保存と読み込みを制限なく行えます。Liberation は _free mode_ のまま何台のコンピューターにでもインストールできます。レーザーを使用する準備ができた時点で、インストールを認証すれば十分です。
{% endhint %}

armed にできるレーザーの数に加えて、DMX フィクスチャも使用できます。詳しくは [DMX / Art-Net](../dmx-control/ "mention") を参照してください。

また、ライセンスプランに応じて、Liberation を認証できるコンピューターの台数にも上限があります。

{% hint style="danger" %}
各ライセンスは、1人のユーザーが同時に1つのショーで使用するためのものです。ユーザー1人につき複数のインストールを持てるのは、認証済みの予備コンピューターを少なくとも1台、**必ず**用意しておくべきだからです。
{% endhint %}

### インターネット接続

Liberation のライセンス管理と認証はインターネット経由で行われます。アプリがサーバーに接続し、インストールを有料ライセンスに関連付けます。

認証が完了すると、継続課金ライセンスの次回支払い日（_About panel_ に表示）までは、再度オンラインにする必要はありません。月額の継続課金ライセンスには通常、短い猶予期間（7日間）がありますが、有料機能を途切れずに利用できるように、支払い日を過ぎたらできるだけ早くオンラインにしてください。

猶予期間は、更新や支払いに関する問題に対応するためのものです。有料期間の終了後にキャンセルされたライセンスや、意図的に一時停止されたライセンスは延長されません。支払いに失敗した場合は、有料機能を継続して利用できるように、猶予期間内にカード情報を更新してください。

現在のライセンスティアもトップバーに表示されます。ティア名、またはライセンス未認証時の _FREE MODE_ をクリックすると、About panel が開きます。
