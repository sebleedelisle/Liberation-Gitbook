# ✅ ライセンスの仕組み

コンピューターに Liberation をインストールすると、_demo mode_ で起動します。使用を開始するには、インストールを_認証_する必要があります。

{% hint style="info" %}
_demo mode_ では、レーザーハードウェアへの出力以外に制限はありません。ショーの練習、タイムラインの作成、ファイルの保存と読み込みを制限なく行えます。Liberation は _demo mode_ のまま何台のコンピューターにでもインストールできます。レーザーを使用する準備ができた時点で、インストールを認証すれば十分です。
{% endhint %}

armed にできるレーザーの数に加えて、DMX フィクスチャも使用できます。詳しくは [DMX](../dmx-control/) を参照してください。

また、サブスクリプションプランに応じて、Liberation を認証できるコンピューターの台数にも上限があります。

{% hint style="danger" %}
各ライセンスは、1人のユーザーが同時に1つのショーで使用するためのものです。ユーザー1人につき複数のインストールを持てるのは、認証済みの予備コンピューターを少なくとも1台、**必ず**用意しておくべきだからです。
{% endhint %}

### インターネット接続

Liberation のライセンス管理と認証はインターネット経由で行われます。アプリがサーバーに接続し、インストールをサブスクリプションに関連付けます。

インストールが認証されると、次回サブスクリプションが更新されるまでオンラインである必要はありません。更新日は _About panel_ で確認できます。短い猶予期間（月額サブスクリプションでは7日間）がありますが、認証が更新され有効な状態を維持できるよう、更新日後はできるだけ早くオンラインにすることをおすすめします。
