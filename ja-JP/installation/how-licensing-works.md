# ✅ ライセンスの仕組み

コンピューターに Liberation をインストールすると、_認証_するまで _demo mode_ で動作します。

{% hint style="info" %}
_demo mode_ では、レーザーハードウェアへの出力以外に制限はありません。ショーの練習、タイムラインの作成、ファイルの保存と読み込みを制限なく行えます。Liberation は _demo mode_ のまま何台のコンピューターにもインストールできます。レーザーを使用する準備ができたときにだけ、インストールを認証すれば十分です。
{% endhint %}

アームできるレーザーの数に加えて、DMXフィクスチャも使用できます。詳しくは [dmx-control](../dmx-control/) を参照してください。

また、Liberation を認証できるコンピューターの台数は、サブスクリプションのティアによって制限されます。

{% hint style="danger" %}
各ライセンスは、1ユーザーが同時に1つのショーで使用するためのものです。ユーザー1人につき複数のインストールを持つことができます。少なくとも1台の予備の認証済みコンピューターを、**常に**すぐ使える状態で用意しておくべきだからです。
{% endhint %}

### インターネット接続

Liberation のライセンス管理と認証はインターネット経由で行われます。アプリがサーバーに接続し、インストールをサブスクリプションに関連付けます。

インストールが認証されると、サブスクリプションが更新されるまで再度オンラインにする必要はありません。更新日は _About panel_ で確認できます。短い猶予期間（月間サブスクリプションの場合は7日間）がありますが、認証が更新され有効な状態を保てるよう、更新日後はできるだけ早くオンラインにすることをおすすめします。
