# ✅ ライセンスのアップグレード / ダウングレード

いつでも上位プランへアップグレードできます。現在の支払い済み期間の残り分は一部返金され、新しいライセンス階層はすぐに開始されます。

ダウングレードもいつでも可能ですが、変更が有効になるのは現在の支払い済み期間の終了後です。

支払いの一時停止が予約されている間、またはライセンスが一時停止中の場合は、プランを変更できません。まず一時停止の予約をキャンセルするか、ライセンスを再開してから、プランを変更してください。

Web サイトにログインし、[_Your licences_](https://liberationlaser.com/account/my-products) ページを開きます。変更したいライセンスを選択して、_UPGRADE / DOWNGRADE_ をクリックします。すると、そのライセンスで利用可能なオプションが表示されます。

<figure><img src="../.gitbook/assets/account-change-licence-plan.png" alt=""><figcaption></figcaption></figure>

該当する _PREVIEW_ ボタンをクリックすると、変更後の支払い内容を確認できます。変更内容に問題がなければ、_Confirm licence change_ をクリックして手続きを完了します。
