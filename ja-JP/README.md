# ✅ Liberation へようこそ



<figure><img src=".gitbook/assets/LiberationTease.jpeg" alt=""><figcaption></figcaption></figure>

Liberation は、レーザーを扱うための新しいワークフローを提供します。**モダン**で**効率的**、**Mac と PC**に対応し、現在のイベント制作現場で実際に求められる使い方を中心に設計されています。すばやく表現力のある制作ツール、内蔵された安全機能、そして流れがよく直感的で、将来を見据えたワークフローを統合し、レーザーソフトウェアを 21 世紀のものにします。

Liberation は、同じ系統のオープンプロトコル controller を使用する、成長中のレーザー制御ソフトウェアエコシステムの一部です。TouchDesigner、MadLaser、LaserShowGen、Liberation のどれを使っていても、特定のメーカーに縛られることはありません。リグ、予算、アイデアに合った hardware を選択できます。

何よりも、Liberation は表現力のあるクリエイティブな楽器のように使えることを目指して作られています。習得しやすく、使っていて楽しく、単一のコンピューターから、大規模なフェスティバル、インスタレーション、アリーナツアーにも確実に対応できる十分な能力を備えています。

このガイドでは、インストールと安全の基本から、Clip の作成、zone のマッピング、フルショーの運用まで、Liberation に慣れるための手順を説明します。重要な概念とワークフローに重点を置いています。Liberation は操作の妨げにならないように設計されているため、ソフトウェアとの格闘に時間を使うのではなく、制作により多くの時間を使えます。

## このガイドの対象者

* 安全を省略せずに、明確な始め方を知りたい新規ユーザー。
* 会場設営やリハーサル中に、信頼できるリファレンスが必要なショーオペレーター。
* Clip Editor、MIDI、DMX、canvas ツールなどの高度な機能を試したいユーザー。

## ガイドの使い方

* hardware を接続する前に、まず [**重要：最初にお読みください**](important.md) を読み、重要な安全上の注意とライセンスに関する情報を確認してください。
* すばやく出力を始めたい場合は [**クイックスタートガイド**](getting-started.md) に沿って進め、その後、より詳しい設定について [**ハードウェア**](hardware/) と [**設定**](setting-up/) を確認してください。
* セクション見出しから目的のトピックへ直接移動できます（例：[**Clips と Clip Deck**](clips/)、[**Outputビュー**](output-view/)、**など**）。正確な情報が必要なときは、[**リファレンス**](reference/) または [**トラブルシューティング**](troubleshooting/) に戻って確認してください。
