# ✅ Liberationへようこそ



<figure><img src=".gitbook/assets/LiberationTease.jpeg" alt=""><figcaption></figcaption></figure>

Liberationは、レーザーを扱うための新しいワークフローを提供します。**モダン**で、**効率的**、**Mac and PC**に対応し、現代のイベント制作の現場で求められる実用的なニーズを中心に設計されています。高速で表現力のある制作ツール、組み込みの安全機能、スムーズで直感的なワークフローをひとつにまとめ、将来を見据えたレーザーソフトウェアとして、レーザー制作を21世紀の環境へ引き上げます。

Liberationは、同じオープンプロトコル系コントローラーを使用する、成長中のレーザー制御ソフトウェアエコシステムの一部です。TouchDesigner、MadLaser、LaserShowGen、Liberationのいずれを使用していても、特定のメーカーに縛られることはありません。自分のリグ、予算、アイデアに合ったハードウェアを選べます。

何よりも、Liberationは表現力の高いクリエイティブな楽器として作られています。習得しやすく、使っていて楽しく、要求の厳しいフェスティバル、インスタレーション、アリーナツアーにも、1台のコンピューターから高い信頼性で対応できる十分なパワーを備えています。

このガイドでは、インストールと安全の基本から、Clipの作成、ゾーンのマッピング、フルショーの実行まで、Liberationを安心して使いこなすための流れを説明します。重要なコンセプトとワークフローに焦点を当てています。Liberationは操作の妨げにならないよう設計されているため、ソフトウェアとの格闘に時間を取られず、制作により多くの時間を使えます。

## このガイドの対象読者

* 安全を省略せず、明確な出発点を求めている新規ユーザー。
* 会場のセットアップやリハーサル中に、信頼できるリファレンスが必要なショーオペレーター。
* Clip Editor、MIDI、DMX、Canvasツールなどの高度な機能を試したい探究心のあるユーザー。

## ガイドの使い方

* ハードウェアを接続する前に、[**重要！最初にお読みください**](important.md)から始め、重要な安全およびライセンスに関する注意事項を確認してください。
* すぐに出力を開始したい場合は、[**クイックスタートガイド**](getting-started.md)に従ってください。その後、より詳しい設定については[**Hardware**](hardware/)と[**Setting up**](setting-up/)を参照してください。
* セクション見出しを使って、目的のトピック（例：[**Clips & Clip deck**](clips/)、[**Output view**](output-view/)、**etc**）へ直接移動できます。正確な答えが必要な場合は、[**Reference**](reference/)または[**Troubleshooting**](troubleshooting/)に戻って確認してください。

## 表記とヒント

* ステータス絵文字は完成度を示します：✅ 完了、🟧 一部完了、🟩 画像が必要、◼️ プレースホルダー。
* リンクは現在のリリースを対象としています。古いビルドを使用している場合、スクリーンショットやメニューが多少異なることがあります。
* ハードウェア手順では、安全優先のデフォルトを明示しています。インターロックを絶対にバイパスせず、ビームを人に向けないでください。

行き詰まった場合は、サポートに問い合わせる前に、まず[**FAQ**](faq.md)を確認し、その後[**Troubleshooting**](troubleshooting/)セクションで該当する対処方法を探してください。
