# 🟩 カラー設定と HSB

Liberation では、色は RGB ではなく HSB として定義されます。最初は慣れないかもしれませんが、慣れるとこちらのほうがずっと直感的な仕組みだと感じられるはずです。

{% hint style="info" %}
RGB を使いたい場合は、任意のカラー設定にある色の四角をクリックしてください。カラーエディターパネルが開き、RGB と HSB のオプションを使用できます。
{% endhint %}

### HSB - Hue、Saturation、Brightness

#### Hue

色相は 0〜255 の範囲です。0 は赤で、値を大きくしていくと、オレンジ、黄色、緑、シアン、青、紫、マゼンタと、虹のすべての色相を通り、255 で再び赤に戻ります。

これはループになっているため、三角波を使ってすべての色を循環させることができます。

#### Saturation

この設定は、色の彩度、つまり鮮やかさを調整します。言い換えると、どれだけ _色味があるか_ を表し、範囲は 0〜255 です。Saturation を 0 にするとグレーになり、255 にすると完全な虹色になります。中間あたりでは、淡く薄いパステル調の色になります。

{% hint style="info" %}
colour という単語に母音が 1 つ多いことについて、米国の皆さんにはお詫びします。Liberation はイングランドで作られているため、デフォルトは British English です。将来的には、フランス語、スペイン語、ドイツ語、イタリア語、簡体字中国語、そしてもちろん US English への翻訳も提供したいと考えています。:innocent:
{% endhint %}

#### Brightness

おそらく最も理解しやすい設定です。0 は完全な黒、255 は最大輝度です。

### 使用例

#### レインボーサイクル：

_Brightness_ と _Saturation_ を 255 に設定します。_Sawtooth_ オシレーターを _Hue_ ソケットに接続し、範囲を 0〜255 に調整します。

#### 明るさのパルス：

_Sawtooth_ オシレーターを _Brightness_ ソケットに接続し、範囲を 255〜0 に調整します。さらに clamp の最小値と最大値を調整して、変化の持続時間を確認できます。その後、イージング機能を使ってアニメーションをさらに細かく調整できます。

#### ハードフラッシュ／ストロボ：

_Hue_ と _Saturation_ を使うか、カラーピッカーをクリックして色を選択します。_Square Wave_ オシレーターを _Brightness_ ソケットに接続し、範囲を 255〜0 に調整します。

#### カスタム色相範囲でのサイクル：

_Brightness_ と _Saturation_ を 255 に設定します。_Triangle Wave_ オシレーターを _Hue_ ソケットに接続し、範囲を調整します。

* 青からシアンにするには、70〜128 の範囲を使用します
* 赤から黄色にするには、0〜40 の範囲を使用します
* 赤からマゼンタにするには、255〜220 の範囲を使用します
