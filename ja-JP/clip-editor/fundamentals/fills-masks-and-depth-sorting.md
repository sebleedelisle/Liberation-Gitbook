# 🟩 フィル、マスク、デプスソート

### ストローク、フィル、マスク

一部の Creator ノードには _Fill state_ オプションがあります。ストローク（アウトライン）として描画したり、マスク（下にあるものを覆う）として描画したり、その両方を使ったりできます。

シェイプをマスクとしてレンダリングすると、黒で塗りつぶされているように扱われ、その下にあるものは覆い隠されます。

{% hint style="info" %}
レーザーで線（または _stroke_）を描くのは比較的簡単です。線の始点から終点までレーザーをスキャンすれば、それが線になります。

ただし、塗りつぶされたシェイプは少し難しくなります。シェイプを色で塗りつぶしたい場合、線を描いて手作業でクロスハッチングし、埋めていくことはできますが、Liberation ではそれを自動では行えません（今のところ）。仮にできたとしても、その下にある他の線が透けて見えてしまいます。

ただし、シェイプを _黒_ で塗りつぶすことはできます。内部では、Liberation が黒で塗りつぶされたシェイプの下にあるコンテンツを取り除くための計算をすべて行っています。正直、かなり細かい処理です。

それでも非常にうまく動作し、黒で塗りつぶされたシェイプのように見せることができます。
{% endhint %}

### デプスソート

一部のシェイプは他のシェイプを _覆い隠す_ ことができるため、Liberation はそれらを奥行き順にソートする必要があります。デフォルトでは、要素は z 位置に基づいてデプスソートされます。同じ z 位置にある場合は、レイヤー位置に基づいてソートされます。このレイヤー位置は、各 creator 内の _MOVE TO FRONT_ および _MOVE TO BACK_ ボタンで変更できます。
