# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_Parameter Control_ は、Clip Editor の外部から、エフェクトや Clip 内の各種設定を制御するための方法です。

* **range min / range max** - パラメーターのレベルが割り当てられる最小値と最大値です。
* **level** - このパラメーターのデフォルト設定です。外部から変更できる値です。手動で調整して、どのような効果になるか確認できます。
* **parameter number** - 各 _parameter control_ には番号を割り当てることができ、この番号は Clip の外部にあるコントロールに反映されます。同じ番号の parameter control を複数設定することもでき、その場合はすべて同時に調整されます。

{% hint style="warning" %}
APC40 および対応するライブコントローラーは、デフォルトで各エフェクトスロットの parameter 1 を制御します。エフェクトおよび Clip 内のすべてのパラメーターは、右クリックメニューからも利用できます。
{% endhint %}
