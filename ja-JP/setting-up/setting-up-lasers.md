# ✅ レーザーセットアップ手順の概要

### レーザーを安全に起動するための手順概要

このマニュアルは、正式なレーザー安全講習の代わりになるものではありません。公共の場でレーザーを使用する前には、必ず適切な講習を受ける必要があります。地域によっては追加の法的要件がある場合もありますが、いずれの場合も、安全性とプロフェッショナリズムに関するベストプラクティスを常に遵守してください。

PLASA では、一般的にベストプラクティスとして受け入れられているレーザー安全ガイドを無料でダウンロードできます: [https://www.plasa.org/guidance-for-display-lasers/](https://www.plasa.org/guidance-for-display-lasers/)

使用前に、レーザーに伴う安全上の影響を必ず理解してください。

#### はじめに

このページでは、レーザーを安全に電源投入する手順の全体像を説明します。各手順を具体的にどのように行うかは、このセクションの後半で詳しく説明しますが、まずは全体像を理解するためにこのページを確認してください。また、レーザーをセットアップするたびに、後から参照用として戻ってくることもできます。

### ハードウェアのセットアップ:

1. レーザーの**アパーチャーカバーを閉じる**
2. **レーザーを確実に設置**し、正しい方向に向ける
3. **ストップボタンを接続**する
4. **レーザーコントローラーをコンピューターに接続**する
5. レーザーの**電源を入れる**

### Liberation のセットアップ:

1. **すべてのレーザーをディスアーム**し、Liberation でコントローラーを検出して接続する
2. **&#x20;**_**Global Brightness**_**&#x20;設定を 0 まで下げる**（アイコンバーのスライダー、または APC40 の _Master Fader_ を使用します）
3. **レーザーをアームする** - アパーチャーカバーを閉じたまま、現在アクティブな Clip がないことを確認してレーザーをアームします（_Laser Overview_ パネルの _Arm_ ボタンを使用します）
4. **テストパターンをオンにする**（アイコンバーの ☒ ボタンを使用し、パターン 1、十字入りの緑の四角を選択します）
5. **出力ゾーンを調整する** - 最も安全と思われるゾーンのサイズと位置を見積もります（例: 天井近くの高い位置に設定するなど。ただし、これは使用環境によって異なります）
6. **レーザーが動作していることを確認する** - アパーチャーウィンドウの奥に光が見えるまで、明るさをゆっくり上げます。その後、明るさを再び 0 まで下げます。
7. **ストップボタンをテスト**し、押したときにすべてのレーザー出力が消えることを確認する

### レーザー出力の開始

1. **曝露エリアを空ける** - 誰もレーザーに曝露されないことを確認し、レーザーのセットアップ中は曝露エリアに入らないよう、すべての関係者に周知します。（カメラやプロジェクターにはカバーをかけるか、レンズキャップを取り付けてください）
2. **アパーチャーカバーを開ける** - 出力方向の横に立ち、出力から離れた状態で、アパーチャーカバーを下にスライドします。ゾーンが高い位置にある場合は、部分的に閉じたままにしてもかまいません。
3. **レーザーがかろうじて見える程度まで明るさを上げる** - ゾーンを確認するために必要な最小限の明るさにします
4. **ゾーンを調整する** - 一般の人が立ち入れるエリアから見て床面から 3m 以上の位置になるように、またレーザーが他の一般立入可能エリアに届かないように、ゾーンのサイズ、形状、位置を設定します
5. **物理マスキングを追加する** - アパーチャーカバーや黒色のフォイルテープを使用して、目的のゾーン外を物理的にマスクします。レーザーハードウェアやソフトウェアはいずれも故障する可能性があるため、これは非常に重要です。
6. **ソフトウェアマスクを追加する** - Liberation 内のソフトウェアマスクは、カメラやプロジェクターの保護に使用できますが、人を保護する目的で物理マスキングの代わりに使用しては**絶対にいけません**。ソフトウェアやハードウェアに完全なものはないため、ソフトウェアマスクを使用する前にリスクを必ず理解してください。

{% hint style="warning" %}
このガイドは屋内でのセットアップを前提としていることに注意してください。屋外で作業する場合は、航空機の安全を確保するために追加の手順が必要です。これには以下が含まれますが、これらに限定されません:

* FAA や CAA などの航空当局から必要な許可を取得する
* 近隣の空港や飛行場と連携する
* 公開されているフライトレーダーを確認し、航空機を監視するスポッターを配置する

安全基準値を大きく下回るレーザーであっても、パイロットに壊滅的な注意散漫を引き起こす可能性があります。

空域にレーザーを向ける前に、必要な資格、ライセンス、許可を取得していることを確認してください。
{% endhint %}
