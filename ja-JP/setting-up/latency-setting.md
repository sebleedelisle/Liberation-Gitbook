# ✅ レイテンシー設定

_Settings_ パネル（_Liberation->Settings_ または CMD/CTRL ,）に、_Latency(ms)._ というラベルのスライダーがあります。（古いバージョンの Liberation では Laser Overview パネル内にありました）

{% hint style="info" %}
デフォルトのレイテンシー設定は 150ms で、ほとんどの場合は問題ありません。ただし、ネットワークに問題がある場合は、値を大きくしてみるとよいでしょう。
{% endhint %}

### 詳しい説明

この「フレームレイテンシー」設定は、Liberation がフレームを生成してから、レーザーがその出力を開始するまでの最大時間です。この値を大きくすると、Liberation とレーザー出力の間にわずかな遅れを感じる場合があります。

フレームレイテンシーを長くする利点は、Liberation ができるだけ早くレーザーコントローラーのバッファーにコンテンツを蓄積できることです。ネットワークが混雑している場合でも、コントローラーがポイント不足になりにくくなります。

これは通常、ネットワーク DAC にのみ関係します。100ms に設定すると、速度とネットワーク遅延への保護のバランスがよくなります。非常に安定したネットワークを使用している場合は、50ms まで下げられるはずです。&#x20;
