# 🟩 ビームゾーンの並べ替え

ビームゾーンは作成した順番で表示されますが、作成後に並べ替えることができます。_View -> Beam Zone Organiser_ を開きます。

ビームゾーンの一覧と、それぞれが接続されているレーザーが表示されます。ゾーンをドラッグ＆ドロップして希望の順番に並べ替えるか、_Autosort by laser_ をクリックして、所属するレーザーごとに連続するよう自動で並べ替えます。

{% hint style="info" %}
ビームゾーンにカスタム名を付けていない場合、_Beam 1, Beam 2,_ などのデフォルト名が使用されます。並べ替えると、並び順に応じてデフォルト名も更新されます。混乱しやすいため、ゾーンを並べ替える際は注意してください。
{% endhint %}
