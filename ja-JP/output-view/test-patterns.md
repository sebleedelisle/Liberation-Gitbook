# 🟩 テストパターン

ゾーンの編集や調整を行うときは、テストパターンを出力すると便利です。上部のアイコンバーにあるトグルボタンを使うと、有効になっているすべてのゾーンにテストパターンを表示できます。その右側にあるスライダーでテストパターンを選択します。

現在ゾーニングしているレーザーにだけテストパターンを表示したい場合は、_Output_ ビュー左側のツールバーにあるトグルボタンを使用します。その下にある番号をクリックまたはドラッグして、テストパターンを選択します。

### テストパターン一覧

以下はテストパターンの全一覧です。特に記載がない限り、各パターンは各ゾーンの出力範囲全体を埋めます。

1. デフォルトパターン - 十字線付きの緑色の長方形です。低輝度でも見やすいように緑色を使用しています。
2. レジストレーションパターン - 青と緑の円に、白と赤のグリッドを組み合わせたパターンです。円の形を使って正しいアスペクト比を合わせられるため、建築物へのマッピングに便利です。
3. 水平の緑線 - 等間隔に配置された 4 本の水平線です。
4. 垂直の緑線 - 等間隔に配置された 4 本の垂直線です。

カラ―キャリブレーション用パターン 5〜8。これらはレーザー設定内のカラー補正システムと併用できます。[カラーキャリブレーション](../advanced/colour-calibration.md) を参照してください。これらのパターンは出力範囲全体を埋めません。

5. カラ―キャリブレーションパターン - 赤。
6. カラ―キャリブレーションパターン - 緑。
7. カラ―キャリブレーションパターン - 青。
8. カラ―キャリブレーションパターン - 白。

{% hint style="danger" %}
**注意** - テストパターン 9 は非常に危険な場合があるため、そのゾーンが安全であることを確認してから使用してください。明るく動かないスポットは、表面を焦がすおそれがあります。空に向けて使用する場合は、必要なライセンスや許可を取得し、航空機がないか常に確認してください。
{% endhint %}

9. 明るい白点 - 各コーナーにフルパワーの点を 1 つずつ表示します。最も視認性の高いパターンなので、明るい場所や空に向けたゾーニングに適しています。
10. スキャンする白線 - ゾーン全幅を左右にスキャンする垂直の白線です。ゾーン全体をカバーするため、マスキングのテストや、ミラーボールなどの危険物を確実に避ける確認に適しています。
11. 白い長方形 - 明るい白色の長方形です。
