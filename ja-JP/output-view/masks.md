# 🟩 Masks

{% hint style="danger" %}
**安全に関する注意**

ソフトウェア上の masks だけを、レーザー出力から人を保護する唯一の手段にしてはいけません。レーザーと、人が曝露する可能性のある場所との間には、必ず物理的なバリアや遮蔽を設置してください。このシステムは二次的な冗長手段、または安全上重要ではない用途（カメラやプロジェクターの保護など）として使用してください。ただし、どのようなソフトウェアでも故障する可能性があることを忘れないでください。
{% endhint %}

***

特定の場所でレーザー出力を除去（または低減）するために、1つ以上の仮想 masks を追加できます。

ツールバーのアイコンをクリックすると mask を追加できます。Output view に赤い四角形が表示され、これが新しい mask になります。クリックしてドラッグすると移動でき、角をドラッグするとサイズを変更できます。均一でない形状にしたい場合は、角をドラッグしながら `Alt / Option` を押します。（一度均一でない形状にすると、その後は `Alt / Option` キーを押す必要はありません。）

mask を右クリックすると、追加設定が表示されます。

* Reduction percentage - mask 内でのレーザーの出力低減率を表します。デフォルトでは 100% に設定されており、つまり mask 内は完全にブラックアウトされます。ただし、任意の量だけレーザー出力を下げることもできます。完全な低減でない場合、mask の色はオレンジに変わります。
* DELETE MASK ボタンも、この右クリックメニュー内にあります。
