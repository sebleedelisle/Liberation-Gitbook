# 🟩 マスク

{% hint style="danger" %}
**安全に関する注意**

ソフトウェアマスクを、レーザー出力から人を保護する唯一の手段として使用することは決してしないでください。レーザーと、人が被曝する可能性のある場所との間には、必ず物理的なバリア／遮蔽を設けてください。このシステムは、二重化のための補助的な手段、または安全上重要ではない用途（カメラやプロジェクターの保護など）として考えてください。ただし、どのようなソフトウェアでも故障する可能性があることを忘れないでください。
{% endhint %}

***

特定の場所のレーザー出力を取り除く（または低減する）ために、1つ以上の仮想マスクを追加できます。

ツールバーのアイコンをクリックするとマスクを追加できます。Outputビューに赤い四角形が表示され、これが新しいマスクになります。クリックしてドラッグすると位置を移動でき、角をドラッグするとサイズを変更できます。均一ではない形状にしたい場合は、角をドラッグしながら `Alt / Option` を押します。（マスクが均一でない形状になった後は、`Alt / Option` キーを押し続ける必要はありません。）

マスクを右クリックすると、追加設定が表示されます。

* Reduction percentage - マスク内のレーザーの出力低減率を表します。デフォルトでは100%に設定されており、つまりマスク内は完全にブラックアウトされます。ただし、任意の量だけレーザー出力を下げることもできます。完全な低減ではない場合、マスクの色がオレンジに変わります。
* DELETE MASK ボタンも、この右クリックメニュー内にあります。
