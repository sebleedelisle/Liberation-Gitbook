# 🟩 ゾーン

ほとんどのプロジェクトで主に使用するゾーンは _Beam zone_ です。これは、空中のアトモスフェリックなビームエフェクト用に設計されたゾーンです。もう 1 つのゾーンタイプは _Canvas zone_ です（[graphics-and-the-canvas-system](../graphics-and-the-canvas-system/) を参照）。

{% hint style="danger" %}
**警告 - レーザー動作中にゾーンを移動するときは、細心の注意を払い**、明るさを可能な限り低くしてください。レーザーを安全に有効化し、ゾーニングするための詳しい手順については、[setting-up-lasers.md](../setting-up/setting-up-lasers.md) を参照してください。
{% endhint %}

ゾーンはマウスでクリックしてドラッグできます。テストパターンをオンにすると、そのゾーンがどこに出力されるか確認できます。

{% hint style="info" %}
矢印キーを使うと、現在選択中のゾーンまたはポイントを**少しずつ移動**できます。`Shift` キーを押しながら操作すると、より大きなステップで移動します。
{% endhint %}

{% hint style="info" %}
便利なヒント：複数のレーザー間でゾーン設定をすばやくコピーできます。[copy-laser-settings.md](../setting-up/copy-laser-settings.md) を参照してください。
{% endhint %}

### 新しい Beam zone を追加する

ツールバー上部の _Add a new beam zone_ ボタンをクリックすると、新しいゾーンが表示されます。Beam zone は追加した順に並びますが、並べ替えることもできます。[re-ordering-beam-zones.md](re-ordering-beam-zones.md) を参照してください。

### 既存の Canvas zone を追加する

_Add existing canvas zone_ ボタンをクリックすると、利用可能な Canvas zone の一覧が表示され、このレーザーで使用するかどうかをオン／オフできます。[graphics-and-the-canvas-system](../graphics-and-the-canvas-system/) を参照してください。

### ゾーンのシェイプタイプ

ゾーンのシェイプタイプは 3 種類あります。

* **Quad** - デフォルトの長方形ゾーンシェイプです。均一な形状（軸に沿った形状）にも、歪ませた形状にもできます。大きめの長方形ゾーンや、パース補正が必要な Canvas zone に適しています。
* **Line/Curve** - 2 点以上のポイントと太さで定義するゾーンです。細いゾーンや、バルコニー、橋、その他の曲線形状でビームを終端させる場合に最適です。
* **Segmented** - 小さな Quad に分割できるゾーンです。建築物へのマッピングに適しています。

任意のゾーンを右クリックすると、その設定を開けます。この右クリックメニューでは、次の操作ができます。

* ゾーン名を変更する（特にゾーンが多い場合、clip deck で識別しやすくなります）
* ゾーンを有効／無効にする
* 位置をロックする
* シェイプタイプを変更する
* デフォルト位置にリセットする
* シェイプタイプ固有の設定にアクセスする
* 削除する
* _Alt Zone_ を追加する（[alt-zone-system.md](alt-zone-system.md) を参照）

{% hint style="danger" %}
**警告 -** レーザーがアクティブな状態でゾーンタイプを変更するときは、十分に注意してください。ゾーンはそのシェイプで最後に使用した位置／サイズに戻るため、出力が突然変わる場合があります。ゾーンタイプを変更する前に、レーザーをオフにすることをおすすめします。
{% endhint %}

### Quad ゾーンシェイプ

Quad の各コーナーはマウスで移動できます。`Alt / Option` キーを押しながらコーナーをクリックすると、他のコーナーとは独立して移動でき、Quad を歪ませることができます。一度 Quad を歪ませると、すべてのコーナーを自由に移動できるようになります。

右クリックメニューの _REMOVE DISTORTION_ ボタンを使うと、歪みを解除して軸に沿った長方形に戻せます。

#### パース補正

このオプションは右クリックメニューのトグルボタンで設定でき、歪ませ方を決定します。ビームではオフのままにしておくのが最適ですが、このゾーンで平面にグラフィックを投影する場合はオンにすると、出力にパース補正が適用されます。

{% hint style="info" %}
_Perspective correction_ がオフの場合、コンテンツは _bi-linear interpolation_ によって歪められます。つまり、コンテンツは Quad 全体に均等に配置されます。そのため、ビームに最適です。

Perspective correction をオンにすると、短縮遠近を補正するパースワープによってコンテンツが歪められます。たとえば、斜めの角度から壁にグラフィックを投影する場合、これを使って出力の歪みを補正し、投影の歪みを修正できます。
{% endhint %}

### Line / Curve ゾーンシェイプ

Line / Curve ゾーンシェイプは、最近のショーで私がよく使う選択肢になっています。Beam zone のデフォルトにしてもよいと言えるかもしれません。

多くの場合、会場の扱いにくい細いスペースや、建物の窓の間に収めるために、ゾーンを細くする必要があります。Quad の 4 つのコーナーが非常に近い状態で調整するのは、とても細かくて面倒だと感じました。そこで Line / Curve ゾーンが生まれました。

直線の場合は、必要なのは 2 つのポイントだけです。その後、右クリックメニューで _Zone thickness_ を調整します。シンプルなゾーンを作成する最速の方法です。

線上を `Alt / Option` キーを押しながらクリックすると、ポイントを追加できます。これらのポイントは自動的にスムージングされ、流れるような形状になります。_Smooth level_ を調整すると、折れや不自然な部分をならすことができます。

ポイントを `Alt / Option` キーを押しながらクリックすると、そのポイントを削除できます。

また、ベクターグラフィックアプリ（Inkscape、Illustrator など）に慣れている場合は、_Manually adjust bezier curves_ オプションを使うことで、すべての制御点を細かく調整できます。

### Segmented ゾーンシェイプ

この分割ゾーンでは、非常に細かな補正が可能です。複雑な形状にマッピングする場合に便利です。右クリックメニューの + ボタンと - ボタンを使って、分割を追加または削除できます。

### 別のゾーンに完全に覆われているゾーンを編集する方法

上にあるゾーンを右クリックし、南京錠ボタンをクリックしてロックします。これで、下にあるゾーンを編集・調整できるようになります。

Output に Beam zone を追加すると、clip deck のクリップに追加できるようになります。
{% endhint %}
