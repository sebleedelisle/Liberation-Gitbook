# 🟩 概要

### クリップとは？

Liberation 内のすべてのレーザーコンテンツは、クリップ内に保存されます。

クリップには、空間的なビームエフェクトやグラフィックアニメーションを含めることができ、通常はループ再生されるサイクルです。任意のゾーン（または Canvas Target Area）へ送ることができ、Clip Deck 内のクリップボタンでトリガーします。

クリップの内容は、柔軟なノードベースのクリップエディターシステムで作成します。（詳しくは [clip-editor-intro.md](../clip-editor/clip-editor-intro.md "mention") を参照してください）

### Clip Deck

<figure><img src="../.gitbook/assets/clips-clip-deck" alt=""><figcaption><p>デフォルトの Clip Deck</p></figcaption></figure>

Clip Deck は、画面上に表示されるクリップのグリッドです。通常、表示されている範囲よりもさらに横方向に広がっています。

Clip Deck を左右にスクロールするには、次の方法を使用します。

* 左右の矢印キー
* トラックパッドの2本指スワイプ（または、対応しているマウスの場合は左右スクロール）
* APC40 の _Cue Level_ スクロールホイール

Clip Deck を1ページ分ずつ左右にスクロールするには、次の方法を使用します。

* `Cmd / Ctrl + Left / Right Arrow`
* APC40 の _Device <- and ->_ ボタン

#### ミニクリップビジュアライザー

Clip Deck の上部には、Clip Deck 全体を小さく表した表示があります。点灯している部分は、現在表示されているクリップを示しています。
