# 🟩 概要

### Clipとは？

Liberation内のすべてのレーザーコンテンツは、Clipの中に保存されます。

Clipには空間的なビームエフェクトやグラフィックアニメーションを含めることができ、通常はループするサイクルとして再生されます。任意のzone（またはCanvas Target Area）へ送ることができ、Clip Deck内のClipボタンを使ってトリガーします。

Clipの内容は、多機能なnodeベースのClip Editorシステムで作成します。（詳しくは[Clip Editorの概要](../clip-editor/clip-editor-intro.md)を参照してください）

### Clip Deck

<figure><img src="../.gitbook/assets/clips-clip-deck" alt=""><figcaption><p>デフォルトのClip Deck</p></figcaption></figure>

Clip Deckは、画面上に表示されるClipのグリッドです。通常、表示中の範囲よりもさらに外側まで広がっています。

Clip Deckを左右にスクロールするには、次の操作を使います。

* 左右の矢印キー
* トラックパッドの2本指スワイプ（マウスが対応している場合は、マウスの左右スクロールも使用できます）
* APC40の_Cue Level_スクロールホイール

Clip Deckを1ページ分ずつ左右にスクロールするには、次の操作を使います。

* `Cmd / Ctrl + Left / Right Arrow`
* APC40の_Device <- and ->_ボタン

#### ミニClipビジュアライザー

Clip Deckの上部には、Clip Deck全体を小さく表示したものがあります。明るく表示されている部分が、現在表示されているClipを表しています。
