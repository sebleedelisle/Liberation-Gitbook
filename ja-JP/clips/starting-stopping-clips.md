# ✅ クリップの開始／停止

{% hint style="info" %}
画面上のボタンでもクリップの開始と停止はできますが、APC40 MIDI コントローラーを使うことを強くおすすめします。Liberation はこのハードウェアで快適に動作するよう最適化されており、レーザーへの投資における重要な一部として考えるべきものです。
{% endhint %}

### クリップの開始と停止

クリップを開始するには、そのクリップのボタンを押します（画面上、または MIDI コントローラー上のボタン）。

クリップを停止するには、もう一度同じボタンを押します。

新しいクリップ（同じグループ内）を開始すると、最初のクリップは自動的に停止します。

他のクリップを停止せずに新しいクリップを開始するには、次のいずれかを行います。

* 新しいクリップを開始するときに `Shift` キー（または APC40 の Shift ボタン）を押す。または
* 現在実行中のクリップを押したまま、新しいクリップを開始する。

クリップグループは互いに独立しているため、あるグループでクリップを開始しても、他のグループのクリップには影響しません。詳しくは [groups.md](groups.md "mention") を参照してください。

### Flash mode

クリップが _flash mode_ に設定されたグループ内にある場合、動作が少し変わります。クリップボタンを押している間だけ、そのクリップは実行され続けます。ボタンを離すとすぐに停止します。（デフォルトでは、クリップグループ 3、赤色のグループが _flash mode_ に設定されています。）

### すべてのクリップを停止する

**実行中のすべてのクリップを停止**するには、**STOP** ボタンを押します。

{% hint style="info" %}
**STOP** ボタンを **2 回**押すと、クリップのフェードアウト時間をバイパスして、すぐにブラックアウトします。
{% endhint %}

特定のグループで実行中のすべてのクリップを停止するには、そのグループボタンを押してから **STOP** ボタンを押します。

1 つを除くすべてのクリップを停止するには、残したい実行中のクリップを押し、そのまま押した状態で **STOP** ボタンも押します。その後、クリップボタンを離します。（これは複数のクリップでも機能します。同時に押さえられる数だけ対応できます！）

### 現在選択されているクリップ

<figure><img src="../.gitbook/assets/clips-selected-active.png" alt="" width="269"><figcaption><p>現在実行中の 2 つのクリップです。白いアウトラインは、右側のクリップが<em>現在選択されているクリップ</em>であることを示しています。</p></figcaption></figure>

クリップが現在実行中の場合、そのクリップは画面上で点灯します（ミニクリップビジュアライザーの表示も点滅します）。また、最後に押したクリップの周囲に白いアウトラインが表示されていることにも気づくでしょう。これは、そのクリップが _現在選択されているクリップ_ であることを示します。

クリップを有効化せずに選択するには、クリップを `Alt / Option`-クリックします。複数のクリップを選択するには、`Alt / Option + Shift`-クリックを使用します。

Clip Deck 上でラッソをクリック＆ドラッグして、複数のクリップを選択することもできます。

{% hint style="info" %}
APC40 にも、選択操作に使う `Alt` ボタンと `Shift` ボタンがあります。
{% endhint %}
