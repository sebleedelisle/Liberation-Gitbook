# ✅ クリップをレーザーゾーンに割り当てる

<figure><img src="../.gitbook/assets/clips-zones" alt=""><figcaption><p>上部のゾーンボタンと、下部の Flip X / Flip Y ボタン。</p></figcaption></figure>

クリップの下には、ゾーンボタンの列が表示されます（画面上と APC40 の両方に反映されます）。デフォルトでは _BEAM 1_ から _BEAM 8_ までのラベルが付いています。これらのボタンを使って、_現在選択中のクリップ_ に対するゾーンのオン／オフを切り替えます。

#### ゾーンをスクロールする

ゾーンが 8 個を超える場合は、次の方法で左右にスクロールできます。

* `Shift + Left / Right Arrow`
* APC40 の _DEV ON/OFF_ ボタンと _DEV LOCK_ ボタン

### Flip X と Y

各ゾーンボタンの下には、_X_ と _Y_ のボタンがあります。これらは、それぞれのゾーンへの出力を水平方向および垂直方向に反転します。

{% hint style="info" %}
ミラー設定を変えるためだけに新しいゾーンを作成する必要はありません。X と Y の反転設定はこのクリップにのみ適用され、ゾーン自体の設定ではありません。
{% endhint %}

### 複数のクリップにゾーンをまとめて割り当てる

ゾーン内のすべてのクリップに対して、ゾーンの割り当てと X/Y の調整を行えます。詳しくは [Clip グループ](groups.md "mention") を参照してください。
