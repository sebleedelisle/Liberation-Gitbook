# 🟩 Clip Deckを整理する

### Clipの移動／複製

Clipを移動するには、クリックしてドラッグします。複数のClipが選択されている場合（[starting-stopping-clips.md](starting-stopping-clips.md "mention")を参照）、選択されたClipはすべて一緒に移動します。複製するには、`Alt / Option`を押したまま操作します。Clipを空いている場所にドロップします。

{% hint style="info" %}
多数のClipを選択する必要がある場合は、クリック／ドラッグで投げ縄選択し、そのままドラッグを続けながらClip Deckをスクロールできます（カーソルキー、APC 40、またはタッチパッド／マウスの横スクロールを使用）。投げ縄選択はClip Deckの端を越えて拡張されます。
{% endhint %}

### Clipの削除

1つのClipを削除するには、右クリックメニューを使用します。

複数のClipを削除するには、それらを選択し（`Alt / Option + Shift`クリック、または投げ縄選択）、Clip Deckの外へドラッグします。

### 空きスペースの追加

Clipボタンを右クリックし、_ADD EMPTY COLUMN TO RIGHT_ を押すと、Clip Deckに空の列を追加できます。

### 複数のClip Deckを結合する

既存のClip Deckをインポートし、現在のClip Deckの末尾に追加できます。メニュー _File -> Clips/Effects -> Append Clip Deck._ を使用します。

Clipを選択し（投げ縄選択、または`Alt / Option + Shift`クリック）、_File -> Clips/Effects -> Export Selected Clips._ を実行します。これにより、選択したClipだけを含むClip Deckがエクスポートされ、別のClip Deckに追加できるようになります。

### TimelineごとにClipを整理する

Timelineをインポートする際には、いくつかのClipオプションも使用できます。現在のClip Deckを置き換える、TimelineのClip Deck全体を追加する、またはそのTimelineで使用されているClipだけを追加する、のいずれかを選べます。

プロジェクトに複数のTimelineがある場合は、TimelineごとにClipを再整理できます。_Timeline selection_ パネルを開き、_ORGANISE CLIPS BY TIMELINE_ をクリックします。これで、すべてのClipがTimelineに基づいたグループで表示されます。1つのClipが複数のTimelineで使用されている場合は、各グループに複製されます。
