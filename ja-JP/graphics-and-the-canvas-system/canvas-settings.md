# 🟩 Canvas settings

左側ツールバー上部の _Canvas settings_ ボタンをクリックすると、_Canvas settings_ パネルが開きます。

### Width and Height

ドラッグ可能なスライダーを使って、キャンバス全体の幅と高さを変更します。これらの寸法は概念的にはピクセルサイズとして考えることもできますが、すべて相対的な値です。

スライダーをダブルクリックすると、値を入力できます。

{% hint style="info" %}
少し細かい技術的な補足です。レーザーはゾーンのサイズに対してではなく、キャンバスの解像度に対して相対的に動きます。これは、キャンバス内の大きなゾーンは、実際の投影でもより大きな範囲に投影される可能性が高いためです。

おそらく多くの場合は気にしなくてよい小さな違いですが、キャンバス内の各ゾーンでパフォーマンスと見え方を均一にしやすくなります。
{% endhint %}

### Zone brightness

Canvas view に表示されるピンク色のゾーン外枠の明るさを変更します。コンテンツを見やすくしたり、このビューが混雑しすぎるのを防いだりするのに役立ちます。
