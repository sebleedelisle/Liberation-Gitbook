# 🟩 Canvas view

上部ツールバーの _CANVAS_ ボタンをクリックして、_CANVAS_ ビューを選択します。

Canvas view は、キャンバス全体を表します。ここはコンテンツを配置して操作する領域です。このビューでは、キャンバスのどの部分をどのゾーンに送るかを選択したり、ガイド画像を追加したりできます。

### Canvas view 内を移動する

何もない場所（またはロックされたゾーン）をクリックしてドラッグすると、ビュー内を移動できます。

### Canvas view をズームイン／ズームアウトする

マウスホイールを使ってズームイン／ズームアウトします。

### ツールバーアイコン

ビューの左側にツールバーがあります。各アイコンにマウスカーソルを合わせると、その機能を確認できます。

* _Canvas settings_ パネルを開く
* 新しいキャンバスゾーンを追加する
* 新しいガイド画像を追加する
* グリッドへのスナップを切り替える
* グリッドサイズを調整する（0 が最小、8 が最大）。（グリッドサイズは 2 の累乗で大きくなるため、グリッドサイズを小さくしても、スナップ済みのポイントはすべて同じライン上に残ります）。
* グリッドを表示／非表示にする
