# 🟩 はじめに

近年、レーザーはサーフェスに映像を投影する用途よりも、空中にビームや空間演出を作り出す用途で使われることが一般的になっています。あえて言えば、レーザーでグラフィックスを投影することは、あまり格好よくないものと見なされることさえあります。その大きな理由は、プリセットとしてよく用意されている低品質なクリップアート風アニメーションや、扱いにくいコンテンツが広く使われてきたためです。

しかし実際には、レーザーグラフィックスは非常に柔軟で、幅広いクリエイティブな可能性を提供します。シンプルにきらめく企業ロゴ（誰もが一度は経験するものです）から、建築ファサード全体をマッピングした演出まで、さまざまな用途に対応できます。

マッピングされたグラフィックスとカスタムアニメーションを使用することで、制作ツールキットに追加できる新しいクリエイティブの可能性が大きく広がります。
