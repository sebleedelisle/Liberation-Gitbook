# ✅ APC40 リファレンス

<figure><img src="../.gitbook/assets/Liberation_APC40_MK2_4.png" alt=""><figcaption></figcaption></figure>

1. 8x5 Clip Deck
2. Zone On/Off
3. Effects 1-8 On/Off
4. Zone X/Y Flip
5. Effects 9-24 On/Off
6. Effect Faders 1-8
7. Global Brightness
8. Effect 1-8 Parameter Adjustment
9. Group Buttons
10. Stop All Clips（2回押すと、フェード設定を使わずにすべてのClipを即座に停止します）
11. Clip Deck Scroll
12. コンテキストに応じた Parameters panel のコントロール
13. Clip Page Left/Right
14. Zone Page Left/Right
15. Alt button（Clip選択に使用）
16. Shift button（複数選択やその他のオプションに使用）
17. Tempo Multiplier Enable/Disable
18. Tempo Multiplier Fader（0% - 200%）
19. Play / Pause
20. Toggle Record Timeline
21. Arm / Disarm（ArmするにはShiftを押す必要があります）
22. Tempo bar reset
23. Tap Tempo
24. Toggle Zone Delay（Shiftと組み合わせるとDelay/Chaseを切り替え、Altと組み合わせるとRetriggerを切り替えます）
25. Nudge tempo + / -
26. Tempo Scroll
27. Foot Pedal、Tap Tempo

Device Control ノブは、画面上の Parameters panel に表示されるコンテキストに応じたパラメーターと連動します。Clip が選択されている場合は、clip shift、zone delay、global spin、global scale を操作します。group button を押している間は、最初のコントロールがそのグループの fade in と fade out の時間に切り替わります。
