# ✅ キーボードショートカット

ショートカットは `Cmd / Ctrl` の形式で記載しています。macOS では `Cmd`、Windows では `Ctrl` を使用します。

| ショートカット                    | 操作                                                  |
| --------------------------------- | ----------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | すべてのレーザーをアームする                          |
| `Escape`                          | すべてのレーザーをディスアームする                    |
| `Tab`                             | 表示を切り替え、レーザーを順に切り替える              |
| `Shift + Tab`                     | 表示を切り替え、レーザーを逆順に切り替える            |
| `1-9`                             | レーザーへジャンプする                                |
| `Left / Right Arrow`              | Clip Deck を左 / 右へ移動する                         |
| `Cmd / Ctrl + Left / Right Arrow` | Clip Deck を 1 ページ左 / 右へ移動する                |
| `Shift + Left / Right Arrow`      | ゾーンを 1 ページ左 / 右へ移動する                    |
| `Space`                           | 再生 / 一時停止（_Timeline_ テンポソースが有効な場合） |
| `T`                               | タップテンポ                                          |
| `R`                               | 小節リセット                                          |
| `Y`                               | テンポを最も近い BPM に丸める                         |
| `Cmd / Ctrl + ,`                  | settings panel を開く                                 |
| `Cmd / Ctrl + F`                  | フルスクリーンモードを切り替える                      |
| `Cmd / Ctrl + L`                  | laser output settings panel を開く                    |
| `Cmd / Ctrl + K`                  | laser controller assignment panel を開く              |
| `Cmd / Ctrl + J`                  | 3D visualiser settings panel を開く                   |
| `Cmd / Ctrl + Shift + Plus (+)`   | インターフェースのスケーリングを大きくする            |
| `Cmd / Ctrl + Shift + Minus (-)`  | インターフェースのスケーリングを小さくする            |
| `Cmd / Ctrl + 0`                  | インターフェースのスケーリングをリセットする          |
| `Cmd / Ctrl + A`                  | Timeline: すべて選択                                  |
| `Delete`                          | Timeline: 選択したブロックを削除する                  |
| `Delete`                          | Clip editor: 選択したノードを削除する                 |
| `Cmd + Q` (macOS)                 | アプリを終了する                                      |
| `Alt + F4` (Windows)              | アプリを終了する                                      |
