# ✅ 出力が断続的に点滅する

_Laser Overview_ パネルを開き、問題が発生しているレーザーの横にある接続ライトを確認してください。

**接続ライトが常に緑色ではない場合：**\
ネットワークまたは CPU パフォーマンスに問題がある可能性があります。

**ネットワークパフォーマンス -**

* Wi-Fi ネットワークに接続していないことを確認してください。必ず有線接続を使用してください。OS が Wi-Fi よりも有線ネットワークを優先していることを確認するか、不明な場合は Wi-Fi を無効にしてください。
* ネットワークアダプターを確認し、別の USB-C アダプターも試してください。
* Windows ユーザーは、ネットワークドライバーを確認／更新し、適切なネットワークテストツールを実行してください。
* すべての配線、スイッチ、ルーターを確認してください。各ケーブルを順番に交換してテストしてください。
* スイッチ、ルーター、すべての Ether Dream を含むネットワーク機器をすべて再起動してください。

**CPU パフォーマンス**

古い、または低スペックのマシンを使用している場合、Liberation を実行するには処理速度が不足している可能性があります。アイコンバー右側のフレームレートインジケーターを確認してください。

そこには 2 つの数値があります。実際のフレームレートと目標フレームレートです。実際のフレームレートが 30 を下回る場合、問題が発生する可能性があります。

以下の対処が役立つ場合があります。

* 使用していないレーザーを削除します。たとえば、接続しているレーザーが 1 台だけなら、他のレーザーを削除してください。
* Output または Canvas ビューに切り替えます。
* 他のすべてのプログラムを閉じ、ネットワークファイアウォール設定を確認し、アンチウイルス、Dropbox などを終了します。
* ディスプレイ解像度を下げ、Liberation ウィンドウを小さくします。

これらを試しても解決しない場合は、コンピューターのアップグレードを検討してください。

***

**接続ライトが常に緑色の場合：**

ハードウェアの問題である可能性が高いです。これは本マニュアルの範囲外ですが、以下の対処を試すことができます。

* SFS（Scan Fail Safety）システムを無効にします。一部のレーザーには、スキャナーの動作が止まった場合、つまり強い静止ビームが出力される状態になった場合に出力を停止する機能があります。この機能はやや過敏、または不安定な場合があります。

{% hint style="danger" %}
Scan Fail Safety システムを無効にする場合は、十分に注意してください。強い静止ビームは発火や焼損の原因になることがあります。停止ボタンと消火器を必ず手元に用意してください。
{% endhint %}

* インターロックケーブルとシステムを確認します。
* コントローラーとレーザー間のすべてのケーブルを確認します。

[ILDA Gem](https://shop.stanwaxlaser.co.uk/ilda-gem-pocket-2020-718-p.asp) は、レーザーの問題をトラブルシューティングするうえで非常に役立つツールです。
