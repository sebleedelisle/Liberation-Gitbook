# 🟧 テンポ / 同期

音楽との同期は Liberation の基本要素です。テンポと拍を音楽に合わせられれば、すべてが同期して動作します。MIDI clock（または Ableton Link）を利用できる場合は、手動で同期する必要はありません。利用できない場合でも問題ありません。_Live_ tempo 機能を使って手動で合わせることができます。

音楽ソフトウェアや照明ソフトウェアの経験がある場合、この手順はなじみのあるものです。初めての場合は、ライブ本番の前に、自宅でシステムに慣れて練習しておくことをおすすめします。

## Tempo パネル

_Tempo_ パネルは常に画面上に表示され、同期に関するすべての設定が含まれます。上部には現在の小節/拍カウンターと、再生/一時停止、巻き戻し/早送りボタンを備えたトランスポートがあります。

その下にはビートマーカーがあります。これは拍に合わせて「パルス」する4つの四角です。この _beat marker_ は非常に便利な視覚表示で、_Live_ tempo システムを使用する際に常に参照することになります。

### テンポの設定

テンポの設定方法はいくつかあります。

* _Tempo_ スライダーをクリックしてドラッグ
* Shift を押しながら _Tempo_ スライダーをクリックしてドラッグし、0.1単位でテンポを変更
* _Tempo_ スライダーをダブルクリックして、数値を手入力
* APC40 の _Tempo_ ノブを使用（0.1単位で調整するには shift を押します）
* Tap Tempo

{% hint style="info" %}
Tempo は「beats per minute（1分あたりの拍数）」で定義され、従来のデフォルトは通常 120 です。
{% endhint %}

## Tap Tempo

拍に合わせて _TAP_ ボタンをクリックすると、テンポを自動設定できます。小節の開始位置は _RESET_ ボタンで設定します。

{% hint style="info" %}
Tap Tempo システムは、しばらくタップを休んだ場合や、数拍タップし損ねた場合も認識できます。倍の速さでタップし始めると、テンポを倍にしたいと判断します。半分の速さでタップした場合も同様です。

また、2人が同時にテンポをタップしている場合（例：1人がキーボード、もう1人が APC40）も判別できます。Liberation は重なったタップを平均化します。
{% endhint %}

#### キーボードコマンド：

T - tap tempo\
R - reset the bar\
Y - テンポを最も近い beats per minute に丸めます。

{% hint style="info" %}
最近の音楽の多くはデジタルで制作されているため、テンポは整数に丸められていることが多いです。タップしたテンポが近いと感じる場合は、Y キーを使う（または APC40 の tempo ノブを1「tick」動かす）ことで、整数に丸められます。
{% endhint %}

#### APC40 コントロール：

APC40 には専用の _TAP TEMPO_ ボタンがあります。また、接続したフットスイッチを使って足でタップすることもできます。

_TEMPO_ ノブを使って調整します。細かく調整するには、_TEMPO_ ノブの操作中に _SHIFT_ を押します。

_METRONOME_ ボタンを使って **reset the bar** します。（_METRONOME_ ボタンは拍に合わせて点灯します）

_TEMPO_ ノブを右または左に1「tick」回すと、**round the tempo** して整数の BPM 値に上下できます。

関連項目：[APC40 リファレンス](reference/apc40-reference.md "mention")

### テンポのナッジ

実際のテンポに十分近いと確信しているものの、少しずつずれていく場合は、_NUDGE_ ボタンで補正します。

Liberation のテンポが音楽より先行している場合は、再び合うまで _NUDGE -_ を長押しして一時的に遅くします。

Liberation のテンポが音楽より遅れている場合は、再び合うまで _NUDGE +_ を長押しして一時的に速くします。

{% hint style="info" %}
画面上の NUDGE ボタン、または APC40 の専用ボタンのどちらでも使用できます。
{% endhint %}

### Half time / double time

_÷2_ ボタンと _x2_ ボタンを使うと、テンポを恒久的に半分または2倍にできます。テンポマルチプライヤーとは異なり、これは現在のテンポに対する恒久的な変更です。

## Tempo Multiplier

_Tempo Multiplier_ システムでは、テンポを一時的に調整し、その後元のテンポに戻すことができます。

_TEMPO MULTIPLIER_ ボタン、または APC40 の _BANK_ ボタンを押して _Tempo Multiplier_ を切り替えます。画面上のスライダー、または APC40 の A-B スライダーで調整します。または、_25%, 50%, 100% 200%_ プリセットボタンを使用します。

## 外部テンポソース

### MIDI Clock

外部 MIDI clock 信号に同期するには、_MIDI CLOCK_ ラジオボタンを選択し、ドロップダウンメニューから MIDI デバイスを選びます。ドロップダウンメニューの横にある色付きステータスライトを確認してください。

* Green - MIDI clock 信号を受信中
* Orange - MIDI デバイスには接続可能だが、現在 clock 信号がない
* Red - MIDI デバイスに接続できない

{% hint style="info" %}
MIDI Clock は一連のフレーム（4分音符あたり24）を送信しますが、メッセージ内に位置データは含まれません。そのため、タイミングを維持するには役立ちますが、小節をリセットする必要がある場合があります。

Liberation の MIDI Clock テンポソースは **MIDI Machine Control (MMC)** メッセージにも反応します。そのため、clock ソースがこれらを送信している場合は、手動で小節をリセットする必要はありません。
{% endhint %}



### Ableton Link

Ableton Link と同期するには、テンポソースとして _ABLETON LINK_ を選択します。Liberation はローカルネットワーク上の Link セッションに参加し、他の Link 対応アプリから共有されるテンポと拍の位相に追従します。

Ableton Link は MIDI ポートを使用せず、絶対的な曲の位置情報も扱いません。Liberation の小節開始位置をショー内の特定のタイミングに合わせる必要がある場合は、小節リセット用のコントロールを使用してください。

### Timeline

各 timeline には独自のテンポがあり、固定値または _Tempo Map_ にできます。_Tempo Map_ を使用すると、timeline 内の特定の時点でテンポを調整できます。

_Timeline_ がテンポソースとして選択されている場合、timeline のテンポが使用されます。

{% hint style="info" %}
timeline は、_any_ のテンポソースと併用できます。クリックに合わせて演奏しないライブバンドの場合でも、timeline を手動で開始し、_LIVE_ テンポソースを使って同期を保つことができます。MIDI clock 信号がある場合は、それを使用することもできます。
{% endhint %}
