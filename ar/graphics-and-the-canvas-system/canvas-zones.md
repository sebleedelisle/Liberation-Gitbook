# 🟩 Canvas zones

يمكن رؤية canvas zone في Canvas view على شكل مستطيل بإطار وردي. تُستخدم لتحديد مساحة مستطيلة داخل Canvas وإرسال محتواها إلى ليزر واحد أو أكثر.

يمكنك النقر على zones وسحبها بالماوس. اسحب الزوايا لتغيير الحجم والشكل.

### إضافة canvas zone جديدة

انقر زر _Add a new canvas zone_ في شريط الأدوات.

### تعيين canvas zone إلى ليزر

انقر بزر الماوس الأيمن على zone، ثم بدّل أزرار الأرقام لاختيار الليزر أو الليزرات التي سيتم تعيين هذه canvas zone إليها.

{% hint style="danger" %}
تحذير - إذا كان الليزر في حالة armed، فقد يبدأ فجأة في عرض محتوى داخل canvas zone افتراضية. من الأفضل وضع الليزر في حالة disarmed قبل تعيين canvas zones إليه.
{% endhint %}

إذا انتقلت الآن إلى عرض _OUTPUT_ لذلك الليزر، فسترى canvas zone هناك، ويمكن تعديلها الآن تمامًا مثل أي beam zone. راجع [zones](../output-view/zones.md "mention").

{% hint style="info" %}
يمكنك أيضًا إضافة canvas zone إلى ليزر مباشرة من عرض _OUTPUT_ - انقر زر _Add existing canvas zone_ في شريط الأدوات الأيسر.
{% endhint %}

### قائمة النقر بزر الماوس الأيمن

انقر بزر الماوس الأيمن على zone، وستظهر لك خيارات لحذف هذه canvas zone أو نقلها إلى الخلف أو قفلها.
