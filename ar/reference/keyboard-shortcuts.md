# ✅ اختصارات لوحة المفاتيح

تُكتب الاختصارات بصيغة `Cmd / Ctrl`: استخدم `Cmd` على macOS و`Ctrl` على Windows.

| الاختصار                          | الإجراء                                                |
| --------------------------------- | ----------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | تنفيذ Arm لجميع أجهزة الليزر                                        |
| `Escape`                          | تنفيذ Disarm لجميع أجهزة الليزر                                     |
| `Tab`                             | تبديل طرق العرض والتنقّل بين أجهزة الليزر                 |
| `Shift + Tab`                     | تبديل طرق العرض والتنقّل بين أجهزة الليزر للخلف        |
| `1-9`                             | الانتقال إلى جهاز ليزر                                         |
| `Left / Right Arrow`              | تحريك Clip Deck إلى اليسار / اليمين                           |
| `Cmd / Ctrl + Left / Right Arrow` | تحريك Clip Deck صفحة واحدة إلى اليسار / اليمين                  |
| `Shift + Left / Right Arrow`      | تحريك zones صفحة واحدة إلى اليسار / اليمين                      |
| `Space`                           | تشغيل / إيقاف مؤقت (عندما يكون مصدر الإيقاع _Timeline_ نشطًا) |
| `T`                               | النقر لتحديد الإيقاع                                             |
| `R`                               | إعادة ضبط المازورة                                             |
| `Y`                               | تقريب الإيقاع إلى أقرب BPM                        |
| `Cmd / Ctrl + ,`                  | فتح لوحة الإعدادات                               |
| `Cmd / Ctrl + F`                  | تبديل وضع ملء الشاشة                               |
| `Cmd / Ctrl + L`                  | فتح لوحة إعدادات خرج الليزر                  |
| `Cmd / Ctrl + K`                  | فتح لوحة تعيين laser controller            |
| `Cmd / Ctrl + J`                  | فتح لوحة إعدادات عارض 3D                 |
| `Cmd / Ctrl + Shift + Plus (+)`   | زيادة تحجيم الواجهة                            |
| `Cmd / Ctrl + Shift + Minus (-)`  | تقليل تحجيم الواجهة                            |
| `Cmd / Ctrl + 0`                  | إعادة ضبط تحجيم الواجهة                               |
| `Cmd / Ctrl + A`                  | Timeline: تحديد الكل                                  |
| `Delete`                          | Timeline: حذف الكتل المحددة                      |
| `Delete`                          | Clip Editor: حذف nodes المحددة                    |
| `Cmd + Q` (macOS)                 | إنهاء التطبيق                                          |
| `Alt + F4` (Windows)              | إنهاء التطبيق                                          |
