# ✅ أين تجد مجلد العمل

#### أين تجد مجلد العمل

لكل إصدار من Liberation مجلد عمل خاص به. على سبيل المثال، إذا كنت تشغّل الإصدار 1.0.3، فسيكون اسم المجلد 1.0.3.

* **macOS**: `~/Library/Application Support/Liberation/1.0.3`
* **Windows**: `AppData\Local\Liberation\1.0.3`

**كيفية فتح المجلد بسرعة**

**macOS**

1. في Finder، اضغط **Shift + Cmd + G**.
2.  الصق هذا المسار واضغط **Enter**:

    ```
    ~/Library/Application Support/Liberation
    ```
3. افتح المجلد المطابق لرقم إصدارك، مثلًا `1.0.3`.

**Windows**

1.  اضغط **Win + R**، ثم الصق هذا واضغط **Enter**:

    ```
    %LOCALAPPDATA%\Liberation
    ```
2. افتح المجلد المطابق لرقم إصدارك، مثلًا `1.0.3`.

> **نصيحة لـ Windows**: إذا كنت تتصفح عبر مستكشف الملفات بدلًا من ذلك، ففعّل إظهار العناصر المخفية: **عرض > إظهار > العناصر المخفية**.
