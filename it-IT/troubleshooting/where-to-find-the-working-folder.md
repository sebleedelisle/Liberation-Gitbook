# ✅ Dove trovare la cartella di lavoro

#### Dove trovare la cartella di lavoro

Ogni versione di Liberation ha la propria cartella di lavoro. Ad esempio, se stai usando la versione 1.0.0, il nome della cartella sarà 1.0.0.

* **macOS**: `~/Library/Application Support/Liberation/1.0.0`
* **Windows**: `AppData\Local\Liberation\1.0.0`

**Come aprire rapidamente la cartella**

**macOS**

1. In Finder, premi **Shift + Cmd + G**.
2.  Incolla questo percorso e premi **Enter**:

    ```
    ~/Library/Application Support/Liberation
    ```
3. Apri la cartella che corrisponde al tuo numero di versione, ad esempio `1.0.0`.

**Windows**

1.  Premi **Win + R**, incolla questo percorso e premi **Enter**:

    ```
    %LOCALAPPDATA%\Liberation
    ```
2. Apri la cartella che corrisponde al tuo numero di versione, ad esempio `1.0.0`.

> **Suggerimento per Windows**: se invece navighi con File Explorer, abilita gli elementi nascosti: **View > Show > Hidden items**.
