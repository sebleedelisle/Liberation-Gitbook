# ✅ Errore “missing resources” all’avvio

Se ricevi un errore “missing resources” all’avvio, significa che mancano file critici necessari per eseguire Liberation. Questo può succedere su Windows se elimini o sposti accidentalmente dei file nella cartella di installazione. Su Mac, può accadere se l’installazione si è danneggiata in qualche modo.

### Come risolvere

Il modo migliore per risolvere questo problema è reinstallare Liberation.
