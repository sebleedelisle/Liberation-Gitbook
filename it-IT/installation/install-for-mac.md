# ✅ Installazione su Mac

### Compatibilità

macOS versione 12 (Monterey) e successive. Apple Silicon è consigliato, soprattutto se vuoi eseguire show di grandi dimensioni. Più veloce è il tuo computer, più fluida sarà l’uscita laser animata.

### Procedura di installazione

Una volta attivata una licenza a pagamento, riceverai un link; in alternativa puoi fare clic su _Download_ nella pagina [_Le tue licenze_](https://liberationlaser.com/account/my-products).

Vedrai questa pagina:

<figure><img src="../.gitbook/assets/account-download-page.png" alt=""><figcaption></figcaption></figure>

Fai clic su _Download for Mac_ (i numeri di versione cambieranno nel tempo) e il file di installazione DMG di Liberation verrà salvato nella tua cartella Downloads. Trova il file e fai doppio clic per aprirlo.

<figure><img src="../.gitbook/assets/mac-dmg-window" alt="" width="375"><figcaption></figcaption></figure>

Trascina l’icona di Liberation a sinistra nella cartella Applications a destra (potrebbe essere necessaria una password di amministratore). Ora puoi trovare Liberation nella cartella Applications: fai doppio clic per aprirlo.

Liberation dovrebbe avviarsi correttamente. Per inviare l’output ai laser, devi completare [Autorizzazione e rimozione dell’autorizzazione](authorising-and-de-authorising.md).
