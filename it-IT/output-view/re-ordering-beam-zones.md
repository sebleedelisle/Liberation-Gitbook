# 🟩 Riordinare le zone beam

Le zone beam vengono visualizzate nell’ordine in cui le crei, ma puoi riordinarle anche in seguito. Apri _View -> Beam Zone Organiser._

Vedrai l’elenco delle zone beam e a quale laser sono collegate. Trascina le zone per disporle nell’ordine che preferisci, oppure fai clic su _Autosort by laser_ per ordinarle automaticamente in modo consecutivo in base al laser in cui si trovano.

{% hint style="info" %}
Se non hai assegnato nomi personalizzati alle zone beam, verrà usato un nome predefinito come _Beam 1, Beam 2,_ ecc. Se le riordini, anche i loro nomi predefiniti verranno aggiornati in base all’ordine. Tienilo presente mentre riordini le zone, perché potrebbe creare confusione.
{% endhint %}
