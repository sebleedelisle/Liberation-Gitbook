# 🟩 Masks

{% hint style="danger" %}
**AVVISO DI SICUREZZA**

Le mask software non dovrebbero MAI essere il tuo unico metodo per proteggere le persone dall’output di un laser. Dovresti sempre prevedere barriere fisiche o mascherature fisiche tra il laser e qualsiasi possibile esposizione umana. Considera questo sistema come un secondo livello di ridondanza o per applicazioni non critiche per la sicurezza (come la protezione di telecamere o proiettori), ma ricorda che qualsiasi software può guastarsi.
{% endhint %}

***

Puoi aggiungere una o più mask virtuali per eliminare (o ridurre) l’output laser in punti specifici.

Aggiungi una mask cliccando sull’icona nella barra degli strumenti; vedrai comparire un quadrato rosso nella Output view: questa è la tua nuova mask. Puoi spostarla cliccando e trascinando, e trascinare gli angoli per modificarne le dimensioni. Se vuoi una forma non uniforme, premi `Alt / Option` mentre trascini un angolo. (Una volta che la mask è non uniforme, non devi più tenere premuto il tasto `Alt / Option`.)

Fai clic destro sulla mask per vedere impostazioni aggiuntive:

* Reduction percentage - rappresenta la riduzione di potenza del laser all’interno della mask. Per impostazione predefinita è impostata al 100%, cioè blackout completo all’interno della mask. Puoi però ridurre la potenza del laser di qualsiasi valore tu scelga. Nota che, quando la riduzione non è completa, la mask cambia colore in arancione.
* Il pulsante DELETE MASK si trova anch’esso in questo menu del clic destro.
