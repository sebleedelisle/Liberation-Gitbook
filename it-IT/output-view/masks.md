# 🟩 Masks

{% hint style="danger" %}
**AVVISO DI SICUREZZA**

Le maschere software non devono MAI essere l’unico metodo per proteggere le persone dall’output di un laser. Dovresti sempre avere barriere fisiche / mascherature tra il laser e qualsiasi possibile esposizione umana. Considera questo sistema come un livello secondario di ridondanza o per applicazioni non critiche per la sicurezza (per esempio per proteggere telecamere o proiettori), ma ricorda che qualsiasi software può guastarsi.
{% endhint %}

***

Puoi aggiungere una o più maschere virtuali per rimuovere (o ridurre) l’output laser in punti specifici.

Aggiungi una maschera facendo clic sull’icona nella barra degli strumenti; vedrai comparire un quadrato rosso nella vista di output: questa è la tua nuova maschera. Puoi spostarla facendo clic e trascinandola, e trascinare gli angoli per modificarne le dimensioni. Se vuoi una forma non uniforme, premi `Alt / Option` mentre trascini un angolo. (Una volta che la maschera è non uniforme, non è più necessario premere il tasto `Alt / Option`.)

Fai clic con il tasto destro sulla maschera per visualizzare impostazioni aggiuntive:

* Percentuale di riduzione: rappresenta la riduzione della potenza del laser all’interno della maschera. Per impostazione predefinita è impostata al 100%, cioè oscuramento totale all’interno della maschera. Puoi però ridurre la potenza del laser di qualsiasi valore tu scelga. Nota che, quando non si tratta di una riduzione totale, la maschera cambia colore e diventa arancione.
* Anche il pulsante DELETE MASK si trova in questo menu contestuale.
