# 🟩 Panoramica del controllo MIDI

Liberation usa MIDI in diversi modi:

* Come controller live, con APC 40. Vedi [Controllo live con APC40](live-control-with-the-apc40.md).
* Come sorgente di sincronizzazione del clock, usando messaggi MIDI clock e MIDI song position. Vedi [#midi-clock](../tempo-synchronisation.md#midi-clock)
* Come input interattivo nel nodo MIDI notes per creare effetti in stile "arpa laser". Vedi [MIDI notes](../clip-editor/operator-nodes/midi-notes.md)
* Come sistema di input/output più generale tramite il sistema MIDI Send/Receive. Vedi [MIDI Send/Receive](midi-send-receive.md)
