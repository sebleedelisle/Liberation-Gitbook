# ✅ Avviare / fermare le clip

{% hint style="info" %}
Puoi usare i pulsanti sullo schermo per avviare e fermare le clip, ma è molto meglio usare il controller MIDI APC40. Liberation è stato ottimizzato per funzionare bene con questo hardware e dovrebbe essere considerato una parte importante del tuo investimento nei laser.
{% endhint %}

### Avviare e fermare le clip

Per avviare una clip, premi il pulsante della clip (sullo schermo o sul tuo controller MIDI).

Per fermare la clip, premilo di nuovo.

Se avvii una nuova clip (nello stesso gruppo), la prima clip si fermerà automaticamente.

Per avviare una nuova clip senza fermare le altre clip puoi:

* premere il tasto `Shift` (o il pulsante Shift dell'APC40) mentre avvii la nuova clip; oppure
* premere di nuovo la clip o le clip attualmente in esecuzione mentre avvii la nuova clip.

I gruppi di clip sono indipendenti tra loro, quindi avviare una clip in un gruppo non influirà sulle clip degli altri gruppi. Vedi [Gruppi di Clip](groups.md)

### Flash mode

Se una clip si trova in un gruppo impostato su _flash mode_, si comporta in modo leggermente diverso: la clip continua a essere eseguita solo finché tieni premuto il pulsante della clip. Appena lo rilasci, si ferma. (Per impostazione predefinita, il gruppo di clip 3, quello rosso, è impostato su _flash mode.)_

### Fermare tutte le clip

Per **fermare tutte le clip in esecuzione**, premi il pulsante **STOP**.

{% hint style="info" %}
Premi il pulsante **STOP** **due volte** per bypassare il tempo di fade out delle clip e mandarle subito a nero.
{% endhint %}

Per fermare tutte le clip in esecuzione in un gruppo specifico, premi il pulsante del gruppo, poi premi il pulsante **STOP**.

Per fermare tutte le clip tranne una, premi la clip in esecuzione che vuoi mantenere, quindi, mentre la tieni premuta, premi anche il pulsante **STOP**. Poi rilascia il pulsante della clip. (Funziona anche con più clip, quante riesci a tenerne premute contemporaneamente!)

### La clip attualmente selezionata

<figure><img src="../.gitbook/assets/clips-selected-active.png" alt="" width="269"><figcaption><p>Due clip attualmente in esecuzione. Il contorno bianco indica che la clip a destra è la <em>clip attualmente selezionata.</em></p></figcaption></figure>

Quando una clip è in esecuzione, si illumina sullo schermo (e la sua mini rappresentazione nel visualizzatore della clip lampeggia). Noterai anche che l'ultima clip che hai premuto ha un contorno bianco. Questo indica che è la _clip attualmente selezionata._

Fai clic su una clip con `Alt / Option` premuto per selezionarla senza attivarla. Usa `Alt / Option + Shift`-clic per selezionare più clip.

Puoi anche fare clic e trascinare un lazo sul Clip Deck per selezionare più clip.

{% hint style="info" %}
Anche l'APC40 ha i pulsanti `Alt` e `Shift` per effettuare selezioni.
{% endhint %}
