# ✅ Scorciatoie da tastiera

Le scorciatoie sono indicate come `Cmd / Ctrl`: usa `Cmd` su macOS e `Ctrl` su Windows.

| Shortcut                          | Action                                                       |
| --------------------------------- | ------------------------------------------------------------ |
| `Cmd / Ctrl + Shift + A`          | Arma tutti i laser                                           |
| `Escape`                          | Disarma tutti i laser                                        |
| `Tab`                             | Cambia vista e passa al laser successivo                     |
| `Shift + Tab`                     | Cambia vista e passa al laser precedente                     |
| `1-9`                             | Passa al laser                                               |
| `Left / Right Arrow`              | Sposta il Clip Deck a sinistra / destra                      |
| `Cmd / Ctrl + Left / Right Arrow` | Sposta il Clip Deck di una pagina a sinistra / destra        |
| `Shift + Left / Right Arrow`      | Sposta le zone di una pagina a sinistra / destra             |
| `Space`                           | Riproduci / pausa (quando la sorgente tempo _Timeline_ è attiva) |
| `T`                               | Tap tempo                                                    |
| `R`                               | Reset della battuta                                          |
| `Y`                               | Arrotonda il tempo al BPM più vicino                         |
| `Cmd / Ctrl + ,`                  | Apri il pannello delle impostazioni                          |
| `Cmd / Ctrl + F`                  | Attiva o disattiva la modalità a schermo intero              |
| `Cmd / Ctrl + L`                  | Apri il pannello delle impostazioni dell’uscita laser        |
| `Cmd / Ctrl + K`                  | Apri il pannello di assegnazione dei controller laser        |
| `Cmd / Ctrl + J`                  | Apri il pannello delle impostazioni del 3D Visualiser        |
| `Cmd / Ctrl + Shift + Plus (+)`   | Aumenta la scala dell’interfaccia                            |
| `Cmd / Ctrl + Shift + Minus (-)`  | Riduci la scala dell’interfaccia                             |
| `Cmd / Ctrl + 0`                  | Reimposta la scala dell’interfaccia                          |
| `Cmd / Ctrl + A`                  | Timeline: seleziona tutto                                    |
| `Delete`                          | Timeline: elimina i blocchi selezionati                      |
| `Delete`                          | Editor Clip: elimina i nodi selezionati                      |
| `Cmd + Q` (macOS)                 | Chiudi l’app                                                 |
| `Alt + F4` (Windows)              | Chiudi l’app                                                 |
