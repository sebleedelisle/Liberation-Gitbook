# 🟩 Aree di destinazione del Canvas

Sappiamo come portare parti del Canvas nelle zone di ciascun laser, ma per inserire contenuti nel Canvas, prima di tutto ti servono le _Canvas target areas_ (nome un po’ ambiguo, ma accurato).

Le _Canvas target areas_ sono sezioni del Canvas in cui puoi disegnare i clip, e nella vista _CANVAS_ sono rappresentate come rettangoli con contorno blu.

Spesso ti basta una sola Canvas target area, che poi puoi suddividere in più zone inviate a laser diversi.

Altre volte potresti voler usare più Canvas target areas per diverse parti di un edificio, oppure per sfruttare lo zone delay tra di esse. (Sì! Lo zone delay funziona anche tra Canvas target areas!).

### Inviare clip alle Canvas target areas

Se guardi nel Clip Deck, accanto ai pulsanti delle beam zone, vedrai i pulsanti delle Canvas target area. Potrebbe essere necessario scorrere i pulsanti di output per visualizzarli: usa `Shift + Left / Right Arrow`, oppure i pulsanti ZONE PAGE sullo schermo, o i pulsanti dell’APC40 (vedi [apc40-reference.md](../reference/apc40-reference.md))

Assegna i clip alle Canvas target areas attivando o disattivando questi pulsanti esattamente come faresti con i pulsanti delle beam zone.

### Aggiungere / modificare Canvas target areas

Nella barra del menu in alto seleziona _View -> Canvas Target Areas_: vedrai tutte le impostazioni per ogni Canvas target area presente nel tuo progetto.

In alto trovi il pulsante _ADD CANVAS TARGET AREA_.

Elimina una Canvas target area usando il pulsante rosso con il segno meno.

Regola dimensione e posizione usando gli slider. Fai doppio clic su uno slider per inserire un valore.

### Modalità di scala

* **FIT TO AREA** - riduce il contenuto in modo che rientri completamente nella Canvas target area, mantenendo le proporzioni. (Questa è l’impostazione predefinita)
* **FILL AREA** - scala il contenuto per riempire la Canvas target area, mantenendo le proporzioni. Il contenuto potrebbe essere tagliato ai bordi.
* **STRETCH TO FIT** - allunga il contenuto per riempire l’intera Canvas target area, ignorando le proporzioni.
