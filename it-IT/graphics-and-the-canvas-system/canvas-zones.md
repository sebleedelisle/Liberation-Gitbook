# 🟩 Zone Canvas

Una zona Canvas appare nella vista Canvas come un rettangolo con contorno rosa. Serve a selezionare un’area rettangolare all’interno del Canvas e a inviarne il contenuto a uno o più laser.

Puoi fare clic sulle zone e trascinarle con il mouse. Trascina gli angoli per modificarne dimensioni e forma.

### Aggiungere una nuova zona Canvas

Fai clic sul pulsante _Add a new canvas zone_ nella barra degli strumenti.

### Assegnare la zona Canvas a un laser

Fai clic con il tasto destro sulla zona e attiva o disattiva i pulsanti numerati per scegliere a quale laser, o a quali laser, assegnare questa zona Canvas.

{% hint style="danger" %}
ATTENZIONE: se il laser è armato, potresti iniziare improvvisamente a proiettare contenuti in una zona Canvas predefinita. È meglio disarmare il laser prima di assegnargli zone Canvas.
{% endhint %}

Se ora vai alla vista _OUTPUT_ di quel laser, vedrai lì la zona Canvas, che potrà essere modificata esattamente come qualsiasi beam zone. Vedi [Zone](../output-view/zones.md).

{% hint style="info" %}
Puoi anche aggiungere una zona Canvas a un laser direttamente dalla vista _OUTPUT_: fai clic sul pulsante _Add existing canvas zone_ nella barra degli strumenti a sinistra.
{% endhint %}

### Menu del tasto destro

Fai clic con il tasto destro sulla zona e vedrai le opzioni per eliminare questa zona Canvas, spostarla dietro e bloccarla.
