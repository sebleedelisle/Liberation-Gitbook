# ✅ 键盘快捷键

快捷键统一写作 `Cmd / Ctrl`：macOS 用 `Cmd`，Windows 用 `Ctrl`。

| 快捷键 | 功能 |
| --- | --- |
| `Cmd / Ctrl + Shift + A` | Arm 所有激光 |
| `Escape` | Disarm 所有激光 |
| `Tab` | 切换视图并在激光间循环 |
| `Shift + Tab` | 切换视图并反向循环激光 |
| `1-9` | 跳转到指定激光 |
| `Left / Right Arrow` | Clip Deck 左右移动 |
| `Cmd / Ctrl + Left / Right Arrow` | Clip Deck 左翻页 / 右翻页 |
| `Shift + Left / Right Arrow` | Zones 左翻页 / 右翻页 |
| `Space` | 播放 / 暂停（当 _Timeline_ tempo source 激活时） |
| `T` | Tap tempo |
| `R` | Bar reset |
| `Y` | 将 tempo 四舍五入到最近 BPM |
| `Cmd / Ctrl + ,` | 打开 settings 面板 |
| `Cmd / Ctrl + F` | 切换全屏模式 |
| `Cmd / Ctrl + L` | 打开 Laser output settings 面板 |
| `Cmd / Ctrl + K` | 打开 Laser controller assignment 面板 |
| `Cmd / Ctrl + J` | 打开 3D visualiser settings 面板 |
| `Cmd / Ctrl + Shift + Plus (+)` | 增加界面缩放 |
| `Cmd / Ctrl + Shift + Minus (-)` | 减少界面缩放 |
| `Cmd / Ctrl + 0` | 重置界面缩放 |
| `Cmd / Ctrl + A` | Timeline：全选 |
| `Delete` | Timeline：删除选中块 |
| `Delete` | Clip editor：删除选中 nodes |
| `Cmd + Q` (macOS) | 退出应用 |
| `Alt + F4` (Windows) | 退出应用 |
