# 🟩 整理你的 Clip Deck

### 移動／複製 Clip

按住並拖曳即可移動 Clip。如果已選取多於一個 Clip（見[啟動／停止 Clip](starting-stopping-clips.md "mention")），它們會一併移動。按住 `Alt / Option` 可複製它們。將 Clip 拖放到空白位置即可。

{% hint style="info" %}
如果需要選取大量 Clip，你可以按住並拖曳出 lasso 框選範圍，然後在仍然拖曳時捲動 Clip Deck（可使用方向鍵、APC40，或觸控板／滑鼠的橫向捲動）。lasso 會延伸到 Clip Deck 邊緣之外。
{% endhint %}

### 刪除 Clip

如要刪除單一 Clip，請使用右鍵選單。

如要刪除多個 Clip，先選取它們（使用 `Alt / Option + Shift` 點擊，或拖曳 lasso 框選），然後將它們拖離 Clip Deck。

### 加入空白位置

你可以在 Clip 按鈕上按右鍵，然後按 _ADD EMPTY COLUMN TO RIGHT_，在 Clip Deck 加入一欄空白位置。

### 合併多個 Clip Deck

你可以匯入現有的 Clip Deck，並將它附加到目前 Clip Deck 的末端。使用選單 _File -> Clips/Effects -> Append Clip Deck._

選取一組 Clip（使用 lasso 或 `Alt / Option + Shift` 點擊），然後選擇 _File -> Clips/Effects -> Export Selected Clips._ 這會匯出一個只包含該些 Clip 的 Clip Deck，之後可附加到另一個 Clip Deck。

### 按 Timeline 整理 Clip

匯入 Timeline 時，你亦會有一些 Clip 選項。你可以選擇取代目前的 Clip Deck、附加 Timeline 的整個 Clip Deck，或只附加它使用的 Clip。

如果你的 project 內有多個 Timeline，可以按 Timeline 重新整理 Clip。開啟 _Timeline selection_ panel，然後按 _ORGANISE CLIPS BY TIMELINE_。你現在會看到所有 Clip 按其 Timeline 分組顯示。如果某個 Clip 用於多個 Timeline，它會在每個分組中各複製一份。
