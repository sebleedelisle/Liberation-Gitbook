# ✅ 將 Clip 指派到激光區域

<figure><img src="../.gitbook/assets/clips-zones" alt=""><figcaption><p>上方的區域按鈕，以及下方的 Flip X / Flip Y 按鈕。</p></figcaption></figure>

在 Clip 下方，你會看到一排區域按鈕（畫面上會顯示，APC40 上亦會對應顯示），預設標示為 _BEAM 1_ 至 _BEAM 8_。使用這些按鈕可為_目前選取的 Clip_ 切換區域。

#### 捲動瀏覽區域

如果有超過 8 個區域，你可以使用以下方式左右捲動：

* `Shift + Left / Right Arrow`
* APC40 上的 _DEV ON/OFF_ 和 _DEV LOCK_ 按鈕

### Flip X 和 Y

在每個區域按鈕下方，你會看到標示為 _X_ 和 _Y_ 的按鈕。它們會分別將輸出到該區域的畫面作水平和垂直翻轉。

{% hint style="info" %}
你不需要為了不同的鏡像設定而建立新區域。X 和 Y 翻轉設定只會套用於這個 Clip，並不是該區域本身的設定。
{% endhint %}

### 一次過將區域指派到多個 Clip

你可以為某個區域內的所有 Clip 指派區域及調整 X/Y。詳情請參閱[群組](groups.md)。
