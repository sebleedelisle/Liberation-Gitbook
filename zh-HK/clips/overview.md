# 🟩 概覽

### 甚麼是 Clip？

Liberation 內所有 laser 內容都儲存在 Clip 之中。

Clip 可以包含空中光束效果或圖像動畫，通常會以循環方式播放。它們可以指向任何 zone（或 Canvas Target Area），並透過 Clip Deck 內的 Clip 按鈕觸發。

Clip 內容是使用靈活的 node-based Clip Editor 系統建立。（詳情請參閱 [Clip Editor 簡介](../clip-editor/clip-editor-intro.md "mention")）

### Clip Deck

<figure><img src="../.gitbook/assets/clips-clip-deck" alt=""><figcaption><p>預設 Clip Deck</p></figcaption></figure>

Clip Deck 是畫面上的 Clip 網格，通常會延伸到目前可見範圍之外。

使用以下方式向左或向右捲動 Clip Deck：

* 向左及向右方向鍵
* 觸控板雙指滑動（如果你的滑鼠支援，也可以用滑鼠向左及向右捲動！）
* APC40 上的 _Cue Level_ 滾輪

使用以下方式將 Clip Deck 向左或向右捲動一整頁：

* `Cmd / Ctrl + Left / Right Arrow`
* APC40 上的 _Device <- and ->_ 按鈕

#### 迷你 Clip visualiser

在 Clip Deck 頂部，你會看到整個 Clip Deck 的小型表示圖。亮起的部分代表目前可見的 Clip。
