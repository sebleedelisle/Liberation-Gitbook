# 🟩 色彩校準

色彩校準可確保投影機的紅、綠、藍激光在所有亮度級別都以平滑而可預測的方式輸出光線。不同投影機可能有非線性的亮度曲線，意思是 50% 紅色看起來可能遠比 100% 紅色一半強度更光或更暗。校準會修正這一點，令顏色混合更乾淨、漸變更平滑，白色亦更平衡。

#### 預熱投影機

激光二極管在升溫時行為會改變。校準前請務必讓投影機穩定下來：

* 投射明亮的畫面，例如 **White rectangle test pattern (11)**，至少 **15–20 分鐘**。
* 這可確保你設定的色彩平衡在演出期間保持一致。

#### 校準測試的運作方式

使用測試圖案進行校準（請參閱 [測試圖案](../output-view/test-patterns.md "mention")）

* **5** – 紅色
* **6** – 綠色
* **7** – 藍色
* **8** – 白色

每個測試圖案都會顯示四條移動線。激光系統中的亮度不只受功率級別影響，亦取決於光束停留在同一位置的時間。當掃描器移動得越快，光束在每個點停留的時間越短，看起來就會較暗。

為了補償這一點，測試圖案會同時按比例調整亮度和速度：

* **最上方線條** – 100% 亮度，全速
* **第二條線** – 75% 亮度，75% 速度
* **第三條線** – 50% 亮度，50% 速度
* **第四條線** – 25% 亮度，25% 速度

由於亮度_和速度_會一同按比例縮放，所有線條看起來應該有相同亮度。如果其中一條線看起來較光或較暗，請調整相應的滑桿，直到它們一致。

每個測試圖案亦有第五條線，亮度為 **0%**，正常情況下不應該可見。這用於修正某些激光在極低級別時不輸出任何光線的情況。如果你的激光在低亮度時仍然看不到，請逐步提高 **0% setting**，直到該線剛好可見，然後稍微調低，直到它再次消失。目標是找出激光開始發亮的臨界點，然後保持在略低於該點的位置——這樣淡入淡出會自然開始，而不會截斷底部範圍。

#### 使用 Colour Calibration 面板

此面板可讓你分別控制每個通道（紅、綠、藍）在 100、75、50、25 和 0% 級別的輸出。

1. **選擇測試圖案**（由紅色開始）。
2. **調整滑桿**，令 100、75、50 和 25% 的線條看起來亮度相同。
3. 如果「off」線仍然隱約可見，請**微調 0% 滑桿**。
4. **對綠色和藍色重複以上步驟。**
5. 切換至 **White test pattern (8)**。四條線看起來應該一致，而白色應呈現中性（沒有偏色）。

#### 調整白平衡

你亦可以使用這個系統調整**白平衡**。分別校準每個通道後，切換至 **White test pattern (8)**。如果輸出看起來有偏色（例如太綠或太藍），請調整紅、綠、藍通道的相對級別，直到線條呈現中性白色。即使你的激光功率相差頗大，校準仍可幫助它們更接近一致，並產生更乾淨、更平衡的色彩混合。

#### 儲存校準

* 使用 **Store** 覆寫目前的 preset。
* 使用 **Store As** 建立新的 preset（如果你使用多部激光投影機，這會很有用）。
