# 🟩 使用 APC40 進行即時控制

<figure><img src="../.gitbook/assets/qs-apc40.jpeg" alt=""><figcaption></figcaption></figure>

### **APC40 控制器**

這是 Liberation 的預設硬件控制器；我們非常建議使用它。可以說，Liberation 從一開始就是圍繞這款硬件設計。只要你插上 APC40，Liberation 就會立即自動連接。

{% hint style="warning" %}
_糟了！演出途中 USB 插頭被拔走了！_

不用擔心——只要重新插上，Liberation 會自動重新連接，無需額外處理。
{% endhint %}

### APC40 Mark 1 還是 Mark 2？

簡單來說，建議使用 Mark 2，因為它有全彩按鈕，能更貼近 Liberation Clip Deck 介面的顏色顯示。Mark 1 版本應急時也可以使用，但會稍為較易混淆，因為它的佈局與畫面上的介面略有不同，而且按鈕只可以顯示紅色、橙色或綠色，無法對應 Clip 的顏色。

{% hint style="info" %}
最初的 APC40 Mark 1 於 2009 年推出（！），至今仍有人偏好它的金屬機身結構，以及堅固、類似控制台的外形。更新版 Mark 2 於 2014 年推出，雖然在 2024 年停產，但因應視覺藝術家（Resolume 等）及 laserists 的需求，將於 2025 年重新投產。
{% endhint %}
