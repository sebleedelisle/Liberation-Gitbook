# ✅ 歡迎使用 Liberation



<figure><img src=".gitbook/assets/LiberationTease.jpeg" alt=""><figcaption></figcaption></figure>

Liberation 開啟了一種全新的激光工作方式：**現代化**、**高效率**、支援 **Mac 和 PC**，並圍繞當代活動製作的實際需要而設計。它結合快速、具表現力的創作工具、內置安全功能，以及流暢、直觀、真正面向未來的工作流程，將激光軟件帶進 21 世紀。

Liberation 是一個不斷發展的激光控制軟件生態系統的一部分；這些軟件全都使用同一系列的開放協議控制器。無論你使用 TouchDesigner、MadLaser、LaserShowGen 還是 Liberation，都不會被綁定於單一製造商。你可以按自己的系統、預算和創作想法，選擇最合適的硬件。

最重要的是，Liberation 旨在成為一件具表現力的創作工具。它容易上手、使用起來真正有趣，同時亦具備足夠強大的能力和穩定可靠性，可由單一電腦處理要求最高的音樂節、裝置和大型場館巡演。

本指南會協助你熟習 Liberation：由安裝和安全基礎，到建立 Clip、映射 Zone，以及運行完整表演。內容集中於真正重要的概念和工作流程；Liberation 的設計目標是盡量不阻礙你的創作，讓你花更多時間創作，少一點時間與軟件糾纏。

## 本指南適合誰閱讀

* 希望有清晰入門路線，同時不忽略安全的新用戶。
* 在場地設定或綵排時，需要可靠參考資料的表演操作員。
* 想探索 Clip Editor、MIDI、DMX 和 Canvas 工具等進階功能的好奇玩家。

## 如何使用本指南

* 連接硬件前，請先閱讀 [**重要！請先閱讀**](important.md)，了解關鍵安全和授權注意事項。
* 如果你想快速開始輸出，請跟隨 [**快速入門指南**](getting-started.md)；之後可深入閱讀 [**硬件**](hardware/) 和 [**設定**](setting-up/)，作更詳細配置。
* 使用章節標題直接跳到相關主題（例如 [**Clip 與 Clip Deck**](clips/)、[**Output 視圖**](output-view/)、**等等**）；需要準確答案時，可返回 [**參考資料**](reference/) 或 [**疑難排解**](troubleshooting/)。

## 慣例與提示

* 狀態 emoji 代表完成程度：✅ 已完成，🟧 部分完成，🟩 需要圖片，◼️ 佔位。
* 連結內容以目前版本為準；如果你使用較舊版本，截圖和選單可能會略有不同。
* 硬件步驟會標明以安全為先的預設做法；切勿繞過安全聯鎖，並保持光束遠離人群。

如果遇到問題，請先查看 [**FAQ**](faq.md)，再到 [**疑難排解**](troubleshooting/) 章節尋找針對性的修正方法；仍未解決才聯絡支援。
