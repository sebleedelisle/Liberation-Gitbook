# 🟩 遮罩

{% hint style="danger" %}
**安全注意事項**

軟件遮罩絕不應作為保護人員免受激光輸出影響的唯一方法。你應該一律在激光與任何可能接觸到人體的位置之間設置實體屏障／遮擋。請把此系統視為第二層冗餘保護，或用於非安全關鍵的用途（例如保護相機或投影機），但請記住任何軟件都有可能失效。
{% endhint %}

***

你可以加入一個或多個虛擬遮罩，在指定位置移除（或降低）激光輸出。

按一下工具列上的圖示即可加入遮罩；你會在 Output view 中看到一個紅色正方形，這就是你的新遮罩。你可以按住並拖曳來移動它，也可以拖曳角位來改變大小。如果想建立非均勻形狀，拖曳角位時按住 `Alt / Option`。（遮罩變成非均勻形狀後，就不再需要按住 `Alt / Option` 鍵。）

在遮罩上按右鍵可查看其他設定：

* Reduction percentage - 這代表遮罩範圍內的激光功率降低百分比。預設為 100%，即遮罩內完全無輸出。不過你可以按需要設定任何降低幅度。請注意，當並非完全降低時，遮罩會變成橙色。
* DELETE MASK 按鈕也在這個右鍵選單內。
