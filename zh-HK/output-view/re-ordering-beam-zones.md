# 🟩 重新排序光束區域

光束區域會按你建立的次序顯示，但之後你亦可以重新排序。開啟 _View -> Beam Zone Organiser_。

你會看到光束區域列表，以及每個區域連接到哪一部 laser。拖放各個區域即可排成你想要的次序；或按 _Autosort by laser_，系統會按區域所在的 laser 自動連續排序。

{% hint style="info" %}
如果你沒有為光束區域設定自訂名稱，它們會使用預設名稱，例如 _Beam 1, Beam 2,_ 等等。如果你重新排序，這些預設名稱亦會按新的排序次序更新。重新排序區域時請留意這點，否則可能會令人混淆。
{% endhint %}
