# ✅ 鳴謝

Liberation 由 [Seb Lee-Delisle](https://app.gitbook.com/u/ylIaEEh1DCgBfWngNL7qhvoUTQV2) 設計及編程。

Windows 版本由 [Dan Moore](https://www.makeitdoathing.com) 協助開發。

本軟件使用以下開源項目：

#### [openFrameworks](https://openframeworks.cc)

openFrameworks 貢獻者的[完整名單](https://github.com/openframeworks/openFrameworks/blob/master/THANKS.md)

openFrameworks 會按你的平台，使用以下[函式庫](https://github.com/openframeworks/openFrameworks/blob/master/docs/libraries.md)的組合：

[OpenGL](http://www.opengl.org/), [GLEW](http://glew.sourceforge.net/), [GLUT](http://www.opengl.org/resources/libraries/glut/), [libtess2](https://code.google.com/p/libtess2/), [cairo](http://cairographics.org/), [rtAudio](http://www.music.mcgill.ca/~gary/rtaudio/), [PortAudio](http://www.portaudio.com/), [FMOD](http://www.fmod.org/), [Kiss FFT](http://kissfft.sourceforge.net/), [FreeType](http://freetype.sourceforge.net/index2.html), [FreeImage](http://freeimage.sourceforge.net/), [Quicktime](https://developer.apple.com/standards/classic-quicktime/), [Unicap](http://unicap-imaging.org/), [GStreamer](http://gstreamer.freedesktop.org/), [videoInput](https://github.com/ofTheo/videoInput), [Poco](http://pocoproject.org/)

#### openFrameworks addons

* [ofxAbletonLink](https://github.com/comoc/ofxAbletonLink)，作者為 [Akihiro Komori](https://github.com/comoc)
* [ofxArtnet](https://github.com/hiroyuki/ofxArtnet)，作者為 [Hiroyuki Hori](https://github.com/hiroyuki)
* [ofxLTC](https://github.com/2bbb/ofxLTC)，作者為 [2bit](https://github.com/2bbb)（使用 [libltc](https://github.com/x42/libltc)）
* [ofxMidi](https://github.com/danomatika/ofxMidi)，作者為 [Dan Wilcox](https://github.com/danomatika)（使用 [Gary Scavone](https://github.com/thestk/rtmidi) 的 [RTMIDI](https://github.com/garyscavone)）
* [ofxNative](https://github.com/kritzikratzi/ofxNative)，作者為 [Hansi Raber](https://github.com/kritzikratzi)
* [ofxSVG](https://github.com/openframeworks/openFrameworks/tree/master/addons/ofxSvg)（使用 [libsvgtiny](https://source.netsurf-browser.org/libsvgtiny.git/)）

#### [Dear ImGui](https://github.com/ocornut/imgui)

UI 部分使用 [Dear ImGui](https://github.com/ocornut/imgui) 建構；Dear ImGui 由 Omar Cornut 及所有直接或間接為 GitHub 項目作出貢獻的[貢獻者](https://github.com/ocornut/imgui/graphs/contributors)共同開發。
