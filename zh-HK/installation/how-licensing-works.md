# ✅ 授權運作方式

在電腦安裝 Liberation 後，軟件會以 _free mode_ 執行，直至你完成_授權_。

{% hint style="info" %}
_free mode_ 除了不能輸出至激光硬件外，沒有其他限制。你可以練習演出、建立 timelines，以及不限次數儲存和載入檔案。你可以在任意數量的電腦上以 _free mode_ 安裝 Liberation；只有在準備使用激光時，才需要為該安裝授權！
{% endhint %}

除了可啟用的激光數量外，你也可以使用 DMX 燈具。請參閱 [DMX / Art-Net](../dmx-control/ "mention")。

你可授權 Liberation 的電腦數量亦會受你的 licence 級別限制。

{% hint style="danger" %}
每個 licence 只供一名用戶使用，並且同一時間只可用於一場演出。每名用戶可以有多個安裝，因為你**務必**隨時準備至少一部已授權的備用電腦！
{% endhint %}

### 互聯網連線

Liberation 的 licence 和授權透過互聯網運作；app 會連線到我們的伺服器，將某個安裝與你的付費 licence 連結起來。

授權完成後，在你的定期 licence 到達下一個付款日期前，都不需要再次連線（日期會顯示於 _About panel_）。每月定期 licence 通常設有短暫寬限期（7 日），但你應在付款日期後盡快上線，以確保付費功能可以不間斷使用。

寬限期適用於續期及付款問題。對於已取消或刻意暫停的 licence，付費期結束後不會因寬限期而延長。如果付款失敗，請在寬限期內更新你的信用卡資料，以保持付費功能可用。

目前的 licence tier 亦會顯示於頂部列。按一下 tier 名稱；如未獲授權，則按 _FREE MODE_，即可開啟 About panel。
