# ✅ 參數控制

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  參數控制

_參數控制_ 是一種可在 Clip Editor 以外，控制效果或 Clip 內各種設定的方法。

* **range min / range max** - 參數 level 會映射到的最小值及最大值
* **level** - 此參數的預設設定。這個值可由外部更改。你亦可手動調整，查看它會產生的效果。
* **parameter number** - 每個 _參數控制_ 都可指定一個編號，並會反映在 Clip 以外的控制項上。你可以有多個使用相同編號的參數控制，而它們會同時被調整。

{% hint style="warning" %}
APC40 及支援的現場控制器預設會控制每個效果欄位的參數 1。效果及 Clip 內的所有參數亦可在右鍵選單中使用。
{% endhint %}
