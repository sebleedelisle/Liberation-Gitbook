# ✅ 鍵盤快捷鍵

快捷鍵以 `Cmd / Ctrl` 表示：macOS 使用 `Cmd`，Windows 使用 `Ctrl`。

| 快捷鍵                            | 動作                                                |
| --------------------------------- | --------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | 啟用所有激光器                                      |
| `Escape`                          | 停用所有激光器                                      |
| `Tab`                             | 切換檢視，並循環切換激光器                          |
| `Shift + Tab`                     | 切換檢視，並反向循環切換激光器                      |
| `1-9`                             | 跳至指定激光器                                      |
| `Left / Right Arrow`              | 將 Clip Deck 向左 / 向右移動                        |
| `Cmd / Ctrl + Left / Right Arrow` | 將 Clip Deck 向左 / 向右移動一頁                    |
| `Shift + Left / Right Arrow`      | 將 zones 向左 / 向右移動一頁                        |
| `Space`                           | 播放 / 暫停（當 _Timeline_ tempo source 啟用時）     |
| `T`                               | Tap tempo                                           |
| `R`                               | 小節重設                                            |
| `Y`                               | 將 tempo 四捨五入至最接近的 BPM                     |
| `Cmd / Ctrl + ,`                  | 開啟設定面板                                        |
| `Cmd / Ctrl + F`                  | 切換全螢幕模式                                      |
| `Cmd / Ctrl + L`                  | 開啟激光輸出設定面板                                |
| `Cmd / Ctrl + K`                  | 開啟激光控制器分配面板                              |
| `Cmd / Ctrl + J`                  | 開啟 3D Visualiser 設定面板                         |
| `Cmd / Ctrl + Shift + Plus (+)`   | 增加介面縮放                                        |
| `Cmd / Ctrl + Shift + Minus (-)`  | 減少介面縮放                                        |
| `Cmd / Ctrl + 0`                  | 重設介面縮放                                        |
| `Cmd / Ctrl + A`                  | Timeline：全選                                      |
| `Delete`                          | Timeline：刪除已選取的 blocks                       |
| `Delete`                          | Clip editor：刪除已選取的 nodes                     |
| `Cmd + Q` (macOS)                 | 結束應用程式                                        |
| `Alt + F4` (Windows)              | 結束應用程式                                        |
