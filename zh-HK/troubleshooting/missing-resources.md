# ✅ 啟動時出現 missing resources 錯誤

如果啟動時出現「missing resources」錯誤，表示 Liberation 執行所需的重要檔案遺失。這種情況可能會在 Windows 發生，例如你不小心刪除或移動了安裝資料夾內的檔案；亦可能會在 Mac 發生，例如安裝內容因某些原因損壞。

### 解決方法

解決此問題的最佳方法是重新安裝 Liberation。
