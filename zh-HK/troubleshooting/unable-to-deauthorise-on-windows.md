# ✅ 無法在 Windows 上取消授權？

#### 無法在 Windows 上取消授權？

如果你無法在 Windows 上為電腦取消授權，請先確保使用當初為該授權進行授權的同一個 Liberation 版本來取消授權，然後才在其他版本中重新授權。

如果這樣仍然無效，而你使用的是 1.0 之前的版本，問題很可能與較舊 Windows 版 Liberation 識別電腦的方式有關。在那些版本中，用來產生機器 ID 的系統可靠性較低；在某些情況下，即使硬件看起來沒有任何變更，ID 也可能會在重新啟動後改變。

如果你在嘗試取消授權時遇到阻礙，而且並沒有切換版本，請聯絡 support@liberationlaser.com，我們可以為你手動取消該機器的授權。

**為甚麼會發生這種情況**

在早期 Windows 版 Liberation（1.0 之前）中，我們使用了 Windows 建議的系統方法來產生機器 ID。可惜，這在某些情況下被證實並不一致。因此，授權系統在 1.0 版中已重寫，改用更穩健的方法組合，現在可以可靠地運作。

因此，較舊版本 Liberation 使用的電腦 ID，可能與目前版本使用的 ID 不同。如果 ID 已經改變，取消授權就需要由支援團隊手動處理。
