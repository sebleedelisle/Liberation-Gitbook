# 🟩 Canvas 檢視

在頂部工具列按一下 _CANVAS_ 按鈕，即可選擇 _CANVAS_ 檢視。

Canvas 檢視會顯示完整的 Canvas，也就是你可用來播放及加入內容的範圍。在此檢視中，你可以選擇 Canvas 的哪些部分送到哪些區域，並加入參考圖像。

### 在 Canvas 檢視中移動

在任何空白位置（或已鎖定的區域）按住並拖曳，即可移動檢視。

### 放大及縮小 Canvas 檢視

使用滑鼠滾輪放大及縮小。

### 工具列圖示

檢視左側會顯示工具列。將滑鼠移到任何圖示上，即可查看其功能。

* 開啟 _Canvas settings_ 面板
* 加入新的 Canvas 區域
* 加入新的參考圖像
* 切換是否對齊格線
* 調整格線大小（0 最小 - 8 最大）。（請注意，格線大小會以 2 的次方增加，因此當你縮小格線大小時，所有已對齊的點仍會保持在同一直線上。）
* 顯示／隱藏格線
