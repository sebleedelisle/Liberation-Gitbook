# 🟩 Canvas 區域

Canvas 區域會在 Canvas 視圖中顯示為粉紅色外框矩形。它用來在 Canvas 內選取一個矩形範圍，並將其內容送到一部或多部激光器。

你可以用滑鼠按住並拖曳區域來移動它。拖曳角位可改變大小和形狀。

### 新增 Canvas 區域

在工具列中按一下 _Add a new canvas zone_ 按鈕。

### 將 Canvas 區域指派到激光器

在區域上按右鍵，然後切換數字按鈕，選擇此 Canvas 區域要指派到哪一部或多部激光器。

{% hint style="danger" %}
警告 — 如果激光器已 armed，你可能會突然開始在預設 Canvas 區域投射內容。建議先將激光器 disarm，再為它指派 Canvas 區域。
{% endhint %}

如果你現在前往該激光器的 _OUTPUT_ 視圖，就會在其中看到這個 Canvas 區域，現在可像任何光束區域一樣進行編輯。請參閱[區域](../output-view/zones.md)。

{% hint style="info" %}
你亦可直接從 _OUTPUT_ 視圖將 Canvas 區域加入激光器 — 按一下左側工具列中的 _Add existing canvas zone_ 按鈕。
{% endhint %}

### 右鍵選單

在區域上按右鍵，你會看到刪除此 Canvas 區域、將它移到後方，以及鎖定它的選項。
