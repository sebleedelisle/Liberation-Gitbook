# 🟩 簡介

現時 laser 較常用於在空中製造光束及氣氛效果，而不是把視覺圖像投射到表面上。我甚至會說，用 laser 投射圖像往往被視為不太型格，主要是因為大量低質素 clip art 動畫和可疑內容經常被當作 preset 提供。

但事實是，laser graphics 可以非常靈活，為你提供大量創作機會。無論是簡單閃亮的公司 logo（對，我們都做過），還是完整 mapping 的建築立面，我們都可以支援。

透過使用 mapped graphics 和自訂動畫，你可以為自己的工具箱加入一整個全新的創作可能世界。
