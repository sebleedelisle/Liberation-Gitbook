# 🟩 Canvas 設定

按一下左側工具列頂部的 _Canvas settings_ 按鈕，即可開啟 _Canvas settings_ 面板。

### Width and Height

使用這些可拖曳的滑桿，調整整個 Canvas 的整體闊度及高度。這些尺寸可以概念上視為像素尺寸，但實際上一切都是相對的。

連按兩下滑桿即可輸入數值。

{% hint style="info" %}
這裡有一點較深入的技術細節：laser 是相對於 Canvas 解像度移動，而不是相對於 zone 大小移動。原因是 Canvas 中較大的 zone，在現實中通常會投影到較大的範圍。

這只是小細節，通常不會對你造成影響，但有助令各個 Canvas zone 的效能及可見度更平均。
{% endhint %}

### Zone brightness

使用此設定可調整 Canvas view 中粉紅色 zone 外框的亮度。這有助令你的內容更清晰，亦可避免此視圖變得過於雜亂。
