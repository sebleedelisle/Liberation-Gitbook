# 🟩 Duplicators

## <img src="../../.gitbook/assets/image (4).png" alt="" data-size="line">  Mirror

生成内容的镜像副本。默认镜像轴为中心的垂直线。

* **angle** - 镜像轴角度
* **offset position** - 平移镜像轴（垂直于轴移动）
* **delay** - 镜像内容的时间延迟。注意只有内容本身带动画时才会生效。

#### 3D 选项（选择 3D 时可用）

* **angle X / angle Y** - 镜像轴变为一个平面，可用这两个设置在 3D 中旋转该平面。

## <img src="../../.gitbook/assets/image (5).png" alt="" data-size="line">  Circular Pattern

将内容以圆形阵列复制。

* **radius** - 旋转前内容与中心点的距离
* **count** - 复制数量
* **use each objects pivot point** - 选中时，每个元素围绕自身中心进行位移和旋转（只有在多个元素时才有效）
* **delay** - 为每个复制元素增加逐渐递增的时间延迟。注意内容需带动画才会明显生效。
* **rotation** - 为元素添加额外的旋转偏移

#### 3D 选项（选择 3D 时可用）

* **rotation x / rotation y** - 将整个圆形阵列绕 x、y 轴旋转。

## <img src="../../.gitbook/assets/image (6).png" alt="" data-size="line">  Grid Pattern

将内容以行列网格复制。

* **spacing** - 元素之间的间距
* **count X**- 水平方向复制数量（列）
* **count Y**- 垂直方向复制数量（行）
* **horizontal alignment** - 列的起点位置，LEFT、CENTRE 或 RIGHT
* **vertical alignment** - 行的起点位置，TOP、MIDDLE 或 BOTTOM
* **delay** - 为每个复制元素增加逐渐递增的时间延迟。注意内容需带动画才会明显生效。
