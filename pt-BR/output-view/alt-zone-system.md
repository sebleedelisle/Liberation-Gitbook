# 🟩 Sistema de zonas Alt

Você já passou por uma situação em que precisa mover ou desativar instantaneamente uma seleção de zonas? Por exemplo:

* Lasers apontando para o céu que você precisa redirecionar para terminar em prédios caso uma aeronave passe por perto.
* Lasers que atingem um nível de varanda sem uso e que você precisa redirecionar em certas partes do show enquanto a equipe está no local.

O sistema de zonas Alt permite criar uma zona alternativa para qualquer zona existente e, com um único botão, alternar para todas as zonas alternativas ao mesmo tempo.

### Adicionando uma zona alternativa

Na visualização Output, clique com o botão direito em uma zone e clique em _ADD ALT ZONE_. Observe que uma nova zone verde aparece. Você pode editar e ajustar essa zone exatamente da mesma forma que qualquer uma das suas zones. Você precisará ativar as Alt zones na barra de ícones superior para ver os padrões de teste.

### Alternando para suas zonas Alt

Assim que você adiciona qualquer zona Alt, um botão de alternância aparece na barra de ícones superior com o rótulo _ALT ZONES._ Clique nesse botão para alternar entre suas zonas Alt e suas zonas normais.

### Como desativar uma zona em vez de movê-la

Adicione uma Alt zone, clique nela com o botão direito e clique no botão _Disable output_ (círculo com uma linha atravessando). Agora, quando você ativar as Alt zones, o conteúdo será redirecionado para essa Alt zone, que agora está desativada.
