# 🟩 Copiar zonas entre lasers

Clique no botão _COPY LASER SETTINGS_ em _Laser Output_ e um novo painel será aberto.

Este é um recurso extremamente útil e que economiza tempo para copiar rapidamente zonas e outras configurações de um laser para os outros.

Certifique-se de ter selecionado o laser do qual você quer copiar as configurações.

Ative todos os outros lasers para os quais você quer copiar as configurações. (Use _ALL_ e _NONE_ para agilizar se você tiver muitos lasers)

Selecione a opção _Copy output zone settings_ e depois clique no botão _COPY SETTINGS_ para concluir o processo.

{% hint style="info" %}
Observe que este processo não adiciona nem remove zonas dos lasers. Ele apenas copia a posição e o tipo de zona para cada zona que já existe nos lasers de destino.
{% endhint %}

Consulte também [Copiar configurações entre lasers](../setting-up/copy-laser-settings.md) para uma descrição completa dos outros recursos deste sistema.
