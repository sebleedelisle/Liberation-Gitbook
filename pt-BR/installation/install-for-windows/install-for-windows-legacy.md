# ✅ Instalação no Windows (legado, antes da versão 0.9)

### Compatibilidade

Máquinas com Windows 10 ou 11 de 64 bits. Máquinas com especificações mais altas são recomendadas se você quiser executar shows grandes. Quanto mais rápido for o seu computador, mais suave será a saída de laser animada.

### Processo de instalação

Depois de ter uma assinatura ativa, você receberá um link, ou pode clicar no link _Download_ na página [_My subscriptions_](https://liberationlaser.com/account/my-products).

Você verá esta página:

<figure><img src="../../.gitbook/assets/windows-download-page.png" alt=""><figcaption></figcaption></figure>

Clique em _Download latest for Windows_ (os números de versão podem mudar) e você verá o indicador de download mostrando o progresso (geralmente no canto superior direito; a captura de tela é do Chrome). Quando terminar, passe o mouse sobre o download do Liberation e clique no ícone de pasta que aparece.

<figure><img src="../../.gitbook/assets/windows-download-chrome.png" alt="" width="260"><figcaption></figcaption></figure>

Isso abrirá sua pasta Downloads. Encontre o arquivo baixado (normalmente no topo), clique com o botão direito e selecione "Extract all".

<figure><img src="../../.gitbook/assets/windows-extract-all.png" alt="" width="248"><figcaption></figcaption></figure>

Um novo painel será aberto perguntando onde você gostaria de instalar o Liberation. Você pode colocá-lo em qualquer lugar do seu disco rígido onde tenha permissão de escrita, mas, se estiver em dúvida, basta colocá-lo em uma pasta no seu drive C:.

<figure><img src="../../.gitbook/assets/windows-select-destination.png" alt="" width="375"><figcaption></figcaption></figure>

Depois que o Windows terminar de descompactar, ele mostrará o arquivo de instalação. Clique duas vezes em Liberation.exe para executar o aplicativo.

<figure><img src="../../.gitbook/assets/windows-open-exe.png" alt="" width="375"><figcaption></figcaption></figure>

O Liberation precisa de permissão para acessar arquivos e a rede, então o Windows provavelmente exibirá algumas solicitações de permissão. Aceite tudo, caso contrário o Liberation pode não funcionar corretamente.

<figure><img src="../../.gitbook/assets/windows-protection-window.png" alt="" width="375"><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

O Liberation deve estar funcionando agora! Para enviar saída para lasers, você precisará passar pelo [processo de autorização.](../authorising-and-de-authorising.md)
