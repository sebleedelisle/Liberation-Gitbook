# ✅ Instalar no Windows

### Compatibilidade

Computadores com Windows 10 ou 11 de 64 bits. Máquinas com especificações mais altas são recomendadas se você quiser executar shows grandes. Quanto mais rápido for o seu computador, mais suave será a saída de laser animada.

### Processo de instalação

Depois que você tiver uma assinatura ativa, você receberá um link, ou poderá clicar no link _Download_ na página [_Your subscriptions_](https://liberationlaser.com/account/my-products).

Você verá esta página:

<figure><img src="../../.gitbook/assets/windows-install-download-page.png" alt=""><figcaption></figcaption></figure>

Clique em _Download latest for Windows_ (os números de versão podem mudar) e você verá o indicador de download mostrando o progresso (geralmente no canto superior direito; a captura de tela é do Chrome). Quando terminar, abra seus downloads e clique no arquivo do instalador baixado.

<figure><img src="../../.gitbook/assets/windows-install-download-list.png" alt=""><figcaption></figcaption></figure>

Isso executará o instalador.

{% hint style="info" %}
Em alguns casos, você pode receber um aviso dizendo que o Windows protegeu seu PC — o Windows é cauteloso ao executar softwares novos! À medida que mais pessoas usarem e confiarem no Liberation, isso deve deixar de aparecer.

Mas, se você receber esse aviso, clique no link _More info_, confirme se o _Publisher_ aparece como _SEB.LY LIMITED_ e então clique em _Run anyway_.

<img src="../../.gitbook/assets/windows-install-windows-protection-warning.png" alt="" data-size="original"><img src="../../.gitbook/assets/windows-install-warning-run-anyway.png" alt="" data-size="original">
{% endhint %}

Antes de o instalador iniciar, você pode receber um aviso perguntando se permite que o instalador faça alterações. Clique em _Yes_.

<figure><img src="../../.gitbook/assets/windows-install-warning-make-changes.png" alt=""><figcaption></figcaption></figure>

Agora o instalador pode começar! Avance pelas telas.

<figure><img src="../../.gitbook/assets/windows-installer-1.png" alt=""><figcaption></figcaption></figure>

É recomendado usar o local de instalação padrão, então basta clicar em _Next_.

<figure><img src="../../.gitbook/assets/windows-installer-2.png" alt=""><figcaption></figcaption></figure>

Em seguida, você terá opções para criar um ícone na área de trabalho e uma pasta no menu Iniciar (recomendado).

<figure><img src="../../.gitbook/assets/windows-installer-3.png" alt=""><figcaption></figcaption></figure>

Aguarde um pouco enquanto o Liberation é instalado.

<figure><img src="../../.gitbook/assets/windows-installer-4.png" alt=""><figcaption></figcaption></figure>

O instalador terminou! Clique em _Finish_ para fechá-lo e iniciar o Liberation.

<figure><img src="../../.gitbook/assets/windows-installer-5.png" alt=""><figcaption></figcaption></figure>

O Liberation precisa de permissão para acessar arquivos e a rede, então o Windows provavelmente exibirá várias solicitações. Aceite tudo para garantir que o Liberation funcione corretamente.

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

O Liberation agora deve estar em execução! Para enviar saída para os lasers, você precisará passar pelo [processo de autorização.](../authorising-and-de-authorising.md)
