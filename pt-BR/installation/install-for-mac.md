# ✅ Instalação no Mac

### Compatibilidade

macOS versão 12 (Monterey) ou superior. Apple Silicon é recomendado, especialmente se você pretende executar shows grandes. Quanto mais rápido for o seu computador, mais suave será a saída de laser animada.

### Processo de instalação

Depois de ter uma assinatura ativa, você receberá um link, ou pode clicar no link _Download_ na página [_My subscriptions_](https://liberationlaser.com/account/my-products).

Você verá esta página:

<figure><img src="../.gitbook/assets/windows-download-page.png" alt=""><figcaption></figcaption></figure>

Clique em _Download latest for Mac_ (os números de versão mudarão com o tempo) e o arquivo de instalação DMG do Liberation será salvo na sua pasta de downloads. Localize o arquivo e dê dois cliques para abri-lo.

<figure><img src="../.gitbook/assets/mac-dmg-window" alt="" width="375"><figcaption></figcaption></figure>

Arraste o ícone do Liberation à esquerda para a pasta Applications à direita (talvez seja necessária uma senha de administrador). Agora você pode encontrar o Liberation na sua pasta Applications; dê dois cliques para abri-lo.

O Liberation agora deve estar em execução! Para enviar saída para lasers, você precisará passar pelo [processo de autorização.](authorising-and-de-authorising.md)
