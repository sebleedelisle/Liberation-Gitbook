# 🟩 Visão geral

### O que é um clip?

Todo o conteúdo de laser no Liberation é armazenado em clips.

Os clips podem conter efeitos atmosféricos de feixe ou animações gráficas e, normalmente, funcionam como um ciclo em loop. Eles podem ser direcionados para qualquer zone (ou Canvas Target Area) e são acionados usando os botões de clip dentro do Clip Deck.

O conteúdo dos clips é criado usando o sistema versátil de editor de clips baseado em nodes. (Veja [clip-editor-intro.md](../clip-editor/clip-editor-intro.md) para mais detalhes)

### O Clip Deck

<figure><img src="../.gitbook/assets/clips-clip-deck" alt=""><figcaption><p>O Clip Deck padrão</p></figcaption></figure>

O Clip Deck é a grade de clips na tela e, normalmente, se estende além da área visível que você consegue ver.

Role o Clip Deck para a esquerda e para a direita usando:

* As teclas de seta para a esquerda e para a direita
* Deslize com 2 dedos no trackpad (ou rolagem para a esquerda e para a direita no mouse, se o seu mouse permitir!)
* A roda de rolagem _Cue Level_ no APC40

Role o Clip Deck uma página inteira para a esquerda e para a direita usando:

* `Cmd / Ctrl + Left / Right Arrow`
* Os botões _Device <- and ->_ no APC40

#### O mini visualizador de clips

Na parte superior do Clip Deck, você verá uma pequena representação de todo o Clip Deck. A seção iluminada representa os clips que estão visíveis no momento.
