# 🟩 Visão geral

### O que é um Clip?

Todo o conteúdo de laser no Liberation é armazenado dentro de Clips.

Clips podem conter efeitos atmosféricos de beam ou animações gráficas, e normalmente funcionam em loop. Eles podem ser direcionados para qualquer zone (ou Canvas Target Area) e são acionados usando os botões de Clip dentro do Clip Deck.

O conteúdo de um Clip é criado usando o sistema versátil do Clip Editor baseado em nodes. (Veja [Introdução ao Clip Editor](../clip-editor/clip-editor-intro.md "mention") para mais detalhes)

### O Clip Deck

<figure><img src="../.gitbook/assets/clips-clip-deck" alt=""><figcaption><p>O Clip Deck padrão</p></figcaption></figure>

O Clip Deck é a grade de Clips exibida na tela e, normalmente, se estende além da área visível.

Role o Clip Deck para a esquerda e para a direita usando:

* As teclas de seta para a esquerda e para a direita
* Deslize com 2 dedos no trackpad (ou rolagem horizontal do mouse, se o seu mouse permitir isso!)
* A roda de rolagem _Cue Level_ no APC40

Role o Clip Deck uma página inteira para a esquerda e para a direita usando:

* `Cmd / Ctrl + Left / Right Arrow`
* Os botões _Device <- and ->_ no APC40

#### O mini visualizador de Clips

Na parte superior do Clip Deck, você verá uma pequena representação de todo o Clip Deck. A seção iluminada representa os Clips que estão visíveis no momento.
