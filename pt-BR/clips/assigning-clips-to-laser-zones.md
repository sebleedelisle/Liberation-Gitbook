# ✅ Atribuindo clips a zonas de laser

<figure><img src="../.gitbook/assets/clips-zones" alt=""><figcaption><p>Os botões de zona na parte superior e os botões Flip X / Flip Y na parte inferior.</p></figcaption></figure>

Abaixo dos clips, você verá uma linha de botões de zona (tanto na tela quanto refletidos no APC40), marcados como _BEAM 1_ até _BEAM 8_ por padrão. Use esses botões para ativar ou desativar zonas para o _clip selecionado no momento_.

#### Navegando pelas zonas

Se houver mais de 8 zonas, você pode rolar para a esquerda e para a direita usando:

* `Shift + Left / Right Arrow`
* Os botões _DEV ON/OFF_ e _DEV LOCK_ no APC40

### Flip X e Y

Abaixo de cada botão de zona, você verá botões marcados como _X_ e _Y._ Eles invertem a saída de cada zona na horizontal e na vertical, respectivamente.

{% hint style="info" %}
Você não precisa criar uma nova zona para ter configurações de espelho diferentes. As configurações de flip X e Y se aplicam apenas a este clip; elas não são uma configuração da zona.
{% endhint %}

### Atribuindo zonas a vários clips de uma vez

Você pode atribuir zonas e ajustar X/Y para todos os clips em uma zona. Consulte [groups.md](groups.md) para mais informações.
