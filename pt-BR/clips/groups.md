# 🟩 Grupos de Clip

Você vai notar que cada Clip tem um contorno colorido, e essa cor representa em qual _grupo_ ele está. Os botões de Clip do APC40 também acendem na cor do grupo correspondente.

O sistema de grupos é bem flexível e permite que você:

* Mantenha os Clips de um grupo tocando enquanto alterna grupos em outro
* Atribua rapidamente zonas e inversões X/Y a todos os Clips dentro de um grupo
* Defina o _Flash mode_ para um Clip (o Group 3 usa _Flash mode_ por padrão)
* Defina os tempos padrão de transição de entrada/saída que podem ser herdados pelos Clips ou substituídos.

Você decide como usar o sistema de _grupos_, mas, se tiver muitas zonas/lasers, pode ser útil usar os diferentes grupos para diferentes conjuntos de lasers. Mas fica a seu critério! Como ponto de partida, é assim que eu normalmente uso os grupos:

### Os botões _Group_

Os botões _Group_ ficam no lado direito do Clip Deck e, ao ativar um deles, você pode:

* atribuir Clips a um grupo
* alterar as configurações de zona e a inversão X/Y para todos os Clips do grupo

Para ativar o botão _group_:

* Pressione e segure o botão _group_ no APC40
* Clique com o mouse no botão _group_ na tela. Clique novamente para desativar.

Você pode atribuir Clips ao grupo facilmente pressionando-os enquanto o botão _group_ estiver ativo. (Outra forma de atribuir um grupo é pelo menu de clique direito do Clip).

### Alterar as configurações de zona para todos os Clips dentro de um grupo

Quando um botão _group_ está ativo, você verá os botões de _zone_ (BEAM 1 - 8, CANVAS 1, etc) acenderem da seguinte forma:

* OFF: esta zona **não está definida para nenhum** dos Clips desse grupo
* FLASHING: esta zona está definida para _**alguns**_**&#x20;mas não&#x20;**_**todos**_ os Clips do grupo
* ON: esta zona está definida para _**todos**_ os Clips do grupo

Pressione o botão de zona para ativar e desativar essa zona em _**todos**_ os Clips do grupo.

Exatamente o mesmo conceito se aplica aos botões de inversão X/Y.
