# ✅ Iniciar / parar clips

{% hint style="info" %}
Você pode usar os botões na tela para iniciar e parar clips, mas é muito melhor usar o controlador MIDI APC40. O Liberation foi otimizado para funcionar bem com esse hardware e ele deve ser considerado uma parte importante do seu investimento em lasers.
{% endhint %}

### Iniciar e parar clips

Para iniciar um clip, pressione o botão desse clip (na tela ou no seu controlador MIDI).

Para parar o clip, pressione-o novamente.

Se você iniciar um novo clip (no mesmo grupo), o primeiro clip será interrompido automaticamente.

Para iniciar um novo clip sem parar outros clips, você pode:

* Pressionar a tecla `Shift` (ou o botão Shift do APC40) enquanto inicia o novo clip; ou
* Pressionar novamente o(s) clip(s) em execução enquanto inicia o novo clip.

Os grupos de clips são independentes entre si, portanto iniciar um clip em um grupo não afeta os clips de nenhum outro grupo. Veja [groups.md](groups.md)

### Flash mode

Se um clip estiver em um grupo configurado como _flash mode_, ele se comporta de forma um pouco diferente: o clip só continua em execução enquanto você mantiver o botão do clip pressionado. Assim que você soltar, ele para. (Por padrão, o grupo de clips 3 — o vermelho — é configurado como _flash mode.)_

### Parar todos os clips

Para **parar todos os clips em execução**, pressione o botão **STOP**.

{% hint style="info" %}
Pressione o botão **STOP** **duas vezes** para ignorar o tempo de fade out dos clips e apagá-los imediatamente.
{% endhint %}

Para parar todos os clips em execução em um grupo específico, pressione o botão do grupo e depois pressione o botão **STOP**.

Para parar todos os clips exceto um, pressione o clip em execução que você deseja manter e, enquanto ele estiver pressionado, pressione também o botão **STOP**. Depois solte o botão do clip. (Isso também funciona com vários clips, tantos quantos você conseguir manter pressionados ao mesmo tempo!)

### O clip selecionado no momento

<figure><img src="../.gitbook/assets/clips-selected-active.png" alt="" width="269"><figcaption><p>Dois clips em execução no momento. O contorno branco indica que o clip à direita é o <em>clip selecionado no momento.</em></p></figcaption></figure>

O clip acende na tela quando está em execução (e sua representação no mini visualizador de clip pisca). Você também perceberá que o último clip que você pressionou tem um contorno branco ao redor. Isso indica que ele é o _clip selecionado no momento._

Clique com `Alt / Option` em um clip para selecioná-lo sem ativá-lo. Use `Alt / Option + Shift`-clique para selecionar vários clips.

Você também pode clicar e arrastar uma seleção por laço no Clip Deck para selecionar vários clips.

{% hint style="info" %}
O APC40 também tem botões `Alt` e `Shift` para fazer seleções.
{% endhint %}
