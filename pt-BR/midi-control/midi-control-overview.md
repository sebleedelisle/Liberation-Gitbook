# 🟩 Visão geral do controle MIDI

Há várias formas de usar MIDI no Liberation:

* Como controlador ao vivo, com o APC 40. Veja [Controle ao vivo com o APC40](live-control-with-the-apc40.md).
* Como fonte de sincronização de clock, usando mensagens de MIDI clock e MIDI song position. Veja [#midi-clock](../tempo-synchronisation.md#midi-clock)
* Como entrada interativa no node MIDI notes para criar efeitos no estilo “harpa laser”. Veja [MIDI notes](../clip-editor/operator-nodes/midi-notes.md)
* Como um sistema mais geral de entrada/saída usando o sistema MIDI Send/Receive. Veja [MIDI Send/Receive](midi-send-receive.md)
