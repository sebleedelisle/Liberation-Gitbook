# 🟩 Zonas do Canvas

Uma zona do Canvas aparece na visualização do Canvas como um retângulo com contorno rosa. Ela é usada para selecionar uma área retangular dentro do Canvas e enviar seu conteúdo para um ou mais lasers.

Você pode clicar e arrastar as zonas com o mouse. Arraste os cantos para alterar o tamanho e o formato.

### Adicionar uma nova zona do Canvas

Clique no botão _Add a new canvas zone_ na barra de ferramentas.

### Atribuir a zona do Canvas a um laser

Clique com o botão direito na zona e alterne os botões numéricos para selecionar a quais laser(s) esta zona do Canvas será atribuída.

{% hint style="danger" %}
AVISO: se o laser estiver armado, você pode começar a projetar conteúdo repentinamente em uma zona padrão do Canvas. O ideal é desarmar o laser antes de atribuir zonas do Canvas a ele.
{% endhint %}

Se agora você acessar a visualização _OUTPUT_ desse laser, verá a zona do Canvas ali, e ela poderá ser editada exatamente como qualquer zona de feixe. Consulte [Zonas](../output-view/zones.md).

{% hint style="info" %}
Você também pode adicionar uma zona do Canvas a um laser diretamente pela visualização _OUTPUT_: clique no botão _Add existing canvas zone_ na barra de ferramentas à esquerda.
{% endhint %}

### Menu do botão direito

Clique com o botão direito na zona e você verá opções para excluir esta zona do Canvas, movê-la para trás e bloqueá-la.
