# ✅ Atalhos de teclado

Os atalhos são escritos como `Cmd / Ctrl`: use `Cmd` no macOS e `Ctrl` no Windows.

| Atalho                            | Ação                                                  |
| --------------------------------- | ----------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Armar todos os lasers                                 |
| `Escape`                          | Desarmar todos os lasers                              |
| `Tab`                             | Alternar visualizações e percorrer os lasers          |
| `Shift + Tab`                     | Alternar visualizações e percorrer os lasers para trás |
| `1-9`                             | Ir para o laser                                       |
| `Left / Right Arrow`              | Mover o Clip Deck para a esquerda / direita           |
| `Cmd / Ctrl + Left / Right Arrow` | Mover o Clip Deck uma página para a esquerda / direita |
| `Shift + Left / Right Arrow`      | Mover as zonas uma página para a esquerda / direita   |
| `Space`                           | Reproduzir / pausar (quando a fonte de tempo _Timeline_ estiver ativa) |
| `T`                               | Tap tempo                                             |
| `R`                               | Reiniciar compasso                                    |
| `Y`                               | Arredondar o tempo para o BPM mais próximo            |
| `Cmd / Ctrl + ,`                  | Abrir o painel de configurações                       |
| `Cmd / Ctrl + F`                  | Alternar modo de tela cheia                           |
| `Cmd / Ctrl + L`                  | Abrir o painel de configurações de saída do laser     |
| `Cmd / Ctrl + K`                  | Abrir o painel de atribuição de controladores de laser |
| `Cmd / Ctrl + J`                  | Abrir o painel de configurações do 3D Visualiser      |
| `Cmd / Ctrl + Shift + Plus (+)`   | Aumentar a escala da interface                        |
| `Cmd / Ctrl + Shift + Minus (-)`  | Diminuir a escala da interface                        |
| `Cmd / Ctrl + 0`                  | Redefinir a escala da interface                       |
| `Cmd / Ctrl + A`                  | Timeline: selecionar tudo                             |
| `Delete`                          | Timeline: excluir blocos selecionados                 |
| `Delete`                          | Editor de Clip: excluir nós selecionados              |
| `Cmd + Q` (macOS)                 | Sair do aplicativo                                    |
| `Alt + F4` (Windows)              | Sair do aplicativo                                    |
