# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

O _Parameter Control_ é um método para controlar várias configurações dentro de um efeito ou de um clip fora do editor de clips.

* **range min / range max** - os valores mínimo e máximo nos quais o nível do parâmetro é mapeado
* **level** - a configuração padrão deste parâmetro. Este é o valor que pode ser alterado externamente. Você pode ajustá-lo manualmente para ver o efeito que ele terá.
* **parameter number** - cada _parameter control_ pode receber um número, e isso é refletido nos controles fora do clip. Você pode ter mais de um parameter control com o mesmo número, e todos eles serão ajustados ao mesmo tempo.

{% hint style="warning" %}
O APC40 e os controllers ao vivo compatíveis controlam o parâmetro 1 de cada effect slot por padrão. Todos os parâmetros dentro de efeitos e Clips também estão disponíveis no menu do botão direito.
{% endhint %}
