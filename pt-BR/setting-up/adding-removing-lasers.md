# ✅ Adicionar / remover lasers

Adicione um novo laser clicando no botão _ADD LASER_ no painel _Laser overview_

<figure><img src="../.gitbook/assets/Remove Laser in Laser Overview.png" alt="" width="292"><figcaption></figcaption></figure>

ou no botão _+_ na visualização _Output_.<br>

<figure><img src="../.gitbook/assets/Laser Output View (1).png" alt="" width="176"><figcaption></figcaption></figure>

Quando você adiciona um novo laser, uma beam zone é adicionada automaticamente a ele. Observe que lasers e zonas são ordenados na sequência em que você os adiciona. Consulte [re-ordering-beam-zones.md](../output-view/re-ordering-beam-zones.md).

Remova um laser clicando no botão vermelho "-" no painel _Laser overview_. Observe que você perderá todas as configurações desse laser!

{% hint style="info" %}
Se quiser copiar zonas ou outras configurações de outros lasers, use [copy-laser-settings.md](copy-laser-settings.md)
{% endhint %}

{% hint style="info" %}
Observe que, sempre que você fizer algo potencialmente perigoso, como excluir um laser, será exibida uma caixa de diálogo de confirmação. Se quiser ignorá-la, pressione `Cmd / Ctrl` enquanto clica no botão. Naturalmente, use isso com cuidado!
{% endhint %}
