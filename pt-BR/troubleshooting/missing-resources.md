# ✅ Erro de recursos ausentes na inicialização

Se você receber um erro de "missing resources" ao iniciar, isso significa que arquivos críticos necessários para o Liberation funcionar estão ausentes. Isso pode acontecer no Windows se você apagar ou mover arquivos acidentalmente na pasta de instalação. Ou no Mac, se a instalação tiver sido corrompida de alguma forma.

### Como corrigir

A melhor maneira de corrigir esse problema é reinstalar o Liberation.
