# ✅ Prečaci na tipkovnici

Prečaci su zapisani kao `Cmd / Ctrl`: upotrijebite `Cmd` na macOS-u i `Ctrl` na Windowsu.

| Prečac                            | Radnja                                                     |
| --------------------------------- | ---------------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Uključi arm za sve lasere                                  |
| `Escape`                          | Isključi arm za sve lasere                                 |
| `Tab`                             | Prebacuj prikaze i kružno prolazi kroz lasere              |
| `Shift + Tab`                     | Prebacuj prikaze i kružno prolazi kroz lasere unatrag      |
| `1-9`                             | Skoči na laser                                             |
| `Left / Right Arrow`              | Pomakni Clip Deck ulijevo / udesno                         |
| `Cmd / Ctrl + Left / Right Arrow` | Pomakni Clip Deck za jednu stranicu ulijevo / udesno       |
| `Shift + Left / Right Arrow`      | Pomakni zones za jednu stranicu ulijevo / udesno           |
| `Space`                           | Reprodukcija / pauza (kada je aktivan izvor tempa _Timeline_) |
| `T`                               | Tap tempo                                                  |
| `R`                               | Reset takta                                                |
| `Y`                               | Zaokruži tempo na najbliži BPM                             |
| `Cmd / Ctrl + ,`                  | Otvori ploču s postavkama                                  |
| `Cmd / Ctrl + F`                  | Uključi/isključi način rada preko cijelog zaslona          |
| `Cmd / Ctrl + L`                  | Otvori ploču s postavkama laserskog izlaza                 |
| `Cmd / Ctrl + K`                  | Otvori ploču za dodjelu laser controller uređaja           |
| `Cmd / Ctrl + J`                  | Otvori ploču s postavkama 3D vizualizatora                 |
| `Cmd / Ctrl + Shift + Plus (+)`   | Povećaj skaliranje sučelja                                 |
| `Cmd / Ctrl + Shift + Minus (-)`  | Smanji skaliranje sučelja                                  |
| `Cmd / Ctrl + 0`                  | Vrati skaliranje sučelja na zadanu vrijednost              |
| `Cmd / Ctrl + A`                  | Timeline: odaberi sve                                      |
| `Delete`                          | Timeline: izbriši odabrane blokove                        |
| `Delete`                          | Clip Editor: izbriši odabrane nodes                        |
| `Cmd + Q` (macOS)                 | Zatvori aplikaciju                                         |
| `Alt + F4` (Windows)              | Zatvori aplikaciju                                         |
