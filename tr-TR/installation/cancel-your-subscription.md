# ✅ Aboneliğinizi iptal edin

Web sitesinde oturum açın, _My subscriptions_ sayfasını açın, iptal etmek istediğiniz aboneliği seçin ve ardından _Cancel Subscription_ bağlantısına tıklayın. Abonelik süresinin sonuna kadar Liberation kullanmaya devam edebilirsiniz.
