# ✅ Klavye kısayolları

Kısayollar `Cmd / Ctrl` şeklinde yazılır: macOS’ta `Cmd`, Windows’ta `Ctrl` kullanın.

| Kısayol                           | Eylem                                                   |
| --------------------------------- | ------------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Tüm lazerleri arm durumuna al                           |
| `Escape`                          | Tüm lazerleri disarm durumuna al                        |
| `Tab`                             | Görünümler arasında geçiş yap ve lazerler arasında dolaş |
| `Shift + Tab`                     | Görünümler arasında geçiş yap ve lazerler arasında geriye doğru dolaş |
| `1-9`                             | Lazere atla                                             |
| `Left / Right Arrow`              | Clip Deck konumunu sola / sağa kaydır                   |
| `Cmd / Ctrl + Left / Right Arrow` | Clip Deck konumunu bir sayfa sola / sağa kaydır         |
| `Shift + Left / Right Arrow`      | Zones alanını bir sayfa sola / sağa kaydır              |
| `Space`                           | Oynat / duraklat (_Timeline_ tempo kaynağı etkin olduğunda) |
| `T`                               | Tap tempo                                               |
| `R`                               | Ölçüyü sıfırla                                          |
| `Y`                               | Tempoyu en yakın BPM değerine yuvarla                   |
| `Cmd / Ctrl + ,`                  | Ayarlar panelini aç                                     |
| `Cmd / Ctrl + F`                  | Tam ekran modunu aç / kapat                             |
| `Cmd / Ctrl + L`                  | Lazer çıkışı ayarları panelini aç                       |
| `Cmd / Ctrl + K`                  | Laser controller atama panelini aç                      |
| `Cmd / Ctrl + J`                  | 3D görselleştirici ayarları panelini aç                 |
| `Cmd / Ctrl + Shift + Plus (+)`   | Arayüz ölçeklendirmesini artır                          |
| `Cmd / Ctrl + Shift + Minus (-)`  | Arayüz ölçeklendirmesini azalt                          |
| `Cmd / Ctrl + 0`                  | Arayüz ölçeklendirmesini sıfırla                        |
| `Cmd / Ctrl + A`                  | Timeline: tümünü seç                                    |
| `Delete`                          | Timeline: seçili blokları sil                           |
| `Delete`                          | Clip Editor: seçili nodes öğelerini sil                 |
| `Cmd + Q` (macOS)                 | Uygulamadan çık                                         |
| `Alt + F4` (Windows)              | Uygulamadan çık                                         |
