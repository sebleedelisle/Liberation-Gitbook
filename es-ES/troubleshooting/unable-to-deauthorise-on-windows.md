# ✅ ¿No puedes desautorizar en Windows?

#### ¿No puedes desautorizar en Windows?

Si no puedes desautorizar un ordenador en Windows, primero asegúrate de desautorizar la licencia usando la misma versión de Liberation con la que se autorizó originalmente, antes de volver a autorizarla en otra versión.

Si esto no funciona y estás usando una versión anterior a la 1.0, es probable que el problema se deba a cómo las versiones antiguas de Liberation para Windows identificaban el ordenador. En esas versiones, el sistema utilizado para generar un ID de máquina era menos fiable y, en algunos casos, el ID podía cambiar entre reinicios, aunque aparentemente no hubiese cambiado ningún hardware.

Si te has quedado atascado intentando desautorizar y no has cambiado de versión, ponte en contacto con support@liberationlaser.com y podremos desautorizar la máquina manualmente por ti.

**Por qué ocurre esto**

En las primeras versiones de Liberation para Windows (anteriores a la 1.0), usábamos el método del sistema Windows recomendado para generar un ID de máquina. Por desgracia, resultó ser inconsistente en algunas situaciones. Por este motivo, el sistema de licencias se reescribió para la versión 1.0, usando una combinación de métodos más robusta que ahora funciona de forma fiable.

Como resultado, el ID de ordenador utilizado por las versiones antiguas de Liberation puede ser distinto del que usan las versiones actuales. Si el ID ya ha cambiado, el equipo de soporte tendrá que gestionar la desautorización manualmente.
