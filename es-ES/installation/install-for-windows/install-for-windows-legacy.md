# ✅ Instalación en Windows (legacy, anterior a la versión 0.9)

### Compatibilidad

Equipos con Windows 10 u 11 de 64 bits. Se recomiendan equipos potentes si quieres ejecutar shows grandes. Cuanto más rápido sea tu ordenador, más fluida será la salida láser animada.

### Proceso de instalación

Cuando tengas una suscripción activa, recibirás un enlace; también puedes hacer clic en el enlace _Download_ de la página [_My subscriptions_](https://liberationlaser.com/account/my-products).

Verás esta página:

<figure><img src="../../.gitbook/assets/windows-download-page.png" alt=""><figcaption></figcaption></figure>

Haz clic en _Download latest for Windows_ (los números de versión pueden cambiar) y verás que el indicador de descarga muestra el progreso (normalmente en la parte superior derecha; la captura es de Chrome). Cuando termine, pasa el cursor sobre la descarga de Liberation y haz clic en el icono de carpeta que aparece.

<figure><img src="../../.gitbook/assets/windows-download-chrome.png" alt="" width="260"><figcaption></figcaption></figure>

Esto abrirá tu carpeta Downloads. Busca el archivo descargado (normalmente estará arriba), haz clic derecho y selecciona "Extract all".

<figure><img src="../../.gitbook/assets/windows-extract-all.png" alt="" width="248"><figcaption></figcaption></figure>

Se abrirá un nuevo panel preguntándote dónde quieres instalar Liberation. Puedes colocarlo en cualquier ubicación de tu disco duro en la que tengas permisos de escritura, pero, si tienes dudas, ponlo en una carpeta de la unidad C:.

<figure><img src="../../.gitbook/assets/windows-select-destination.png" alt="" width="375"><figcaption></figcaption></figure>

Cuando Windows haya terminado de descomprimirlo, te mostrará el archivo de instalación. Haz doble clic en Liberation.exe para ejecutar la aplicación.

<figure><img src="../../.gitbook/assets/windows-open-exe.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation necesita permiso para acceder a archivos y a la red, así que es probable que Windows muestre algunas solicitudes de permiso. Asegúrate de aceptarlas todas o puede que Liberation no funcione correctamente.

<figure><img src="../../.gitbook/assets/windows-protection-window.png" alt="" width="375"><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

¡Liberation debería ejecutarse ahora! Para enviar salida a los láseres, tendrás que seguir el [proceso de autorización.](../authorising-and-de-authorising.md)
