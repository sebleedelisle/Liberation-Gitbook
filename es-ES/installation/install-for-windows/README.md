# ✅ Instalación en Windows

### Compatibilidad

Equipos con Windows 10 u 11 de 64 bits. Se recomiendan equipos potentes si quieres ejecutar shows grandes. Cuanto más rápido sea tu ordenador, más fluida será la salida láser animada.

### Proceso de instalación

Cuando tengas una suscripción activa, recibirás un enlace, o puedes hacer clic en el enlace _Download_ de la página[ _Your subscriptions_](https://liberationlaser.com/account/my-products).

Verás esta página:

<figure><img src="../../.gitbook/assets/windows-install-download-page.png" alt=""><figcaption></figcaption></figure>

Haz clic en _Download latest for Windows_ (los números de versión pueden cambiar) y verás que el indicador de descarga muestra el progreso (normalmente en la parte superior derecha; la captura es de Chrome). Cuando termine, abre tus descargas y haz clic en el archivo del instalador descargado.

<figure><img src="../../.gitbook/assets/windows-install-download-list.png" alt=""><figcaption></figcaption></figure>

Esto ejecutará el instalador.

{% hint style="info" %}
En algunos casos puede aparecer una advertencia indicando que Windows protegió tu PC: Windows es prudente al ejecutar software nuevo. A medida que más personas usen Liberation y confíen en él, esta advertencia debería desaparecer.

Si aparece esta advertencia, haz clic en el enlace _More info_, comprueba que _Publisher_ aparece como _SEB.LY LIMITED_ y después haz clic en _Run anyway_.

<img src="../../.gitbook/assets/windows-install-windows-protection-warning.png" alt="" data-size="original"><img src="../../.gitbook/assets/windows-install-warning-run-anyway.png" alt="" data-size="original">
{% endhint %}

Antes de que empiece el instalador, puede que aparezca una advertencia preguntando si permites que el instalador haga cambios. Haz clic en _Yes_.

<figure><img src="../../.gitbook/assets/windows-install-warning-make-changes.png" alt=""><figcaption></figcaption></figure>

Ahora el instalador puede empezar. Avanza por las páginas.

<figure><img src="../../.gitbook/assets/windows-installer-1.png" alt=""><figcaption></figcaption></figure>

Se recomienda usar la ubicación de instalación predeterminada, así que haz clic en _Next_.

<figure><img src="../../.gitbook/assets/windows-installer-2.png" alt=""><figcaption></figcaption></figure>

A continuación tendrás opciones para crear un icono en el escritorio y una carpeta en el menú Inicio (recomendado).

<figure><img src="../../.gitbook/assets/windows-installer-3.png" alt=""><figcaption></figcaption></figure>

Espera un momento mientras se instala Liberation.

<figure><img src="../../.gitbook/assets/windows-installer-4.png" alt=""><figcaption></figcaption></figure>

El instalador ha terminado. Haz clic en _Finish_ para cerrarlo e iniciar Liberation.

<figure><img src="../../.gitbook/assets/windows-installer-5.png" alt=""><figcaption></figcaption></figure>

Liberation necesita permiso para acceder a los archivos y a la red, así que probablemente Windows mostrará algunos avisos de permisos. Asegúrate de aceptar todo o puede que Liberation no funcione correctamente.

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation debería ejecutarse ahora. Para enviar salida a los láseres, ahora tendrás que seguir el [proceso de autorización.](../authorising-and-de-authorising.md)
