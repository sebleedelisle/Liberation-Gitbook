# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_Parameter Control_ es un método para controlar distintos ajustes dentro de un efecto o de un clip desde fuera del editor de clips.

* **range min / range max** - los valores mínimo y máximo a los que se asigna el nivel del parámetro
* **level** - el ajuste predeterminado para este parámetro. Este es el valor que se puede cambiar desde fuera. Puedes ajustarlo manualmente para ver el efecto que tendrá.
* **parameter number** - a cada _parameter control_ se le puede asignar un número, y esto se refleja en los controles fuera del clip. Puedes tener más de un parameter control con el mismo número, y todos se ajustarán al mismo tiempo.

{% hint style="warning" %}
Actualmente, solo el parámetro 1 de un efecto está disponible para controlarlo con el APC40. Todos los parámetros dentro de efectos y clips están disponibles en el menú contextual del botón derecho. En el futuro, más de estos parámetros podrán asignarse a controladores MIDI.
{% endhint %}
