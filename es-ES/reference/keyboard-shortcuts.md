# ✅ Atajos de teclado

Los atajos se muestran como `Cmd / Ctrl`: usa `Cmd` en macOS y `Ctrl` en Windows.

| Shortcut                          | Action                                                   |
| --------------------------------- | -------------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Armar todos los láseres                                  |
| `Escape`                          | Desarmar todos los láseres                               |
| `Tab`                             | Alternar vistas y recorrer los láseres                   |
| `Shift + Tab`                     | Alternar vistas y recorrer los láseres hacia atrás       |
| `1-9`                             | Saltar al láser                                          |
| `Left / Right Arrow`              | Mover Clip Deck a la izquierda / derecha                 |
| `Cmd / Ctrl + Left / Right Arrow` | Mover Clip Deck una página a la izquierda / derecha      |
| `Shift + Left / Right Arrow`      | Mover las zonas una página a la izquierda / derecha      |
| `Space`                           | Reproducir / pausar (cuando la fuente de tempo _Timeline_ está activa) |
| `T`                               | Marcar el tempo                                          |
| `R`                               | Reiniciar el compás                                      |
| `Y`                               | Redondear el tempo al BPM más cercano                    |
| `Cmd / Ctrl + ,`                  | Abrir el panel de ajustes                                |
| `Cmd / Ctrl + F`                  | Activar o desactivar el modo de pantalla completa        |
| `Cmd / Ctrl + L`                  | Abrir el panel de ajustes de salida láser                |
| `Cmd / Ctrl + K`                  | Abrir el panel de asignación de controladores láser      |
| `Cmd / Ctrl + J`                  | Abrir el panel de ajustes de 3D Visualiser               |
| `Cmd / Ctrl + Shift + Plus (+)`   | Aumentar el escalado de la interfaz                      |
| `Cmd / Ctrl + Shift + Minus (-)`  | Reducir el escalado de la interfaz                       |
| `Cmd / Ctrl + 0`                  | Restablecer el escalado de la interfaz                   |
| `Cmd / Ctrl + A`                  | Timeline: seleccionar todo                               |
| `Delete`                          | Timeline: eliminar los bloques seleccionados             |
| `Delete`                          | Editor de Clip: eliminar los nodos seleccionados         |
| `Cmd + Q` (macOS)                 | Salir de la app                                          |
| `Alt + F4` (Windows)              | Salir de la app                                          |
