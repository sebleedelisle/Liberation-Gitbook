# ✅ Iniciar / detener clips

{% hint style="info" %}
Puedes usar los botones en pantalla para iniciar y detener Clips, pero es mucho mejor usar el controlador MIDI APC40. Liberation está optimizado para funcionar bien con este hardware y debería considerarse una parte importante de tu inversión en láseres.
{% endhint %}

### Iniciar y detener clips

Para iniciar un clip, pulsa el botón de ese clip (en pantalla o en tu controlador MIDI).

Para detener el clip, púlsalo de nuevo.

Si inicias un clip nuevo (en el mismo grupo), el primer clip se detendrá automáticamente.

Para iniciar un clip nuevo sin detener otros clips, puedes hacer una de estas dos cosas:

* Mantén pulsada la tecla `Shift` (o el botón Shift del APC40) mientras inicias el clip nuevo; o
* Vuelve a pulsar el clip o los clips que estén en reproducción mientras inicias el clip nuevo.

Los grupos de clips son independientes entre sí, así que iniciar un clip en un grupo no afectará a los clips de ninguno de los demás. Consulta [Grupos de Clip](groups.md "mention")

### Flash mode

Si un clip está dentro de un grupo configurado en _flash mode_, se comporta de forma ligeramente distinta: el clip solo seguirá ejecutándose mientras mantengas pulsado su botón. En cuanto lo sueltes, se detendrá. (De forma predeterminada, el grupo de clips 3 —el rojo— está configurado en _flash mode_).

### Detener todos los clips

Para **detener todos los clips en reproducción**, pulsa el botón **STOP**.

{% hint style="info" %}
Pulsa el botón **STOP** **dos veces** para omitir el tiempo de fundido de salida de los clips y dejarlos en negro inmediatamente.
{% endhint %}

Para detener todos los clips en reproducción de un grupo concreto, pulsa el botón de ese grupo y después pulsa el botón **STOP**.

Para detener todos los clips excepto uno, pulsa el clip en reproducción que quieras conservar y, mientras lo mantienes pulsado, pulsa también el botón **STOP**. Después suelta el botón del clip. (Esto también funciona con varios clips: tantos como puedas mantener pulsados a la vez).

### El clip seleccionado actualmente

<figure><img src="../.gitbook/assets/clips-selected-active.png" alt="" width="269"><figcaption><p>Dos clips que se están reproduciendo actualmente. El contorno blanco indica que el clip de la derecha es el <em>clip seleccionado actualmente.</em></p></figcaption></figure>

El clip se ilumina en pantalla cuando se está reproduciendo (y su representación en el mini visualizador del clip parpadea). También verás que el último clip que has pulsado tiene un contorno blanco alrededor. Esto indica que es el _clip seleccionado actualmente._

Haz `Alt / Option`-clic en un clip para seleccionarlo sin activarlo. Usa `Alt / Option + Shift`-clic para seleccionar varios clips.

También puedes hacer clic y arrastrar un lazo sobre el Clip Deck para seleccionar varios clips.

{% hint style="info" %}
El APC40 también tiene botones `Alt` y `Shift` para hacer selecciones.
{% endhint %}
