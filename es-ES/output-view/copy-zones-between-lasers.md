# 🟩 Copiar zonas entre láseres

Haz clic en el botón _COPY LASER SETTINGS_ en _Laser Output_ y se abrirá un nuevo panel.

Esta función es muy útil y ahorra mucho tiempo para copiar rápidamente zonas y otros ajustes de un láser a los demás.

Asegúrate de haber seleccionado el láser _desde_ el que quieres copiar los ajustes.

Activa todos los demás láseres _a los que_ quieres copiar los ajustes. (Usa _ALL_ y _NONE_ para hacerlo más rápido si tienes muchos láseres)

Selecciona la opción _Copy output zone settings_ y después haz clic en el botón _COPY SETTINGS_ para completar el proceso.

{% hint style="info" %}
Ten en cuenta que este proceso no añadirá ni eliminará zonas de los láseres. Simplemente copiará la posición y el tipo de zona de cada zona que exista en los láseres de destino.
{% endhint %}

Consulta también [copy-laser-settings.md](../setting-up/copy-laser-settings.md "mention") para ver una descripción completa de las demás funciones de este sistema.
