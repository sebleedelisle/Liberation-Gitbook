# 🟩 Máscaras

{% hint style="danger" %}
**AVISO DE SEGURIDAD**

Las máscaras de software NUNCA deben ser tu único método para proteger a las personas de la salida de un láser. Debes contar siempre con barreras físicas o enmascaramiento entre el láser y cualquier posible exposición humana. Considera este sistema como un nivel secundario de redundancia o para aplicaciones no críticas para la seguridad, como proteger cámaras o proyectores, pero recuerda que cualquier software puede fallar.
{% endhint %}

***

Puedes añadir una o varias máscaras virtuales para eliminar o reducir la salida láser en zonas concretas.

Añade una máscara haciendo clic en el icono de la barra de herramientas; verás aparecer un cuadrado rojo en la vista de salida: esa es tu nueva máscara. Puedes moverla haciendo clic y arrastrando, y arrastrar las esquinas para cambiar su tamaño. Si quieres una forma no uniforme, pulsa `Alt / Option` mientras arrastras una esquina. Una vez que la máscara ya no sea uniforme, no tendrás que seguir pulsando la tecla `Alt / Option`.

Haz clic derecho sobre la máscara para ver ajustes adicionales:

* Porcentaje de reducción: representa la reducción de potencia del láser dentro de la máscara. Por defecto está configurado al 100%, es decir, apagado completo dentro de la máscara. Pero puedes reducir la potencia del láser en la cantidad que elijas. Ten en cuenta que, cuando no es una reducción completa, la máscara cambia de color a naranja.
* El botón DELETE MASK también está en este menú de clic derecho.
