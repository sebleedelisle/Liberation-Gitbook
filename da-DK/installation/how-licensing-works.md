# ✅ Sådan fungerer licensering

Når du har installeret Liberation på din computer, kører programmet i _demotilstand_, indtil du _autoriserer_ det.

{% hint style="info" %}
Der er ingen begrænsninger i _demotilstand_ bortset fra output til laserhardware. Du kan øve shows, opbygge timelines og gemme og indlæse dine filer uden begrænsninger. Du kan installere Liberation i _demotilstand_ på så mange computere, du vil, og du behøver først at autorisere din installation, når du er klar til at bruge lasere!
{% endhint %}

Ud over antallet af lasere, du kan armere, kan du også bruge DMX-fixtures. Se [dmx-control](../dmx-control/ "mention").

Der er også en grænse for, hvor mange computere du kan autorisere Liberation på, afhængigt af dit abonnement.

{% hint style="danger" %}
Hver licens er kun til én bruger og ét show ad gangen. Du kan have mere end én installation pr. bruger, fordi du **altid** bør have mindst én ekstra autoriseret computer klar!
{% endhint %}

### Internetforbindelse

Licensering og autorisation i Liberation foregår via internettet; appen opretter forbindelse til vores server for at knytte en installation til dit abonnement.

Når din installation er autoriseret, behøver den ikke at være online igen, før dit abonnement fornyes. Du kan se fornyelsesdatoen i _About panel._ Der er en kort henstandsperiode (7 dage for månedlige abonnementer), men det anbefales at gå online så hurtigt som muligt efter fornyelsesdatoen for at sikre, at din autorisation fornyes og forbliver aktiv.
