# ✅ Tastaturgenveje

Genveje er skrevet som `Cmd / Ctrl`: brug `Cmd` på macOS og `Ctrl` på Windows.

| Genvej                            | Handling                                              |
| --------------------------------- | ----------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Armér alle lasere                                     |
| `Escape`                          | Slå armering fra for alle lasere                      |
| `Tab`                             | Skift visning og gå frem gennem laserne               |
| `Shift + Tab`                     | Skift visning og gå baglæns gennem laserne            |
| `1-9`                             | Gå direkte til laser                                  |
| `Left / Right Arrow`              | Flyt Clip Deck til venstre / højre                    |
| `Cmd / Ctrl + Left / Right Arrow` | Flyt Clip Deck én side til venstre / højre            |
| `Shift + Left / Right Arrow`      | Flyt zoner én side til venstre / højre                |
| `Space`                           | Afspil / pause (når tempokilden _Timeline_ er aktiv)  |
| `T`                               | Tap tempo                                             |
| `R`                               | Nulstil takt                                          |
| `Y`                               | Afrund tempo til nærmeste BPM                         |
| `Cmd / Ctrl + ,`                  | Åbn indstillingspanelet                               |
| `Cmd / Ctrl + F`                  | Slå fuldskærmstilstand til/fra                        |
| `Cmd / Ctrl + L`                  | Åbn panelet med indstillinger for laser-output        |
| `Cmd / Ctrl + K`                  | Åbn panelet til tildeling af laser-controller         |
| `Cmd / Ctrl + J`                  | Åbn indstillingspanelet for 3D Visualiser             |
| `Cmd / Ctrl + Shift + Plus (+)`   | Forøg skalering af brugerfladen                       |
| `Cmd / Ctrl + Shift + Minus (-)`  | Formindsk skalering af brugerfladen                   |
| `Cmd / Ctrl + 0`                  | Nulstil skalering af brugerfladen                     |
| `Cmd / Ctrl + A`                  | Timeline: vælg alle                                   |
| `Delete`                          | Timeline: slet markerede blokke                       |
| `Delete`                          | Clip editor: slet markerede noder                     |
| `Cmd + Q` (macOS)                 | Afslut appen                                          |
| `Alt + F4` (Windows)              | Afslut appen                                          |
