# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_Parameter Control_ er en metode til at styre forskellige indstillinger i en effekt eller et Clip uden for clip-editoren.

* **range min / range max** - de minimums- og maksimumværdier, som parameterens niveau mappes til
* **level** - standardindstillingen for denne parameter. Dette er værdien, der kan ændres udefra. Du kan justere den manuelt for at se, hvilken effekt den har.
* **parameter number** - hvert _parameter control_ kan tildeles et nummer, og dette afspejles i kontrollerne uden for clippet. Du kan have mere end én parameter control med samme nummer, og de justeres alle på samme tid.

{% hint style="warning" %}
I øjeblikket er kun parameter 1 for en effekt tilgængelig til styring med APC40. Alle parametre i effekter og Clips er tilgængelige i højreklikmenuen. I fremtiden vil flere af disse parametre kunne mappes med MIDI-controllere.
{% endhint %}
