# 🟩 Canvas settings

Нажмите кнопку _Canvas settings_ в верхней части левой панели инструментов, чтобы открыть панель _Canvas settings_.

### Width and Height

Используйте эти перетаскиваемые ползунки, чтобы изменить общую ширину и высоту всего Canvas. Эти размеры можно условно считать размером в пикселях, но всё здесь относительно.

Дважды щёлкните по ползунку, чтобы ввести значение.

{% hint style="info" %}
Немного дополнительных технических подробностей: лазер перемещается относительно разрешения Canvas, а не относительно размера zone. Это связано с тем, что большая zone на Canvas, скорее всего, будет проецироваться на большую область в реальности.

Это небольшая деталь, которая, скорее всего, не будет для вас критичной, но она помогает выровнять производительность и видимость во всех canvas zones.
{% endhint %}

### Zone brightness

Используйте этот параметр, чтобы изменить яркость розовых контуров zones в Canvas view. Это помогает сделать содержимое понятнее и не перегружать этот вид.
