# 🟩 Целевые области Canvas

Мы уже знаем, как отправлять части Canvas в zones каждого лазера, но чтобы сначала разместить контент в Canvas, вам понадобятся _целевые области Canvas_ — название немного сбивает с толку, но точно описывает функцию.

_Целевые области Canvas_ — это участки Canvas, в которые можно выводить Clips. В представлении _CANVAS_ они отображаются синими прямоугольными контурами.

Во многих случаях достаточно одной целевой области Canvas, которую затем можно разделить на несколько zones и отправить на разные лазеры.

Иногда может понадобиться несколько целевых областей Canvas: например, для разных частей здания или чтобы использовать между ними zone delay. Да, zone delay работает и между целевыми областями Canvas!

### Отправка Clips в целевые области Canvas

Если посмотреть в Clip Deck, рядом с кнопками beam zone вы увидите кнопки целевых областей Canvas. Возможно, понадобится прокрутить кнопки выходов, чтобы увидеть их: используйте `Shift + Left / Right Arrow`, кнопки ZONE PAGE на экране или кнопки APC40 (см. [справочник по APC40](../reference/apc40-reference.md "mention")).

Назначайте Clips на целевые области Canvas, включая и выключая эти кнопки точно так же, как кнопки beam zone.

### Добавление и редактирование целевых областей Canvas

В верхней строке меню выберите _View -> Canvas Target Areas_ — вы увидите все настройки для каждой целевой области Canvas в вашем проекте.

Вверху находится кнопка _ADD CANVAS TARGET AREA_.

Удалить целевую область Canvas можно красной кнопкой со знаком минус.

Настройте размер и положение с помощью ползунков. Дважды щёлкните ползунок, чтобы ввести значение.

### Режим масштабирования

* **FIT TO AREA** — уменьшает контент так, чтобы он полностью помещался внутри целевой области Canvas, сохраняя пропорции. Это настройка по умолчанию.
* **FILL AREA** — масштабирует контент так, чтобы он заполнил целевую область Canvas, сохраняя пропорции. Часть контента может быть обрезана по краям.
* **STRETCH TO FIT** — растягивает контент на всю целевую область Canvas без сохранения пропорций.
