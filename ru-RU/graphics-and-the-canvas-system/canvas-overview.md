# 🟩 Обзор Canvas

Система Canvas в Liberation устроена довольно просто, но поначалу может сбивать с толку. Ниже — общий обзор, который поможет начать работу.

{% hint style="info" %}
**Подождите, а мне вообще нужна система Canvas?**

Возможно, нет! Если вы просто проецируете одну графику на один лазер, это легко сделать через beam zone. Но учтите: по умолчанию содержимое beam zone отражается по горизонтали, поэтому для Clip нужно включить X flip.

Но если вы хотите распределить графический контент между несколькими лазерами или разделить его на разные секции для мэппинга на архитектуру, система Canvas как раз для этого и предназначена.
{% endhint %}

### Canvas

Прежде всего, есть сам Canvas. Это то, что вы видите в представлении _CANVAS_: большая рабочая область, своего рода холст, внутри которого можно размещать контент где угодно.

### Целевые области Canvas

В Canvas view они отображаются синими прямоугольными контурами. Это области, в которые можно отправлять контент. Вы отправляете содержимое Clip в целевую область Canvas так же, как отправляете Clip в beam zone. Кнопки целевых областей Canvas находятся в Clip Deck справа от кнопок beam zone.

{% hint style="info" %}
Если вы не видите кнопки Canvas в Clip Deck, попробуйте прокрутить кнопки beam zone с помощью `Shift + Left / Right Arrow`. Вы должны увидеть кнопку для каждой целевой области Canvas с подписью _CANVAS 1, CANVAS 2_ и так далее.
{% endhint %}

### Canvas zones

Canvas zones — это области внутри Canvas, которые вы выбираете для отправки на лазер. В Canvas view они отображаются розовыми прямоугольными контурами. Можно щёлкнуть правой кнопкой по каждой zone и выбрать лазеры, которым она должна быть назначена. Если затем переключиться в _OUTPUT_ view для этого лазера, вы увидите, что появилась новая zone.

{% hint style="danger" %}
ВНИМАНИЕ: если лазер находится в состоянии armed, он может внезапно начать проецировать контент в canvas zone по умолчанию. Лучше перевести лазер в состояние disarmed перед назначением ему canvas zones.
{% endhint %}

{% hint style="info" %}
Canvas zone также можно назначить лазеру, нажав кнопку _add canvas zone_ в _OUTPUT_ view. См. [Zones](../output-view/zones.md).
{% endhint %}

### Направляющие изображения

В Canvas можно добавить направляющее изображение и использовать его как шаблон для графики. Рекомендуется настроить цветовой оттенок направляющего изображения через меню правой кнопки мыши и затемнить его, чтобы поверх него было проще видеть ваш контент.

{% hint style="info" %}
Для архитектурного мэппинга мне было удобно подготовить «развёрнутое» изображение здания, где все поверхности представлены как плоская неискажённая картинка. Перспективную коррекцию для отдельных секций можно выполнить с помощью canvas zone в _OUTPUT_ view.
{% endhint %}

### Canvas в 3D-визуализаторе

Воссоздавать сложную проекционную систему с несколькими лазерами в 3D-визуализаторе было бы, мягко говоря, неудобно. Поэтому вместо этого можно разместить Canvas в 3D-пространстве. Включите флажок _Show canvas_ на панели _3D visualiser settings_. Все направляющие изображения, которые есть в Canvas, также появятся в визуализаторе.

{% hint style="info" %}
Обратите внимание: визуализатор всё равно будет показывать проекции Canvas как атмосферные эффекты, исходящие от лазеров. Их можно просто сдвинуть за пределы вида или, если хотите сделать всё аккуратно, совместить с Canvas.
{% endhint %}
