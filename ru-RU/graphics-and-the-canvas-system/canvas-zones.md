# 🟩 Canvas zones

Canvas zone отображается в Canvas view как розовый прямоугольный контур. Она используется для выбора прямоугольной области на Canvas и отправки её содержимого на один или несколько лазеров.

Зоны можно перетаскивать мышью. Чтобы изменить размер и форму, перетаскивайте углы.

### Добавление новой Canvas zone

Нажмите кнопку _Add a new canvas zone_ на панели инструментов.

### Назначение Canvas zone лазеру

Щёлкните по зоне правой кнопкой мыши и включите кнопки с номерами, чтобы выбрать, каким лазерам будет назначена эта Canvas zone.

{% hint style="danger" %}
ПРЕДУПРЕЖДЕНИЕ: если лазер находится в состоянии armed, вы можете внезапно начать проецировать содержимое в стандартной Canvas zone. Лучше перевести лазер в состояние disarmed перед назначением ему Canvas zones.
{% endhint %}

Если теперь перейти в _OUTPUT_ view для этого лазера, вы увидите там Canvas zone. Её можно редактировать точно так же, как любую beam zone. См. [Zones](../output-view/zones.md "mention").

{% hint style="info" %}
Canvas zone также можно добавить к лазеру напрямую из _OUTPUT_ view: нажмите кнопку _Add existing canvas zone_ на левой панели инструментов.
{% endhint %}

### Меню правой кнопки мыши

Щёлкните по зоне правой кнопкой мыши, и вы увидите параметры для удаления этой Canvas zone, перемещения её на задний план и блокировки.
