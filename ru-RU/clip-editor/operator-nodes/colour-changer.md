# 🟩 Colour change

## <img src="../../.gitbook/assets/image.png" alt="" data-size="line"> Colour change

Изменяет цвета всего входящего содержимого. Можно задать фиксированные значения HSB или переключиться на систему градиента и брать цвета из пользовательского градиента.

* **hue, saturation, brightness** — значения цвета, см. [Настройки цвета и HSB](../fundamentals/colour-settings-and-hsb.md "mention")
* **hue mode** —
  * OFF — тон не изменяется
  * FIXED — тон элементов устанавливается на значение hue
  * SHIFT — тон элементов смещается на указанное значение, поэтому элементы разных цветов останутся разными, но будут сдвинуты по шкале тона.
* **saturation mode** —
  * OFF — насыщенность не изменяется
  * FIXED — насыщенность фиксируется на указанном значении.
* **brightness mode** —
  * OFF — яркость не изменяется
  * FIXED — яркость элементов устанавливается на значение brightness
  * MULTIPLY — яркость элементов комбинируется со значением brightness, поэтому если они мигают, они продолжат мигать, но только до яркости, указанной здесь.
* **gradient mode** — переключает фиксированные ползунки HSB на редактор градиента. node берет один цвет из градиента, а затем применяет его с помощью режимов оттенка, насыщенности и яркости, описанных выше.
* **gradient position** — выбирает точку градиента, из которой берется цвет. Анимируйте это значение от 0% до 100% с помощью Sawtooth Oscillator, чтобы со временем проходить по градиенту циклически.
* **blend** — насколько сильно применяется Colour change: 0% — совсем не применяется, 100% — применяется полностью, а 50% — это сочетание существующего цвета и новых значений.

{% hint style="info" %}
node Colour Change берет один цвет из градиента для всего входного содержимого. Если нужно, чтобы градиент проходил по форме в зависимости от положения, используйте [изменения на основе положения](position-based-changers.md "mention").
{% endhint %}

### Редактор градиента

Когда включен **gradient mode**, редактор градиента появляется под основными элементами управления.

* Нажмите на полосу градиента, чтобы добавить цветовую точку.
* Нажмите левой кнопкой мыши на точку, чтобы выбрать ее, затем перетащите в сторону, чтобы переместить.
* Чтобы удалить выбранную точку, перетащите ее вниз от полосы или нажмите Delete/Backspace. В градиенте всегда остается как минимум две точки.
* Нажмите правой кнопкой мыши на точку, чтобы изменить ее в палитре выбора цвета.
* Используйте **Position**, **Hue**, **Saturation** и **Brightness**, чтобы точно настроить выбранную точку.
* **interpolation** выбирает способ смешивания цветов между точками:
* **HSB** — смешивает оттенок, насыщенность и яркость. Лучше всего подходит для плавного радужного движения по цветовому кругу.
* **RGB** — напрямую смешивает значения красного, зеленого и синего. Часто воспринимается как цветовой переход на экране или световом пульте.
* **NONE** — переключает цвет от одной точки к следующей без смешивания.
* **hue direction** доступен при интерполяции HSB:
* **AUTO** — выбирает кратчайший путь по кругу оттенков.
* **FORWARDS** — всегда движется вперед по значениям оттенка.
* **BACKWARDS** — всегда движется назад по значениям оттенка.
