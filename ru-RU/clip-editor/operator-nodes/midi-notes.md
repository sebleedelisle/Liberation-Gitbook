# 🟩 MIDI notes

## MIDI notes

Создаёт эффекты в стиле «лазерной арфы», где входящие MIDI-ноты запускают лучи или фигуры в заданном диапазоне. Node использует любой контент, который вы подаёте на вход, как _источник_ для каждой ноты: подайте точку — получите ряд точек. Подайте фигуру, например круг, — получите ряд кругов; более сложные фигуры будут точно так же дублироваться.

Вы можете выбрать, какой MIDI-интерфейс Liberation будет слушать, в **Liberation → Settings** (`Cmd / Ctrl + ,`)

* **midi channel** – MIDI-канал для приёма данных (0 = все каналы, 1–16 = конкретный канал)
* **width** – общая ширина, на которую распределяются ноты.
* **midi note min / max** – минимальное и максимальное значения MIDI-нот в диапазоне.
* **ignore out of range notes** – отфильтровывает все ноты за пределами заданного диапазона. Если параметр отключён, ноты вне диапазона «прижимаются» к ближайшей доступной ноте: высокие ноты запускают верх диапазона, низкие — низ диапазона.
* **auto extend range** – автоматически расширяет диапазон, если играются ноты за его пределами.

{% hint style="info" %}
Не уверены, какой диапазон нот поступает? Включите **auto extend range**, задайте для **midi note min** очень высокое значение, а для **midi note max** очень низкое, затем сыграйте нужные ноты. Система поймает их все и расширит диапазон за вас. Когда диапазон будет настроен, просто выключите **auto extend range**, чтобы зафиксировать его.
{% endhint %}

* **leave all notes visible** – создаёт лучи или фигуры для всех нот в диапазоне, независимо от того, звучат они сейчас или нет, создавая эффект «лазерной арфы».
* **hit colour** – цвет, который появляется при срабатывании ноты.
* **hit colour hold time** – как долго цвет срабатывания остаётся на полной яркости перед затуханием. Значение указывается в секундах (0–1). _100% = 1 секунда._
* **hit colour decay time** – сколько времени занимает затухание цвета срабатывания после времени удержания. Значение указывается в секундах (0–1). _100% = 1 секунда._

{% hint style="info" %}
Если ваш контент уже чисто белый, установка белого цвета для **hit colour** ничего не изменит. Для лучшего результата используйте насыщенный цвет контента и задайте для **hit colour** белый — так при срабатывании нот получится красивый эффект вспышки.
{% endhint %}

* **note off fade out time** – сколько времени занимает затухание ноты после отпускания. Значение указывается в секундах (0–1). _100% = 1 секунда._
* **hit scale factor** – насколько увеличивается нота при срабатывании (например, 2 = двойной размер).
* **hit scale hold time** – как долго нота остаётся увеличенной перед возвратом к обычному размеру. Значение указывается в секундах (0–1). _100% = 1 секунда._
* **hit scale decay time** – сколько времени занимает возврат ноты к исходному размеру. Значение указывается в секундах (0–1). _100% = 1 секунда._
* **note off shrink time** – сколько времени занимает возврат к исходному размеру после отпускания ноты. Значение указывается в секундах (0–1). _100% = 1 секунда._ (Не действует, когда включён **leave all notes visible**.)

{% hint style="info" %}
Масштабирование: обратите внимание, что если ваш контент — одиночная точка, масштабирование не даст эффекта!
{% endhint %}
