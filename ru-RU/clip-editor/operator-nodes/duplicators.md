# 🟩 Duplicators

## <img src="../../.gitbook/assets/image (5).png" alt="" data-size="line"> Mirror

Создаёт зеркальную копию всего содержимого. По умолчанию ось отражения — вертикальная линия по центру.

* **angle** — угол линии оси отражения.
* **offset position** — смещает ось отражения (перемещает её перпендикулярно оси).
* **delay** — задержка по времени для отражённого содержимого. Обратите внимание: параметр будет заметен только если содержимое имеет какую-либо анимацию.

#### Параметры 3D (доступны, когда выбрано 3D)

* **angle X / angle Y** — ось отражения становится плоскостью, и эти параметры позволяют поворачивать её в 3D.

## <img src="../../.gitbook/assets/image (6).png" alt="" data-size="line"> Circular Pattern

Дублирует содержимое по круговому шаблону.

* **radius** — расстояние от центральной точки, на которое содержимое смещается перед поворотом.
* **count** — количество создаваемых копий объекта.
* **use each objects pivot point** — если включено, каждый элемент будет смещаться и поворачиваться вокруг собственного центра. (Влияет только при дублировании нескольких элементов.)
* **delay** — добавляет каждому следующему дублированному элементу всё большую задержку по времени. Обратите внимание: чтобы эффект был заметен, содержимое должно иметь какую-либо анимацию.
* **rotation** — дополнительный поворот, применяемый к элементам.

#### Параметры 3D (доступны, когда выбрано 3D)

* **rotation x / rotation y** — поворачивает весь круговой шаблон вокруг осей x и y.

## <img src="../../.gitbook/assets/image (7).png" alt="" data-size="line"> Grid Pattern

Дублирует содержимое в виде сетки из строк и столбцов.

* **spacing** — расстояние между элементами.
* **count X**- количество копий по горизонтали (столбцы).
* **count Y**- количество копий по вертикали (строки).
* **horizontal alignment** — начальная точка столбцов: LEFT, CENTRE или RIGHT.
* **vertical alignment** — начальная точка строк: TOP, MIDDLE или BOTTOM.
* **delay** — добавляет каждому следующему дублированному элементу всё большую задержку по времени. Обратите внимание: чтобы эффект был заметен, содержимое должно иметь какую-либо анимацию.
