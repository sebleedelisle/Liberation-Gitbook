# 🟩 Модификаторы пути

## &#x20;Dotter

Этот node заменяет линии и фигуры равномерно расположенными точками (существующие точки не изменяются).

* **Colour** — цвет точек. Игнорируется, если включен _Inherit Colour_; см. ниже. _См. также_ [Настройки цвета и HSB](../fundamentals/colour-settings-and-hsb.md)
* **Spacing** — расстояние между точками в пикселях. Меньшие значения = больше точек, большие значения = меньше точек.
* **Offset** — смещает начальную позицию точек в процентах от значения Spacing. Можно анимировать (например, с помощью пилообразного Oscillator Node), чтобы создавать эффекты «движущихся» точек.
* **Keep Original** — если включено, исходные линии/фигуры сохраняются, а точки рисуются поверх них.
* **Render Profile** — выбирает качество рендеринга. _См._ [Профиль рендеринга](../fundamentals/render-profile.md)
* **Length Auto Divisible by Spacing** — автоматически подстраивает Spacing так, чтобы длина пути делилась на него без остатка.
* **Fade Out Ends** — постепенно уменьшает яркость точек к началу и концу пути. Полезно при анимации **Offset** с помощью пилообразного Oscillator Node: точки будут плавно появляться и исчезать при движении к концу фигуры.

## &#x20;Trimmer

Этот node обрезает видимую длину линий и фигур, позволяя постепенно показывать, скрывать или анимировать их во времени.

* **Offset** — управляет тем, где начинается видимая часть фигуры. Даже если _Trim Size_ установлен на 0%, анимация Offset от 0% → 100% заставляет фигуру прорисоваться, стать полностью видимой на 50%, а затем снова исчезнуть.
* **Trim Size** — задает, какая часть фигуры сохраняется, в процентах от ее общей длины.
* **Loop** — обрабатывает фигуру как непрерывный цикл: конец соединяется с началом, а не исчезает.
* **All Shapes** — объединяет все входные фигуры и обрезает их как единый путь. Если выключено, каждая фигура обрезается отдельно.
* **Add Dot at Start / Add Dot at End** — добавляет точку выбранного цвета в точках обрезки. (Если обрезка не применяется, точки не добавляются.)
* **Colour** — цвет точек обрезки. _См. также_ [Настройки цвета и HSB](../fundamentals/colour-settings-and-hsb.md)
* **Render Profile** — выбирает профиль рендеринга для точек. _См._ [Профиль рендеринга](../fundamentals/render-profile.md)
