# 🟩 Преобразования

## &#x20;Перемещение

Перемещает всё содержимое по осям x, y и/или z. Обратите внимание, что система координат центрирована и по осям x и y простирается от -200 до +200. См. [Система координат](../fundamentals/co-ordinate-system.md).

* **x** — расстояние перемещения по оси x (слева направо).
* **y** — расстояние перемещения по оси y (сверху вниз).

#### Параметры 3D (доступны, когда выбран режим 3D)

* **z** — расстояние перемещения по оси z (назад и вперёд в глубину экрана).

## <img src="../../.gitbook/assets/image (3).png" alt="" data-size="line"> Поворот

Поворачивает всё содержимое. Значения указываются в градусах. См. [Система координат](../fundamentals/co-ordinate-system.md).

* **rotation** — величина поворота содержимого по часовой стрелке в градусах. Всё содержимое поворачивается вокруг начала координат (0,0), то есть центра.
* **pivot point x / pivot point y** — используйте эти значения, чтобы сместить центр поворота.

#### Параметры 3D (доступны, когда выбран режим 3D)

* **rotation x** — поворот вокруг оси x (pitch).
* **rotation y** — поворот вокруг оси y (yaw).
* **pivot point z** — смещение точки поворота по оси z.

## <img src="../../.gitbook/assets/image (4).png" alt="" data-size="line"> Масштабирование

Масштабирует всё содержимое.

* **scale** — процент масштаба.
* **scale x / scale y** — используйте эти параметры, если нужно масштабировать содержимое по горизонтали и/или вертикали.

{% hint style="warning" %}
Если масштаб по любой оси задан как 0%, содержимое исчезает!
{% endhint %}
