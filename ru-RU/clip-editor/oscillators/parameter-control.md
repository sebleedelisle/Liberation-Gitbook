# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_Parameter Control_ — это способ управлять различными настройками внутри эффекта или Clip за пределами Clip Editor.

* **range min / range max** — минимальное и максимальное значения, на которые отображается уровень параметра.
* **level** — значение по умолчанию для этого параметра. Именно это значение можно менять извне. Его можно настроить вручную, чтобы увидеть, как оно влияет на результат.
* **parameter number** — каждому _parameter control_ можно назначить номер, и этот номер отображается в элементах управления за пределами Clip. Можно использовать несколько parameter control с одним и тем же номером — в этом случае они будут изменяться одновременно.

{% hint style="warning" %}
Сейчас через APC40 можно управлять только параметром 1 эффекта. Все параметры внутри эффектов и Clips доступны в меню по правому щелчку. В будущем больше таких параметров можно будет привязывать к MIDI-контроллерам.
{% endhint %}
