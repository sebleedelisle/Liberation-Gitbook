# 🟩 Загрузка и сохранение

Liberation постоянно сохраняет своё состояние на диск, поэтому при отключении питания или системной ошибке приложение запустится с того же места, где остановилось. Вы не должны потерять свои zones, таймлайны или другой контент.

При этом вы можете экспортировать свою конфигурацию для резервного копирования или переноса на другой компьютер.

### Импорт/экспорт проекта

Файл проекта хранит почти всё из текущей конфигурации, включая:

* Всё, что перечислено ниже в разделе [Импорт/экспорт Laser Settings](loading-and-saving.md#laser-settings-import-export "mention")
* Clips, эффекты и настройки групп
* Все ваши таймлайны (без аудио- и видеомедиа)
* Настройки Art-Net
* Настройки отправки/приёма MIDI
* Настройки темпа и синхронизации

Сейчас он не сохраняет и не загружает:

* Настройки звукового и MIDI-входа, используемые в MIDI notes node и Sound Input Oscillator (при этом настройки отправки/приёма MIDI, а также звуковой вход тайм-кода _сохраняются_)
* Масштаб интерфейса
* Медиафайлы для направляющих изображений Canvas
* Аудио- и видеомедиа для таймлайнов
* Шрифты, используемые в Text node

{% hint style="danger" %}
Аудио- и видеофайлы в таймлайне не сохраняются вместе с файлами проекта, поэтому сохраните их отдельно, если хотите перенести проект на другой компьютер. См. [Важное примечание о медиафайлах таймлайна](loading-and-saving.md#important-note-about-timeline-media-files "mention")
{% endhint %}

### Импорт/экспорт Laser Settings

* Laser Settings для каждого лазера
* Beam zones
* Целевые области Canvas
* DMX zones
* Назначения laser controller (и псевдонимы для любых controller, которые вы переименовали)
* Настройки и пресеты сканера лазера и калибровки цвета
* Настройки и пресеты визуализатора 3D

### Импорт/экспорт Clip Deck

* Все Clips, назначенные им zones, настройки и параметры
* Все настройки групп, flash mode, времена fade in/out и т. д.

Сейчас он не сохраняет и не загружает:

* Все эффекты, их параметры и настройки

{% hint style="info" %}
**Загрузить Clips из файла проекта без загрузки всего проекта**

Чтобы импортировать из проекта только Clips, выберите _**Clips->Import Clip Deck**_ и вместо файла Clip Deck (.cpdk) выберите файл проекта.
{% endhint %}

### Добавление Clip Deck

Вы можете добавить Clips из экспортированного файла Clip Deck в текущий проект с помощью _Append Clip Deck_. Clips будут добавлены в конец текущего Clip Deck, но эффекты и настройки групп из файла импортированы не будут.

### Export Selected Clips

Все выбранные в данный момент Clips будут экспортированы в файл. Настройки групп и эффекты не сохраняются — только Clips. Обратите внимание: активные Clips, которые сейчас запущены, не экспортируются, если они также не выбраны.

{% hint style="info" %}
Чтобы выбрать Clips, нажимайте по ним с удержанием Option/Alt-Shift (или используйте лассо). Выбранные Clips можно определить по толстой белой обводке вокруг них. См. [Запуск и остановка Clips](clips/starting-stopping-clips.md "mention")
{% endhint %}

### Импорт/экспорт эффектов

Загружает и сохраняет все эффекты вместе с их настройками групп и параметрами.

{% hint style="info" %}
**Загрузить эффекты из файла проекта без загрузки всего проекта**

Чтобы импортировать из проекта только эффекты, выберите _**Effects->Import Effects**_ и вместо файла эффектов (.efts) выберите файл проекта.
{% endhint %}

### Экспорт таймлайна

Экспортирует файл таймлайна с одним или несколькими таймлайнами. Обратите внимание: Clip Deck всегда включается в экспортированные файлы таймлайна (хотя при обратном импорте можно выбрать, какие Clips импортировать; см. [Импорт таймлайна](loading-and-saving.md#timeline-import "mention"))

Если в файле проекта больше одного таймлайна, откроется панель, где можно выбрать таймлайны для экспорта.

{% hint style="danger" %}
Аудио- и видеофайлы в таймлайне не сохраняются вместе с файлами таймлайна, поэтому сохраните их отдельно, если хотите перенести контент на другой компьютер. См. [Важное примечание о медиафайлах таймлайна](loading-and-saving.md#important-note-about-timeline-media-files "mention")
{% endhint %}

### Импорт таймлайна

Импортирует один или несколько таймлайнов из одного файла таймлайна. После выбора файла таймлайна откроется панель с несколькими вариантами импорта.

Если файл таймлайна содержит больше одного таймлайна, все они будут показаны в списке. Отметьте те, которые хотите включить.

* Replace existing timelines\
  Удалит все текущие таймлайны и заменит их импортированными
* Import used clips only\
  Импортирует только используемые Clips и распределит Clips по группам — по одной группе для каждого таймлайна. Если этот параметр не выбран, весь Clip Deck из файла таймлайна будет добавлен к существующим Clips.
* Replace existing clip deck\
  Заменяет текущий Clip Deck на Clips из файла таймлайна. Доступно только если выбран параметр _Replace existing timelines_.

{% hint style="info" %}
**Загрузить таймлайны из файла проекта без загрузки всего проекта**

Чтобы импортировать из проекта только таймлайны, выберите _**Timeline->Import Timeline(s)**_ и вместо файла таймлайна (.ltml) выберите файл проекта.
{% endhint %}

### Импорт/экспорт DMX / Art-Net

Сохраняет и загружает Art-Net nodes вместе с их IP-адресами. Также включает DMX zones и все ваши DMX-пресеты.

### Важное примечание о медиафайлах таймлайна

Аудио- и видеофайлы сейчас **не** экспортируются вместе с файлом таймлайна, поэтому если нужно перенести контент на другой компьютер, обязательно включите их отдельно.

{% hint style="info" %}
**Как таймлайн ищет медиафайлы**

При загрузке таймлайн будет искать файлы в той же папке, где находится файл таймлайна (или проекта), а также во всех её подпапках. Поэтому если файлы лежат в той же папке или в подпапке (например, _/videos_ или _/sound_), они будут найдены при загрузке.
{% endhint %}
