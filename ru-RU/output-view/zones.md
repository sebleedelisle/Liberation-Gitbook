# 🟩 Zones

Основной тип zone, который вы будете использовать в большинстве проектов, — _Beam zone_. Этот тип zone предназначен для атмосферных лучевых эффектов в воздухе. Другой тип — _Canvas zone_ (см. [Графика и система Canvas](../graphics-and-the-canvas-system/ "mention")).

{% hint style="danger" %}
**ПРЕДУПРЕЖДЕНИЕ — соблюдайте крайнюю осторожность при перемещении zones во время работы лазера** и уменьшите яркость до минимально возможного уровня. Подробное руководство по безопасному включению лазеров и настройке zones см. в разделе [Настройка лазеров](../setting-up/setting-up-lasers.md "mention")
{% endhint %}

Zones можно перетаскивать мышью. Включите test pattern, чтобы увидеть, куда попадает эта zone.

{% hint style="info" %}
Используйте клавиши со стрелками, чтобы **слегка сдвигать** выбранную zone или точку. Удерживайте `Shift`, чтобы сдвигать более крупными шагами.
{% endhint %}

{% hint style="info" %}
Полезный совет: вы можете быстро копировать настройки zones между несколькими лазерами! См. [Копирование Laser Settings](../setting-up/copy-laser-settings.md "mention")
{% endhint %}

### Добавление новой Beam zone

Нажмите кнопку _Add a new beam zone_ в верхней части панели инструментов — появится новая zone. Обратите внимание: beam zones сортируются в порядке добавления, но этот порядок можно изменить. См. [Изменение порядка Beam zones](re-ordering-beam-zones.md "mention")

### Добавление существующей Canvas zone

Нажмите кнопку _Add existing canvas zone_, и вы увидите список доступных canvas zones. Для этого лазера их можно включать и выключать. См. [Графика и система Canvas](../graphics-and-the-canvas-system/ "mention")

### Типы формы zone

Есть 3 типа формы zone:

* **Quad** — стандартная прямоугольная форма zone, которая может быть обычной (выровненной по осям) или искажённой. Лучше всего подходит для крупных прямоугольных zones или canvas zones, где нужна коррекция перспективы.
* **Line/Curve** — zone, заданная 2 или более точками и толщиной. Идеально подходит для узких zones или для ограничения лучей по балконам, мостам и другим изогнутым формам.
* **Segmented** — zone, которую можно разделить на более мелкие четырёхугольники. Идеально подходит для архитектурного мэппинга.

Щёлкните правой кнопкой мыши по любой zone, чтобы открыть её настройки. В этом контекстном меню можно:

* Переименовать zone (это помогает отличать её в Clip Deck, особенно если zones много!)
* Включить или отключить zone
* Заблокировать её положение
* Изменить тип формы
* Сбросить её в положение по умолчанию
* Открыть настройки, относящиеся к выбранному типу формы
* Удалить её
* Добавить _Alt Zone_ (см. [Система Alt Zone](alt-zone-system.md "mention"))

{% hint style="danger" %}
**ПРЕДУПРЕЖДЕНИЕ —** будьте очень осторожны при изменении типа zone, когда лазер активен. Zone вернётся к последнему положению и размеру для этой формы, поэтому Output может резко измениться. Лучше выключить лазер перед изменением типа zone.
{% endhint %}

### Форма Quad zone

Каждый угол Quad можно перемещать мышью. Щёлкните по углу с `Alt / Option`, чтобы переместить его независимо от остальных и исказить Quad. После искажения Quad все углы можно перемещать свободно.

Убрать искажение и вернуть форму к прямоугольнику, выровненному по осям, можно кнопкой _REMOVE DISTORTION_ в контекстном меню.

#### Коррекция перспективы

Эта опция задаётся переключателем в контекстном меню и определяет метод искажения. Для лучей её обычно лучше оставить выключенной, но если эта zone проецирует графику на плоскую поверхность, включите её — Output будет скорректирован с учётом перспективы.

{% hint style="info" %}
Если _Perspective correction_ выключена, контент искажается с помощью _би-линейной интерполяции_. Другими словами, контент равномерно распределяется по Quad. Поэтому этот режим лучше подходит для лучей.

Когда коррекция перспективы включена, контент искажается с помощью перспективного преобразования, которое учитывает перспективное сокращение. Поэтому, если вы проецируете графику на стену под косым углом, эта опция позволяет выровнять Output и исправить искажение проекции.
{% endhint %}

### Форма Line / Curve zone

Форма Line / Curve zone стала моим основным вариантом в последних шоу, и можно сказать, что именно она должна быть формой по умолчанию для beam zones.

Чаще всего мои zones должны быть узкими, чтобы помещаться в неудобные тонкие пространства на площадках или между окнами зданий. Я обнаружил, что настраивать четыре угла Quad, когда они расположены так близко друг к другу, бывает очень неудобно. Так и появилась Line / Curve zone!

Для прямых линий нужны всего две точки, а затем достаточно настроить _Zone thickness_ в контекстном меню. Это самый быстрый способ создать простые zones.

Щёлкните по линии с `Alt / Option`, чтобы создать дополнительные точки. Эти точки автоматически сглаживаются, формируя плавную линию; параметром _Smooth level_ можно убрать перегибы.

Щёлкните по точке с `Alt / Option`, чтобы удалить её.

Если у вас есть опыт работы с векторными редакторами (Inkscape, Illustrator и т. п.), можно использовать опцию _Manually adjust bezier curves_ для точной ручной настройки всех контрольных точек.

### Форма Segmented zone

Эта subdivided zone позволяет выполнять очень детальную коррекцию и полезна при мэппинге на сложные формы. Добавлять и удалять подразделения можно кнопками + и - в контекстном меню.

### Как редактировать zone, полностью закрытую другой zone

Щёлкните правой кнопкой мыши по верхней zone и нажмите кнопку с замком, чтобы заблокировать её. После этого вы сможете редактировать и настраивать zone под ней.

<br>

{% hint style="info" %}
После добавления Beam zone в Output она станет доступна для добавления в Clip в Clip Deck.
{% endhint %}
