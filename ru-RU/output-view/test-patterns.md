# 🟩 Test patterns

При редактировании и настройке zone полезно вывести test pattern. Включить test pattern для всех включённых zones можно переключателем на верхней панели значков, а выбрать нужный test pattern — ползунком справа от него.

Если test pattern нужен только на лазере, для которого вы сейчас настраиваете zone, используйте переключатель на панели инструментов слева в _Output_ view. Чтобы выбрать test pattern, нажмите на число под ним и перетащите его.

### Список test patterns

Ниже приведён полный список test patterns. Если не указано иное, pattern заполняет всё пространство вывода каждой zone.

1. Default pattern — зелёный прямоугольник с крестом внутри. Используется зелёный цвет, чтобы pattern был лучше виден при низкой яркости.
2. Registration pattern — сочетание синих и зелёных окружностей, а также белой и красной сетки. Полезен для архитектурного мэппинга: по форме окружности удобно выставлять правильное соотношение сторон.
3. Horizontal green lines — 4 горизонтальные линии с равными интервалами.
4. Vertical green lines — 4 вертикальные линии с равными интервалами.

Patterns 5–8 для калибровки цвета. Их можно использовать вместе с системой цветокоррекции в Laser Settings. См. [Калибровка цвета](../advanced/colour-calibration.md "mention"). Эти patterns не заполняют всё пространство вывода.

5. Colour calibration pattern — красный.
6. Colour calibration pattern — зелёный.
7. Colour calibration pattern — синий.
8. Colour calibration pattern — белый.

{% hint style="danger" %}
**Внимание** — test pattern 9 может быть очень опасен, и его следует использовать только после того, как вы убедились, что zone безопасна. Яркие неподвижные точки могут прожигать поверхности. При использовании в небе убедитесь, что у вас есть необходимые лицензии/разрешения, и постоянно следите за воздушными судами.
{% endhint %}

9. Bright white dots — по одной точке полной мощности в каждом углу. Это самый заметный pattern, поэтому он хорошо подходит для настройки zone в ярко освещённых местах или при выводе в небо.
10. Scanning white line — вертикальная белая линия, которая сканирует слева направо по всей ширине zone. Она покрывает всю zone, поэтому отлично подходит для проверки masks и для того, чтобы убедиться, что зеркальные шары или другие опасные объекты не попадают в вывод.
11. White rectangle — яркий белый прямоугольник.
