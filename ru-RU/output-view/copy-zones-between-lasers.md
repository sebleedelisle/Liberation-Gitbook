# 🟩 Копирование zones между лазерами

Нажмите кнопку _COPY LASER SETTINGS_ в _Laser Output_ — откроется новая панель.

Это очень полезная функция, которая экономит время, когда нужно быстро скопировать zones и другие настройки с одного лазера на остальные.

Убедитесь, что выбран лазер, _с которого_ нужно скопировать настройки.

Включите все остальные лазеры, _на которые_ нужно скопировать настройки. (Если лазеров много, используйте _ALL_ и _NONE_, чтобы сделать это быстрее.)

Выберите опцию _Copy output zone settings_, затем нажмите кнопку _COPY SETTINGS_, чтобы завершить процесс.

{% hint style="info" %}
Обратите внимание: этот процесс не добавляет и не удаляет zones у лазеров. Он только копирует положение и тип каждой zone, которая уже существует на целевых лазерах.
{% endhint %}

См. также [Копирование настроек лазера](../setting-up/copy-laser-settings.md) для полного описания других возможностей этой системы.
