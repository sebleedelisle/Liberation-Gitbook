# ✅ Обзор

Output view показывает полный диапазон вывода каждого лазера. В этом представлении можно настраивать его zones и добавлять области mask.

<figure><img src="../.gitbook/assets/Output View.png" alt="" width="375"><figcaption><p>Представление <em>OUTPUT</em> с beam zone, canvas zone и mask.</p></figcaption></figure>

Чтобы открыть Output view, нажмите кнопку _OUTPUT_ на верхней панели инструментов.

<figure><img src="../.gitbook/assets/Top icon bar with output selected.png" alt=""><figcaption></figcaption></figure>

### Выбор лазера

Используйте кнопки с номерами вверху, чтобы выбрать лазер, который вы редактируете. Также можно переключаться между ними клавишей `Tab` или нажать клавишу с нужным номером.

### Навигация в Output view

Нажмите и перетащите в любом пустом месте (или на заблокированных zones), чтобы переместить вид.

### Увеличение и уменьшение масштаба Output view

Используйте колесо мыши для увеличения и уменьшения масштаба.

### Значки панели инструментов

<figure><img src="../.gitbook/assets/Output Tool Bar Icon.png" alt="" width="269"><figcaption><p>Панель инструментов слева в представлении <em>OUTPUT</em></p></figcaption></figure>

Слева в этом представлении находится панель инструментов. Наведите указатель мыши на любой значок, чтобы узнать, что он делает.

* Добавить beam zone для лазера
* Выбрать, какие canvas zones входят в этот лазер
* Добавить mask для лазера
* Включить или выключить test pattern для этого лазера
* Выбрать test pattern для этого лазера (переопределяет основной test pattern)
* Включить или выключить привязку к сетке
* Настроить размер сетки (0 — самый маленький, 8 — самый большой). (Обратите внимание: размеры сетки увеличиваются в степени 2, поэтому при уменьшении размера сетки все точки с привязкой остаются на одной линии).
