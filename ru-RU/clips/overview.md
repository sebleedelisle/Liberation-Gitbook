# 🟩 Обзор

### Что такое Clip?

Всё лазерное содержимое в Liberation хранится внутри Clip.

Clip может содержать атмосферные лучевые эффекты или графические анимации и обычно представляет собой зацикленную последовательность. Clip можно направить в любую zone (или Canvas Target Area) и запускать с помощью кнопок Clip внутри Clip Deck.

Содержимое Clip создаётся в универсальной системе Clip Editor, построенной на nodes. (Подробнее см. [Введение в Clip Editor](../clip-editor/clip-editor-intro.md))

### Clip Deck

<figure><img src="../.gitbook/assets/clips-clip-deck" alt=""><figcaption><p>Clip Deck по умолчанию</p></figcaption></figure>

Clip Deck — это экранная сетка Clip. Обычно она выходит за пределы видимой области экрана.

Прокручивать Clip Deck влево и вправо можно с помощью:

* Клавиш со стрелками влево и вправо
* Жеста двумя пальцами по трекпаду (или горизонтальной прокрутки мышью, если она это поддерживает!)
* Колеса прокрутки _Cue Level_ на APC40

Прокручивать Clip Deck влево и вправо на целую страницу можно с помощью:

* `Cmd / Ctrl + Left / Right Arrow`
* Кнопок _Device <- and ->_ на APC40

#### Мини-визуализатор Clip

В верхней части Clip Deck находится уменьшенное представление всего Clip Deck. Подсвеченная область показывает Clip, которые сейчас видны.
