# 🟩 Организация Clip Deck

### Перемещение и дублирование Clips

Щелкните Clip и перетащите его. Если выбрано несколько Clips (см. [запуск и остановка Clips](starting-stopping-clips.md "mention")), они будут перемещены вместе. Удерживайте `Alt / Option`, чтобы продублировать их. Перетащите Clip или Clips на пустое место.

{% hint style="info" %}
Если нужно выбрать большое количество Clips, щелкните и протяните рамку выделения, а затем прокручивайте Clip Deck, не отпуская кнопку мыши (с помощью клавиш со стрелками, APC40 или горизонтальной прокрутки на тачпаде/мыши). Рамка выделения будет продолжаться за край Clip Deck.
{% endhint %}

### Удаление Clips

Чтобы удалить один Clip, используйте контекстное меню по правой кнопке мыши.

Чтобы удалить несколько Clips, выберите их (щелчком с удержанием `Alt / Option + Shift` или рамкой выделения) и перетащите за пределы Clip Deck.

### Добавление пустого пространства

Щелкните правой кнопкой мыши по кнопке Clip и выберите _ADD EMPTY COLUMN TO RIGHT_, чтобы добавить в Clip Deck пустой столбец.

### Объединение нескольких Clip Deck

Вы можете импортировать существующий Clip Deck и добавить его в конец текущего Clip Deck. Используйте меню _File -> Clips/Effects -> Append Clip Deck._

Выберите Clips (рамкой выделения или щелчком с удержанием `Alt / Option + Shift`), затем выберите _File -> Clips/Effects -> Export Selected Clips._ Будет экспортирован Clip Deck, содержащий только выбранные Clips; затем его можно добавить к другому Clip Deck.

### Организация Clips по таймлайнам

При импорте таймлайна доступны дополнительные параметры Clips. Вы можете заменить текущий Clip Deck, добавить весь Clip Deck таймлайна или добавить только используемые в нем Clips.

Если в проекте несколько таймлайнов, можно реорганизовать Clips по таймлайнам. Откройте панель _Timeline selection_ и нажмите _ORGANISE CLIPS BY TIMELINE_. После этого все Clips будут сгруппированы по таймлайнам. Если Clip используется в нескольких таймлайнах, он будет продублирован для каждой группы.
