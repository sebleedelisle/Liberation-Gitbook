# ✅ Назначение Clips для zones

<figure><img src="../.gitbook/assets/clips-zones" alt=""><figcaption><p>Кнопки zones вверху и кнопки Flip X / Flip Y внизу.</p></figcaption></figure>

Под Clips виден ряд кнопок zones (на экране и на APC40), которые по умолчанию подписаны от _BEAM 1_ до _BEAM 8_. Используйте эти кнопки, чтобы включать и отключать zones для _текущего выбранного Clip_.

#### Прокрутка списка zones

Если zones больше 8, можно прокручивать список влево и вправо с помощью:

* `Shift + Left / Right Arrow`
* Кнопок _DEV ON/OFF_ и _DEV LOCK_ на APC40

### Flip X и Y

Под каждой кнопкой zone находятся кнопки _X_ и _Y._ Они отражают Output для каждой zone по горизонтали и вертикали соответственно.

{% hint style="info" %}
Не нужно создавать новую zone, чтобы использовать другие настройки отражения. Настройки Flip X и Flip Y применяются только к этому Clip и не являются настройками самой zone.
{% endhint %}

### Назначение zones сразу для нескольких Clips

Вы можете назначать zones и настраивать X/Y для всех Clips в группе. Подробнее см. [Группы](groups.md).
