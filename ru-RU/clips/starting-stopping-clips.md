# ✅ Запуск и остановка Clips

{% hint style="info" %}
Вы можете запускать и останавливать Clips с помощью экранных кнопок, но гораздо удобнее использовать MIDI controller APC40. Liberation оптимизирована для работы с этим оборудованием, и его стоит считать важной частью ваших вложений в лазеры.
{% endhint %}

### Запуск и остановка Clips

Чтобы запустить Clip, нажмите кнопку этого Clip — на экране или на MIDI controller.

Чтобы остановить Clip, нажмите её ещё раз.

Если вы запустите новый Clip в той же группе, первый Clip автоматически остановится.

Чтобы запустить новый Clip, не останавливая другие Clips, можно:

* Удерживать клавишу `Shift` или кнопку Shift на APC40 при запуске нового Clip; или
* Повторно нажать текущие запущенные Clip(s) в момент запуска нового Clip.

Группы Clips независимы друг от друга, поэтому запуск Clip в одной группе не влияет на Clips в других группах. См. [Группы](groups.md "mention")

### Режим flash mode

Если Clip находится в группе, для которой включён _flash mode_, он ведёт себя немного иначе: Clip продолжает работать только пока вы удерживаете его кнопку. Как только вы отпускаете кнопку, Clip останавливается. По умолчанию группа Clips 3 — красная — работает в _flash mode._

### Остановить все Clips

Чтобы **остановить все запущенные Clips**, нажмите кнопку **STOP**.

{% hint style="info" %}
Нажмите кнопку **STOP** **дважды**, чтобы пропустить время плавного затухания Clips и сразу погасить их.
{% endhint %}

Чтобы остановить все запущенные Clips в определённой группе, нажмите кнопку этой группы, затем нажмите кнопку **STOP**.

Чтобы остановить все Clips, кроме одного, нажмите запущенный Clip, который хотите оставить, и, удерживая его кнопку, нажмите кнопку **STOP**. Затем отпустите кнопку Clip. Это также работает с несколькими Clips — с таким количеством, сколько вы сможете удерживать одновременно!

### Выбранный в данный момент Clip

<figure><img src="../.gitbook/assets/clips-selected-active.png" alt="" width="269"><figcaption><p>Два Clip, которые сейчас запущены. Белая рамка показывает, что Clip справа — это <em>выбранный в данный момент Clip.</em></p></figcaption></figure>

Когда Clip запущен, он подсвечивается на экране, а его мини-представление в визуализаторе Clip мигает. Вы также заметите, что последний Clip, который вы нажали, обведён белой рамкой. Это означает, что он является _выбранным в данный момент Clip._

Нажмите `Alt / Option` и щёлкните Clip, чтобы выбрать его без запуска. Используйте `Alt / Option + Shift` и щелчок, чтобы выбрать несколько Clips.

Вы также можете щёлкнуть и протянуть лассо по Clip Deck, чтобы выбрать несколько Clips.

{% hint style="info" %}
На APC40 также есть кнопки `Alt` и `Shift` для выбора.
{% endhint %}
