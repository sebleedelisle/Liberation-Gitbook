# ✅ Ошибка «missing resources» при запуске

Если при запуске появляется ошибка «missing resources», это означает, что отсутствуют критически важные файлы, необходимые для работы Liberation. В Windows такое может произойти, если вы случайно удалили или переместили файлы в папке установки. На Mac — если установка по какой-то причине была повреждена.

### Как исправить

Лучший способ исправить эту проблему — переустановить Liberation.
