# ✅ Где найти рабочую папку

#### Где найти рабочую папку

У каждой версии Liberation есть собственная рабочая папка. Например, если вы используете версию 1.0.3, папка будет называться 1.0.3.

* **macOS**: `~/Library/Application Support/Liberation/1.0.3`
* **Windows**: `AppData\Local\Liberation\1.0.3`

**Как быстро открыть папку**

**macOS**

1. В Finder нажмите **Shift + Cmd + G**.
2.  Вставьте этот путь и нажмите **Enter**:

    ```
    ~/Library/Application Support/Liberation
    ```
3. Откройте папку с номером вашей версии, например `1.0.3`.

**Windows**

1.  Нажмите **Win + R**, вставьте этот путь и нажмите **Enter**:

    ```
    %LOCALAPPDATA%\Liberation
    ```
2. Откройте папку с номером вашей версии, например `1.0.3`.

> **Совет для Windows**: если вы открываете папку через Проводник, включите показ скрытых элементов: **Вид > Показать > Скрытые элементы**.
