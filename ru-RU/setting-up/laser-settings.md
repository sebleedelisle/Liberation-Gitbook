# ✅ Панель настроек лазерного вывода

Откройте панель настроек _Laser output_ через меню _View -> Laser Output Settings._

<figure><img src="../.gitbook/assets/Copy Laser Setting in Laser Output.png" alt="" width="326"><figcaption></figcaption></figure>

Откроются настройки выбранного лазера. Выбрать другой лазер можно:

* кнопкой с его номером в панели _Laser overview_
* цифровой клавишей на клавиатуре: клавиши 1–0 открывают лазеры 1–10
* клавишей `Tab` для перехода между лазерами (`Shift + Tab` — в обратном направлении).

В верхней части панели находятся:

* кнопка с номером — нажмите её, чтобы включить или отключить готовность этого лазера к выводу. Кнопка красная, когда лазер находится в состоянии готовности.
* ползунок _Brightness_ только для этого лазера. Обратите внимание, что он работает совместно с Global Brightness.
* переключатель _Test Pattern_ и выбор шаблона. Они позволяют выбрать отдельный тестовый шаблон только для этого лазера. (Эти элементы также доступны на панели инструментов Output view.)

### Ориентация вывода / коррекция зеркального отображения

Следующие элементы нужны для коррекции настройки лазера, чтобы он работал в Liberation предсказуемо и согласованно.

* **Flip horizontal / vertical** — эти параметры позволяют исправить вывод лазера

{% hint style="info" %}
Обычно не нужно менять настройки горизонтального или вертикального отражения, если только лазер не подключён неправильно или если на его задней панели есть кнопки X/Y flip и они выставлены неверно. Если нужно отразить вывод только для определённого Clip, это можно сделать в настройках самого Clip.
{% endhint %}

* **Orientation** — если лазер установлен на боку или вверх ногами, этим параметром можно исправить поворот.
* **Fine position adjustments** — можно использовать для очень небольшой коррекции смещения или поворота. Эти настройки рассчитаны на компенсацию дрейфа или усадки, если лазер оставался включённым на ночь или на длительное время.

{% hint style="info" %}
Обратите внимание: коррекции ориентации и зеркального отображения ничего не меняют в 3D Visualiser. Их следует использовать, чтобы фактический вывод лазера совпадал с тем, что показано в 3D Visualiser.
{% endhint %}

### Настройки сканера

<figure><img src="../.gitbook/assets/Laser output scanner settings.png" alt=""><figcaption></figcaption></figure>

#### **Speed**

Параметр скорости определяет, как быстро двигаются сканеры.

{% hint style="danger" %}
Хотя настройки по умолчанию довольно консервативные, сканеры всё равно можно повредить, если заставлять их работать слишком быстро. Будьте осторожны, особенно при увеличении скорости.
{% endhint %}

{% hint style="info" %}
Этот параметр скорости не меняет частоту точек. Вместо этого он регулирует, насколько далеко точки разнесены друг от друга. Подробнее см. [Как Liberation создаёт лазерный контент](../advanced/how-liberation-generates-laser-content.md)
{% endhint %}

#### **Scanner sync (Colour shift / blank shift)**

Во время движения сканеров луч меняет цвет, включается и выключается, и эти процессы обычно не идеально синхронизированы друг с другом. Настройте этот параметр, чтобы снова выровнять их по времени.

{% hint style="info" %}
Иногда это называют _blank shift_, но лично я предпочитаю термин _scanner sync_ — он немного точнее, потому что параметр регулирует тайминг всех изменений цвета относительно движения сканера.
{% endhint %}

Colour shift настроен хорошо!</p></figcaption></figure></div>

Если на лазерном выводе видны небольшие «хвосты», скорее всего, нужно настроить синхронизацию сканера. Если «хвосты» остаются при любых настройках, вероятно, вы заставляете сканеры или драйверы лазера работать быстрее, чем они способны. Попробуйте уменьшить скорость сканера.

#### Предустановки сканера

Используйте этот параметр, чтобы выбрать заранее подготовленную настройку сканера. Обычно вариант по умолчанию подходит, поэтому менять его не нужно, если только у вас не особенно плохие (или хорошие) сканеры. Если хотите разобраться подробнее, см. [Предустановки сканера](../advanced/scanner-presets.md)

#### Калибровка цвета

Эту систему можно использовать для коррекции кривой яркости и баланса белого вашего лазера. См. [Калибровка цвета](../advanced/colour-calibration.md)

#### Расширенные настройки

Обычно эти параметры трогать не нужно, но если вам интересно, см. [Расширенные настройки лазера](../advanced/advanced-laser-settings.md)
