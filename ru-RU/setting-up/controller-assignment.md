# ✅ Controller Assignment

После того как вы настроили лазеры в Liberation, можно назначить каждому из них физический лазерный контроллер. (См. [совместимые лазеры и контроллеры (DAC)](../hardware/compatible-lasers-and-controllers-dacs.md), чтобы проверить, какое оборудование можно использовать.) Контроллеры подключаются либо по USB, либо по сети.

* Откройте панель _Controller Assignment_ через пункт меню _View -> Controller Assignment_. (Также можно использовать кнопку _ASSIGN LASER CONTROLLERS_ на панели _Laser Overview_.)

<figure><img src="../.gitbook/assets/Screenshot 2024-12-31 at 14.30.18.png" alt="Панель Controller Assignment"><figcaption></figcaption></figure>

* Панель разделена на две части: слева находится список лазеров, справа — список доступных контроллеров. Если вы не видите свой лазерный контроллер в списке, нажмите кнопку _REFRESH_. Если проблема сохраняется, см. [Устранение неполадок](../troubleshooting/).
* Чтобы назначить контроллер лазеру, перетащите его из списка справа на свободный слот лазера слева. Так Liberation понимает, какой контроллер нужно использовать для какого лазера. (Если вы передумаете, контроллеры можно свободно перетаскивать между лазерами вверх и вниз.)

<figure><img src="../.gitbook/assets/Screenshot 2024-12-31 at 14.33.23.png" alt="Список контроллеров" width="375"><figcaption></figcaption></figure>

* Если рядом с контроллером отображается зелёный квадрат, значит Liberation успешно подключился к нему.

<figure><img src="../.gitbook/assets/controller-assignment-laser-list" alt="" width="338"><figcaption><p>Ether Dream и Helios DAC назначены лазерам 2 и 3 соответственно</p></figcaption></figure>

{% hint style="info" %}
Обратите внимание: при каждом подключении к контроллеру лазер автоматически снимается с готовности.
{% endhint %}

* Оранжевый квадрат 🟧 означает, что у контроллера периодически возникают проблемы с соединением. Обычно это связано с сетью; см. [Устранение неполадок](../troubleshooting/).
* Красный квадрат 🟥 означает, что контроллер недоступен; см. [Устранение неполадок](../troubleshooting/).
* _Кнопка отключения_ (X) отключает контроллер, но не удаляет его назначение лазеру. После этого можно нажать _кнопку повторного подключения_ (значок стрелки обновления), чтобы подключить его снова, или ещё раз нажать _кнопку отключения_, чтобы удалить назначение.
* _Дополнительная функция:_ откройте панель аналитики контроллера, нажав кнопку, похожую на график. Это расширенная функция: она показывает подробную информацию о потоке данных и помогает устранять проблемы. (Эта опция может быть недоступна для некоторых типов контроллеров.)
* С помощью _кнопки переименования_ (карандаш) можно задать для контроллера любое имя. Лучше выбрать имя, по которому легко связать контроллер с конкретным оборудованием. Если контроллер встроен в лазер, можно назвать его соответствующим образом, например _LaserCube Ultra #1_ или _Triton T5 #3_. Эти имена сохранятся в вашей установке Liberation и будут отображаться в дальнейшем; это очень помогает быстро находить нужные лазеры.

{% hint style="info" %}
Совет: **дважды щёлкните** контроллер справа, чтобы автоматически назначить его следующему доступному лазеру слева. Это заметно экономит время, если нужно назначить много лазеров.
{% endhint %}

Кнопки _DISCONNECT ALL_ и _RECONNECT ALL_ позволяют быстро сбросить все подключения. Это полезно при проблемах с сетью.
