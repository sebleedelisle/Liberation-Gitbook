# ✅ Панель настроек лазерного вывода

Откройте панель настроек _Laser output_ через меню _View -> Laser Output Settings._

<figure><img src="../../.gitbook/assets/Copy Laser Setting in Laser Output.png" alt="" width="326"><figcaption></figcaption></figure>

Откроются настройки выбранного лазера. Выбрать другой лазер можно:

* кнопкой с его номером на панели _Laser Overview_
* цифровой клавишей на клавиатуре: клавиши 1–0 открывают лазеры 1–10
* клавишей `Tab` для последовательного переключения между лазерами (`Shift + Tab` переключает в обратном направлении).

В верхней части этой панели находятся:

* кнопка с номером — нажмите её, чтобы включить или отключить готовность этого лазера. Когда лазер готов к выводу, кнопка красная.
* слайдер _Brightness_ только для этого лазера. Учтите, что это значение объединяется с Global Brightness.
* переключатель _Test Pattern_ и выбор шаблона. Они позволяют выбрать отдельный тестовый шаблон только для этого лазера. Эти элементы также дублируются на панели инструментов Output view.

### Коррекция ориентации вывода и зеркального отражения

Следующие элементы нужны для корректировки настройки лазера, чтобы он работал в Liberation предсказуемо и согласованно.

* **Flip horizontal / vertical** — эти параметры позволяют скорректировать вывод лазера

{% hint style="info" %}
Обычно настройки горизонтального и вертикального отражения менять не нужно, если только лазер не подключён неправильно или на его задней панели есть кнопки X/Y flip, установленные неверно. Если нужно отразить вывод только для конкретного Clip, это можно сделать в настройках самого Clip.
{% endhint %}

* **Orientation** — если лазер установлен на боку или вверх ногами, этой настройкой можно скорректировать поворот.
* **Fine position adjustments** — можно использовать для очень небольших сдвигов или поворотов. Предназначено для компенсации дрейфа или смещения, если лазер был оставлен на ночь или на длительное время.

{% hint style="info" %}
Обратите внимание: коррекция ориентации и зеркального отражения ничего не меняет в 3D Visualiser. Эти настройки нужны, чтобы привести вывод реального лазера в соответствие с тем, что отображается в 3D Visualiser.
{% endhint %}

### Настройки сканеров

<figure><img src="../../.gitbook/assets/Laser output scanner settings.png" alt=""><figcaption></figcaption></figure>

#### **Speed**

Настройка Speed определяет, насколько быстро движутся сканеры.

{% hint style="danger" %}
Хотя настройки по умолчанию довольно консервативны, сканеры всё равно можно повредить, если заставлять их работать слишком быстро. Будьте осторожны, особенно при увеличении скорости.
{% endhint %}

{% hint style="info" %}
Эта настройка скорости не меняет частоту точек, а регулирует расстояние между ними. Подробнее см. в разделе [Как Liberation генерирует лазерный контент](../../advanced/how-liberation-generates-laser-content.md "mention")
{% endhint %}

#### **Scanner sync (Colour shift / blank shift)**

Луч меняет цвет, включается и выключается во время движения сканеров, и обычно эти процессы не идеально синхронизированы между собой. Отрегулируйте эту настройку, чтобы снова выровнять их.

{% hint style="info" %}
Иногда это называют _blank shift_, но я предпочитаю термин _scanner sync_: он немного точнее, потому что настройка изменяет синхронизацию всех изменений цвета относительно движения сканеров.
{% endhint %}

<div><figure><img src="../../.gitbook/assets/Colour shift tails.jpeg" alt="" width="320"><figcaption><p>Лазерные «хвосты»: Colour shift настроен неправильно</p></figcaption></figure> <figure><img src="../../.gitbook/assets/Colour shift no tails.jpeg" alt="" width="320"><figcaption><p>Лазерных «хвостов» нет! Colour shift настроен хорошо!</p></figcaption></figure></div>

Если на лазерном выводе видны небольшие «хвосты», скорее всего, нужно настроить Scanner sync. Если «хвосты» остаются при любых настройках, вероятно, вы управляете сканерами или драйверами лазера быстрее, чем они способны отработать. Попробуйте снизить скорость сканеров.

#### Пресеты сканеров

Используйте эту настройку, чтобы выбрать заранее подготовленные параметры сканеров. Обычно вариант по умолчанию подходит, поэтому менять эту настройку не нужно, если только у вас не особенно плохие или, наоборот, хорошие сканеры. Если хотите разобраться подробнее, см. [Пресеты сканеров](../../advanced/scanner-presets.md "mention")
