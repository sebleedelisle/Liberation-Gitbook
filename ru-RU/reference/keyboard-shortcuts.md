# ✅ Сочетания клавиш

Сочетания записаны как `Cmd / Ctrl`: используйте `Cmd` в macOS и `Ctrl` в Windows.

| Сочетание                         | Действие                                                   |
| --------------------------------- | ---------------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Arm для всех лазеров                                       |
| `Escape`                          | Disarm для всех лазеров                                    |
| `Tab`                             | Переключать режимы просмотра и переходить между лазерами   |
| `Shift + Tab`                     | Переключать режимы просмотра и переходить между лазерами в обратном порядке |
| `1-9`                             | Перейти к лазеру                                           |
| `Left / Right Arrow`              | Переместить Clip Deck влево / вправо                       |
| `Cmd / Ctrl + Left / Right Arrow` | Переместить Clip Deck на одну страницу влево / вправо      |
| `Shift + Left / Right Arrow`      | Переместить zones на одну страницу влево / вправо          |
| `Space`                           | Воспроизведение / пауза (когда активен источник темпа _Timeline_) |
| `T`                               | Задать темп нажатием                                       |
| `R`                               | Сброс такта                                                |
| `Y`                               | Округлить темп до ближайшего BPM                           |
| `Cmd / Ctrl + ,`                  | Открыть панель настроек                                    |
| `Cmd / Ctrl + F`                  | Переключить полноэкранный режим                            |
| `Cmd / Ctrl + L`                  | Открыть панель настроек лазерного вывода                   |
| `Cmd / Ctrl + K`                  | Открыть панель назначения laser controller                  |
| `Cmd / Ctrl + J`                  | Открыть панель настроек 3D-визуализатора                   |
| `Cmd / Ctrl + Shift + Plus (+)`   | Увеличить масштаб интерфейса                               |
| `Cmd / Ctrl + Shift + Minus (-)`  | Уменьшить масштаб интерфейса                               |
| `Cmd / Ctrl + 0`                  | Сбросить масштаб интерфейса                                |
| `Cmd / Ctrl + A`                  | Timeline: выбрать всё                                      |
| `Delete`                          | Timeline: удалить выбранные блоки                          |
| `Delete`                          | Clip Editor: удалить выбранные nodes                       |
| `Cmd + Q` (macOS)                 | Закрыть приложение                                         |
| `Alt + F4` (Windows)              | Закрыть приложение                                         |
