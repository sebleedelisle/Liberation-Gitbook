# ✅ Стандартное сопоставление MIDI для отправки и приёма

**Включение и выключение Clip запускается сообщениями MIDI Note On / Note Off на каналах с 1 по 14**

У каждого Clip есть горизонтальная (x) и вертикальная (y) позиция. Щёлкните Clip правой кнопкой мыши, и вы увидите его позицию. MIDI может запускать Clips, начиная с координат 0,0.

{% hint style="info" %}
Обратите внимание: управление Clip в этой системе является абсолютным, и позиции Clip не меняются при прокрутке Clip Deck.
{% endhint %}

MIDI-канал 1, нота 1 — это Clip 0,0, нота 2 — Clip 0,1, нота 3 — Clip 0,2, далее вниз по строкам и затем по столбцам. Когда счёт доходит до 128, он переходит на следующий канал и начинается заново. Итого через MIDI доступно 128 x 14 = 1792 Clips.

Обратите внимание: событие MIDI Note On запускает Clip, а соответствующее событие Note Off останавливает Clip. Это не зависит от режима запуска в группе. Изначально система была разработана для воспроизведения и записи, поэтому переключение Clip одной и той же нотой было бы нежелательным.

### **Эффекты**

Сообщения MIDI Control Change (CC) на **канале 15** регулируют эффекты.\
Эффект 1 использует CC 0-3, значение 0-127\
Эффект 2 использует CC 4-7, значение 0-127\
Эффект 3 использует CC 8-11, значение 0-127\
… и так далее.

Каждая группа из четырёх элементов управления отвечает за уровень и 3 параметра этого эффекта:

### **Глобальные настройки**

Сообщения MIDI Control Change на **канале 16** изменяют глобальные настройки:\
CC 1 : Shift X (по горизонтали) 0 -127, 64 — середина\
CC 2 : Shift Y (по вертикали) 0 -127, 64 — середина\
CC 4 : Scale X\
CC 5 : Scale Y\
CC 8 : Rotation 3D (Y)\
CC 9 : Rotation 2D (Z)

И особенно важно:\
CC 15 : Global Brightness

Обратите внимание: эта система изначально была разработана для записи и воспроизведения, чтобы вы могли использовать Logic, Ableton или другую DAW для создания анимаций на таймлайне. Её можно использовать для управления вживую, но она не оптимизирована для такого сценария, и некоторые функции живого управления отсутствуют по сравнению с настройкой APC40.
