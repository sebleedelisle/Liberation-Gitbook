# ✅ Установка для Windows (устаревшая, до версии 0.9)

### Совместимость

Компьютеры с Windows 10 или 11, 64-бит. Для больших шоу рекомендуются компьютеры с высокой производительностью. Чем быстрее компьютер, тем плавнее будет анимированный лазерный вывод.

### Процесс установки

После оформления активной подписки вам отправят ссылку. Также вы можете нажать ссылку _Download_ на странице [_My subscriptions_](https://liberationlaser.com/account/my-products).

Вы увидите такую страницу:

<figure><img src="../../.gitbook/assets/windows-download-page.png" alt=""><figcaption></figcaption></figure>

Нажмите _Download latest for Windows_ (номера версий могут отличаться), и вы увидите индикатор загрузки с ходом выполнения (обычно в правом верхнем углу; на снимке экрана показан Chrome). Когда загрузка завершится, наведите указатель на загруженный файл Liberation и нажмите появившийся значок папки.

<figure><img src="../../.gitbook/assets/windows-download-chrome.png" alt="" width="260"><figcaption></figcaption></figure>

Откроется папка загрузок. Найдите загруженный файл (обычно он находится вверху), щёлкните по нему правой кнопкой мыши и выберите "Extract all".

<figure><img src="../../.gitbook/assets/windows-extract-all.png" alt="" width="248"><figcaption></figcaption></figure>

Откроется новая панель, где нужно выбрать, куда установить Liberation. Вы можете поместить его в любое место на жёстком диске, для которого у вас есть права на запись. Если не уверены, просто создайте папку на диске C:.

<figure><img src="../../.gitbook/assets/windows-select-destination.png" alt="" width="375"><figcaption></figcaption></figure>

После завершения распаковки Windows покажет установочный файл. Дважды щёлкните Liberation.exe, чтобы запустить приложение.

<figure><img src="../../.gitbook/assets/windows-open-exe.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation нужны разрешения на доступ к файлам и сети, поэтому Windows, скорее всего, покажет несколько запросов. Обязательно разрешите всё, иначе Liberation может работать некорректно.

<figure><img src="../../.gitbook/assets/windows-protection-window.png" alt="" width="375"><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

Теперь Liberation должен запуститься. Чтобы выводить изображение на лазеры, далее нужно пройти [процесс авторизации.](../authorising-and-de-authorising.md)
