# ✅ Установка для Windows

### Совместимость

Компьютеры с 64-битной Windows 10 или 11. Если вы планируете запускать большие шоу, рекомендуется мощный компьютер. Чем быстрее компьютер, тем плавнее будет анимированный лазерный вывод.

### Процесс установки

После оформления активной подписки вам отправят ссылку. Также можно нажать ссылку _Download_ на странице [ _Your subscriptions_](https://liberationlaser.com/account/my-products).

Вы увидите эту страницу:

<figure><img src="../../.gitbook/assets/windows-install-download-page.png" alt=""><figcaption></figcaption></figure>

Нажмите _Download latest for Windows_ (номера версий могут измениться), и индикатор загрузки покажет ход скачивания (обычно он находится в правом верхнем углу; скриншот сделан в Chrome). Когда загрузка завершится, откройте папку загрузок и нажмите на скачанный файл установщика.

<figure><img src="../../.gitbook/assets/windows-install-download-list.png" alt=""><figcaption></figcaption></figure>

Запустится установщик.

{% hint style="info" %}
В некоторых случаях может появиться предупреждение _Windows protected your PC_ — Windows осторожно относится к запуску нового программного обеспечения. По мере того как больше пользователей будут запускать Liberation и доверять ему, это предупреждение должно исчезнуть.

Если вы увидите такое предупреждение, нажмите ссылку _More info_, убедитесь, что в поле _Publisher_ указано _SEB.LY LIMITED_, а затем нажмите _Run anyway_.

<img src="../../.gitbook/assets/windows-install-windows-protection-warning.png" alt="" data-size="original"><img src="../../.gitbook/assets/windows-install-warning-run-anyway.png" alt="" data-size="original">
{% endhint %}

Перед запуском установщика может появиться предупреждение с запросом разрешения на внесение изменений. Нажмите _Yes_.

<figure><img src="../../.gitbook/assets/windows-install-warning-make-changes.png" alt=""><figcaption></figcaption></figure>

Теперь установщик может начать работу. Пройдите по страницам мастера установки.

<figure><img src="../../.gitbook/assets/windows-installer-1.png" alt=""><figcaption></figcaption></figure>

Рекомендуется использовать папку установки по умолчанию, поэтому просто нажмите _Next_.

<figure><img src="../../.gitbook/assets/windows-installer-2.png" alt=""><figcaption></figcaption></figure>

Далее можно выбрать создание значка на рабочем столе и папки в меню «Пуск» (рекомендуется).

<figure><img src="../../.gitbook/assets/windows-installer-3.png" alt=""><figcaption></figcaption></figure>

Подождите немного, пока Liberation установится.

<figure><img src="../../.gitbook/assets/windows-installer-4.png" alt=""><figcaption></figcaption></figure>

Установка завершена. Нажмите _Finish_, чтобы закрыть установщик и запустить Liberation.

<figure><img src="../../.gitbook/assets/windows-installer-5.png" alt=""><figcaption></figcaption></figure>

Liberation требуется доступ к файлам и сети, поэтому Windows, скорее всего, покажет несколько запросов. Обязательно разрешите всё, иначе Liberation может работать неправильно.

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

Теперь Liberation должен запуститься. Чтобы выводить изображение на лазеры, нужно выполнить действия, описанные в разделе [Авторизация и деавторизация](../authorising-and-de-authorising.md).
