# 🟩 Koordinatsystem

Clip-innehåll använder ett x/y-koordinatsystem där origo (0,0) ligger i mitten av skärmen.

* På den horisontella x-axeln är vänster negativt och höger positivt.
* På den vertikala y-axeln är upp negativt och ned positivt.

Clip har en bredd och höjd på 400 pixlar, så koordinaterna för det synliga området går från -200 till 200.

{% hint style="info" %}
Clip-editorn skapar _vektorformer_. Det betyder att de inte lagras som pixlar, utan som uppsättningar av koordinater som definierar hur formerna ritas. Det liknar hur Inkscape eller Illustrator definierar innehåll, till skillnad från Photoshop.
{% endhint %}

### 3D

Du kan dessutom röra dig i 3D-rymden, framåt och bakåt längs z-axeln. Som standard ligger allt på z-positionen 0.

* På z-axeln är bakåt, bort från dig, negativt och framåt, mot dig, positivt.
