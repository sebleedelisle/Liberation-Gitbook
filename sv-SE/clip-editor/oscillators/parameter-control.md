# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_Parameter Control_ är en metod för att styra olika inställningar i en effekt eller ett clip utanför clip-editorn.

* **range min / range max** - de minsta och största värden som parameterns nivå mappas till
* **level** - standardinställningen för den här parametern. Det är värdet som kan ändras utifrån. Du kan justera det manuellt för att se vilken effekt det får.
* **parameter number** - varje _parameter control_ kan tilldelas ett nummer, och detta återspeglas i kontrollerna utanför clipet. Du kan ha fler än en parameter control med samma nummer, och då justeras alla samtidigt.

{% hint style="warning" %}
APC40 och live-kontroller som stöds styr som standard parameter 1 för varje effektplats. Alla parametrar i effekter och Clips är också tillgängliga i högerklicksmenyn.
{% endhint %}
