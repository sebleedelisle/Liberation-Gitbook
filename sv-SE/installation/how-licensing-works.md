# ✅ Så fungerar licensieringen

När du har installerat Liberation på din dator körs det i _demoläge_ tills du _auktoriserar_ det.

{% hint style="info" %}
Det finns inga begränsningar i _demoläge_ förutom utmatning till laserhårdvara. Du kan öva på shower, bygga tidslinjer samt spara och läsa in dina filer utan begränsningar. Du kan installera Liberation i _demoläge_ på hur många datorer du vill, och du behöver bara auktorisera installationen när du är redo att använda lasrar!
{% endhint %}

Utöver antalet lasrar du kan arma kan du även använda DMX-armaturer. Se [dmx-control](../dmx-control/ "mention").

Det finns också en gräns för hur många datorer du kan auktorisera Liberation på, beroende på din abonnemangsnivå.

{% hint style="danger" %}
Varje licens gäller endast för en användare och en show åt gången. Du kan ha mer än en installation per användare eftersom du **alltid** bör ha minst en extra auktoriserad dator redo!
{% endhint %}

### Internetanslutning

Liberations licensiering och auktorisering fungerar via internet; appen ansluter till vår server för att koppla en installation till ditt abonnemang.

När din installation är auktoriserad behöver den inte vara online igen förrän ditt abonnemang förnyas. Du kan se förnyelsedatumet i _About panel._ Det finns en kort respitperiod (7 dagar för månadsabonnemang), men vi rekommenderar att du går online så snart som möjligt efter förnyelsedatumet för att säkerställa att din auktorisering förnyas och förblir aktiv.
