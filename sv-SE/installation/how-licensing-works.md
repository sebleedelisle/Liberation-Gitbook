# ✅ Så fungerar licensiering

När du har installerat Liberation på din dator körs programmet i _friläge_ tills du _auktoriserar_ det.

{% hint style="info" %}
Det finns inga begränsningar i _friläge_ förutom att du inte kan skicka Output till laserhårdvara. Du kan öva på shower, bygga timelines samt spara och läsa in dina filer utan begränsningar. Du kan installera Liberation i _friläge_ på så många datorer du vill, och du behöver bara auktorisera installationen när du är redo att använda lasrar!
{% endhint %}

Utöver det antal lasrar du kan aktivera kan du också använda DMX-enheter. Se [DMX / Art-Net](../dmx-control/ "mention").

Det finns också en gräns för hur många datorer du kan auktorisera Liberation på, beroende på din licensnivå.

{% hint style="danger" %}
Varje licens gäller endast en användare och en show åt gången. Du kan ha fler än en installation per användare eftersom du **alltid** bör ha minst en auktoriserad reservdator redo!
{% endhint %}

### Internetanslutning

Liberations licensiering och auktorisering fungerar via internet. Appen ansluter till vår server för att koppla en installation till din betalda licens.

När du är auktoriserad behöver du inte vara online igen förrän din återkommande licens når nästa betalningsdatum (visas i _About panel_). Återkommande månadslicenser har vanligtvis en kort respitperiod (7 dagar), men du bör gå online så snart som möjligt efter betalningsdatumet för att hålla dina betalda funktioner tillgängliga utan avbrott.

Respitperioden är avsedd för problem med förnyelse och betalning. Den förlänger inte licenser som har sagts upp eller avsiktligt pausats efter att den betalda perioden har löpt ut. Om en betalning misslyckas ska du uppdatera dina kortuppgifter inom respitperioden för att behålla tillgången till betalda funktioner.

Den aktuella licensnivån visas också i det översta fältet. Klicka på nivåns namn, eller _FREE MODE_ om ingen licens är aktiverad, för att öppna About panel.
