# ✅ Så fungerar licensiering

När du har installerat Liberation på din dator körs programmet i _demoläge_ tills du _auktoriserar_ det.

{% hint style="info" %}
Det finns inga begränsningar i _demoläge_ förutom att du inte kan skicka Output till laserhårdvara. Du kan öva på shower, bygga timelines samt spara och läsa in dina filer utan begränsningar. Du kan installera Liberation i _demoläge_ på så många datorer du vill, och du behöver bara auktorisera installationen när du är redo att använda lasrar!
{% endhint %}

Utöver hur många lasrar som kan vara armed kan du också använda DMX-enheter. Se [DMX / Art-Net](../dmx-control/).

Det finns också en gräns för hur många datorer du kan auktorisera Liberation på, beroende på din abonnemangsnivå.

{% hint style="danger" %}
Varje licens gäller endast en användare och en show åt gången. Du kan ha fler än en installation per användare eftersom du **alltid** bör ha minst en auktoriserad reservdator redo!
{% endhint %}

### Internetanslutning

Liberations licensiering och auktorisering fungerar via internet. Appen ansluter till vår server för att koppla en installation till ditt abonnemang.

När din installation är auktoriserad behöver den inte vara online igen förrän ditt abonnemang förnyas. Du kan se förnyelsedatumet i _About panel_. Det finns en kort respitperiod (7 dagar för månadsabonnemang), men vi rekommenderar att du går online så snart som möjligt efter förnyelsedatumet för att säkerställa att auktoriseringen förnyas och förblir aktiv.
