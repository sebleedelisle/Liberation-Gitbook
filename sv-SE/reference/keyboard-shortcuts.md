# ✅ Tangentbordsgenvägar

Genvägar skrivs som `Cmd / Ctrl`: använd `Cmd` på macOS och `Ctrl` på Windows.

| Genväg                            | Åtgärd                                               |
| --------------------------------- | ---------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Armera alla lasrar                                   |
| `Escape`                          | Desarmera alla lasrar                                |
| `Tab`                             | Växla vyer och bläddra genom lasrar                  |
| `Shift + Tab`                     | Växla vyer och bläddra bakåt genom lasrar            |
| `1-9`                             | Hoppa till laser                                     |
| `Left / Right Arrow`              | Flytta Clip Deck åt vänster/höger                    |
| `Cmd / Ctrl + Left / Right Arrow` | Flytta Clip Deck en sida åt vänster/höger            |
| `Shift + Left / Right Arrow`      | Flytta zoner en sida åt vänster/höger                |
| `Space`                           | Spela upp/pausa (när tempokällan _Timeline_ är aktiv) |
| `T`                               | Tryck in tempo                                       |
| `R`                               | Nollställ takt                                       |
| `Y`                               | Avrunda tempot till närmaste BPM                     |
| `Cmd / Ctrl + ,`                  | Öppna inställningspanelen                            |
| `Cmd / Ctrl + F`                  | Växla helskärmsläge                                  |
| `Cmd / Ctrl + L`                  | Öppna inställningspanelen för laserutgång            |
| `Cmd / Ctrl + K`                  | Öppna panelen för tilldelning av laserkontroller     |
| `Cmd / Ctrl + J`                  | Öppna inställningspanelen för 3D Visualiser          |
| `Cmd / Ctrl + Shift + Plus (+)`   | Öka gränssnittets skalning                           |
| `Cmd / Ctrl + Shift + Minus (-)`  | Minska gränssnittets skalning                        |
| `Cmd / Ctrl + 0`                  | Återställ gränssnittets skalning                     |
| `Cmd / Ctrl + A`                  | Timeline: markera allt                               |
| `Delete`                          | Timeline: ta bort markerade block                    |
| `Delete`                          | Clip editor: ta bort markerade noder                 |
| `Cmd + Q` (macOS)                 | Avsluta appen                                        |
| `Alt + F4` (Windows)              | Avsluta appen                                        |
