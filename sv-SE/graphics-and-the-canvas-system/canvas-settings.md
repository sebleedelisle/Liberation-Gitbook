# 🟩 Canvas settings

Klicka på knappen _Canvas settings_ högst upp i det vänstra verktygsfältet för att öppna panelen _Canvas settings_.

### Width and Height

Använd dessa dragbara reglage för att ändra den totala bredden och höjden för hela canvasen. Måtten kan i princip ses som pixelstorlek, men allt är relativt.

Dubbelklicka på ett reglage för att skriva in ett värde.

{% hint style="info" %}
Här är en extra nördig detalj: lasern rör sig relativt canvasens upplösning, inte relativt zonens storlek. Det beror på att en stor zon i canvasen sannolikt projiceras över en större yta i verkligheten.

Det är en liten detalj som troligen inte spelar någon större roll för dig, men den hjälper till att jämna ut prestanda och synlighet mellan dina canvas-zoner.
{% endhint %}

### Zone brightness

Använd detta för att ändra hur ljusa de rosa zonkonturerna visas i Canvas-vyn. Det kan göra ditt innehåll tydligare och förhindra att vyn blir för rörig.
