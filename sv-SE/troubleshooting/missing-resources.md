# ✅ Felet "missing resources" vid start

Om du får ett "missing resources"-fel när du startar programmet betyder det att viktiga filer som Liberation behöver för att kunna köras saknas. Det kan hända i Windows om du råkar ta bort eller flytta filer i installationsmappen. På Mac kan det hända om installationen på något sätt har blivit skadad.

### Så åtgärdar du det

Det bästa sättet att lösa problemet är att installera om Liberation.
