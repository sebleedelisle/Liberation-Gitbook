# 🟩 Ändra ordning på beam zones

Beam zones visas i den ordning du skapar dem, men du kan ändra ordningen i efterhand. Öppna _View -> Beam Zone Organiser._

Du ser listan över beam zones och vilken laser de är anslutna till. Dra och släpp zonerna för att lägga dem i den ordning du vill ha, eller klicka på _Autosort by laser_ för att automatiskt sortera dem i följd utifrån vilken laser de tillhör.

{% hint style="info" %}
Om du inte har gett dina beam zones egna namn använder de standardnamn som _Beam 1, Beam 2,_ osv. Om du ändrar ordningen på dem uppdateras även standardnamnen utifrån sorteringsordningen. Tänk på detta när du ändrar ordningen på zoner, eftersom det annars kan bli förvirrande.
{% endhint %}
