# 🟩 Organisera ditt Clip Deck

### Flytta/duplicera klipp

Klicka och dra för att flytta ett klipp. Om fler än ett klipp är markerat (se [Starta/stoppa clips](starting-stopping-clips.md "mention")) flyttas alla. Håll ned `Alt / Option` för att duplicera dem. Släpp klippet eller klippen på en tom plats.

{% hint style="info" %}
Om du behöver markera många klipp kan du klicka och dra ett markeringslasso och sedan rulla i Clip Deck medan du fortfarande drar (antingen med piltangenterna, APC 40 eller sidscrollning på pekplatta/mus). Lassot förlängs utanför kanten på Clip Deck.
{% endhint %}

### Ta bort klipp

Använd högerklicksmenyn för att ta bort ett enskilt klipp.

För att ta bort flera klipp markerar du dem (antingen med `Alt / Option + Shift`-klick eller genom att dra lassot) och drar dem ut från Clip Deck.

### Lägga till tomt utrymme

Du kan högerklicka på en clip-knapp och trycka på _ADD EMPTY COLUMN TO RIGHT_ för att lägga till en tom kolumn med utrymme i Clip Deck.

### Kombinera flera Clip Decks

Du kan importera ett befintligt Clip Deck och lägga till det i slutet av det aktuella Clip Deck. Använd menyn _File -> Clips/Effects -> Append Clip Deck._

Gör en markering av klipp (antingen med lassot eller `Alt / Option + Shift`-klick) och välj sedan _File -> Clips/Effects -> Export Selected Clips._ Detta exporterar ett Clip Deck med endast de klippen, som sedan kan läggas till i ett annat Clip Deck.

### Organisera klipp efter timeline

När du importerar en timeline får du även några klippalternativ. Du kan antingen ersätta det aktuella Clip Deck, lägga till hela clip deck från timeline, eller bara de klipp som används.

Om du har flera timelines i ditt projekt kan du omorganisera dina klipp efter timeline. Öppna panelen _Timeline selection_ och klicka på _ORGANISE CLIPS BY TIMELINE_. Du ser nu alla dina klipp i grupper baserade på deras timeline. Om ett klipp används i flera timelines dupliceras det för varje grupp.
