# ✅ Sneltoetsen

Sneltoetsen worden geschreven als `Cmd / Ctrl`: gebruik `Cmd` op macOS en `Ctrl` op Windows.

| Shortcut                          | Actie                                                 |
| --------------------------------- | ----------------------------------------------------- |
| `Cmd / Ctrl + Shift + A`          | Alle lasers activeren                                 |
| `Escape`                          | Alle lasers deactiveren                               |
| `Tab`                             | Weergaven wisselen en door lasers bladeren            |
| `Shift + Tab`                     | Weergaven wisselen en achteruit door lasers bladeren  |
| `1-9`                             | Naar laser springen                                   |
| `Left / Right Arrow`              | Clip Deck naar links / rechts verplaatsen             |
| `Cmd / Ctrl + Left / Right Arrow` | Clip Deck één pagina naar links / rechts verplaatsen  |
| `Shift + Left / Right Arrow`      | Zones één pagina naar links / rechts verplaatsen      |
| `Space`                           | Afspelen / pauzeren (wanneer _Timeline_ als tempobron actief is) |
| `T`                               | Tempo tikken                                          |
| `R`                               | Maat resetten                                         |
| `Y`                               | Tempo afronden naar de dichtstbijzijnde BPM           |
| `Cmd / Ctrl + ,`                  | Het instellingenpaneel openen                         |
| `Cmd / Ctrl + F`                  | Volledig scherm in- of uitschakelen                   |
| `Cmd / Ctrl + L`                  | Het paneel voor laseroutput-instellingen openen       |
| `Cmd / Ctrl + K`                  | Het paneel voor toewijzing van lasercontrollers openen |
| `Cmd / Ctrl + J`                  | Het instellingenpaneel voor 3D Visualiser openen      |
| `Cmd / Ctrl + Shift + Plus (+)`   | Interfaceschaal vergroten                             |
| `Cmd / Ctrl + Shift + Minus (-)`  | Interfaceschaal verkleinen                            |
| `Cmd / Ctrl + 0`                  | Interfaceschaal resetten                              |
| `Cmd / Ctrl + A`                  | Timeline: alles selecteren                            |
| `Delete`                          | Timeline: geselecteerde blokken verwijderen           |
| `Delete`                          | Clip editor: geselecteerde nodes verwijderen          |
| `Cmd + Q` (macOS)                 | De app afsluiten                                      |
| `Alt + F4` (Windows)              | De app afsluiten                                      |
