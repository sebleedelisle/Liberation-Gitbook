# ✅ Parameter Control

## <img src="../../.gitbook/assets/image (9).png" alt="" data-size="line">  Parameter Control

_De Parameter Control_ is een methode om verschillende instellingen binnen een effect of clip buiten de clip editor te bedienen.

* **range min / range max** - de minimale en maximale waarden waarop het parameterniveau wordt gemapt
* **level** - de standaardinstelling voor deze parameter. Dit is de waarde die van buitenaf kan worden gewijzigd. Je kunt deze handmatig aanpassen om te zien welk effect dat heeft.
* **parameter number** - elke _parameter control_ kan een nummer krijgen, en dit zie je terug in de controls buiten de clip. Je kunt meerdere parameter controls met hetzelfde nummer hebben; ze worden dan allemaal tegelijk aangepast.

{% hint style="warning" %}
De APC40 en ondersteunde live controllers bedienen standaard parameter 1 voor elke effectsleuf. Alle parameters binnen effecten en Clips zijn ook beschikbaar via het rechtermuisknopmenu.
{% endhint %}
