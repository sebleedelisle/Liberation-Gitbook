# ✅ Sound Input-oscillator

## <img src="../../.gitbook/assets/sound-input-icon.png" alt="" data-size="line"> Sound input

Zet het geluidsinputniveau om naar een eigenschapswaarde.

{% hint style="info" %}
De Sound Input-oscillator gebruikt de standaard geluidsinterface, maar je kunt dit wijzigen in _Preferences_. Open het menu _Liberation -> Preferences._

Bij de _Sound Input_-instellingen kun je kiezen welke geluidsinterface je wilt gebruiken, samen met enkele andere instellingen om het volumeniveau aan te passen.
{% endhint %}

* **range min / range max** - de minimale en maximale waarden waarop de golfvorm wordt gemapt
* **channel** - als je geluidsinterface meer dan één inputkanaal heeft, kun je hier kiezen welk kanaal je gebruikt.

{% hint style="info" %}
Een leuke techniek is om meerdere geluidsfeeds vanaf de mengtafel te gebruiken. Zo kun je verschillende clips laten reageren op verschillende muziekinstrumenten.
{% endhint %}

{% hint style="info" %}
Je kunt voor alle _Sound Inputs_ maar één geluidsinterface tegelijk gebruiken (geselecteerd in het _App Settings_-paneel). Als je hiervoor meer dan één interface wilt gebruiken, kun je op macOS een [Aggregate Device aanmaken](https://support.apple.com/en-gb/HT202000) om inputs te combineren tot één virtuele geluidsbron. (Er zijn ook veel Windows-apps die dit kunnen, maar die heb ik niet geprobeerd).
{% endhint %}

* **clamp min / clamp max** - gebruik dit om te kiezen op welk bereik van niveaus je wilt reageren. Je hoeft dit normaal niet aan te passen als de instellingen voor gate en limit (in het _App Settings_-paneel) goed zijn ingesteld, maar je kunt ze wel gebruiken voor creatieve effecten.

{% hint style="info" %}
Je ziet een klein microfoonpictogram op de clipknop wanneer die een _Sound Input_-oscillator heeft.
