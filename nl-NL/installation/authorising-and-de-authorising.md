# ✅ Autoriseren en de-autoriseren

### Liberation autoriseren

Wanneer je Liberation voor het eerst opent, draait het in _demo mode_ en zie je het _About panel:_

<figure><img src="../.gitbook/assets/authorisation-about-panel.png" alt="" width="375"><figcaption></figcaption></figure>

Klik op de knop _AUTHORISE ONLINE_. Je webbrowser wordt geopend. Als je nog niet bent ingelogd, wordt je nu gevraagd dat te doen.

Het systeem autoriseert je installatie nu automatisch met je abonnement. Daarna zie je dit bericht:

<figure><img src="../.gitbook/assets/authorisation-machine-added" alt=""><figcaption></figcaption></figure>

Wanneer je teruggaat naar Liberation, zie je dat het _About panel_ is bijgewerkt (mogelijk moet je een paar seconden wachten).

<figure><img src="../.gitbook/assets/authorise-about-panel-authorised" alt="" width="375"><figcaption></figcaption></figure>

{% hint style="warning" %}
Als je al het maximale aantal computers voor je abonnement hebt geautoriseerd, moet je een van je andere machines de-autoriseren of je abonnement upgraden.
{% endhint %}

{% hint style="info" %}
Als je meerdere abonnementen hebt, wordt je gevraagd het abonnement te kiezen waaraan je de computer wilt koppelen.
{% endhint %}

Gefeliciteerd! Je Liberation-installatie is nu geautoriseerd en je kunt output naar lasers sturen. Lees wel eerst [Quickstartgids](../getting-started.md "mention") en [Overzicht van het instelproces voor lasers](../setting-up/setting-up-lasers.md "mention") voordat je je lasers voor output activeert.

{% hint style="info" %}
Je kunt het _About panel_ op elk moment openen via het menu _Liberation -> About Liberation_ of _Liberation -> Authorise/Deauthorise this computer_
{% endhint %}

### Liberation de-autoriseren

**Vanuit Liberation** - Open het menu _Liberation -> Authorise / De-authorise this Computer_ en klik op de knop _DEAUTHORISE COMPUTER_. Dat is alles! Hiervoor moet je online zijn.

<figure><img src="../.gitbook/assets/Screenshot 2024-08-28 at 10.18.57.png" alt=""><figcaption></figcaption></figure>

Je kunt dit ook via de website doen. Selecteer _Your subscriptions_ in het menu en klik daarna op _Manage Subscription_ om je abonnementspagina te openen. Je ziet informatie over je abonnement en een lijst met de computers die je hebt geautoriseerd.

Klik op de link _De-authorise_ naast de machine die je wilt de-autoriseren.

<figure><img src="../.gitbook/assets/Screenshot 2024-08-28 at 10.16.40.png" alt=""><figcaption></figcaption></figure>

Als je machine sinds je laatste verlenging niet online is geweest, wordt deze direct gedeautoriseerd. Zo niet, dan wordt de machine in de wachtrij gezet voor de-autorisatie. Dit betekent dat de de-autorisatie automatisch wordt uitgevoerd zodra de machine de volgende keer verbinding maakt met internet, of op je volgende verlengingsdatum, afhankelijk van wat het eerst gebeurt.
