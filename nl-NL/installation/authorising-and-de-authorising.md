# ✅ Autoriseren en de-autoriseren

### Liberation autoriseren

Wanneer je Liberation voor het eerst opent, draait het in _free mode_ en zie je het _About panel:_

<figure><img src="../.gitbook/assets/authorisation-about-panel.png" alt="" width="375"><figcaption></figcaption></figure>

Klik op de knop _AUTHORISE ONLINE_. Je webbrowser wordt geopend. Als je nog niet bent ingelogd, wordt je nu gevraagd dat te doen.

Het systeem autoriseert je installatie nu automatisch met je licentie. Daarna zie je een bevestigingsbericht.

Wanneer je teruggaat naar Liberation, zie je dat het _About panel_ is bijgewerkt (mogelijk moet je een paar seconden wachten).

<figure><img src="../.gitbook/assets/authorise-about-panel-authorised" alt="" width="375"><figcaption></figcaption></figure>

{% hint style="warning" %}
Als je al het maximale aantal computers voor je licentie hebt geautoriseerd, moet je een van je andere machines de-autoriseren of je licentie upgraden.
{% endhint %}

{% hint style="info" %}
Als je meerdere licenties hebt, wordt je gevraagd de licentie te kiezen waaraan je de computer wilt koppelen.
{% endhint %}

Gefeliciteerd! Je Liberation-installatie is nu geautoriseerd en je kunt output naar lasers sturen. Lees wel eerst [Quickstartgids](../getting-started.md "mention") en [Overzicht van het instelproces voor lasers](../setting-up/setting-up-lasers.md "mention") voordat je je lasers voor output activeert.

{% hint style="info" %}
Je kunt het _About panel_ op elk moment openen via het menu _Liberation -> About Liberation_ of _Liberation -> Authorise/Deauthorise this computer_
{% endhint %}

### Liberation de-autoriseren

**Vanuit Liberation** - Open het menu _Liberation -> Authorise / De-authorise this Computer_ en klik op de knop _DEAUTHORISE COMPUTER_. Dat is alles! Hiervoor moet je online zijn.

<figure><img src="../.gitbook/assets/Screenshot 2024-08-28 at 10.18.57.png" alt=""><figcaption></figcaption></figure>

Je kunt dit ook via de website doen. Selecteer _Your licences_ in het menu en klik daarna op _MANAGE_ voor de betreffende licentie. Als je account maar één licentie heeft, brengt _Your licences_ je direct naar die licentiepagina. Je ziet informatie over je licentie en een lijst met de computers die je hebt geautoriseerd.

<figure><img src="../.gitbook/assets/account-your-licences.png" alt=""><figcaption></figcaption></figure>

Klik op de link _De-authorise_ naast de machine die je wilt de-autoriseren.

Als je machine sinds je laatste licentievernieuwing niet online is geweest, wordt deze direct gedeautoriseerd. Zo niet, dan wordt de machine in de wachtrij gezet voor de-autorisatie. Dit betekent dat de de-autorisatie automatisch plaatsvindt wanneer de machine de volgende keer verbinding maakt met internet, of op de volgende datum voor licentievernieuwing, afhankelijk van wat het eerst gebeurt.
