# ✅ Installeren voor Windows

### Compatibiliteit

Windows 10- of 11-machines met 64 bit. Een computer met hoge specificaties wordt aanbevolen als je grote shows wilt draaien. Hoe sneller je computer is, hoe vloeiender de geanimeerde laseroutput zal zijn.

### Installatieproces

Zodra je een actief abonnement hebt, krijg je een link toegestuurd. Je kunt ook op de link _Download_ klikken op de pagina [ _Your subscriptions_](https://liberationlaser.com/account/my-products).

Je ziet dan deze pagina:

<figure><img src="../../.gitbook/assets/windows-install-download-page.png" alt=""><figcaption></figcaption></figure>

Klik op _Download latest for Windows_ (de versienummers kunnen veranderen). Je ziet dan de downloadindicator met de voortgang (meestal rechtsboven; de schermafbeelding is uit Chrome). Als de download klaar is, open je je downloads en klik je op het gedownloade installatiebestand.

<figure><img src="../../.gitbook/assets/windows-install-download-list.png" alt=""><figcaption></figcaption></figure>

Hiermee start je het installatieprogramma.

{% hint style="info" %}
In sommige gevallen krijg je mogelijk een waarschuwing dat Windows je pc heeft beveiligd - Windows is voorzichtig met het uitvoeren van nieuwe software! Naarmate meer mensen Liberation gebruiken en vertrouwen, zou dit moeten verdwijnen.

Krijg je deze waarschuwing, klik dan op de link _More info_, controleer of _Publisher_ wordt weergegeven als _SEB.LY LIMITED_ en klik daarna op _Run anyway_.

<img src="../../.gitbook/assets/windows-install-windows-protection-warning.png" alt="" data-size="original"><img src="../../.gitbook/assets/windows-install-warning-run-anyway.png" alt="" data-size="original">
{% endhint %}

Voordat het installatieprogramma start, krijg je mogelijk een waarschuwing waarin wordt gevraagd of je toestaat dat het installatieprogramma wijzigingen aanbrengt. Klik op _Yes_.

<figure><img src="../../.gitbook/assets/windows-install-warning-make-changes.png" alt=""><figcaption></figcaption></figure>

Nu kan het installatieprogramma starten! Klik door de pagina’s heen.

<figure><img src="../../.gitbook/assets/windows-installer-1.png" alt=""><figcaption></figcaption></figure>

Het wordt aanbevolen om de standaardinstallatielocatie te gebruiken, dus klik gewoon op _Next_.

<figure><img src="../../.gitbook/assets/windows-installer-2.png" alt=""><figcaption></figcaption></figure>

Daarna krijg je opties voor een bureaubladpictogram en een map in het startmenu (aanbevolen).

<figure><img src="../../.gitbook/assets/windows-installer-3.png" alt=""><figcaption></figcaption></figure>

Wacht kort terwijl Liberation wordt geïnstalleerd.

<figure><img src="../../.gitbook/assets/windows-installer-4.png" alt=""><figcaption></figcaption></figure>

Het installatieprogramma is klaar! Klik op _Finish_ om het te sluiten en Liberation te starten.

<figure><img src="../../.gitbook/assets/windows-installer-5.png" alt=""><figcaption></figcaption></figure>

Liberation heeft toestemming nodig om bestanden en het netwerk te openen, dus Windows zal waarschijnlijk enkele toestemmingsmeldingen tonen. Zorg dat je alles accepteert, anders werkt Liberation mogelijk niet goed.

<figure><img src="../../.gitbook/assets/windows-defender-firewall-block-message.png" alt="" width="375"><figcaption></figcaption></figure>

Liberation zou nu moeten starten! Om lasers aan te sturen, moet je nu het [Autoriseren en de-autoriseren](../authorising-and-de-authorising.md) doorlopen.
