# ✅ Lasers toevoegen / verwijderen

Voeg een nieuwe laser toe door op de knop _ADD LASER_ te klikken in het paneel _Laser overview_

<figure><img src="../.gitbook/assets/Remove Laser in Laser Overview.png" alt="" width="292"><figcaption></figcaption></figure>

of op de knop _+_ in de _Output_-weergave.<br>

<figure><img src="../.gitbook/assets/Laser Output View (1).png" alt="" width="176"><figcaption></figcaption></figure>

Wanneer je een nieuwe laser toevoegt, wordt er automatisch een beam zone aan toegevoegd. Let op: lasers en zones worden gesorteerd in de volgorde waarin je ze toevoegt. Zie [Beamzones opnieuw ordenen](../output-view/re-ordering-beam-zones.md "mention").

Verwijder een laser door op de rode knop "-" te klikken in het paneel _Laser overview_. Let op: je verliest al je laserinstellingen!

{% hint style="info" %}
Als je zones of andere instellingen van andere lasers wilt kopiëren, gebruik dan [Instellingen kopiëren tussen lasers](copy-laser-settings.md "mention")
{% endhint %}

{% hint style="info" %}
Let op: wanneer je iets doet dat mogelijk gevaarlijk is, zoals een laser verwijderen, krijg je een bevestigingsvenster. Als je dit wilt overslaan, houd dan `Cmd / Ctrl` ingedrukt terwijl je op de knop klikt. Gebruik dit uiteraard met voorzichtigheid!
{% endhint %}
